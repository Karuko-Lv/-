在计算机系统结构中，这几个概念是衡量 CPU 性能和时钟周期的核心指标。它们的定义、计算关系以及对应的**标准单位**如下表所示：

## 指标单位一览表

| 物理量 / 指标   | 英文全称                                         | 核心定义                      | **标准单位 / 常见单位**                                        |
| ---------- | -------------------------------------------- | ------------------------- | ------------------------------------------------------ |
| **时间周期**   | Clock Period                                 | CPU 执行一个最基础时钟脉冲所需要的时间。    | 秒 (**s**)、毫秒 (**ms**)、微秒 ($\mu$s)、纳秒 (**ns**)          |
| **时间频率**   | Clock Frequency (Rate)                       | CPU 一秒钟内产生的时钟脉冲个数（主频）。    | 赫兹 (**Hz**)、兆赫 (**MHz**)、吉赫 (**GHz**)                  |
| **CPI**    | Cycles Per Instruction                       | 每条指令执行所需要的平均时钟周期数。        | **周期 / 指令** (Cycles/Instruction)  <br>_(通常直接写纯数字，无量纲)_ |
| **IPS**    | Instructions Per Second                      | CPU 每秒钟能执行的指令条数。          | **条 / 秒** (Instructions/Second)                        |
| **MIPS**   | Million Instructions Per Second              | CPU 每秒钟能执行的**百万条**指令数。    | **MIPS** 或 **百万条 / 秒**                                 |
| **MFLOPS** | Million Floating-point Operations Per Second | CPU 每秒钟能执行的**百万次浮点运算**次数。 | **MFLOPS** 或 **百万次浮点运算 / 秒**                           |


    
    $$\text{频率 } (f) = \frac{1}{\text{周期 } (T)}$$$$\text{MIPS} = \frac{\text{主频 } (f)}{\text{CPI} \times 10^6}$$
