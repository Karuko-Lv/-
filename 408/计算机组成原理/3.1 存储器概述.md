##### 磁带存储器
20 世纪 50 年代初开始应用于计算机领域，20 世纪 90 年代流行的录音带录像带都是磁带存储器
优点：成本相对低存储容量相对大
计算机早期年代由于硬盘光盘等存储技术的高昂成本，磁带存储成为大量数据备份和归档的主要方式
🧚‍♀️磁带存储适合用来做监控备份或者日志备份；要是保存录像和，重要资料还是要用磁盘存储器，SSD 容易丢失

###### 软盘
20 世纪 90 年代广泛应用，常用的 5 寸软盘和 3 寸软盘的最大容量分别为 1.2 MB 和 1.44 MB

### 存取方式
#### 顺序存储器
地址顺序即物理位置顺序
磁带存储 器是典型的顺序存储器
🧑‍🏫磁带机对磁带存储器的内容进行存取：磁带存储器中的磁带被磁带机中的**旋转装置**带动旋转，磁带机中的磁头贴近磁带对其进行存取操作，显然要读取磁带某段区域存储的内容时需要等到旋转装置带动磁带旋转到这段区域到达磁头位置才可以![[截屏2026-06-12 15.59.31.png]]
🧚‍♀️昏了？ 链表怎么按地址顺序访问链表是随机地址的数组是按地址存储的；倒带
#### 随机存储器
早起的磁芯存储器和当前广泛使用的半导体存储器都是随机存取存储器
🧑‍🏫访问内存中某个存储单元的内容只需要将该存储单元的地址写入到存储器地址寄存器 MAR 中，即可实现对该存储单元进行读取或写入操作
![[截屏2026-06-12 16.02.46.png]]
#### 直接存储器
🧚‍♀️直接存储器是通过地址快速定位大致区域，再顺序读取目标数据。就像翻书时先找章节，再细读某一页。
🧑‍🏫磁盘存储器是典型的直接存储器。由于磁盘存在机械寻道和旋转延迟，因此数据访问时间与磁头和目标扇区的距离有关

### 可改写性
#### 读写存储器
磁芯、磁盘、磁带存储器和内存、固态硬盘都是读写存储器![[截屏2026-06-12 16.31.40.png]]
#### 只读存储器 ROM
只能读出而不能改写，可以写![[截屏2026-06-12 16.36.12.png]]
🧑‍🏫半导体只读存储器需要使用专门工具进行改写，用来存放不需要修改的程序和常量，例如 BIOS 程序，BIOS 配置界面喜欢 CPU 超频和内存超频的人都知道，BIOS 配置的程序代码和配置参数存储在主办上一颗半导体只读存储器芯片中，现在主流的计算机主板厂商都提供了更新这种芯片存储信息的便捷方法，但是对于普通的计算机用户很少用到

以太网卡上固化有 MAC 地址，例如⬇️网卡使用 EEPROM![[截屏2026-06-12 16.36.50.png]]
即带电可擦可编程只读存储器，例如 93 C 46 芯片存储 MAC 地址及产品相关信息
🧚‍♀️只读存储器原因是在一开始厂家把系统内部写好了个人不能改不能往里面写，对于用户来说是只读，读写是后面才改进的，你搞个只写存储意思是厂家让用户花钱自己再写系统吗，只写存储器没意义啊，写进去不给人看？；谁说只读存储器没有意义啦，主**板上的 BIOS 芯片就是只读存储器**！BIOS 存放开机自检程序和 OS 系统首地址导入程序，这些程序决不能随意更改，否则无法正常开机！故使用只读储存器进行存储.；一次性读取存储器用完就报废，所以很耗钱

#### 寄存器存储器
CPU 内部多个寄存器构成的存储器
存储器地址寄存器 MAR、存储器数据寄存器 MDR、累加器 ACC 和乘商寄存器 MQ 
速度极快容量极小![[截屏2026-06-12 16.47.01.png]]
🧚‍♀️可以压栈压入内存然后恢复以实现比如多任务切换
#### 高速缓冲寄存器
Cache 也位于 CPU 内部，是 CPU 内部的寄存器与 CPU 外部主存之间的高速小容量存储器![[截屏2026-06-12 16.48.05.png]]
#### 主存储器
简称主存或内存，存放指令和数据，CPU 可以通过主存地址随机读写主存
主存的存取速度低于高速缓存，但高于辅存
![[截屏2026-06-12 16.50.31.png]]
#### 辅助存储器
简称辅存或外存，存放不参与运行的程序和数据以及一些需要长期保存的信息
容量很大但是存取速度相对低![[截屏2026-06-12 16.51.11.png]]
