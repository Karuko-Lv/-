## 链栈的定义
```cpp
typedef struct Linknode{
	int data;//数据域
	struct Linknode *next;//指针域
} *LiStack;//栈类型定义
```

与单链表一样链栈可以分为带头结点和不带头结点两种设计方式，判断是否为空不同

进栈/出栈对应插入数据元素和删除数据元素的操作，只不过只能在表头的位置进行（单链表有代码）

课本中推荐使用不带头结点来实现链栈

## 链栈🆚顺序栈
如果**知道最多存多少**且**追求速度**，用**顺序栈**；如果**完全猜不到会有多少数据**，或者**数据量波动极大**，用**链栈**。

| **比较维度**  | **顺序栈 (数组)**        | **链栈 (链表)**          |
| --------- | ------------------- | -------------------- |
| **内存分配**  | 静态预分配（或成倍动态扩容）      | 动态实时分配（按需申请）         |
| **空间利用率** | 容易浪费（未满时）或溢出        | 100% 贴合实际数据量（但有指针开销） |
| **时间效率**  | 略高（Cache命中率高，无分配开销） | 略低（需频繁分配/释放节点内存）     |
| **扩容代价**  | 高（需重新开辟大块内存并拷贝旧数据）  | 零（直接连接新节点即可）         |
| **长度限制**  | 受限于预设大小（或最大连续内存）    | 无限制（仅受限于系统总内存）       |
