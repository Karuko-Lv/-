新的数据结构——串，特殊的线性表（线性表是多个数据元素组成的有序序列），数据元素被限定为字符
普通线性表更关注**单个元素**的增删改查，而“串”更关注**整个子串**的操作
![[截屏2026-05-03 16.48.19.png]]
## 定义
串的定义即探讨逻辑结构
🧚‍♀️串可以理解为数据元素为字符的有限集合
![[截屏2026-05-03 16.50.06.png]]
Python 单双都可以表示串，Java 和 C 是双引号表示字符串，单双引号都不是字符串的内容只是一个边界符用来表示头尾
## 术语

![[截屏2026-05-03 16.58.02.png]]**子串**：重点是连续的，且空串是任意字符串的子串
**主串**：包含子串的串
**字符在主串中的位置**：⚠️字符的编号是从 1 开始的而不是从 0 开始的，和线性表中的位序是一样的；没有特别说明探讨的是这个字符第一次出现的位置；注意空格也是一个字符
**子串在主串中的位置**：以子串的第一个字符在主串中的位置作为子串在在主串中的位置
**空串**：没有元素的字符串，即长度为 0 的字符串
**空格串**：两个单引号之间有很多空格字符，且每个字符会占一个字节，即8bit（8 个二进制位）

## 串🆚线性表
![[截屏2026-05-03 17.01.05.png]]
1️⃣线性表（书 P 12）是由 n 个**相同数据类型**的元素组成的有限序列
串这种数据结构对里面存储的数据元素的数据类型有限制
2️⃣线性表基本操作的增删改查通常是某一个单独的数据元素
字符串的增删改查通常是子串（串中某几个连续的数据元素的有序序列）为操作对象，即对一堆数据元素进行增删改查
![[截屏2026-05-03 17.12.33.png]]
e.g: 使用搜索引擎搜索想要的信息不能只输入一个字母，人类的语言需要将多个字母或者汉字拼接起来表示相应信息==>往搜索引擎输入一个东西相当于让其找哪些文章包含了这个字符串
网络中一段段的文字信息本身就是很长的字符串，搜索引擎做的是从这些大量字符串中找到包含给出子串的信息
