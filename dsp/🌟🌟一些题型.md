### 第一类：采样与重建（A/D与D/A）的频谱分析

**真题样子：** 给出一个方框图，信号 $x(t)$ 经过 A/D 变成 $x(k)$，再经过数字滤波器 $h(k)$，最后 D/A 重建为 $y(t)$。让你画各点的频谱。

- **例子：** 2017年第一题 1、2019年第五题 2、2022年第二题 3。
    

**解题思路（陈教授点拨）：**

1. **A/D (采样)：** 记住口诀“时域离散，频域周期延拓”。模拟频谱 $X(j\omega)$ 会以采样频率 $\Omega_s = \frac{2\pi}{T}$ 为周期进行搬移和叠加。注意横坐标轴的变化（模拟角频率 $\omega$ vs 数字角频率 $\Omega = \omega T$）。
    
2. **数字滤波：** 在数字域 $Y(e^{j\Omega}) = X(e^{j\Omega}) H(e^{j\Omega})$。
    
3. **D/A (重建)：** 相当于通过一个理想低通滤波器（截止频率通常是 $\pi/T$ 或 $\Omega_s/2$）。把周期性的频谱滤得只剩中间一个主值，再乘以系数 $T$（有时题目会有增益要求）。
    

**对应教材内容：**

- **第1章 1.8节 信号时域抽样与信号重建** 4。特别是图1-26（抽样频谱）5和 图1-30（重建模型）6。

### 第二类：线性卷积 vs 圆周卷积

**真题样子：** 给两个序列 $x(n)$ 和 $h(n)$，第一问求线性卷积，第二问求 $N$ 点圆周卷积。

- **例子：** 2017年第二题 7、2018年第二题 8、2022年第一题 9。
    

**解题思路：**

1. **线性卷积：** 列表法或者多项式乘法。长度 $L = N_1 + N_2 - 1$。
    
2. **圆周卷积：**
    
    - **方法一（时域）：** 补零对齐到 $N$ 点 -> 周期延拓 -> 翻转 -> 移位 -> 相乘相加 -> 取主值序列。
        
    - **方法二（频域/DFT）：** 两个序列分别做 DFT -> 频域相乘 -> IDFT。
        
    - **重要结论：** 当圆周卷积点数 $L \ge N_1 + N_2 - 1$ 时，圆周卷积 = 线性卷积。如果点数不够，就会发生**时域混叠**（这是考点！）。
        

**对应教材内容：**

- **线性卷积：** 第1章 1.1.2节 10。
    
- **圆周卷积：** 第2章 2.2.4节 11以及 例2-5 12。

### 第三类：DFT 的性质与计算

**真题样子：** 给一个序列，问你它的 DFT $X(k)$ 的某些特定值（如 $X(0)$），或者对称性，或者利用 DFT 性质求另一个序列的 DFT。

- **例子：** 2018年第六题（已知前5点求后3点）13、2020年第二题 14。
    

**解题思路：**

1. **定义法：** $X(m) = \sum x(n) W_N^{mn}$。
    
    - $X(0) = \sum x(n)$ （所有样点之和）。
        
    - $X(N/2) = \sum x(n) (-1)^n$ （交替求和）。
        
2. **对称性（必考）：** 实数序列的 DFT 具有**共轭对称性**：$X(m) = X^*(N-m)$。如果题目给你前一半数据，让你求后一半，就是考这个！15。
    

**对应教材内容：**

- **第2章 2.2节 离散 Fourier 变换的性质** 16，特别是表2-1 17。

### 第四类：滤波器设计（IIR 与 FIR）

**真题样子：**

- **IIR：** 给定模拟指标，用**双线性变换法**或**脉冲响应不变法**设计数字滤波器。
    
    - 例子：2017年第十题 18、2019年第九题 19。
        
- **FIR：** 给定指标，用**窗函数法**设计线性相位 FIR 滤波器。
    
    - 例子：2020年第五题 20、2022年第十三题 21。
        

**解题思路：**

1. **IIR (双线性变换法)：** 预畸变（$\Omega = \frac{2}{T}\tan(\omega/2)$）-> 查表归一化模拟滤波器 -> 去归一化 -> 代入 $s = \frac{2}{T}\frac{1-z^{-1}}{1+z^{-1}}$。
    
2. **FIR (窗函数法)：** 根据阻带衰减选择窗函数（矩形、汉宁、汉明等）-> 计算理想脉冲响应 $h_d(n)$ （通常是 Sinc 函数）-> 乘以窗函数 $w(n)$ -> 移位保证因果性。
    

**对应教材内容：**

- **IIR：** 第4章 4.4节 双线性变换法 22。
    
- **FIR：** 第5章 5.2节 窗函数法设计 23。

### 第五类：差分方程与 Z 变换

**真题样子：** 给一个差分方程（如 $y(n) - 0.5y(n-1) = x(n)$），求系统函数 $H(z)$、单位脉冲响应 $h(n)$、零极点图、判断稳定性。

- **例子：** 2017年第四题 24、2021年第二题 25。
    

**解题思路：**

1. **求 H(z)：** 两边做 Z 变换，利用位移性质 $Z\{y(n-k)\} = z^{-k}Y(z)$。
    
2. **求 h(n)：** 对 $H(z)$ 进行部分分式展开，查表做逆 Z 变换。
    
3. **稳定性：** 极点必须在单位圆**内**（对于因果系统）。
    

**对应教材内容：**

- **第1章 1.5节 离散时间信号的复频域分析 (Z变换)** 26和 1.6节 系统函数 27。
