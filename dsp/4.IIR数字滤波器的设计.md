0. 知识点回忆：
	1. 系统函数 $H(z)$ 是单位脉冲响应 $h[n]$ 的 z 变换。系统频率响应函数$H(e^{j\omega})$是单位脉冲响应 $h[n]$ 的 DTFT 变换
1. 数字滤波器（Digital Filter DF）:
	1. 定义：是一种离散时间**系统**，特殊在可以根据需要有选择性的滤除输入信号的某些频率分量，实现对输入信号的处理（模拟滤波器是一种连续时间系统）
	2. 框图：（数字滤波器通常从频域角度对信号特性进行描述，轴是频率）$$x[k] \longrightarrow \boxed{H(e^{j\Omega})} \longrightarrow y[k]$$
	3. 分类
		1. 功能
			1. 低通滤波器
			2. 高通滤波器
			3. 带通滤波器
			4. 带阻滤波器
		2. 单位脉冲响应长度
			1. FIR 滤波器（有限脉冲响应）：系统函数只有零点，无反馈，总是稳定的，易实现线性相位。
			2. IIR 滤波器（无限脉冲响应）：系统函数既有零点又有极点，有反馈，相位非线性，设计利用模拟滤波器原型。
			- **==[考点]== IIR vs FIR**：IIR 利用模拟原型设计，阶数低但相位非线性；FIR 阶数高但可实现严格线性相位。
	4. 指标要求：
		1. 理想低通数字滤波器：只有通带和阻带，且阻带等于 0、通带等于 1。物理上无法实现
		2. 物理上可实现的：![[截屏2025-12-02 23.57.06.png|200]]
			1. 有通带、阻带和过渡带（一定有过渡带）
			2. 通带不可能始终为常数 1，阻带不可能始终为常数 0
		3. 具体指标
			1. $\Omega_p$ : 通带截止频率
			2. $\Omega_s$：阻带截止频率
			3. $\delta_p$：通带波动
			4. $\Omega_p$：阻带波动
			PS：工程应用当中往往将通带阻带波动用衰耗来表示，转换成 dB 值，将通带边界频率处对应的幅度值转换成 dB 值，一般是通带最大衰耗和阻带最小衰耗
2. IIR 系统：无限长单位脉冲响应 (Infinite Impulse Response)
	1. 系统函数形式：$$H(z) = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 - \sum_{k=1}^{N} a_k z^{-k}}$$
	2. **特点**：存在反馈环路（分母多项式不为1），极点不全在原点。
	3. 设计思路：（模拟的技术成熟）$$\Omega_p, \Omega_s \xrightarrow[\text{变换}]{\text{频率}} \boxed{\omega_p, \omega_s \xrightarrow[\text{滤波器}]{\text{设计模拟}} H(s)} \xrightarrow[\substack{\text{双线性} \\ \text{变换法}}]{\substack{\text{脉冲响应} \\ \text{不变法}}} H(z)$$ $$\omega_p, \omega_s \xrightarrow[\text{变换}]{\text{频率}} \boxed{\bar{\omega}_p, \bar{\omega}_s \xrightarrow[\substack{\text{低通} \\ \text{滤波器}}]{\text{设计原型}} H_L (\bar{s}) }\xrightarrow[\text{变换}]{\text{复频率}} H (s)$$ 有 BW、CB、C 三个低通滤波器模版，模拟低通滤波器是设计核心
		 - **间接法**：
			 - 数字滤波器
		         1. 将数字滤波器指标 ($\Omega$) 转换为模拟滤波器指标 ($\omega$)。
		         2. 设计模拟滤波器 $H(s)$。
		         3. 将任意滤波器转换成数字滤波器，将 $H(s)$ 映射为数字滤波器 $H(z)$
			- 模拟滤波器
				1. 将任意模拟滤波器指标($\omega$) 转换成低通的模拟滤波器指标（$\bar{\omega}$）
				2. 设计低通模拟滤波器 $H_L (\bar{s})$ 
				3. 将低通滤波器转换成任意模拟滤波器，即将 $H_L (\bar{s})$ 映射为数字滤波器 $H(s)$
		- **指标定义**：通带截频 $\Omega_p$，阻带截频 $\Omega_s$，通带最大衰减 $A_p$ (dB)，阻带最小衰减 $A_s$ (dB)。
	4. 技术指标定义
		- **数字频率**：$\Omega$ (单位：rad)，范围 $[0, \pi]$。
		- **四项核心指标**：
		     1. **通带截止频率** ($\Omega_p$)：通带边缘。
		     2. **阻带截止频率** ($\Omega_s$)：阻带边缘。
		     3. **通带波动** ($\delta_p$) 或 **通带最大衰减** ($A_p$, dB)。
		     4. **阻带波动** ($\delta_s$) 或 **阻带最小衰减** ($A_s$, dB)。    
		     - **dB与线性值的转换** 【高频考点】：
			     - 通带衰减（ dB ）：$A_p = -20 \lg(1-\delta_p) \approx -20 \lg |H(e^{j\Omega_p})|$ (因为前面说了衰减两个字所以加上了符号 $1-\delta_p<1$, 所以 $A_p >0$)
			     - 阻带衰减（ dB ）：$A_s = -20 \lg(\delta_s) \approx -20 \lg |H(e^{j\Omega_s})|$
			     - _注意：考试中常给出dB值，需熟练转换。_   
3. 模拟低通滤波器设计 (原型设计)
	1. 幅频特性转换成衰减函数：
		1. 公式：$A (\omega)=-10 lg|H(j\omega)|^2=-20lg|H(j\omega)|$ 
		2. 图形： ![[Pasted image 20251203093104.png|300]]
		3. 增益函数（常用）：最后设计的系统根据它的 $H (s)$ 来求幅频特性时，即验证它的频率响应特性。 
			0. 知识点回忆：$频率响应 = 幅频特性 + 相频特性$
				-  $H(j\omega) = |H(j\omega)| \cdot e^{j\varphi(\omega)}$
			1. 关系：$G(\omega)=-A (\omega)=20lg|H(j\omega)|$ 没有本质区别只是习惯表述方式
			2. 图形：![[4.0 引论.jpeg]]
	3. 巴特沃斯 (Butterworth) 滤波器(BW 型)
		1. 幅度平方函数：$$|H(j\omega)|^2 = \frac{1}{1 + (\frac{\omega}{\omega_c})^{2N}}$$ 也称之为模方函数
			- $N$：滤波器阶数
			- $\omega_c$：3 dB 截频（PS：与通\阻带截止频率没有任何必然联系）🧚‍♀️模值平方减到1/2处。
		2. BW 模拟低通滤波器特性曲线 ![[截屏2025-12-03 10.03.04.png|350]]
			特性：都是单调变换的，调整 $N$ 和 $\omega_c$ 的值可以得到无限多低通滤波器
			1. $|H(j0)|=1$， $|H(j∞)|=0$ ，$A(\omega_c) = -10 \lg |H(j\omega_c)|^2 = 10 \lg 2 \approx 3\text{dB}$，故 $\omega_c$ 被称为 **3dB 截频**当频率 $\omega = \omega_c$ 时，模值平方为 $1/2$（即图中 $10 \lg 2$ 的来源），这时候信号衰减了约 3dB。如果 $\omega_c = 1$，则称为**归一化**巴特沃斯滤波器 BWF。
					PS：\omega_c 一般不是截止频率\omega_p
			2. **幅度响应单调下降**。
					- 巴特沃斯滤波器在通带内**没有纹波**（Ripple）。这与切比雪夫滤波器不同，切比雪夫滤波器为了获得更陡峭的下降，在通带内会有波浪状的起伏。巴特沃斯则是最平滑的）
			3. $|H(j\omega)|^2$ 在 $\omega=0$ 点的 1 到 $2N-1$ 阶导数均为零。🧚‍♀️处处可导 没有跳变
				- 巴特沃斯滤波器被称为 **"Maximally Flat" (最大平坦)** 滤波器的原因，是通带频率响应曲线最平坦的
		3. BW 模拟低通滤波器的设计步骤
			1. 确定阶数 $N$：根据 $A_p, A_s$ 计算。
				- 阶数 N 的计算 【考点】：$N \ge \frac{\lg(\frac{10^{0.1 A_s} - 1}{10^{0.1 A_p} - 1})}{2 \lg(\frac{\omega_s}{\omega_p})}$（N 只能取证书）
			2. 确定 $3 dB$ 截频 $\omega_c$：通常取通带和阻带计算出的交集范围。
			3. 计算系统函数 $H ({s})$ 极点
			4. 得到系统函数 $H_L ({s})$（设计滤波器最终拿相应的系统函数达到设计目标）

## 2. 模拟低通滤波器原型设计 (4.1)

IIR设计通常借助模拟滤波器为原型。

### 2.1 巴特沃斯 (Butterworth) 滤波器

- **特点**：
    
    - 通带内**最平坦** (Maximally Flat)，无纹波。
        
    - 幅频特性随频率单调下降。
        
- 幅度平方函数：
    
    $$|H(j\omega)|^2 = \frac{1}{1 + (\frac{\omega}{\omega_c})^{2N}}$$
    
- 阶数 N 的计算 【考点】：
    
    $$N \ge \frac{\lg(\frac{10^{0.1 A_s} - 1}{10^{0.1 A_p} - 1})}{2 \lg(\frac{\omega_s}{\omega_p})}$$
    
    - _计算结果需向上取整。_
        
- 3dB截止频率 $\omega_c$：
    
    $$\omega_c = \frac{\omega_p}{(10^{0.1 A_p} - 1)^{1/2N}}$$
    
- **极点分布** 【小众点】：
    
    - 极点均匀分布在s平面的左半圆上，角度间隔 $\frac{\pi}{N}$。
        

### 2.2 切比雪夫 (Chebyshev) 滤波器

- **I型 (CBI)**：
    
    - **通带等波纹** (Equi-ripple)，阻带单调。
        
    - 系统函数涉及切比雪夫多项式 $C_N(x)$。
        
    - _优势_：在相同指标下，阶数比BW低。
        
- **II型 (CBII / 逆切比雪夫)** 【小众点】：
    
    - 通带单调（最平坦），**阻带等波纹**。
        
- **参数 $\epsilon$**：与通带波动有关，$\epsilon^2 = 10^{0.1 A_p} - 1$。
    

### 2.3 椭圆 (Elliptic) 滤波器

- **特点**：
    
    - 通带和阻带**均有波纹**。
        
    - 过渡带**最陡峭**。
        
- **优势**：达到同样指标所需的**阶数最低**。
    
- **缺点**：相位非线性最严重（群时延变化大）。
    

### 2.4 三种滤波器对比总结 【考点】

1. **阶数 (N)**：椭圆 < 切比雪夫 < 巴特沃斯。
    
2. **相位线性度**：巴特沃斯最好，椭圆最差。
    
3. **幅频特性**：BW单调；Chebyshev一侧波纹；Elliptic两侧波纹。

## 3. 模拟域频率变换 (4.2)

如何从**模拟低通原型** ($H_L(s)$) 得到模拟高通、带通、带阻？

### 3.1 变换步骤

1. 将待设计滤波器的指标（如高通 $\omega_p, \omega_s$）转换为**归一化低通原型**的指标（$\bar{\omega}_p, \bar{\omega}_s$）。
    
2. 设计归一化低通 $H_L(\bar{s})$。
    
3. 利用复变量代换 $s = f(\bar{s})$ 转换回目标滤波器。
    

### 3.2 常见变换公式 【考点：公式记忆】

设 $H_L(\bar{s})$ 为低通原型，目标为 $H(s)$：

1. **低通 $\to$ 低通**：$\bar{s} = \frac{s}{\omega_c}$
    
2. **低通 $\to$ 高通**：$\bar{s} = \frac{\omega_c}{s}$ （频率倒数关系）
    
3. 低通 $\to$ 带通 【难点】：
    
    $$\bar{s} = \frac{s^2 + \omega_0^2}{Bs}$$
    
    - 其中 $\omega_0$ 为中心频率，$B$ 为带宽。
        
4. 低通 $\to$ 带阻：
    
    $$\bar{s} = \frac{Bs}{s^2 + \omega_0^2}$$

## 4. 脉冲响应不变法 (4.3)

### 4.1 基本原理

- 核心思想：使数字滤波器的单位脉冲响应 $h[k]$ 等于模拟滤波器响应 $h(t)$ 的等间隔采样。
    
    $$h[k] = h(t)|_{t=kT}$$
    
- **变换关系**：
    
    - $s$平面极点 $p_i$ 映射到 $z$平面极点 $e^{p_i T}$。
        
    - 部分分式展开法：
        
        $$H(s) = \sum \frac{A_i}{s - p_i} \implies H(z) = \sum \frac{A_i}{1 - e^{p_i T} z^{-1}}$$
        

### 4.2 频率关系与混叠 【核心考点】

- **频率映射**：线性关系 $\omega = \Omega / T$。
    
- **致命缺陷**：**频谱混叠 (Aliasing)**。
    
    - 模拟滤波器的频响必须是带限的（高频衰减够快）。
        
    - **适用范围**：仅适用于**低通**和**带通**滤波器设计。
        
    - **禁区**：**不能**用于设计**高通**和**带阻**滤波器（因为高频部分不为0，采样后会严重混叠）。
        

### 4.3 优缺点总结

- **优点**：频率线性映射，模拟频率 $\omega$ 和数字频率 $\Omega$ 呈线性，保留了模拟域的相位特性。
    
- **缺点**：频谱混叠，限制了适用类型。

## 5. 双线性变换法 (4.4)

### 5.1 基本原理

- **核心思想**：利用梯形积分公式近似微分，实现 $s$ 平面到 $z$ 平面的非线性一一映射。
    
- 映射公式 【考点】：
    
    $$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$
    

### 5.2 频率预畸变 (Pre-warping) 【高频考点】

- 频率映射关系：
    
    $$\omega = \frac{2}{T} \tan(\frac{\Omega}{2})$$
    
    - $\Omega$：数字频率（待设计指标）。
        
    - $\omega$：模拟频率（设计原型时用的指标）。
        
- **非线性压缩**：将模拟频率的 $(-\infty, +\infty)$ 压缩映射到数字频率的 $(-\pi, \pi)$。
    
- **无混叠**：解决了脉冲响应不变法的混叠问题。
    
- **设计步骤修正**：
    
    - 拿到数字指标 $\Omega_p, \Omega_s$ 后，必须先用 $\omega = \frac{2}{T} \tan(\frac{\Omega}{2})$ 转换成模拟指标，再设计模拟滤波器。这叫**预畸变**。
        

### 5.3 优缺点总结

- **优点**：彻底消除频谱混叠，适用于**所有类型**（低通、高通、带通、带阻）的滤波器设计。
    
- **缺点**：频率非线性压缩，导致高频部分的**相位失真**严重（频率刻度被扭曲）。

## 6. 设计全流程总结 (综合)

### 通用设计路径 (Flowchart)

1. **确定指标**：确定数字滤波器的性能指标 ($\Omega_p, \Omega_s, A_p, A_s$)。
    
2. **方法选择**：
    
    - 若为高通/带阻 $\to$ 必须用双线性变换法。
        
    - 若要求波形失真小（相位线性）且为低通 $\to$ 脉冲响应不变法可能更好。
        
3. **频率转换 (关键步骤)**：
    
    - **脉冲响应不变法**：$\omega = \Omega / T$
        
    - **双线性变换法**：$\omega = \frac{2}{T} \tan(\frac{\Omega}{2})$ （预畸变）
        
4. **设计模拟原型**：
    
    - 计算阶数 $N$ 和截止频率 $\omega_c$。
        
    - 查表或计算得到归一化传输函数 $H(s)$。
        
    - 若是高通/带通，需进行模拟域频带变换。
        
5. **离散化**：将设计好的 $H(s)$ 变换为 $H(z)$。

## 7. 实例与细节补充 (4.6及其他)

### 7.1 脉搏信号处理案例

- **背景**：脉搏信号含基线漂移（低频）和工频干扰（高频）。
    
- **处理**：
    
    - **高通滤波器**：去除基线漂移（0.5Hz以下）。
        
    - **低通滤波器**：去除高频噪声。
        
- **结论**：在阶数相同情况下，椭圆滤波器过渡带最窄，但相位非线性导致时域波形可能有过冲；BW滤波器波形保持较好但频率选择性差。
    

### 7.2 易错点与小众点

- **【小众点】参数 T 的选取**：
    
    - 在双线性变换法中，最终 $H(z)$ 的系数其实与 $T$ 无关（设计过程中 $T$ 会被约掉）。为了计算方便，通常取 $T=2$，这样 $\frac{2}{T}=1$。
        
    - 但在脉冲响应不变法中，增益会差一个 $1/T$ 因子，通常需要在最后乘以 $T$ 来保持直流增益不变。
        
- **【考点】模拟滤波器阶数计算**：
    
    - 注意 $\lg$ (以10为底) 和 $\ln$ (自然对数) 的区别，公式中通常是 $\lg$。
        
- **【考点】全通滤波器与最小相位系统**：
    
    - 虽然课件未大篇幅展开，但IIR极点必须在单位圆内才稳定（考稳定性）。
        

### 7.3 IIR vs FIR 简要对比 (引论部分)

- **IIR**：极点固定，反馈结构，相位非线性，设计简单（利用模拟原型），阶数低，效率高。
    
- **FIR**：全零点，无反馈，容易实现严格线性相位，总是稳定的，但阶数高，计算量大。

### 💡 复习建议

1. **背诵公式**：BW滤波器的 $N$ 计算公式、双线性变换的频率映射公式 $\tan(\Omega/2)$。
    
2. **掌握流程**：拿到一个设计题目，要能分清是用哪种方法，第一步永远是**先把数字频率 $\Omega$ 换算成模拟频率 $\omega$**。
    
3. **区分方法**：看到“高通”或“带阻”，立刻排除脉冲响应不变法。

## 2. 模拟低通滤波器设计 (原型设计)

这一步是所有设计的基石，后续的高通、带通、带阻都以此为基础。

### 2.1 Butterworth (巴特沃斯) 滤波器 (BW型)

1. **频域特性**：
    
    - **最大平坦性**：通带内幅频响应最平滑，单调下降，无波纹。
        
    - **==[考点]== 3dB 截频**：$\omega_c$ 处衰减约为 3dB。
        
    - 幅度模方：$|H(j\omega)|^2 = \frac{1}{1+(\omega/\omega_c)^{2N}}$。
        
2. **设计步骤**：
    
    - 确定阶数 $N$：根据 $A_p, A_s$ 计算。
        
    - 确定截止频率 $\omega_c$：通常取通带和阻带计算出的交集范围。
        
3. **极点分布**：均匀分布在 $s$ 平面左半平面的圆上，间隔 $\pi/N$。
    

### 2.2 Chebyshev (切比雪夫) 滤波器 (CB型)

1. **Chebyshev I 型 (CBI)**：
    
    - **==[考点]==**：通带**等波纹**，阻带单调下降。
        
    - 全极点系统。
        
2. **Chebyshev II 型 (CBII)**：
    
    - **==[考点]==**：通带单调下降，阻带**等波纹**。
        
    - 既有极点也有零点。
        
3. **特性对比**：相同指标下，Chebyshev 阶数比 Butterworth 低，但在通带/阻带会有波动。
    

### 2.3 椭圆 (Elliptic) 滤波器

1. **特性**：
    
    - 通带和阻带**均为等波纹**。
        
    - **==[考点]== 阶数最低**：在相同指标下，椭圆滤波器的阶数 $N$ 最小，过渡带最窄。
        
    - **==[小众点]==**：实现最困难，且极点离虚轴最近，对参数灵敏度高。
        

### 2.4 三种滤波器总结对比 (重要考点)

|**特性**|**Butterworth**|**Chebyshev I**|**椭圆 (Elliptic)**|
|---|---|---|---|
|**通带特性**|最平坦 (单调)|等波纹|等波纹|
|**阻带特性**|单调下降|单调下降|等波纹|
|**过渡带**|最宽 (下降慢)|较窄|最窄 (最陡峭)|
|**阶数 N**|最高|中等|最低|
|**相位线性**|相对最好|较差|最差|

## 3. 模拟域频率变换 (非低通设计)

设计高通、带通、带阻时，先设计**归一化原型低通滤波器** ($H_L(\bar{s})$，$\bar{\omega}_p=1$)，再变换。

1. **低通 $\to$ 高通 (LP $\to$ HP)**
    
    - 变换公式：$\bar{s} = \frac{1}{s}$ 或 $\bar{\omega} = -1/\omega$。
        
    - **==[考点]==**：将低通的原点映射到高通的无穷远处。
        
2. **低通 $\to$ 带通 (LP $\to$ BP)**
    
    - 变换公式：$\bar{s} = \frac{s^2 + \omega_0^2}{Bs}$。
        
    - 参数：$B$ (带宽)，$\omega_0$ (中心频率 $\sqrt{\omega_{p1}\omega_{p2}}$)。
        
    - **==[考点/难点]==**：由于模拟带通往往不对称，映射到原型低通时，阻带截频 $\bar{\omega}_s$ 需取 $\min(|\bar{\omega}_{s1}|, |\bar{\omega}_{s2}|)$ 以保证严谨满足指标。
        
    - 阶数变化：带通滤波器阶数是原型低通的 **2倍**。
        
3. **低通 $\to$ 带阻 (LP $\to$ BS)**
    
    - 变换公式：$\bar{s} = \frac{Bs}{s^2 + \omega_0^2}$。
        
    - **==[考点/难点]==**：映射到原型低通时，阻带截频 $\bar{\omega}_p$ 需取 $\max(|\bar{\omega}_{p1}|, |\bar{\omega}_{p2}|)$。

## 4. 模拟 $\to$ 数字滤波器转换方法 (核心考点)

### 4.1 脉冲响应不变法 (Impulse Invariance Method)

1. **原理**：时域逼近。使数字滤波器的单位脉冲响应 $h[k]$ 等于模拟滤波器响应的采样 $h(kT)$。
    
2. **映射关系**：$z = e^{sT}$。
    
    - $s$ 平面左半平面 $\to$ $z$ 平面单位圆内。
        
    - 线性频率关系：$\omega = \Omega / T$。
        
3. **缺点 (致命弱点)**：
    
    - **==[考点]== 频谱混叠 (Aliasing)**：模拟滤波器的频响不是带限的，采样后高频部分会折叠到低频。
        
    - **适用性**：仅适用于**低通**和**带通**。
        
    - **==[考点]==**：绝对**不能**用于设计高通和带阻滤波器 (因为高频部分不衰减，混叠严重)。
        
4. **==[小众点]== 增益修正**：通常需要乘以 $T$，以消除采样带来的幅度 $1/T$ 缩放，即 $H(z) \approx T \cdot H(s)$。
    

### 4.2 双线性变换法 (Bilinear Transformation Method)

1. **原理**：通过非线性频率压缩，将整个 $j\omega$ 轴压缩到单位圆上，避免混叠。
    
2. **映射关系**：$s = \frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}}$。
    
3. **频率预畸变 (Pre-warping) ==[核心考点]==**：
    
    - 模拟频率 $\omega$ 与数字频率 $\Omega$ 是非线性正切关系。
        
    - 公式：$\omega = \frac{2}{T} \tan(\frac{\Omega}{2})$。
        
    - **设计步骤**：先将数字指标 $\Omega$ 用上述公式“预畸变”为模拟指标 $\omega$，设计模拟滤波器，最后代入变换公式。
        
4. **优缺点**：
    
    - 优点：无混叠，适用于所有类型 (LP, HP, BP, BS)。
        
    - 缺点：频率轴是非线性的，高频部分压缩严重 (但在分段常数的滤波器设计中影响不大)。

    - `lp2hp`, `lp2bp`, `lp2bs`：将模拟低通原型转换为其他类型。
        
4. **==[小众点]== 注意事项**：MATLAB 中的 `butter` 等函数若不加参数 `'s'`，默认设计的是数字滤波器 (即内部自动包含了双线性变换步骤)；加上 `'s'` 才是设计模拟滤波器。

## 6. 应用实例：脉搏信号处理

1. **信号特征**：
    
    - 有效频率：1Hz ~ 20Hz。
        
    - 主要噪声：
        
        - 基线漂移 (0 ~ 1Hz)。
            
        - 工频干扰 (50Hz)。
            
        - 高频肌电干扰 (>20Hz)。
            
2. **滤波方案**：
    
    - **方案一**：设计一个通带为 [1Hz, 20Hz] 的带通滤波器。
        
    - **方案二 (级联)**：设计一个截止频率 1Hz 的高通 (去基线漂移) + 一个截止频率 20Hz 的低通 (去高频)。
        
    - **对比**：方案二级联方式调试更灵活；BW 型相位较好，CB 型阶数更低但波形可能有失真。

|**比较项目**|**脉冲响应不变法**|**双线性变换法**|
|---|---|---|
|**频率映射**|线性 $\omega = \Omega/T$|非线性 $\omega = \frac{2}{T}\tan(\frac{\Omega}{2})$|
|**频谱混叠**|**有** (严重缺陷)|**无** (主要优势)|
|**适用范围**|仅限带限信号 (低通/带通)|所有类型 (低/高/带通/带阻)|
|**s 平面映射**|多条带状区域映射到单位圆内|整个左半平面单值映射到单位圆内|
|**设计核心**|保持 $h(t)$ 采样值不变|保持代数转换关系|
