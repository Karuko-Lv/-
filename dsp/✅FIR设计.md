# 线性相位 FIR 滤波器零点特性：真题解题指南

**(Linear Phase FIR Filter Properties & Zero Locations)**

针对东南大学 937/837 生物信号处理考研真题中反复出现的“线性相位系统零点”题型，这里总结了一套通用的解题逻辑和核心考点。

## 一、 核心考点速记 (Cheat Sheet)

解这类题，必须熟记两个核心概念：**零点的对称性** 和 **四种线性相位滤波器的约束**。

### 1. 零点的对称性 (Symmetry of Zeros)

对于实系数线性相位 FIR 滤波器，零点在 $z$ 平面上呈现特定的对称分布。如果 $z_0$ 是一个零点，那么根据以下规则，其他的点也必然是零点：

|零点类型|分布规律|必有零点组|说明|
|---|---|---|---|
|**复数零点** (不在单位圆上)|**四重共轭倒数对称**|$\{z_0, z_0^*, 1/z_0, 1/z_0^*\}$|这种零点总是 **4个** 一组出现。|
|**实数零点** (不在单位圆上)|**倒数对称**|$\{z_0, 1/z_0\}$|这种零点总是 **2个** 一组出现。|
|**复数零点** (在单位圆上)|**共轭对称**|$\{z_0, z_0^*\}$|模为1 ($|
|**实数零点** (在单位圆上)|**自对称**|$z=1$ 或 $z=-1$|可以单独存在 (但个数受滤波器类型限制)。|

### 2. 四种线性相位 FIR 类型 (The Four Types)

题目通常会给出“阶数 (Order)”或“长度 (Length)”。

- **注意：** 阶数 $M$ = 长度 $N - 1$。
    
- **关键：** 不同类型强制要求在 $z=1$ 或 $z=-1$ 处有零点。
    

|类型|对称性 $h(n)$|长度 $N$|阶数 $M$|**固定零点约束**|典型特征|
|---|---|---|---|---|---|
|**Type I**|偶对称 (Sym)|奇数|偶数|**无强制零点**|各种滤波器均可设计 (低/高/带通/带阻)|
|**Type II**|偶对称 (Sym)|偶数|奇数|$z = -1$ **必为零点**|**不能**做高通 (高频处为0)|
|**Type III**|奇对称 (Anti)|奇数|偶数|$z=1, z=-1$ **均必为零点**|只能做带通 (两端均为0)|
|**Type IV**|奇对称 (Anti)|偶数|奇数|$z = 1$ **必为零点**|**不能**做低通 (低频处为0)|

> **记忆口诀：**
> 
> - 偶长度 (Even Length) 必有一个固定零点 (II型在-1, IV型在1)。
>     
> - 奇对称 (Anti-Sym) 在 $z=1$ 处必为零 (直流增益为0)。
>     
> - Type III (奇长+奇对) 最“惨”，两头 ($1, -1$) 都有零点。
>     

## 二、 通用解题步骤 (Algorithm)

**第一步：确定参数**

- 确定阶数 $M$ 和长度 $N$ ($N=M+1$)。
    
- 确定类型 (I, II, III, IV)。题目通常会直接给出“线性相位”，你需要根据 $h(n)$ 的对称性或阶数来判断，或者题目直接告诉你类型。
    

**第二步：列出固定零点**

- 根据类型，先写下必须存在的零点 (1 或 -1)。
    

**第三步：扩展已知零点**

- 对题目给出的每个 $z_k$，应用“四重对称”规则找全它的伙伴。
    
    - 给 $z$，找 $z^*$ (共轭)。
        
    - 给 $z$，找 $1/z$ (倒数)。
        
    - 给 $z$，找 $1/z^*$ (共轭倒数)。
        

**第四步：核对总数**

- 所有零点加起来的总数必须等于阶数 $M$。
    
- 利用这一点可以检查是否漏掉了固定零点，或者是否存在重根。
    

**第五步：写出系统函数 H(z)**

- $H(z) = A \cdot \prod_{k=1}^{M} (1 - z_k z^{-1})$
    
- 利用题目给的额外条件 (如 $H(1)=2$) 求出增益常数 $A$ (即 $h[0]$ 或其他比例系数)。
    

## 三、 真题实战演练 (Walkthrough)

### 1. 2022年 第12题 (典型代表)

**题目：** 现有一8阶III型线性相位系统，其部分零点为：$z_1=-5, z_2=-1.25j$。 (1) 求其它零点。 (2) 若 $h[0]=1$，求系统的传输函数 $H(z)$。

**解析：**

1. **分析类型：**
    
    - **8阶 (Order M=8)** $\rightarrow$ 长度 $N=9$ (奇数)。
        
    - **III型 (Type III)** $\rightarrow$ 奇对称。
        
    - **推论：** Type III 必然在 $z=1$ **和** $z=-1$ 处有零点。
        
    - 目前已知零点数：2个 (固定)。剩余 6 个名额。
        
2. **扩展零点：**
    
    - **处理** $z_1 = -5$ **(实数)：**
        
        - 它是实数，不在单位圆上。
            
        - 必须有倒数对称：$1/z_1 = -1/5 = -0.2$。
            
        - 组：$\{-5, -0.2\}$。(消耗2个名额)
            
    - **处理** $z_2 = -1.25j$ **(纯虚数)：**
        
        - 它是复数，不在单位圆上 ($1.25 \neq 1$)。
            
        - 找共轭 $z_2^* = +1.25j$。
            
        - 找倒数 $1/z_2 = 1/(-1.25j) = \frac{1}{-1.25} \cdot \frac{1}{j} = -0.8 \cdot (-j) = +0.8j$。
            
        - 找共轭倒数 $1/z_2^* = -0.8j$。
            
        - 组：$\{-1.25j, 1.25j, 0.8j, -0.8j\}$。(消耗4个名额)
            
3. **汇总零点：**
    
    - 固定：$1, -1$
        
    - 实数对：$-5, -0.2$
        
    - 虚数四重组：$-1.25j, 1.25j, 0.8j, -0.8j$
        
    - 总数：$2 + 2 + 4 = 8$。与8阶吻合。
        
    
    **答案(1)：** 其他零点为 $1, -1, -0.2, 1.25j, 0.8j, -0.8j$。
    
4. **求 H(z)：**
    
    - $H(z) = h[0] \prod_{k=1}^{8} (1 - z_k z^{-1})$
        
    - 展开时可以分组相乘：
        
        - $(1-z^{-1})(1+z^{-1}) = 1 - z^{-2}$
            
        - $(1+5z^{-1})(1+0.2z^{-1}) = 1 + 5.2z^{-1} + z^{-2}$
            
        - $(1+1.25jz^{-1})(1-1.25jz^{-1}) = 1 + 1.5625z^{-2}$
            
        - $(1-0.8jz^{-1})(1+0.8jz^{-1}) = 1 + 0.64z^{-2}$
            
    - 题目给 $h[0]=1$，直接代入即可。
        

### 2. 2017年 第12题

**题目：** 线性相位FIR系统，其三个零点分别为 $-1, 0.5, 1+\sqrt{2}i$。(1) 求最低阶数。

**解析：**

1. **扩展零点：**
    
    - $z_1 = -1$：单位圆上实零点，可能是固定的，也可能成对。
        
    - $z_2 = 0.5$：实数，需配对 $1/0.5 = 2$。 $\rightarrow \{0.5, 2\}$ (2个)
        
    - $z_3 = 1+\sqrt{2}i$：复数。
        
        - 模值检查：$|z|^2 = 1^2 + (\sqrt{2})^2 = 3 \neq 1$。不在圆上。
            
        - 需四重对称：$\{1+\sqrt{2}i, 1-\sqrt{2}i, \frac{1}{1+\sqrt{2}i}, \frac{1}{1-\sqrt{2}i}\}$。 $\rightarrow$ (4个)
            
2. **统计与定阶：**
    
    - 目前确定存在的零点：$z_1(-1)$ + 实数对(2个) + 复数四重组(4个) = 7个零点。
        
    - 如果阶数是7 (长度8)，偶数长度线性相位只有 Type II 或 IV。
        
    - **验证：** Type II (偶长度，偶对称) 要求 $z=-1$ 处有零点。我们确实有 $z_1=-1$。
        
    - 所以，**最低阶数 = 7** (即长度8，Type II)。
        

### 3. 2020年 第4题

**题目：** 8阶 III型，部分零点 $z_1=-0.2, z_2=j0.8$。

**解析：**

- **III型约束：** 必有 $z=1$ 和 $z=-1$。 (2个)
    
- $z_1=-0.2$**：** 实数，配对 $1/(-0.2)=-5$。 $\rightarrow \{-0.2, -5\}$ (2个)
    
- $z_2=j0.8$**：** 纯虚数，配对 $z^*=-j0.8$, $1/z=-j1.25$, $1/z^*=j1.25$。 $\rightarrow \{\pm j0.8, \pm j1.25\}$ (4个)
    
- **总计：** $2+2+4=8$。刚好填满8阶。
    
- **答案：** 其他零点为 $1, -1, -5, -j0.8, -j1.25, j1.25$。
    

### 4. 2024年 第1题(8) (注：此题需注意题目数据准确性)

**题目：** 5阶线性相位FIR，两个零点 $z_1=1+j, z_2=2-j$。

**分析陷阱：**

- **5阶 (Order 5)** $\rightarrow$ 长度 6。偶数长度，必然是 Type II 或 Type IV。
    
- 必有一个固定零点 ($z=-1$ 或 $z=1$)。
    
- 剩余名额：$5 - 1 = 4$ 个。
    
- 这意味着只能容纳 **一组** 四重零点。
    
- **检查数据：**
    
    - $z_1 = 1+j$ 产生的四重组是：$1+j, 1-j, \frac{1-j}{2}, \frac{1+j}{2}$。
        
    - $z_2 = 2-j$ 产生的四重组是：$2-j, 2+j, \frac{2+j}{5}, \frac{2-j}{5}$。
        
    - 这两组互不重叠。如果同时存在，需要 $4+4=8$ 个名额，超过了5阶。
        
- **考试策略：**
    
    1. **检查是否看错题：** 是否是 $z_2$ 是 $z_1$ 的倒数？(如 $z_2 = 0.5-0.5j$)。
        
    2. **如果题目数字确凿：** 说明该题可能存在印刷错误（阶数可能是8阶），或者 $H(z)$ 系数是复数（生物信号处理不常见）。
        
    3. **按规则作答：** 写出 $z_1$ 对应的另外3个，指出 $z_2$ 对应的另外3个，并备注“若为5阶系统，只能容纳其中一组四重零点及一个固定零点”。
        
    4. _(推测：真题中_ $z_2$ _很可能是_ $\frac{1}{1+j}$ _的某种变体，考查点依然是四重对称)_。
        

## 四、 总结：做题“三板斧”

1. **看阶数，定固定零点：**
    
    - 奇数阶(偶长) $\rightarrow$ 找 $1$ 或 $-1$。
        
    - III型 $\rightarrow$ 找 $1$ 和 $-1$。
        
2. **看单体，找伙伴：**
    
    - 复数 $\rightarrow$ 找4个。
        
    - 实数 $\rightarrow$ 找2个。
        
3. **凑总数，写结果：**
    
    - 缺谁补谁，直到凑齐 $M$ 个。
