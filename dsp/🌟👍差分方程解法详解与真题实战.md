# 差分方程解法详解与真题实战

在考研真题中，求解差分方程的核心方法是**Z变换法**。它能条理清晰地求出**零输入响应 (**$y_{zi}$**)**、**零状态响应 (**$y_{zs}$**)** 和 **全响应 (**$y$**)**。

## 一、 核心工具：Z变换的移位性质

解题的关键在于正确处理初始条件。根据方程的形式（延迟或超前），选择不同的性质公式。

### 1. 延迟性质 (适用于 $y(n-k)$ 形式)

**场景**：方程中出现 $y(n-1), y(n-2)$，初始条件给出 $y(-1), y(-2)$。

- $Z[y(n)] = Y(z)$
    
- $Z[y(n-1)] = z^{-1}Y(z) + y(-1)$
    
- $Z[y(n-2)] = z^{-2}Y(z) + z^{-1}y(-1) + y(-2)$
    

### 2. 超前性质 (适用于 $y(n+k)$ 形式)

**场景**：方程中出现 $y(n+1), y(n+2)$，初始条件给出 $y(0), y(1)$。

- $Z[y(n)] = Y(z)$
    
- $Z[y(n+1)] = zY(z) - z\cdot y(0)$
    
- $Z[y(n+2)] = z^2Y(z) - z^2\cdot y(0) - z\cdot y(1)$
    

## 二、 解题标准步骤 (模板)

### 第一步：对方程两边取Z变换

代入上述移位性质公式。

### 第二步：整理方程，分离 $Y(z)$

将方程整理为如下形式：

$$A(z)Y(z) = B(z)X(z) + C(z)$$

其中：

- $B(z)X(z)$ 部分与输入信号有关（对应零状态响应）。
    
- $C(z)$ 部分仅与初始条件有关（对应零输入响应）。
    

### 第三步：求解 $Y(z)$

$$Y(z) = \underbrace{\frac{B(z)X(z)}{A(z)}}_{Y_{zs}(z)} + \underbrace{\frac{C(z)}{A(z)}}_{Y_{zi}(z)}$$

### 第四步：进行逆Z变换 (IZT)

通常使用**部分分式展开法**，查表求出 $y_{zs}(n)$ 和 $y_{zi}(n)$。

## 三、 真题实战演练

### 题型一：延迟型 ($y(n-k)$)

**【2018年 真题第5题】** **题目：** 差分方程：$y(n) + 3y(n-1) + 2y(n-2) = 2f(n) + f(n-1)$ 初始状态：$y(-1)=1/2, y(-2)=1/4$ 输入信号：$f(n)=u(n)$ 求：零输入响应 $y_{zi}(n)$ 和 零状态响应 $y_{zs}(n)$。

**解析：**

1. **取Z变换 (代入初始值** $y(-1)=0.5, y(-2)=0.25$**)**：
    
    $$Y(z) + 3[z^{-1}Y(z) + 0.5] + 2[z^{-2}Y(z) + 0.5z^{-1} + 0.25] = 2F(z) + [z^{-1}F(z) + f(-1)]$$
    
    _(注意：题目通常隐含_ $f(n)$ _为因果信号，故_ $f(-1)=0$_)_
    
    整理方程：
    
    $$Y(z)(1 + 3z^{-1} + 2z^{-2}) + (1.5 + z^{-1} + 0.5) = (2 + z^{-1})F(z)$$$$Y(z)(1 + 3z^{-1} + 2z^{-2}) + (2 + z^{-1}) = (2 + z^{-1})F(z)$$
2. **分离响应**：
    
    $$Y(z) = \underbrace{\frac{(2+z^{-1})F(z)}{1+3z^{-1}+2z^{-2}}}_{Y_{zs}(z)} - \underbrace{\frac{2+z^{-1}}{1+3z^{-1}+2z^{-2}}}_{Y_{zi}(z)}$$
3. **求零输入响应** $y_{zi}(n)$：
    
    $$Y_{zi}(z) = \frac{-(2+z^{-1})}{1+3z^{-1}+2z^{-2}} = \frac{-(2z^2+z)}{z^2+3z+2} = \frac{-z(2z+1)}{(z+1)(z+2)}$$
    
    部分分式展开：
    
    $$\frac{Y_{zi}(z)}{z} = \frac{-(2z+1)}{(z+1)(z+2)} = \frac{A}{z+1} + \frac{B}{z+2}$$
    
    解得 $A=1, B=-3$。
    
    $$Y_{zi}(z) = \frac{z}{z+1} - \frac{3z}{z+2}$$$$\therefore y_{zi}(n) = [(-1)^n - 3(-2)^n]u(n)$$
4. **求零状态响应** $y_{zs}(n)$： 输入 $F(z) = \frac{z}{z-1} = \frac{1}{1-z^{-1}}$。
    
    $$Y_{zs}(z) = \frac{2+z^{-1}}{(1+z^{-1})(1+2z^{-1})} \cdot \frac{1}{1-z^{-1}}$$
    
    展开求解系数（过程略），最后做逆变换即可。
    

### 题型二：超前型 ($y(n+k)$)

**【2020年 真题第9题】** **题目：** 方程：$y(k+2) + 0.1y(k+1) - 0.06y(k) = 2x(k+2) + 0.1x(k+1)$ 初始条件：$y_{zi}(0)=0.02, y_{zi}(1)=0.124$ 求：$H(z)$ 及零输入响应 $y_{zi}(k)$。

**解析：**

1. **求系统函数** $H(z)$ **(零状态，不考虑初始条件)**： 对原式取Z变换（忽略初始项）：
    
    $$z^2Y(z) + 0.1zY(z) - 0.06Y(z) = 2z^2X(z) + 0.1zX(z)$$$$H(z) = \frac{Y(z)}{X(z)} = \frac{2z^2 + 0.1z}{z^2 + 0.1z - 0.06} = \frac{z(2z+0.1)}{(z+0.3)(z-0.2)}$$
2. **求零输入响应** $y_{zi}(k)$ **(令** $X(z)=0$**)**： 利用超前性质代入初始值：
    
    $$[z^2Y(z) - z^2y(0) - zy(1)] + 0.1[zY(z) - zy(0)] - 0.06Y(z) = 0$$$$Y(z)(z^2 + 0.1z - 0.06) = z^2y(0) + zy(1) + 0.1zy(0)$$
    
    代入 $y(0)=0.02, y(1)=0.124$：
    
    $$Y_{zi}(z) = \frac{0.02z^2 + 0.124z + 0.002z}{z^2 + 0.1z - 0.06} = \frac{0.02z^2 + 0.126z}{(z+0.3)(z-0.2)}$$
    
    部分分式展开：
    
    $$\frac{Y_{zi}(z)}{z} = \frac{0.02z + 0.126}{(z+0.3)(z-0.2)} = \frac{A}{z+0.3} + \frac{B}{z-0.2}$$$$A = \frac{0.02(-0.3)+0.126}{-0.3-0.2} = \frac{0.12}{-0.5} = -0.24$$$$B = \frac{0.02(0.2)+0.126}{0.2+0.3} = \frac{0.13}{0.5} = 0.26$$$$Y_{zi}(z) = -0.24\frac{z}{z+0.3} + 0.26\frac{z}{z-0.2}$$$$\therefore y_{zi}(k) = [-0.24(-0.3)^k + 0.26(0.2)^k]u(k)$$

## 四、 极简口诀

1. 看到 $y(n-1)$，想着 $z^{-1}Y + y(-1)$。
    
2. 看到 $y(n+1)$，想着 $zY - z y(0)$。
    
3. **零状态**：初始条件全当0，只管输入。
    
4. **零输入**：输入信号全当0，只管初始。
    
5. **全响应**：两者相加。
