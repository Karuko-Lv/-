您好！这是一个在信号处理领域非常重要且具有优美对偶性的傅里叶变换。

**核心结论：周期冲激信号（也称“冲激串”或“狄拉克梳”）的傅里叶变换，结果是另一个“冲激串”。**

也就是说，一个在**时域**上无限延伸的、离散的冲激序列，其频谱在**频域**上也是一个无限延伸的、离散的冲激序列。

---

### 1. 信号的定义

首先，我们定义一个周期为 $T_0$ 的周期冲激信号 $s(t)$，它的数学表达式是：
$$s(t) = \sum_{n=-\infty}^{\infty} \delta(t - nT_0)$$
这个函数表示在时间轴的整数倍 $T_0$ 位置（..., $-2T_0, -T_0, 0, T_0, 2T_0$, ...）上，各有一个单位冲激。

![时域冲激串](https://storage.googleapis.com/gemini-generations-prod-public/i/p/time_domain_dirac_comb.png)

### 2. 傅里叶变换的结果

对这个周期冲激信号 $s(t)$ 进行傅里叶变换，得到其频谱 $S(\omega)$：
$$S(\omega) = \mathcal{F}\{s(t)\} = \omega_0 \sum_{k=-\infty}^{\infty} \delta(\omega - k\omega_0)$$
其中，$\omega_0 = \frac{2\pi}{T_0}$ 是基波角频率。

---

### 3. 频谱图是什么样子的？

根据上面的公式，我们可以详细描述其频谱图的特征：

1.  **频谱图的整体形态**:
    它**也是一个冲激串**。

2.  **冲激的位置**:
    这些冲激（谱线）出现在频率轴的**整数倍基频**位置上，即：
    $$\omega = k\omega_0 \quad (\text{其中 } k = \dots, -2, -1, 0, 1, 2, \dots)$$
    所以，你会在 $0, \pm\omega_0, \pm2\omega_0, \pm3\omega_0, \dots$ 这些离散的频率点上看到谱线。

3.  **谱线的幅度 (Strength)**:
    这是一个非常有趣且重要的结果：**所有谱线的幅度都是相同的**！
    每一根谱线（冲激）的“强度”或“权重”都是一个常数，这个常数等于**基波角频率 $\omega_0$**。

4.  **谱线的间隔**:
    频域中相邻两根谱线之间的距离是恒定的，这个距离就是**基波角频率 $\omega_0$**。

---

### 4. 重要的对偶关系 (Duality)

这个变换对揭示了一个深刻的对偶关系：

| 时域 (Time Domain) | 频域 (Frequency Domain) |
| :--- | :--- |
| 周期冲激串 | 周期冲激串 |
| 冲激间隔为 $T_0$ | 谱线间隔为 $\omega_0 = \frac{2\pi}{T_0}$ |
| 冲激幅度为 1 | 谱线幅度为 $\omega_0$ |

**关键结论**：
* 时域上的**周期** $T_0$ 决定了频域上的**间隔** $\omega_0$。**时域越“宽疏”，频域越“密集”**。
* 时域上的**间隔** $T_0$ 也决定了频域上的**幅度** $\omega_0$。**时域越“宽疏”，频域幅度越“小”**。

### 5. 这个结果的意义是什么？

这个变换对是**采样定理 (Sampling Theorem)** 的数学基础。在对一个连续信号进行采样时，我们就是将原始信号乘以一个周期冲激串。根据卷积定理的对偶性质，这个时域的乘法操作，就对应于频域中原始信号的频谱与另一个冲激串的**卷积**。

频域中的卷积会导致原始信号的频谱以 $\omega_0$ 为周期进行**周期性延拓**。这完美地解释了为什么采样后的数字信号的频谱是周期性的，也揭示了奈奎斯特采样频率为什么必须足够大以避免频谱混叠。
