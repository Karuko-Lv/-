针对东南大学937生物信号处理考研真题中反复出现的**IIR数字滤波器设计**（双线性变换法、脉冲响应不变法、巴特沃斯滤波器设计及预畸变），这是一类**送分题**，因为其解题步骤非常固定。只要掌握了“标准套路”，就能拿到满分。

以下是这类题型的**标准解题逻辑和步骤详解**，结合了你提供的真题（如2017-2024年相关题目）进行总结。

---

### 核心考点分类

这类题目主要分为两种情形：

1. **给定模拟传递函数 $H(s)$，求数字传递函数 $H(z)$**。
    
    - 这是最基础的题型（如2017 Q14, 2019 Q9, 2022 Q6）。
        
2. **给定数字滤波器指标（截止频率等），设计巴特沃斯滤波器**。
    
    - 这是进阶题型，通常指定用“双线性变换法”并要求“预畸变”（如2020 Q6, 2024 Q7）。
        

---

### 方法一：脉冲响应不变法 (Impulse Invariance Method)

这种方法的核心思想是：数字滤波器的单位脉冲响应 $h(n)$ 是模拟滤波器单位冲激响应 $h(t)$ 的等间隔采样。

#### **标准步骤：**

1. 对 $H(s)$ 进行部分分式展开
    
    首先必须将 $H(s)$ 化为单极点形式：
    
    $$H(s) = \sum_{k=1}^{N} \frac{A_k}{s - p_k}$$
    
    - _技巧：如果 $H(s)$ 分母是二次多项式（如 $s^2+3s+2$），先因式分解为 $(s+1)(s+2)$，然后用待定系数法求 $A_k$。_
        
2. 利用变换公式写出 $H(z)$
    
    直接套用变换对：
    
    $$\frac{1}{s - p_k} \xrightarrow{\text{变换}} \frac{1}{1 - e^{p_k T} z^{-1}}$$
    
    则最终结果为：
    
    $$H(z) = \sum_{k=1}^{N} \frac{A_k}{1 - e^{p_k T} z^{-1}}$$
    
    - _注意：有的教材会在分子乘 $T$，即 $\frac{T}{1 - e^{p_k T} z^{-1}}$，目的是保持增益不变。但若题目未特别说明增益要求，通常使用不乘 $T$ 的标准变换。_
        
3. **主要优缺点（理论考点，如2021 Q5）：**
    
    - **优点**：频率响应的线性关系好，模拟频率 $\Omega$ 和数字频率 $\omega$ 是线性关系 ($\omega = \Omega T$)。
        
    - **缺点**：**频谱混叠 (Aliasing)**。高频部分会折叠到低频，因此只适用于带限的低通或带通滤波器，绝对不能用于高通。
        

---

### 方法二：双线性变换法 (Bilinear Transformation Method)

这是考研中最常考的方法，尤其是配合巴特沃斯滤波器设计。核心是用代数替换避免混叠。

#### **情形 A：直接转换（已知 $H(s)$ 求 $H(z)$）**

1. 代入变换公式
    
    直接将 $s$ 替换为 $z$ 的函数：
    
    $$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
    - _真题技巧：注意看题目给的 $T$。如2022年 Q6 提到 T=2，此时 $2/T = 1$，公式简化为 $s = \frac{1 - z^{-1}}{1 + z^{-1}}$，计算量大幅降低。_
        
2. 化简整理
    
    将代入后的式子通分，整理成 $z^{-1}$ 的有理多项式形式（标准形式）：
    
    $$H(z) = \frac{b_0 + b_1 z^{-1} + \dots}{1 + a_1 z^{-1} + \dots}$$
    

#### **情形 B：完整设计（给定指标 + 预畸变）**

_对应真题：2020 Q6, 2024 Q7_

这是一套完整的**Design Flow**，步骤必须严格遵守：

1. 指标预畸变 (Pre-warping)
    
    双线性变换存在频率扭曲（非线性映射），必须先把数字指标（通带截止频率 $f_c$ 或 $\omega_c$）转换成“预畸变”后的模拟频率 $\Omega_c$。
    
    $$\Omega = \frac{2}{T} \tan\left( \frac{\omega}{2} \right)$$
    
    其中 $\omega = 2\pi f / f_s$ 或 $\omega = 2\pi f T$。
    
    - _注意：如果题目直接给了数字频率 $\omega_c$（如 $0.4\pi$），直接代入。如果给了物理频率 $f$（Hz），先转为 $\omega$。_
        
2. 设计模拟低通滤波器 $H(s)$ (通常是巴特沃斯)
    
    根据计算出的 $\Omega_c$ 和阶数 $N$ 写出 $H(s)$。
    
    - **N=1**: $H(s) = \frac{\Omega_c}{s + \Omega_c}$
        
    - **N=2**: $H(s) = \frac{\Omega_c^2}{s^2 + \sqrt{2}\Omega_c s + \Omega_c^2}$
        
    - **N=3**: $H(s) = \frac{\Omega_c^3}{(s + \Omega_c)(s^2 + \Omega_c s + \Omega_c^2)}$
        
    - _考点记忆：你需要背诵巴特沃斯归一化多项式（当 $\Omega_c=1$ 时），然后将 $s$ 换成 $s/\Omega_c$。_
        
    - 真题中常给表格（如2017 Q10），直接查表得到归一化系数，然后去归一化。
        
3. 将 $H(s)$ 转换为 $H(z)$
    
    使用双线性变换公式：
    
    $$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
    代入上一步得到的 $H(s)$ 中，化简即可。
    
4. **主要优缺点（理论考点）：**
    
    - **优点**：消除了频谱混叠（单值映射）。
        
    - **缺点**：频率轴发生了非线性畸变（Warping），高频部分被压缩，因此必须进行“预畸变”校正。

### 真题实战演练 (以2022年真题 Q6 为例)

**题目**：$H(s) = \frac{2}{s^2 + 3s + 2}$， $T=2$。分别用两种方法求 $H(z)$。

#### **解法 1：脉冲响应不变法**

1. 部分分式展开：
    
    $$H(s) = \frac{2}{(s+1)(s+2)} = \frac{2}{s+1} - \frac{2}{s+2}$$
    
    这里极点 $p_1 = -1, p_2 = -2$。
    
2. 变换：
    
    对应项：$\frac{1}{s+a} \to \frac{1}{1 - e^{-aT} z^{-1}}$
    
    代入 $T=2$：
    
    $$H(z) = \frac{2}{1 - e^{-1\times2} z^{-1}} - \frac{2}{1 - e^{-2\times2} z^{-1}}$$
    
    $$H(z) = \frac{2}{1 - e^{-2} z^{-1}} - \frac{2}{1 - e^{-4} z^{-1}}$$
    
3. 通分（可选）：
    
    如果题目要求写成一般式，则通分合并。
    

#### **解法 2：双线性变换法**

1. 准备公式：
    
    $T=2 \Rightarrow \frac{2}{T} = 1$。
    
    $$s = \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
2. 代入 $H(s)$：
    
    $$H(z) = \frac{2}{ (\frac{1 - z^{-1}}{1 + z^{-1}})^2 + 3(\frac{1 - z^{-1}}{1 + z^{-1}}) + 2 }$$
    
3. 化简：
    
    分子分母同乘 $(1 + z^{-1})^2$：
    
    $$\text{分母} = (1 - z^{-1})^2 + 3(1 - z^{-1})(1 + z^{-1}) + 2(1 + z^{-1})^2$$
    
    $$\text{分母} = (1 - 2z^{-1} + z^{-2}) + 3(1 - z^{-2}) + 2(1 + 2z^{-1} + z^{-2})$$
    
    $$\text{分母} = 1 - 2z^{-1} + z^{-2} + 3 - 3z^{-2} + 2 + 4z^{-1} + 2z^{-2}$$
    
    $$\text{分母} = (1+3+2) + (-2+4)z^{-1} + (1-3+2)z^{-2} = 6 + 2z^{-1}$$
    
    $$\text{分子} = 2(1 + z^{-1})^2 = 2(1 + 2z^{-1} + z^{-2})$$
    
    $$H(z) = \frac{2(1 + 2z^{-1} + z^{-2})}{6 + 2z^{-1}} = \frac{1 + 2z^{-1} + z^{-2}}{3 + z^{-1}}$$
    
    最后整理成标准形式（分母常数项为1）：
    
    $$H(z) = \frac{\frac{1}{3} + \frac{2}{3}z^{-1} + \frac{1}{3}z^{-2}}{1 + \frac{1}{3}z^{-1}}$$

### 总结：考试答题模板

遇到这类题，请按以下模块书写，逻辑清晰，阅卷老师给分快：

1. **写出已知条件**：明确 $H(s)$、采样周期 $T$ 或 截止频率指标。
    
2. **声明使用的方法**：写出对应的公式（如 $s = \dots$）。
    
3. **预畸变计算（仅限双线性设计题）**：计算 $\Omega_c$。
    
4. **确定模拟滤波器 $H(s)$**：写出具体表达式。
    
5. **代入变换与化简**：这是计算最容易出错的地方，建议分步写，不要跳步。
    
6. **最终结果**：写成 $H(z) = \frac{N(z^{-1})}{D(z^{-1})}$ 的形式。
