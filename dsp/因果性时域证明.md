**一个LTI系统是因果的 $\iff$ 它的单位脉冲响应 $h[k] = 0, \text{ for } k < 0$** 。

---

### 1. 证明方向一：(充分性)

**如果 $h[k] = 0, \text{ for } k < 0$，那么系统是因果的。**

这个方向是说，只要 $h[k]$ 满足“$k<0$时为0”这个条件，我们就能保证这个系统一定是因果的。

- **证明：**
    
    1. 我们从LTI系统的卷积公式（输入输出关系）出发 ：
        $$y[k] = \sum_{n=-\infty}^{\infty} h[n]x[k-n]$$
        
        这个公式是推导因果性的最佳形式。
        
    1. 我们把这个无穷的求和拆分成两部分：$n < 0$ 的部分和 $n \ge 0$ 的部分 ：
        $$y[k] = \sum_{n=-\infty}^{-1} h[n]x[k-n] + \sum_{n=0}^{\infty} h[n]x[k-n]$$
        
    1. 现在，我们用上我们的**前提条件**：$h[k] = 0, \text{ for } k < 0$ 。
        
        - 这意味着，对于第一个求和 $\sum_{n=-\infty}^{-1}$，所有的 $h[n]$（比如 $h[-1], h[-2] \dots$）**全部都等于0** 。
            
        - 因此，第一个求和变成了 $0$。
            
    2. 公式简化为：
        
        $$y[k] = \sum_{n=0}^{\infty} h[n]x[k-n]$$
        
    3. 我们把这个剩下的求和展开看看：
        $$y[k] = h[0]x[k-0] + h[1]x[k-1] + h[2]x[k-2] + \dots$$
        
    4. **分析结果：**
        
        - $y[k]$（**当前**的输出）
            
        - 取决于 $x[k]$（**当前**的输入）
            
        - 取决于 $x[k-1]$（**过去**的输入）
            
        - 取决于 $x[k-2]$（**过去**的输入）
            
        - ...
            
        - 它**绝不**取决于 $x[k+1], x[k+2] \dots$ 这样的**未来**输入。
            
    5. 结论： 这完全符合因果系统的定义（输出只取决于当前和过去的输入）。
        
        所以，方向一证明完毕。
        

---

### 2. 证明方向二：(必要性)

**如果一个LTI系统是因果的，那么 $h[k]$ 必须等于 $0, \text{ for } k < 0$。**

这个方向是说，只要你告诉我一个LTI系统是因果的，我就敢断定它的 $h[k]$ 在 $k<0$ 时必为0。

- **证明：**
    
    1. 我们从**因果系统**的定义出发：**任何**时刻的输出 $y[k]$ 只能由**当前和过去**的输入（即 $x[n], n \le k$）决定 。
        
    2. 我们再使用**单位脉冲响应 $h[k]$** 的定义：$h[k]$ 是系统在**零状态**下，对输入 $x[k] = \delta[k]$ 所产生的**输出** 。
        
        - 即： $h[k] = T\{\delta[k]\}$ 
            
    3. 现在，我们把**因果性**的定义，应用到 $h[k]$ 的计算上。
        
        - **输入是：** $x[k] = \delta[k]$。
            
        - **输出是：** $y[k] = h[k]$。
            
        - **因果性要求：** $h[k]$ 在任意时刻 $k$ 的值，只能由 $\delta[n]$ 在 $n \le k$ 时的值决定。
            
    4. 我们来**检验 $k < 0$ 的时刻**，比如 $k = -1$：
        
        - 根据因果性，$h[-1]$ 的值只能由 $n \le -1$ 的输入 $\delta[n]$ 决定。
            
        - 根据 $\delta[n]$ 的定义，当 $n = -1, -2, -3, \dots$ 时，**输入 $\delta[n]$ 全部为 0**。
            
        - 由于系统是线性的（LTI系统的前提），在零状态下，输入为0，**输出 $h[-1]$ 必然为 0**。
            
    5. **同理**，我们检验 $k = -2$：
        
        - 根据因果性，$h[-2]$ 的值只能由 $n \le -2$ 的输入 $\delta[n]$ 决定。
            
        - 当 $n = -2, -3, \dots$ 时，**输入 $\delta[n]$ 全部为 0**。
            
        - 因此，**输出 $h[-2]$ 必然为 0**。
            
    6. 结论： 对任意 $k < 0$ 的时刻，因果性都要求 $h[k]$ 只能由 $n \le k$（此时 $n$ 也是负数）的输入 $\delta[n]$ 决定。而 $\delta[n]$ 在所有 $n < 0$ 的时刻值都为 0。因此， $h[k]$ 在所有 $k < 0$ 的时刻必然为 0 。
        
        所以，方向二证明完毕。
        

---

总证明：

由于我们证明了两个方向，因此 $LTI$ 系统是因果的 $\iff h[k] = 0, \text{ for } k < 0$ 成立。
