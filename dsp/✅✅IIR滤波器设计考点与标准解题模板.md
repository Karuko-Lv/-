# 837生物信号处理：IIR滤波器设计考点与标准解题模板

## 一、 历年考情分析

IIR滤波器设计是837试卷中的必考/高频题，通常出现在计算题部分（10-15分）。

- **高频考点**：双线性变换法（Bilinear Transformation）、巴特沃斯（Butterworth）低通滤波器。
    
- **难点趋势**：近年（2020-2024）特别强调**“预畸变（Pre-warping）”**以及**两种方法的优缺点对比**。
    
- **常见阶数**：$N=2$ 或 $N=3$（需熟记归一化巴特沃斯多项式）。
    

## 二、 核心解题模板 1：双线性变换法 (Bilinear Transformation)

这是考试中最常用的方法，尤其是题目提到“预畸变”时。

### 1. 标准解题步骤

**Step 1: 确定技术指标与预畸变 (Pre-warping)** 题目通常给出数字频率 $\omega_c$ 或模拟截止频率 $f_c$ 与采样频率 $f_s$。

- **情况 A (给出数字频率** $\omega_c$**)**： 直接计算模拟预畸变频率 $\Omega_c$：
    
    $$\Omega_c = \frac{2}{T} \tan\left(\frac{\omega_c}{2}\right)$$
- **情况 B (给出模拟频率** $f_c$ **和** $f_s$**)**： 先转数字频率 $\omega_c = \frac{2\pi f_c}{f_s}$，再预畸变：
    
    $$\Omega_c = \frac{2}{T} \tan\left(\frac{\pi f_c}{f_s}\right)$$

**Step 2: 设计模拟巴特沃斯滤波器** $H_a(s)$ 根据阶数 $N$ 和截止频率 $\Omega_c$，写出归一化低通原型 $H_{normalized}(p)$，然后去归一化。

- **归一化极点分布**：$p_k = e^{j[\frac{\pi}{2} + \frac{2k-1}{2N}\pi]}, \quad k=1, \dots, N$
    
- **常用归一化多项式 (必须背诵)**：
    
    - **N=1**: $p + 1$
        
    - **N=2**: $p^2 + \sqrt{2}p + 1$
        
    - **N=3**: $(p+1)(p^2 + p + 1) = p^3 + 2p^2 + 2p + 1$
        
- **去归一化 (Denormalization)**： 将 $p = \frac{s}{\Omega_c}$ 代入归一化传输函数，得到 $H_a(s)$。
    

**Step 3: 双线性变换** 将 $s$ 替换为 $z$ 的表达式：

$$s = \frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}}$$

代入 $H_a(s)$ 并化简整理，最终整理为标准 $H(z)$ 形式（通常分子分母同乘 $(1+z^{-1})^N$）。

### 2. 真题实战 (以2024年第7题为例)

**题目概要**：双线性变换法设计3阶巴特沃斯低通，需预畸变。$f_c=1\text{kHz}, f_s=6\text{kHz}$ (假设题目意图为截止频率1k)。

**标准解答示范**： **(1) 计算模拟截止频率 (预畸变)**

$$\omega_c = 2\pi \frac{f_c}{f_s} = 2\pi \frac{1}{6} = \frac{\pi}{3}$$$$\Omega_c = \frac{2}{T} \tan\left(\frac{\omega_c}{2}\right) = \frac{2}{T} \tan\left(\frac{\pi}{6}\right) = \frac{2}{T} \frac{1}{\sqrt{3}}$$

_(注：考试中若未给出T的具体数值，最后T通常会被约掉；若需计算系数，T可保留符号或取1，但双线性变换中系数_ $\frac{2}{T}$ _会在去归一化时抵消)_

**(2) 构造模拟滤波器** $H_a(s)$ 3阶巴特沃斯归一化传递函数为：

$$H_n(p) = \frac{1}{p^3 + 2p^2 + 2p + 1}$$

令 $p = \frac{s}{\Omega_c}$，代入得：

$$H_a(s) = \frac{1}{(\frac{s}{\Omega_c})^3 + 2(\frac{s}{\Omega_c})^2 + 2(\frac{s}{\Omega_c}) + 1} = \frac{\Omega_c^3}{s^3 + 2\Omega_c s^2 + 2\Omega_c^2 s + \Omega_c^3}$$

**(3) 变换为数字滤波器** $H(z)$ 代入 $s = \frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}}$。 注意到 $\frac{s}{\Omega_c} = \frac{\frac{2}{T}\frac{1-z^{-1}}{1+z^{-1}}}{\frac{2}{T}\frac{1}{\sqrt{3}}} = \sqrt{3} \frac{1-z^{-1}}{1+z^{-1}}$ (系数 $\frac{2}{T}$ 完美约掉)。 令 $C = \sqrt{3}$，代入 $H_n(p)$：

$$H(z) = \frac{1}{C^3(\frac{1-z^{-1}}{1+z^{-1}})^3 + 2C^2(\frac{1-z^{-1}}{1+z^{-1}})^2 + 2C(\frac{1-z^{-1}}{1+z^{-1}}) + 1}$$

最后通分化简，整理成 $H(z) = \frac{\sum b_k z^{-k}}{1 - \sum a_k z^{-k}}$ 的形式。

## 三、 核心解题模板 2：脉冲响应不变法 (Impulse Invariance Method)

考频略低，但2023年和2024年均有涉及。通常用于给定了 $H(s)$ 表达式的题目。

### 1. 标准解题步骤

**Step 1: 部分分式展开** 将给定的 $H_a(s)$ 展开为部分分式和的形式（单极点形式）：

$$H_a(s) = \sum_{k=1}^{N} \frac{A_k}{s - p_k}$$

**Step 2: 变换极点** 利用变换对：

$$\frac{1}{s - p_k} \longrightarrow \frac{1}{1 - e^{p_k T} z^{-1}}$$

得到：

$$H(z) = \sum_{k=1}^{N} \frac{A_k}{1 - e^{p_k T} z^{-1}}$$

**Step 3: 通分整理** 将求和式合并为一个分式。

### 2. 真题实战 (以2022年第6题为例)

**题目**：$H(s) = \frac{2}{s^2 + 3s + 2}$， $T=2$。 **解答**：

1. **因式分解**：$s^2 + 3s + 2 = (s+1)(s+2)$
    
2. **部分分式展开**：
    
    $$H(s) = \frac{2}{(s+1)(s+2)} = \frac{2}{s+1} - \frac{2}{s+2}$$
3. **变换**： 极点 $p_1 = -1, p_2 = -2$。
    
    $$H(z) = \frac{2}{1 - e^{-1\cdot T} z^{-1}} - \frac{2}{1 - e^{-2\cdot T} z^{-1}}$$
    
    代入 $T=2$：
    
    $$H(z) = \frac{2}{1 - e^{-2} z^{-1}} - \frac{2}{1 - e^{-4} z^{-1}}$$

## 四、 理论对比题 (简答/分析题)

近年常考“两种方法的对比” (2020 Q3, 2021 Q5, 2022 Q6)。标准回答要点如下：

|特性|脉冲响应不变法 (Impulse Invariance)|双线性变换法 (Bilinear Transformation)|
|---|---|---|
|**映射关系**|s平面多带映射到z平面（多值映射）|s平面单值映射到z平面（一一映射）|
|**混叠 (Aliasing)**|**有频谱混叠** (Aliasing)，只适合带限信号|**无混叠** (因采用了频率压缩)|
|**频率关系**|线性关系 $\omega = \Omega T$ (在低频处)|非线性关系 $\omega = 2\arctan(\frac{\Omega T}{2})$ (频率卷曲/Warping)|
|**适用范围**|仅适用于**低通、带通** (高频衰减快)|适用于**任何类型** (低通、高通、带阻、带通)|
|**设计核心**|保持冲激响应 $h[n] = h_a(nT)$|需进行**预畸变**以校正频率卷曲|

**答题话术**：

> "脉冲响应不变法的主要缺点是会产生频谱混叠，因此只适用于设计带限滤波器（如低通），不适合高通。双线性变换法通过非线性频率压缩消除了混叠现象，但引入了频率卷曲，因此在设计时必须对截止频率进行预畸变处理。"

## 五、 备考速记表

**1. 常用公式**

- **预畸变**: $\Omega = \frac{2}{T}\tan(\frac{\omega}{2})$
    
- **双线性代换**: $s = \frac{2}{T}\frac{1-z^{-1}}{1+z^{-1}}$
    
- **脉冲响应代换**: $\frac{1}{s-a} \to \frac{1}{1-e^{aT}z^{-1}}$
    

**2. 常用三角函数值 (预畸变计算用)**

- $\tan(\pi/4) = 1$
    
- $\tan(\pi/6) = 1/\sqrt{3} \approx 0.577$
    
- $\tan(\pi/3) = \sqrt{3} \approx 1.732$
