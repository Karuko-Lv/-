针对东南大学937生物信号处理考研真题中反复出现的**IIR数字滤波器设计**（双线性变换法、脉冲响应不变法、巴特沃斯滤波器设计及预畸变），这是一类**送分题**，因为其解题步骤非常固定。只要掌握了“标准套路”，就能拿到满分。

以下是这类题型的**标准解题逻辑和步骤详解**，结合了你提供的真题（如2017-2024年相关题目）进行总结。

---

### 核心考点分类

这类题目主要分为两种情形：

1. **给定模拟传递函数 $H(s)$，求数字传递函数 $H(z)$**。
    
    - 这是最基础的题型（如2017 Q14, 2019 Q9, 2022 Q6）。
        
2. **给定数字滤波器指标（截止频率等），设计巴特沃斯滤波器**。
    
    - 这是进阶题型，通常指定用“双线性变换法”并要求“预畸变”（如2020 Q6, 2024 Q7）。
        

---

### 方法一：脉冲响应不变法 (Impulse Invariance Method)

这种方法的核心思想是：数字滤波器的单位脉冲响应 $h(n)$ 是模拟滤波器单位冲激响应 $h(t)$ 的等间隔采样。

#### **标准步骤：**

1. 对 $H(s)$ 进行部分分式展开
    
    首先必须将 $H(s)$ 化为单极点形式：
    
    $$H(s) = \sum_{k=1}^{N} \frac{A_k}{s - p_k}$$
    
    - _技巧：如果 $H(s)$ 分母是二次多项式（如 $s^2+3s+2$），先因式分解为 $(s+1)(s+2)$，然后用待定系数法求 $A_k$。_
        
2. 利用变换公式写出 $H(z)$
    
    直接套用变换对：
    
    $$\frac{1}{s - p_k} \xrightarrow{\text{变换}} \frac{1}{1 - e^{p_k T} z^{-1}}$$
    
    则最终结果为：
    
    $$H(z) = \sum_{k=1}^{N} \frac{A_k}{1 - e^{p_k T} z^{-1}}$$
    
    - _注意：有的教材会在分子乘 $T$，即 $\frac{T}{1 - e^{p_k T} z^{-1}}$，目的是保持增益不变。但若题目未特别说明增益要求，通常使用不乘 $T$ 的标准变换。_
        
3. **主要优缺点（理论考点，如2021 Q5）：**
    
    - **优点**：频率响应的线性关系好，模拟频率 $\Omega$ 和数字频率 $\omega$ 是线性关系 ($\omega = \Omega T$)。
        
    - **缺点**：**频谱混叠 (Aliasing)**。高频部分会折叠到低频，因此只适用于带限的低通或带通滤波器，绝对不能用于高通。
        

---

### 方法二：双线性变换法 (Bilinear Transformation Method)

这是考研中最常考的方法，尤其是配合巴特沃斯滤波器设计。核心是用代数替换避免混叠。

#### **情形 A：直接转换（已知 $H(s)$ 求 $H(z)$）**

1. 代入变换公式
    
    直接将 $s$ 替换为 $z$ 的函数：
    
    $$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
    - _真题技巧：注意看题目给的 $T$。如2022年 Q6 提到 T=2，此时 $2/T = 1$，公式简化为 $s = \frac{1 - z^{-1}}{1 + z^{-1}}$，计算量大幅降低。_
        
2. 化简整理
    
    将代入后的式子通分，整理成 $z^{-1}$ 的有理多项式形式（标准形式）：
    
    $$H(z) = \frac{b_0 + b_1 z^{-1} + \dots}{1 + a_1 z^{-1} + \dots}$$
    

#### **情形 B：完整设计（给定指标 + 预畸变）**

_对应真题：2020 Q6, 2024 Q7_

这是一套完整的**Design Flow**，步骤必须严格遵守：

1. 指标预畸变 (Pre-warping)
    
    双线性变换存在频率扭曲（非线性映射），必须先把数字指标（通带截止频率 $f_c$ 或 $\omega_c$）转换成“预畸变”后的模拟频率 $\Omega_c$。
    
    $$\Omega = \frac{2}{T} \tan\left( \frac{\omega}{2} \right)$$
    
    其中 $\omega = 2\pi f / f_s$ 或 $\omega = 2\pi f T$。
    
    - _注意：如果题目直接给了数字频率 $\omega_c$（如 $0.4\pi$），直接代入。如果给了物理频率 $f$（Hz），先转为 $\omega$。_
        
2. 设计模拟低通滤波器 $H(s)$ (通常是巴特沃斯)
    
    根据计算出的 $\Omega_c$ 和阶数 $N$ 写出 $H(s)$。
    
    - **N=1**: $H(s) = \frac{\Omega_c}{s + \Omega_c}$
        
    - **N=2**: $H(s) = \frac{\Omega_c^2}{s^2 + \sqrt{2}\Omega_c s + \Omega_c^2}$
        
    - **N=3**: $H(s) = \frac{\Omega_c^3}{(s + \Omega_c)(s^2 + \Omega_c s + \Omega_c^2)}$
        
    - _考点记忆：你需要背诵巴特沃斯归一化多项式（当 $\Omega_c=1$ 时），然后将 $s$ 换成 $s/\Omega_c$。_
        
    - 真题中常给表格（如2017 Q10），直接查表得到归一化系数，然后去归一化。
        
3. 将 $H(s)$ 转换为 $H(z)$
    
    使用双线性变换公式：
    
    $$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
    代入上一步得到的 $H(s)$ 中，化简即可。
    
4. **主要优缺点（理论考点）：**
    
    - **优点**：消除了频谱混叠（单值映射）。
        
    - **缺点**：频率轴发生了非线性畸变（Warping），高频部分被压缩，因此必须进行“预畸变”校正。

### 真题实战演练 (以2022年真题 Q6 为例)

**题目**：$H(s) = \frac{2}{s^2 + 3s + 2}$， $T=2$。分别用两种方法求 $H(z)$。

#### **解法 1：脉冲响应不变法**

1. 部分分式展开：
    
    $$H(s) = \frac{2}{(s+1)(s+2)} = \frac{2}{s+1} - \frac{2}{s+2}$$
    
    这里极点 $p_1 = -1, p_2 = -2$。
    
2. 变换：
    
    对应项：$\frac{1}{s+a} \to \frac{1}{1 - e^{-aT} z^{-1}}$
    
    代入 $T=2$：
    
    $$H(z) = \frac{2}{1 - e^{-1\times2} z^{-1}} - \frac{2}{1 - e^{-2\times2} z^{-1}}$$
    
    $$H(z) = \frac{2}{1 - e^{-2} z^{-1}} - \frac{2}{1 - e^{-4} z^{-1}}$$
    
3. 通分（可选）：
    
    如果题目要求写成一般式，则通分合并。
    

#### **解法 2：双线性变换法**

1. 准备公式：
    
    $T=2 \Rightarrow \frac{2}{T} = 1$。
    
    $$s = \frac{1 - z^{-1}}{1 + z^{-1}}$$
    
2. 代入 $H(s)$：
    
    $$H(z) = \frac{2}{ (\frac{1 - z^{-1}}{1 + z^{-1}})^2 + 3(\frac{1 - z^{-1}}{1 + z^{-1}}) + 2 }$$
    
3. 化简：
    
    分子分母同乘 $(1 + z^{-1})^2$：
    
    $$\text{分母} = (1 - z^{-1})^2 + 3(1 - z^{-1})(1 + z^{-1}) + 2(1 + z^{-1})^2$$
    
    $$\text{分母} = (1 - 2z^{-1} + z^{-2}) + 3(1 - z^{-2}) + 2(1 + 2z^{-1} + z^{-2})$$
    
    $$\text{分母} = 1 - 2z^{-1} + z^{-2} + 3 - 3z^{-2} + 2 + 4z^{-1} + 2z^{-2}$$
    
    $$\text{分母} = (1+3+2) + (-2+4)z^{-1} + (1-3+2)z^{-2} = 6 + 2z^{-1}$$
    
    $$\text{分子} = 2(1 + z^{-1})^2 = 2(1 + 2z^{-1} + z^{-2})$$
    
    $$H(z) = \frac{2(1 + 2z^{-1} + z^{-2})}{6 + 2z^{-1}} = \frac{1 + 2z^{-1} + z^{-2}}{3 + z^{-1}}$$
    
    最后整理成标准形式（分母常数项为1）：
    
    $$H(z) = \frac{\frac{1}{3} + \frac{2}{3}z^{-1} + \frac{1}{3}z^{-2}}{1 + \frac{1}{3}z^{-1}}$$

### 总结：考试答题模板

遇到这类题，请按以下模块书写，逻辑清晰，阅卷老师给分快：

1. **写出已知条件**：明确 $H(s)$、采样周期 $T$ 或 截止频率指标。
    
2. **声明使用的方法**：写出对应的公式（如 $s = \dots$）。
    
3. **预畸变计算（仅限双线性设计题）**：计算 $\Omega_c$。
    
4. **确定模拟滤波器 $H(s)$**：写出具体表达式。
    
5. **代入变换与化简**：这是计算最容易出错的地方，建议分步写，不要跳步。
    
6. **最终结果**：写成 $H(z) = \frac{N(z^{-1})}{D(z^{-1})}$ 的形式。

以下为您总结的**“标准解题套路”**（Standard Operating Procedure），涵盖了双线性变换法、脉冲响应不变法以及巴特沃斯滤波器的设计流程。您可以将其作为考场上的答题模板。

### 一、 核心解题逻辑图谱

在拿到题目后，首先确认以下三个关键信息：

1. **目标方法**：双线性变换法 (Bilinear Transformation) 还是 脉冲响应不变法 (Impulse Invariance)？
    
2. **模拟原型**：是否指定了巴特沃斯 (Butterworth)？是否直接给出了 $H(s)$？
    
3. **关键参数**：采样周期 $T$（或 $f_s$）、截止频率（通带/阻带）。**注意是否需要“预畸变”！**
    

---

### 二、 标准解题步骤详解

#### 场景 A：双线性变换法设计 (Bilinear Transformation) —— **最高频考点**

这是最常用的方法，核心在于解决了频谱混叠问题，但引入了频率畸变（Warping），因此**预畸变**是关键步骤。

**Step 1: 频率转换（预畸变 Pre-warping）**

- 若题目给定数字频率指标 $\omega_c$ (rad) 或 $f_c$ (Hz)：
    
    必须先将其转换为模拟域频率 $\Omega$。
    
    - 公式：$\Omega = \frac{2}{T} \tan(\frac{\omega}{2})$
        
    - 若给的是模拟指标 $f_c$ 且题目明确要求“预畸变”或“修正失真”，也需先算数字频率 $\omega = 2\pi f_c T$，再用上式反求校正后的模拟频率 $\Omega$。
        
    - _注：历年真题（如2020、2024）常考此点，务必写出计算过程。_
        

**Step 2: 设计模拟滤波器 $H(s)$**

- **情况 1：直接给出 $H(s)$**（如2017年第14题 $1/s$，2019年第9题）。
    
    - 直接跳到 Step 3。
        
- **情况 2：要求设计巴特沃斯滤波器**（如2019年第10题，2017年第10题）。
    
    1. **确定阶数 $N$**：根据通带衰减 $\alpha_p$ 和阻带衰减 $\alpha_s$ 计算（通常考研题会直接给出N或指标简单，可直接查表）。
        
        - _公式（备用）_：$N \ge \frac{\lg(\sqrt{\frac{10^{0.1\alpha_s}-1}{10^{0.1\alpha_p}-1}})}{\lg(\Omega_s/\Omega_p)}$
            
    2. **查表或写出归一化 $H_n(p)$**：
        
        - N=1: $p+1$
            
        - N=2: $p^2 + 1.414p + 1$
            
        - N=3: $(p+1)(p^2+p+1) = p^3 + 2p^2 + 2p + 1$
            
    3. **去归一化**：将 $p = \frac{s}{\Omega_c}$ 代入 $H_n(p)$ 得到实际的 $H(s)$。
        
        - 即：$H(s) = H_n(p) \Big|_{p = s/\Omega_c}$
            

**Step 3: S域到Z域的变换**

- 代入双线性变换公式：
    
    $$s = \frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}}$$
    
- 化简技巧：
    
    将代入后的式子上下同乘 $(1+z^{-1})^N$，整理成 $H(z) = \frac{\sum b_k z^{-k}}{1 - \sum a_k z^{-k}}$ 的形式。
    
    - _考试提示_：计算量较大，建议分步化简，不要跳步。
        

**Step 4: 写出差分方程（如果题目要求）**

- 将 $H(z) = \frac{Y(z)}{X(z)}$ 展开，利用 $z^{-k} Y(z) \leftrightarrow y(n-k)$ 写出时域表达式。
    

---

#### 场景 B：脉冲响应不变法 (Impulse Invariance) —— **易错点**

核心是时域采样，$h(n) = h_a(nT)$。**缺点是有频谱混叠**，只能用于低通或带通，不能用于高通。

**Step 1: 频率转换（线性关系）**

- 模拟频率 $\Omega$ 与数字频率 $\omega$ 为线性关系：
    
    $$\Omega = \frac{\omega}{T}$$
    
- _注意：此处没有 $\tan$ 函数，不要与双线性混淆。_
    

**Step 2: 设计/获取模拟滤波器 $H(s)$**

- 同双线性变换法，得到 $H(s)$。
    

**Step 3: 部分分式展开 (Partial Fraction Expansion)**

- 这是该方法的灵魂步骤。必须将 $H(s)$ 展开为单极点形式：
    
    $$H(s) = \sum_{k=1}^{N} \frac{A_k}{s - p_k}$$
    
- _注意：如果有多重极点（考研较少见），公式会变复杂。_
    

**Step 4: S域到Z域的变换**

- 利用变换对进行映射：
    
    $$\frac{1}{s - p_k} \implies \frac{1}{1 - e^{p_k T} z^{-1}}$$
    
- 最终得到：
    
    $$H(z) = \sum_{k=1}^{N} \frac{A_k}{1 - e^{p_k T} z^{-1}}$$

### 三、 理论简答题标准话术 (针对2020, 2021, 2022, 2024真题)

真题经常要求“比较两种方法”或“说明优缺点”，请背诵以下标准答案：

|**特性**|**脉冲响应不变法 (Impulse Invariance)**|**双线性变换法 (Bilinear Transformation)**|
|---|---|---|
|**映射关系**|$s$平面到 $z$平面的**多值映射**（多对一）。|$s$平面到 $z$平面的**单值映射**（一一对应）。|
|**频谱混叠**|**有混叠**（Aliasing）。$S$平面的多个条带映射到 $Z$平面同一个圆内。|**无混叠**。$S$平面的整个左半平面压缩到 $Z$平面的单位圆内。|
|**频率关系**|线性：$\omega = \Omega T$|非线性（畸变）：$\omega = 2 \arctan(\frac{\Omega T}{2})$|
|**适用范围**|仅适用于**带限信号**（如低通、带通），**不适用于高通、带阻**（因为高频部分混叠严重）。|适用于**各种类型**滤波器（低通、高通、带通、带阻）。|
|**优缺点**|**优**：模拟频率与数字频率线性对应，保持了冲激响应的时域特性。**缺**：有频谱混叠，高频响应失真。|**优**：彻底消除混叠，可设计任意类型滤波器。**缺**：高频部分频率压缩严重（Warping），需预畸变校正。|

### 四、 真题中的“特殊陷阱”与应对

1. **$H(s) = 1/s$ 的变换 (2017-Q14)**
    
    - 这是一个积分器。
        
    - **双线性法**：$H(z) = \frac{1}{\frac{2}{T} \frac{1-z^{-1}}{1+z^{-1}}} = \frac{T}{2} \frac{1+z^{-1}}{1-z^{-1}}$。
        
    - **差分方程**：$y(n) - y(n-1) = \frac{T}{2}[x(n) + x(n-1)]$（梯形积分公式）。
        
2. **关于 $T$ 的取值**
    
    - 有的题目 $T=1$，有的 $T=2$，甚至 $T=0.01$。
        
    - 在双线性变换中，如果未指定T，有时可设 $T=2$，这样 $\frac{2}{T}=1$，公式简化为 $s = \frac{1-z^{-1}}{1+z^{-1}}$（但需注明）。**但在真题中，通常必须代入给定的T值计算，不可随意约去。**
        
3. **预畸变 (Pre-warping) 的计算时刻**
    
    - **口诀**：“先畸变，后设计”。
        
    - 千万不要把模拟滤波器设计好了，最后再把截止频率拿去畸变，那是错的。必须用**畸变后的模拟截止频率**去设计 $H(s)$。
        

### 五、 备考建议

1. **熟记巴特沃斯归一化表格**（N=1, 2, 3 即可）。
    
2. **熟练掌握部分分式展开**（为了脉冲响应不变法）。
    
3. **计算器使用**：双线性变换经常涉及复数运算或繁琐的多项式乘法，平时练习务必动手算到底，不要只列式子。
    
4. **关注2024年趋势**：2024年第4题同时考了两种方法，第7题考了双线性+预畸变。这说明**计算的准确性**和**对两种方法流程的清晰区分**是当前的考察重点。
