4. 当在时域将信号表示为 $\delta$ 函数的时候，揭示出当信号通过系统的时候输入和输出是一种卷积的关系；当在频域将信号表示为正弦信号（虚指数信号）的时候，输入输出的关系是一种乘积的关系；当在复频域将信号表示为复指数信号的时候，输入输出的关系是一种乘积的关系==>信号作用于系统的机理，信号在不同域下输入输出的关系
	1. 信号卷积：将一个任意信号 $x(t)$ 表达为 $\delta(t)$ 时，$\begin{aligned}&x(t)=\lim_{\Delta\tau\to0}\sum_{k=-\infty}^\infty x(k\Delta\tau)\underline{\delta(t-k\Delta\tau)}\Delta\tau\\&y(t)=\lim_{\Delta\tau\to0}\sum_{k=-\infty}^{+\infty}x(k\Delta\tau)\cdot\underline{h(t-k\Delta\tau)}\Delta\tau\\&=\int_{-\infty}^{+\infty}x(\tau)h(t-\tau)\mathrm{d}\tau=x(t)*h(t)\end{aligned}$ 当所有信号都可以表示为 $\delta (t)$ 之后若 $\delta (t)$ 作用于系统的响应是 $h(t)$（单位冲激响应）。利用线性时不变特性，得到 $x(t)$ 产生的零状态响应是无限个无限小量和的极限是积分，根据将信号表示为 $\delta(t)$ 输出与输入和系统之间的关系是卷积的关系（连续卷积积分，离散卷积和），时域关系是分析其他域的基础
	2. 频域相乘：信号表达为虚指数信号（正弦信号）时，基本信号作用系统产生的响应和 $h(t)$ 是卷积的关系，$H (j\omega)$ 是 $h(t)$ 的傅里叶变换

13. 【背✅】**~={red}常用基本离散信号=~**
	1. **单位脉冲序列** $\delta[k]=\begin{cases}1&k=0\\0&k\neq0&\end{cases}$
		- 延时单位脉冲序列 $\delta[k-n]=\begin{cases}1&k=n\\0&k\neq n&\end{cases}$
		- 记忆：[ ]内整体的自变量等于 0 的点有值
		- 作用：信号的时域表示是将任意的离散信号表示为 $\delta[k]$ 这个信号，即任意离散信号都可以用 $\delta[k]$ 和它的延时通过加权表示 ![[截屏2025-11-03 15.28.13.png|300]]

	2. **单位阶跃序列** $u[k]=\begin{cases}1&k\geq0\\0&k<0&&\end{cases}$ ⚠️0 处有值；幅值为 1 故称单位，如果幅值不为 1 则为阶跃序列
		- 延时单位阶跃序列 $u[k-n]=\begin{cases}1&k\geq n\\0&k<n&&\end{cases}$

    5. **正弦类信号**$x[k] = A\cos(\Omega_0 k + \phi)$
	    1. 虚指数序列/正弦序列/余弦信号统称为**正弦类信号**，因为利用 Euler 公式可以将它们联系起来 $\begin{gathered}\mathbf{e}^{\mathrm{j}\Omega_0k}=\cos\Omega_0k+\mathrm{j}\sin\Omega_0k\\\cos\Omega_0k=\frac{1}{2}(\mathrm{e}^{\mathrm{j}\Omega_0k}+\mathrm{e}^{-\mathrm{j}\Omega_0k})\\\sin\Omega_0k=\frac{1}{2\mathrm{j}}(\mathrm{e}^{\mathrm{j}\Omega_0k}-\mathrm{e}^{-\mathrm{j}\Omega_0k})\end{gathered}$
	    2. 虚指数序列 ${e}^{\mathrm{j}\Omega_0k}$ 是一个复数，它的**实部**是一个**余弦序列** $\cos(\Omega k)$；**虚部**是一个**正弦序列** $\sin(\Omega k)$。它只改变**相位**，代表着纯粹的**振荡**。
		    1.  $k$ 是自变量，代表**离散时间**或**采样序号**，是一个整数
		    2.  $\Omega_0$ 代表的是**数字角频率**，对于一个**特定**的信号， $\Omega_0$ 是一个**常数**或**参数**。它定义了这个信号的“身份”——即它振荡得多快。
	    3. **周期性 (考点)**：离散正弦序列**不一定**是周期的。连续正弦类信号都是周期的信号。
	         - 周期信号定义：一个离散序列 $\tilde{x}[k]$，如果存在一个**正整数 $N$**，使得对于所有的整数 $k$ 都满足：$\tilde{x}[k] = \tilde{x}[k+N]$，则这个序列就称为周期序列。满足该条件的**最小正整数 $N$** 称为该序列的**基本周期**。
	         - **周期条件**: 仅当 $\frac{|\Omega_0|}{2\pi} = \frac{M}{N}$ (一个有理数) 时才为周期序列，周期为 $N$ ($M, N$ 为互质整数)。🧚‍♀️可以把 $e$ 的指数看成相角，如果那一项等于1，那么1的相角就是 $2\pi$ 的整数倍
	         - **考题** ：正弦型信号是否为周期信号
				1. 对 $x(t)=\cos(t)$ 以 $T=\pi/6$ (s) 抽样得到的序列 $x_1[k]$：$x_1[k] = \cos(k\pi/6)$，$\Omega_0 = \pi/6$；$\Omega_0/2\pi = 1/12$ 为有理数，序列 $x_1[k]$ 的周期 $N=12$。
				==>**抽样：原连续信号的 $t$ 换成 $Tn/Tk$**
				2. 对 $x(t)=\cos(t)$ 以 $T=1/6$ (s) 抽样得到的序列 $x_2[k]$：$x_2[k] = \cos(k/6)$，$\Omega_0 = 1/6$；$\Omega_0/2\pi = 1/12\pi$ 为非有理数，序列 $x_2[k]$ 是非周期序列 ![[Pasted image 20251110104612.png]]
				==>只有对原函数进行抽样，抽样点能抽到周期点上的时候，离散序列才可能成为周期序列🧚‍♀️因为用求解出来的 M/N 是一个无理数，无理数抽不到周期点
				==>离散正余弦信号或虚指数信号不一定是周期信号

	6. 复指数信号 $x[k] = c^k,c = a e^{j\Omega_0}$ ⚠️其中，$a$  是 $c$ 的**幅度**（一个实数）。$\Omega_0$ 是 $c$ 的**相位/幅角**（一个实数），复指数信号才会改变幅度

14. 【背✅】~={red}**基本运算**=~ 【**必考作图题**】
	1. **位/平移 (Shift)**: $x[k \pm n]$ (左加右减)。
		1. $k+n$ 提前、左移； $k-n$ 延时、右移
		2. $k+n$ 左移，原点值 $0$ 变成 $-n$
		3. 在 $k=n$ 时刻才开始的 $x[k]$ 则正是 $x[k-n]$。
	2. **翻转 (Reversal)**: $x[-k]$ (以 $k=0$ 纵轴为中心翻转)。$k=0$ 不动，其余点左右翻转
		==>平移和翻转都只是涉及到自变量的改变，不涉及到函数值的调整
		‼️序列在**时域翻转**（Time Reversal），对应在**频域也是翻转**。
		例如：如果 $x[n] \longleftrightarrow X(e^{j\Omega})$，那么：$x[-n] \longleftrightarrow X(e^{-j\Omega})$
		‼️在实际应用（特别是针对**实数信号**）中，这个性质有一个更有用的物理含义：**取共轭**。对于~={orange}实数信号=~，有一个非常重要的对称性：$X(e^{-j\Omega}) = X^*(e^{j\Omega})$(即：~={orange}频率取反 = 频谱取共轭=~)对于实数信号，结论变成了：时域翻转 $x[-n]$ $\longleftrightarrow$ 频域取共轭 $X^*(e^{j\Omega})$
	3. **抽取 (Decimation)**: $y[k] = x[Mk]$，(M为正整数)，从 $x[k]$ 中每 $M$ 个点（每隔 $M-1$ 个点）抽一个。
		1. 可以理解成对离散信号进行**压缩**，压缩之后坐标点非整数点的坐标就舍去了🧚‍♀️一样啊，连续是压缩，离散的抽取相当于是采样压缩后的值
		2. $n$ 倍抽样相当于信号的采样速率降低了 $n$ 倍
		3. 抽取之后由于抽样速率的改变同样一段原来的连续时间信号得到的样点（抽样数） $N$ 是不一样的
		4. $y[k] = x[Mk+n]$ (M 为正整数)也是从 $x[k+n]$ 中每 $M$ 个点（每隔 $M-1$ 个点）抽一个，⚠️不是 $x[Mk+Mn]$
		5. ⚠️注意抽取的含义，一旦抽取，原有的信息就损失一部分了。所以离散的抽取和连续伸缩还是有区别的。
	4. **内插 (Interpolation)**: $x_I[k] = \begin{cases} x[k/L] & k \text{ 是 } L \text{ 的整数倍} \\ 0 & \text{其它} \end{cases}$，在原序列点之间插 $L-1$ 个零。
		1. 可以理解成对离散信号进行**扩展**，扩展的点的值为 0
	==>无论翻转、抽取或内插 $k=0$ 不动
	5. **复合运算 (考点)**: 如 $y[k] = x[-3k+2]$。
	    - **标准步骤**: **先位移，再抽取/内插，最后翻转**。[[运算的非交换性]]
	    - $x[k] \xrightarrow{\text{左移} 2} x[k+2] \xrightarrow{3\text{倍抽取}} x[3k+2] \xrightarrow{\text{翻转}} x[-3k+2]$。

		3. **与卷积的关系 (考点)**: $R_{xy}[n] = x[-n] * y[n]$。**反转后再卷积**
		4. 性质: 
			1. $R_{xy}[n] = R_{yx}[-n]$；（翻转）
			2. $R_x[n]$ 是**偶对称**，即 $R_x[n]=R_x[-n]$；
			3. $|R_x[n]| \le R_x[0]$。（自相关函数的零点值一定是最大的，因为 0 点是两个信号完全重叠之后再乘积🧚‍♀️自相关=翻转（-n 翻转）之后乘以自己，当重叠的时候肯定是最大的）
				PS：2 和 3 可以用来检验自相关计算结果对不对
			4. $R_x(e^{j\Omega}) = |X(e^{j\Omega})|^2$

				2. 噪声信号的自相关函数大部分值比较小，个别点可能比较大
				3. 降噪耳机

15.  离散系统
	1. 定义：当一个系统的输入和输出都是离散的信号，称这样的系统为离散系统。即离散系统是处理离散信号的；若用 $x(n)$ 表示系统的输入, $y(n)$ 表示系统的输出, $T[\cdot]$ 表示为了得到 $y(n)$ 而对 $x(n)$ 施加的处理, 则系统的输入与输出之间的关系可以表示为: $y(n) = T[x(n)]$ 或者表示为: $x(n) \xrightarrow{T} y(n)$ ![[Pasted image 20251111111517.png|200]]
	2. 🌟离散系统的分类：
		1. 线性和非线性系统
			1. **线性 (Linearity)**: 满足叠加（可加性）和均匀（齐次、比例）特性，即线性特性：$T\{ax_1[k]+bx_2[k]\} = aT\{x_1[k]\} + bT\{x_2[k]\}$。输入为两个信号相加的时候，输出是分别两个输入单独作用于系统产生的响应值叠加。输入放大一定倍数，输出也放大同样的倍数[[单位延时器]]
			2. 非线性：不具有线性特性的系统🧚‍♀️这个就是矩阵论里的。
				- 例题：[[平方器系统]]
		2. 时变与非时变系统
			1. 时不变系统 Time-Invariance $x[k-n] \implies y_{zs}[k-n]$。：系统的零状态响应和输入激励的关系不随输入激励作用于系统的起点而改变/输入延时，输出也响应延时同样的单位（🐱如果输入起点变了 $\rightarrow$ 输出和输入的关系不变）
				输入是 $x[k]$ 的时候产生的零状态响应是 $x[k] \rightarrow y_{zs}[k]$，输入延时 $n$ 个单位输出也响应延时相同的 $n$ 个单位[[单位延时器]]
			2. 时变系统：[[调制器]]
			==>常见的线性非时变系统 
				1.  MA（移动平均Moving Average System）模型：$y[k] = a_0 x[k] + a_1 x[k-1] + \dots + a_M x[k-M]$ 该系统是线性非时变系统。🧚‍♀️前面加权系数不是k是常数就是非时变
		3. **因果与非因果系统**
			1. 因果系统：系统的输出响应不超前于系统的输入信号。==>输出可以滞后，不可以超前即“**没有输入就不应该有输出，有输入的时候才会有输出**” ![[Pasted image 20251111093109.png|300]]
			2. 非因果系统：
				1. 定义
					1. 不具有因果特性的系统称为非因果系统。
					2. **当前的输出 $y[k]$ 取决于未来的输入 $x[k+1], x[k+2] \dots$**。
				2. 非因果信号卷积需要用的未来的信号值，这个在现实中无法实时实现。
		4. **稳定与不稳定系统**
			1. 稳定系统：系统对任意的有界输入其输出也有界(有界即有最大值最小值)，称为BIBO稳定系统。BIBO: Bounded Input, Bounded Output 🧚‍♀️级数收敛![[截屏2025-11-11 09.32.35.png]]
			2. 不稳定系统：系统输入有界时，输出可能无界。
		‼️【背✅】总结： 
		 - **线性 (Linearity)**: 满足叠加原理。$T\{ax_1[k]+bx_2[k]\} = aT\{x_1[k]\} + bT\{x_2[k]\}$。
		 - **时不变 (Time-Invariance)**: $x[k-n] \implies y_{zs}[k-n]$。
		 - **LTI系统的时域特性 (必考)**:
			   1. **因果性 (Causality)**: $\iff h[k] = 0, \text{ for } k < 0$。
				- 证明：线性非时变系统输入输出关系 $y[k]=x[k]*h[k]$; 且**一个系统输出的起点等于两个序列的起点之和**，只要 $h[k]$ 的起点在等于 0 或者 0 之后，输出 $y[k]$ 的起点永远不可能超前于输入 $x[k]$ [[因果性时域证明]]
			2. **稳定性 (Stability)**: $\iff h[k]$ 绝对可和，$\sum_{k=-\infty}^{\infty} |h[k]| < \infty$。
				==>用系统的单位脉冲响应分析系统的特性
	3. 离散 LTI系统**数学模型**（~={red}**时域描述**=~）：
		1. **线性常系数差分方程**: $\sum_{i=0}^{N} a_i y[k-i] = \sum_{j=0}^{M} b_j x[k-j]$。（复杂）
			1. 延迟/差分 (Delays/Differences)，如 $y[k-n]$
			2. 描述系统在**当前时刻**的值与**过去时刻**值的关系
			3. 线性常系数微分方程通用公式：$\sum_{i=0}^{N} a_i \frac{d^i y(t)}{dt^i} = \sum_{j=0}^{M} b_j \frac{d^j x(t)}{dt^j}$（这个 $\sum$ 符号，不是对时间 $t$ 求和，而是在累加“不同阶的导数项”。）
		 2. **单位脉冲响应 $h[k]$**: $h[k] = T\{\delta[k]\}$ ($h[k]$ 是单位脉冲序列 $\delta[k]$ 激励系统所产生的**零状态响应**, 是系统的时域描述)。![[Pasted image 20251111094435.png|200]]
			Q：为什么用单位脉冲序列作为系统的时域描述呢
			A：任意的离散序列都可以表示为 $\delta[k]$，只用分析 $\delta[k]$ 作用于系统的响应就可以描述系统的时域特性
	4. 离散 LTI 系统响应的分析
		1. 原理： $x[k] = \sum_{n=-\infty}^{\infty} x[n]\delta[k-n]\rightarrow y[k] = \sum_{n=-\infty}^{\infty} x[n]h[k-n] = x[k] * h[k]$ 
			1. 公式 $x[k] = \sum_{n=-\infty}^{\infty} x[n]\delta[k-n]$ 看起来是把 $x[k]$ 写成了一个无穷多项的和，但实际上，对于任意一个 $k$ 值，这个无穷多项的和中**只有一项不为零，** 这个推导的核心，就是 $\delta[k]$ 的“筛选特性” (Sifting Property)。）
			2. 公式 $y_{zs}[k] = \sum_{n=-\infty}^{\infty} x[n]h[k-n] = x[k] * h[k]$ 通过将任意的信号表达为信号 $\delta[k]$ 来分析系统零状态响应的目的不仅仅是求系统的零状态响应，而是通过信号的表示揭示输入和系统之间的相互作用机理，为信号分析和处理奠定理论基础；离散时间LTI系统的零状态响应是输入信号与系统单位脉冲响应的卷积，此揭示了信号与系统在时域相互作用的机理==>知二求三
				PS: $y_{zs}[k]=x[k] * h[k]$ 是信号作用于线性非时变系统的机理，根据机理对信号进行分析和处理
				1. $x+H \to y \quad$ Analysis 分析：知道输入和系统来求输出-->求系统响应
				2. $y+H \to x \quad$ Control 控制 ：电磁信号收到之后，信号在传递过程中收到了噪声干扰，从输出当中重新估算出原来的输入🧚‍♀️自控原理？ 解卷积[[🌟解卷积]]
				3. $x+y \to H \quad$ Design 设计：滤波器的设计 
					(a) 已知输入$x$及系统$H$，求解输出$y$，称为正问题。
					(b) 已知输出 $y$ 及系统 $H$，求解输入 $x$；
					或已知输出 $y$ 及输入 $x$，求解系统 $H$；称为逆问题。
					(c) 已知输出 $y$ 及部分系统 $H$，求解输入 $x$；
					或已知输出 $y$ 及部分输入 $x$，求解系统 $H$，称为灰盒问题。
					(d) 已知输出 $y$，未知输入 $x$ 和系统 $H$，称为黑盒问题。
16. 离散周期信号频域分析：时域分析某些信号时会存在局限性；时域卷积运算复杂，频域分析（如滤波、辨识）更直观。  
		- 离散非周期序列频谱周期化，离散周期序列**频谱离散化**
		- $\tilde{X}[m]$ 是 $X(e^{j\omega})$ 在频率轴上的“采样值”。
		- 🌟 $\tilde{X}[m] = X(e^{j\omega}) \Big|_{\omega = \frac{2\pi}{N} m}$，若 $X(e^{j\omega})$ 这条连续的曲线，将区间 $[0, 2\pi]$ 平均切成 $N$ 份，在每一个切分点（$\frac{2\pi}{N} \cdot 0, \frac{2\pi}{N} \cdot 1, \dots$）上取一个值，值就是 $\tilde{X}[m]$。
	1. 频域表达：将信号表示为正弦类信号从而提供一种新的信号分析和处理的途径和方法
	2. 时域和频域的自变量分别是第几个采样点和第几次谐波（频率分量）
	3. 离散周期信号的 DFS（离散傅里叶级数 Discrete Fourier Series） 表示：周期为 N的任意周期序列可用 $N$ 个虚指数序列表示 $\tilde{x}[k] = \frac{1}{N} \sum_{m=0}^{N-1} \tilde{X}[m]e^{j\Omega_0 mk} \qquad \Omega_0 = \frac{2\pi}{N}$ ==>不同的周期信号只是前面的加权函数 $\tilde{X}[m]$ 不一样
		- PS：波浪号代表周期信号，信号像波一样周期延拓
		1. **对象**: 周期为 $N$ 的离散序列 $\tilde{x}[k]$。
		2. $\tilde{X}[m]$ 与 $\tilde{x}[k]$ 是一一对应的：时域信号不同，虚指数序列前面的加权系数 $\tilde{X}[m]$ 不同。![[Pasted image 20251111151045.png]]
		3. **变换对 (DFS Pair)**~={red}**【背公式】** =~$\tilde{x}[k] \xleftrightarrow{\text{DFS}} \tilde{X}[m]$
			- DFS: $\tilde{X}[m] = \sum_{k=0}^{N-1} \tilde{x}[k] e^{-j\frac{2\pi}{N}mk}$ ==>加权函数可以利用虚指数序列的正交特性计算得到
		    - IDFS: $\tilde{x}[k] = \frac{1}{N} \sum_{m=0}^{N-1} \tilde{X}[m] e^{j\frac{2\pi}{N}mk}$ ==> $\tilde{x}[k]$ 可以表示为由 $N$ 个虚指数信号的线性表示
			    - PS：~={orange}周期信号才有除，非周期的是频谱密度，没有除，连续积分中，那个无限小的微分 $d\omega$ 本质上是 $\frac{1}{N}$，“乘以一个无穷小的 $d\omega$” 在数学效果上等同于 “除以一个无穷大的 $N$”=~
			    - 正变换（DFT）是一个求和过程 $\sum_{k=0}^{N-1}$。当你把 $N$ 个 $1$ 加在一起，结果是 $N$。意味着信号的幅度被放大了 $N$ 倍。为了把这个数值“还原”回时域原本的大小，必须除以总点数 $N$
			    - 在离散信号处理中，“**对 $N$ 取模**” ($k \pmod N$)是 “**周期延拓**” 在数学操作上的具体实现方式。
		4. **频谱**: 加权系数称之为离散周期信号的频谱，也是**离散的**、**周期为 N** 的序列。（可以用欧拉公式化简后得到一部分为一，然后证明周期）
			1. $\underbrace{X(j\omega)}_{\text{频谱 (复数)}} = \underbrace{|X(j\omega)|}_{\text{幅度谱 (实数)}} \cdot e^{j \underbrace{\varphi(\omega)}_{\text{相位谱 (实数)}}}$
			2. $\underbrace{\tilde{X}[m]}_{\text{离散频谱 (复数)}} = \underbrace{|\tilde{X}[m]|}_{\text{幅度谱 (实数)}} \cdot e^{j \underbrace{\varphi[m]}_{\text{相位谱 (实数)}}}$
	4. DFS 性质：
		 所有傅立叶变换都是表示为正弦类信号或虚指数类信号，很多性质是相似的
		 所有傅立叶变换时域频域都是一一对应的
		1. 线性特性：两个周期均为 $N$ 的序列 $\tilde{x}_1[k]$ 和 $\tilde{x}_2[k]$ 的DFS分别为 $\tilde{X}_1[m]$ 和 $\tilde{X}_2[m]$ 7。那么对于任意常数 $a$ 和 $b$：$a\tilde{x}_1[k] + b\tilde{x}_2[k] \longleftrightarrow a\tilde{X}_1[m] + b\tilde{X}_2[m]$
		2. 位移特性：周期信号位移之后周期不改变，只需要观察位移之后序列一个周期的情况即可
			1. **时域位移**：频域怎么改变？==>幅度函数不发生改变，相位发生改变，即序列在时域的位移对应频域的相移（相移是单位圆相位的移动） $\tilde{x}[k+n] \longleftrightarrow e^{j\frac{2\pi}{N}mn} \tilde{X}[m]$
			2. **频域位移**：序列在时域的相移对应频域的位移， $e^{-j\frac{2\pi}{N}lk} \tilde{x}[k] \longleftrightarrow \tilde{X}[m+l]$
				- **调制**（Modulation）的数学原理。在时域将信号乘以一个复指数（$e^{-j\frac{2\pi}{N}lk}$），就相当于在频域将其频谱搬移了 $l$ 个位置。
				- 在时域乘以一个正指数 $e^{j\frac{2\pi}{N}lk}$，会导致频谱负向位移。$e^{j\frac{2\pi}{N}lk} \tilde{x}[k] \longleftrightarrow \tilde{X}[m-l]$
		3.  对称特性：
			1. 偶对称： $\tilde{x}[k] = \tilde{x}[-k] = \tilde{x}[N-k]$ 
				1. 周期 $N$ 为偶数：周期序列满足偶对称时，在一个周期内也具有对称特性 
					- DFT 会用到相关的对称关系 
				 ![[Pasted image 20251113111527.png|300]]
				2. 周期 $N$ 为奇数：![[Pasted image 20251113111602.png|300]]
				3. 一个周期内，$N/2$ 处也是一个对称点，不仅关于0点对称也关于 $N/2$ 对称
			2. 奇对称：$\tilde{x}[k] = -\tilde{x}[-k] = -\tilde{x}[N-k]$
			==>若周期序列是偶（奇）对称的，则在任一周期N内也是偶（奇）对称的 🧚‍♀️就是半波镜像和半波重叠嘛；可类比正余弦函数
			3. 满足一定的对称关系，频谱会怎么改变？
				1. 答：若 $\tilde x[k]$ 对应的频谱是 $\tilde{X}[m]$，则共轭 $\tilde x^*[k]$ 对应的频谱是原来频谱的共轭，同时 $m$ 取相反数 $\tilde{X}^*[-m]$，相对于对频谱 $\tilde{X}[m]$ 翻转再共轭；若对原信号进行翻转再共轭 $\tilde x^*[-k]$，其对应的频谱是原来 $\tilde x[k]$ 频谱的共轭 $\tilde{X}^*[m]$
				2. 如果 $x[k]$ 是实序列，则有 $\tilde{X}[m] = \tilde{X}^*[-m]$ 
					1. $\tilde{X}[m]$ 表示为幅度与相位则 $\tilde{X}[m]=|\tilde{X}[m]|e^{j\varphi[m]}$
						1. 幅度谱是偶对称 $|\tilde{X}[m]|=|\tilde{X}[-m]|$**幅偶**
						2. 相位谱是奇对称 $\varphi[m]=-\varphi[-m]$**相奇**
					2. $\tilde{X}[m]$ 表示为实部与虚部则 $\tilde{X}[m]=\tilde{X}_r[m]+j\tilde{X}_i[m]$
						1. 实部是偶对称的：$\tilde{X}_r[m] = \tilde{X}_r[-m]$ 。**实偶**
						2. 虚部是奇对称的： $\tilde{X}_i[m] = -\tilde{X}_i[-m]$ 。**虚奇**
				3. 原理：实序列 $x[k]$（无论它是否周期），都可以被唯一地分解为一个“实偶部” $x_e[k]$ 和一个“实奇部” $x_o[k]$ 的和。$x[k] = x_e[k] + x_o[k]$ ==> $\tilde{X}[m] = \tilde{X}^*[-m]$
					1. 频谱的**实部** ($R[m]$) 对应时域信号的 **偶分量**，所以实部必须是 **偶对称** 的。
					2. 频谱的**虚部** ($I[m]$) 对应时域信号的 **奇分量**，所以虚部必须是 **奇对称** 的。
		    ==>实际意义:对于一个实数序列，**幅偶相奇实偶虚奇**【背✅】，只需要计算和存储它**一半**的频谱（例如 $m=0$ 到 $N/2$），另一半的信息是冗余的，可以通过这个对称性直接推导出来。
			==>实偶部变换后还是实偶部，实奇部变换后会变为虚奇部
			==>【验证】分析实序列信号频谱时如果幅度谱不满足偶对称，相位谱不满足奇对称，则频谱计算有误🙅
		4. **周期卷积定理 (考点)**：揭示了时域和频域之间的对应关系和物理概念
			1. 时域周期卷积定理： 时域周期卷积 $\iff$ 频域乘积。
				1. 文字叙述：两个周期序列在时域的卷积对应的DFS（离散傅里叶级数）是等于两个序列各自离散傅立叶级数的乘积
				2. 公式：$\tilde{x}_1[k] * \tilde{x}_2[k] \xleftrightarrow{\text{DFS}} \tilde{X}_1[m]\cdot \tilde{X}_2[m]$
			2. 频域周期卷积定理：频域周期卷积 $\iff$ 时域乘积。
				1. 文字描述：两个周期序列在时域的乘积对应的频谱是各自频谱的周期卷积，前面有一个系数 $\frac{1}{N}$（ %% $\frac{1}{N}$ 是 DFS/DFT 特有的系数。在连续时间傅里叶变换 (CTFT) 中，这个系数通常是 $\frac{1}{2\pi}$ %%）![[Pasted image 20251122202841.png]]
				2. 公式：$\tilde{x}_1[k] \cdot \tilde{x}_2[k] \xleftrightarrow{\text{DFS}} \frac{1}{N} \tilde{X}_1[m] * \tilde{X}_2[m]$
			3. 周期卷积：和一般信号卷积求和相比上下限从-∞到+∞变成了一个周期里求和

			1. $l$ 代表着第几个“周期副本”的平移倍数。
			2. $\sum x[k+lN]$ 能变成周期信号 $\begin{aligned} \tilde{x}[k+N] &= \sum_{l=-\infty}^{\infty} x[(k+N) + lN] \\ &= \sum_{l=-\infty}^{\infty} x[k + (l+1)N] = \sum_{m=-\infty}^{\infty} x[k + mN] = \tilde{x}[k]\end{aligned}$
		5. 文字：序列 $x[k]$ 的频谱 $X(\mathrm{e}^{\mathrm{j}\Omega})$ 离散化，对应其时域序列 $x[k]$ 的周期化
		6. 频域离散化，时域周期化，周期化过程中 $\tilde{x}[k]$ 的周期序列是否包含原来的序列是有条件的。频域离散化导致时域周期化无条件，周期化之后的序列当中是否包含原来的 $x[k]$ 是需要条件的

18. 离散 LTI 系统的频域描述：信号与系统紧密相关，信号表示为频域时，相应系统描述也有频域描述
	1. 离散 LTI 系统频率响应 $H(e^{j\Omega})$：
		1. 语言定义： LTI 系统单位脉冲响应 $h[k]$ 的离散时间傅里叶变换（DTFT）。（输入是 $e^{j\Omega k}$ 时输出依然含有 $e^{j\Omega k}$，再与系统频率响应相乘）
		2. 公式定义：$H(e^{j\Omega}) = \text{DTFT}\{h[k]\} = \sum h[k]e^{-j\Omega k}$
		3. **核心关系**$Y(e^{j\Omega}) = X(e^{j\Omega}) \cdot H(e^{j\Omega})$
		4. ~={red} 物理含义=~： $H(e^{j\Omega})$ 反映了离散系统对输入信号当中不同的频率**分量（虚指数信号）** 的传输特性 $$\begin{aligned} \text{Input} & \xrightarrow{\quad \text{LTI System} \quad} \text{Output} \\ \\ e^{\mathrm{j}\Omega_0 k} & \longrightarrow e^{\mathrm{j}\Omega_0 k} H(\mathrm{e}^{\mathrm{j}\Omega_0}) \\ e^{\mathrm{j}\Omega_1 k} & \longrightarrow e^{\mathrm{j}\Omega_1 k} H(\mathrm{e}^{\mathrm{j}\Omega_1}) \\ &\vdots \end{aligned}$$
		5. $H(e^{j\Omega})$ 一般是 $\Omega$ 的**复函数**，可以表示为幅度和相位的形式：（复数，可以写成“幅度 $\times$ 相位旋转因子”的极坐标形式）
			1. 公式：$H(\mathrm{e}^{\mathrm{j}\Omega}) = \underbrace{|H(\mathrm{e}^{\mathrm{j}\Omega})|}_{\text{幅度响应}} \cdot \mathrm{e}^{\mathrm{j}\underbrace{\varphi(\Omega)}_{\text{相位响应}}}$
			2. $h[k]$ 是实序列时，则：
				1. 幅度响应 $|H(\mathrm{e}^{\mathrm{j}\Omega})|$ 是 $\Omega$ 的偶函数 ：$|H(\mathrm{e}^{\mathrm{j}\Omega})| = |H(\mathrm{e}^{-\mathrm{j}\Omega})|$
				2. 相位响应 $\varphi(\Omega)$ 是 $\Omega$ 的奇函数：$\varphi(\Omega) = -\varphi(-\Omega)$
			3. $e^{\mathrm{j}\Omega_0 k} H(\mathrm{e}^{\mathrm{j}\Omega_0})$ 代入 $H(\mathrm{e}^{\mathrm{j}\Omega_0}) = |H(\mathrm{e}^{\mathrm{j}\Omega_0})|\mathrm{e}^{\mathrm{j}{\varphi(\Omega_0)}}$ 得到 $|H(\mathrm{e}^{\mathrm{j}\Omega_0})|\mathrm{e}^{\mathrm{j}({\varphi(\Omega_0)+\Omega_0k})}$, 幅度响应改变幅度，相位响应改变相位
			4. 正余弦信号通过系统响应时会进一步看出幅度响应和相位响应不同的特性
		6. 离散LTI 系统描述
			1. 时域描述：通过时域的单位脉冲响应进行描述 $h[k]$
			2. 频域描述：通过频域的系统频率响应 $H(e^{j\Omega})$
			3. 离散时间傅立叶变换对：$h[k] \xleftrightarrow{\text{DTFT}} H(\mathrm{e}^{\mathrm{j}\Omega})$
		7. 非周期序列通过离散 LTI 系统的输出（响应）：
			1. 离散非周期信号**满足一定条件下**可以表示为 $e^{j\Omega k}$ 的形式，且前面加权函数称之为这个信号的频谱$x[k] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(\mathrm{e}^{\mathrm{j}\Omega}) \mathrm{e}^{\mathrm{j}\Omega k} \mathrm{d}\Omega$
			2. 响应关系+线性时不变系统的特性， $x[k] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(\mathrm{e}^{\mathrm{j}\Omega}) \, \underline{\mathrm{e}^{\mathrm{j}\Omega k}} \, \mathrm{d}\Omega$  $\xrightarrow{\quad \text{LTI System} \quad}$ $y_{\mathrm{zs}}[k] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(\mathrm{e}^{\mathrm{j}\Omega}) \, \underline{H(\mathrm{e}^{\mathrm{j}\Omega})\mathrm{e}^{\mathrm{j}\Omega k}} \, \mathrm{d}\Omega$ 根据傅立叶变换定义输出信号 $\mathrm{e}^{\mathrm{j}\Omega k}$ 前面是对应的频谱函数 ==> 输出信号频谱表达式 $Y(e^{j\Omega}) = X(e^{j\Omega}) \cdot H(e^{j\Omega})$ 该结论和前面傅立叶变换性质的卷积特性相呼应==>揭示信号通过系统时在频域的相互作用关系，不只是用来简化运算的==>不同域中信号和系统之间的相互关系，知道在频域中信号和系统的处理关系更有利于分析和处理信号🧚‍♀️频谱就是频率谱，在频域里研究一个信号
		8. 周期序列通过离散 LTI 系统的输出（响应）：
			1. 离散周期序列$\tilde{x}[k]$可以表达为 N 项的虚指数序列$e^{j \frac{2\pi}{N} mk}$的叠加，加权函数称之为其离散傅立叶级数（频谱）$\underbrace{\tilde{x}[k]}_{\text{复原的序列}} = \sum_{m=0}^{N-1} \underbrace{\tilde{X}[m]}_{\text{加权函数(权重)}} \cdot \underbrace{e^{j \frac{2\pi}{N} mk}}_{\text{N项虚指数}}$
			2. 响应关系+线性时不变系统的特性，将周期信号 $\tilde{x}[k]$ 分解为一系列不同频率（$\Omega_m = \frac{2\pi m}{N}$）的复指数信号的加权和 $\tilde{x}[k] = \frac{1}{N} \sum_{m=\langle N \rangle} \tilde{X}[m] \, \underline{\mathrm{e}^{\mathrm{j}\frac{2\pi m}{N}k}}$ $\xrightarrow{\quad \text{LTI System} \quad}$ $\tilde{y}[k] = \frac{1}{N} \sum_{m=\langle N \rangle} \tilde{X}[m] \, \underline{H\left(\mathrm{e}^{\mathrm{j}\frac{2\pi m}{N}}\right)} \, \mathrm{e}^{\mathrm{j}\frac{2\pi m}{N}k}$，通过求系统响应给出频域输入输出系统之间的关系$\tilde{Y}[m] = \tilde{X}[m] \cdot H\left(\mathrm{e}^{\mathrm{j}\frac{2\pi m}{N}}\right)$
		9. 离散线性时不变系统
			1. 通过**差分方程**描述输入输出的关系
			2. 通过时域的**单位脉冲响应**来描述输入输出的关系
			3. 通过**系统频率响应**描述输入输出的关系
			-  例如：当前的输出，等于一半的上次输出，加上当前的输入。假设输入是一个单位脉冲 $\delta[k]$（即 $x[0]=1$，后面全是 $0$），初始状态 $y[-1]=0$。
				- 差分方程：$y[k] - 0.5y[k-1] = x[k]$ 
				- 单位脉冲响应：$h[k] = (0.5)^k u[k]$，$y[k] = x[k] * h[k] = \sum_{m=-\infty}^{+\infty} x[m] (0.5)^{k-m} u[k-m]$
				- 频率响应描述：对 $h[k]$ 做离散时间傅里叶变换 DTFT：$H(\mathrm{e}^{\mathrm{j}\Omega}) = \sum_{k=-\infty}^{+\infty} (0.5)^k u[k] \mathrm{e}^{-\mathrm{j}\Omega k} = \sum_{k=0}^{+\infty} (0.5 \mathrm{e}^{-\mathrm{j}\Omega})^k$
					1. 低频 ($\Omega = 0$, 直流)：$H(\mathrm{e}^{\mathrm{j}0}) = \frac{1}{1 - 0.5} = \mathbf{2}$ (信号被放大了 2 倍)
					2. 高频 ($\Omega = \pi$, 震荡最快)：$H(\mathrm{e}^{\mathrm{j}\pi}) = \frac{1}{1 - 0.5(-1)} = \frac{1}{1.5} = \mathbf{0.67}$ (信号被衰减了，只剩 0.67 倍)
					3. **特点：** 一眼就能看出这是一个 **低通滤波器 (Low Pass Filter)**。它让低频通过（放大），压制高频。是差分方程和时域波形很难直接看出来的。
				🌟三者关系：差分方程进行 DTFT很容易得到 $Y (e^{j\Omega})$ 和 $X (e^{j\Omega})$ 的关系，之后通过 $H (e^{j\Omega})=\frac{Y (e^{j\Omega})} { X (e^{j\Omega}) }$ 得到频域响应描述，之后再通过 IDTFT 得到单位脉冲响应$h[k]$ 的关系式
		10. **DTFT** 专门处理**无限长**的非周期信号（只要它满足收敛条件）；**DFT** 专门处理**有限长**的序列（比如长度为 $N$ 的数组）。
		11. **稳定**意味着系统极点都在单位圆内。
			- **结论：** 当系统稳定时，收敛域 (ROC) 包含单位圆。此时，$z = e^{j\Omega}$ 的 Z 变换就是 DTFT。
			- **做题习惯：** 当题目直接问频率响应时，直接设 $z = e^{j\Omega}$ 进行 DTFT 运算会更直接（虽然本质上和算 Z 变换再代入是一样的）。

			2. **相位**叠加了系统的相位响应 $\varphi(\Omega_0)$。
			印证了“正弦进，正弦出”的特征函数性质。幅值和相位恰恰是系统频率响应在 $\Omega_0$ 这一点频率响应的幅度响应和相位响应进行了改变，这也是 $|H(\mathrm{e}^{\mathrm{j}\Omega_0})|$ 、$\varphi(\Omega_0)$ 叫幅度函数和相位函数的原因：可以改变正余弦信号的幅度和相位
			【总结】系统频率响应的幅度函数改变的是幅度，相位响应改变的是相位，**频率 $\Omega_0$** 是身份证，永远不变

19. 离散时间信号的复频域分析即 Z 域分析 (Z-Transform)
	0. 信号与系统预备知识：
		1. **拉普拉斯变换**和**Z变换**本质上都是将信号从“时域”（时间 $t$ 或 $n$）转换到“复频域”（$s$ 或 $z$）的工具
		2. 为什么引入双边 z 变换：单边序列的变换对象都是单边的，有时会输入信号或者单位脉冲响应不是单边信号，所以引用双倍z变换
		3. $s$ 域 $\sigma < 0$ 映射成了 Z 平面上**单位圆内部的一个同心圆**; $\sigma > 0$ 映射成了 Z 平面上**单位圆外部的一个同心圆**；单位圆Z 域中是 S 域虚轴的“替身”。
		4. $z$ 复指数形式：$z = r e^{j\omega}$，$z^k = (r e^{j\omega})^k = \underbrace{r^k}_{\text{模（幅度）}} \cdot \underbrace{e^{j\omega k}}_{\text{相角（频率）}}$
			1. 幅度部分 $r^k$ (增长或衰减)： 是 $z^k$ 区别于普通正弦波（傅里叶变换）的关键。**是为什么它是“非周期”的** ，除非 $r=1$，否则信号的幅度一直在变大或变小，波形永远不会重复之前的值。
				- **若 $r > 1$（极点在圆外）：** $r^k$ 随时间 $k$ 指数**爆炸增长**
				- **若 $r < 1$（极点在圆内）：** $r^k$ 随时间 $k$ 指数**衰减趋零**
				- **若 $r = 1$（极点在圆上）：** 幅度不变（就是傅里叶变换的情况）
			2. 旋转部分 $e^{j\omega k}$ (震荡频率)：代表在复平面上的旋转。根据欧拉公式：$e^{j\omega k} = \cos(\omega k) + j\sin(\omega k)$。它决定了信号震荡的快慢（频率 $\omega$）。**这就是为什么它是“非周期”的：** 除非 $r=1$，否则信号的幅度一直在变大或变小，波形永远不会重复之前的值。
	1. Z 变换是拉普拉斯变换的离散对应，能处理更广泛的序列 (如不满足DTFT收敛条件的)，并将**卷积运算**变为**乘积运算**。
	2. 双边 **Z 变换**及收敛域 (ROC)：
		1. 双边 $z$ 正变换: ~={red} $X(z) = \sum_{k=-\infty}^{\infty} x[k] z^{-k}$ =~，PS：$x[k]$ 不一定要求是因果序列，可以是双边序列; 注意是 $z^{-k}$ 🧚‍♀️对 $x[k]$ 作Z变换应为对 $x[k]z^{-k}$ 的求和，$k$ 取-∞到+∞
		2. 双边 $z$ 反变换：$x[k] = \frac{1}{2\pi \mathrm{j}} \oint_{C} X(z) z^{k-1} \mathrm{d}z$ ，PS：C为 $X(z)$ 的**收敛域中**的一闭合曲线
		3. 收敛域 （Region Of Convergence）：使得正变换收敛的 $z$ 值区域，用 $z$ 的模表示
	3. **序列类型与 ROC 形状 (必考)**：
		1. 有限长序列：$$X(z) = \sum_{k=N_1}^{N_2} x[k] z^{-k}$$有 $0 < |z| < \infty$ (可能除去 $z=0$ 或 $z=\infty$)。
			1.  是有限项求和
			2.  $z$ 变换 ROC 特点：$0 < |z| < \infty$
				0. $|z| =0$ 时，这个等比数列不成立，具体是使用大于0还是小于0那段看变换式决定
				1. 例如 $x[k]=\delta[k]$，$X (z) = 1$，ROC：$|z| \geq 0$，即充满整个 z 的平面
				2. 例如 $x[k]=R_N[k]$，$X(z) = \sum_{k=-\infty}^{+\infty} R_N[k] z^{-k} = \sum_{k=0}^{N-1} 1 \cdot z^{-k} = \frac{1-z^{-N}}{1-z^{-1}}, \quad |z| > 0$
				3. 例如 $x[k] = \{ -2, 1, \overset{\downarrow}{3}, 2, 4 \}$（非因果序列），$X(z) = -2z^{2} + 1 \cdot z^{1} + 3 + 2z^{-1} + 4z^{-2}$，$|z| < +\infty$，强调性的数学表达，收敛域不包含无穷远点 ($z = \infty$)
			3. 时限信号全平面收敛，是性质
			4. $z^{-k}$ 在 $z$ 模大于零时收敛
		2. **右边序列**（因果序列） (如 $a^k u[k]$): $$X(z) = \sum_{k=N_1}^{\infty} x[k] z^{-k}$$$\implies$ **圆外**，ROC 为 $|z| > |a|$。$|z| > R_{x-}$
			1. 语言描述：从某一个起点开始，向右持续到**无限远**
			2.  $z$ 变换 ROC 特点：$|z| > R_{x-}$ ![[截屏2025-11-25 11.25.18.png|100]]
				1. 例如：$N_1 \geq 0$，指数序列 $a^k u[k]$ 的双边 $z$ 变换及其收敛域 $X(z) = \sum_{k=0}^{\infty} a^k z^{-k} = \frac{1}{1 - az^{-1}} \quad |z| > |a|$，收敛域是以 $R_{x-} = |a|$ 为半径的圆外区域 
				2. 例如：$N_1 < 0$，$X(z) = x[N_1] z^{-N_1} + \dots + x[-1]z + \sum_{k=0}^{\infty} x[k] z^{-k}$，$|z| \neq \infty$,  变换的收敛域是 $z$ 变换的收敛域是 $R_{x-} < |z| < \infty$
		3. 左边序列 (如 $-b^k u[-k-1]$): $$X(z) = \sum_{k=-\infty}^{N_2} x[k] z^{-k}$$$\implies$ **圆内**，ROC 为 $|z| < |b|$。$|z| < R_{x+}$
			1. 语言描述：从某一个起点开始，持续到**负无限远**
			2. $z$ 变换 ROC 特点： $|z| < R_{x+}$ ![[截屏2025-11-25 13.36.06.png|100]]
				1. 例如： $N_2 \leq 0$，   $X(z) = \sum_{k=-\infty}^{-1} -b^k z^{-k} = \frac{1}{1 - bz^{-1}} \quad |z| < |b|$，收敛域是以 $R_{x+} = |b|$ 为半径的圆内区域
				2. 例如：$N_2 > 0$ 时， $X(z)$ 可表示为: $X(z) = \sum_{k=-\infty}^{-1} x[k] z^{-k} + x[0] + x[1]z^{-1} + \dots + x[N_2] z^{-N_2}$，$|z| \neq 0$，左边序列 $z$ 变换的收敛域为 ：$0 < |z| < R_{x+}$ 🧚‍♀️等于0的话，会存在 0的-1次方这一项
		4. 双边序列 (如 $a^k u[k] - b^k u[-k-1]$): $$X(z) = \sum_{k=-\infty}^{\infty} x[k] z^{-k}$$$\implies$ **圆环**，ROC 为 $|a| < |z| < |b|$，$R_{x-} < |z| < R_{x+}$。 (假设 $|a|<|b|$)。
			1. 容易和有限长混淆，双边序列一定是从负无穷远持续到正无穷远
			2. $z$ 变换 ROC 特点： $R_{x-} < |z| < R_{x+}$ ![[截屏2025-11-25 13.50.41.png|100]]
				1.  $a^k u[k] - b^k u[-k-1]$ 的双边 $z$ 变换，$X(z) = \frac{1}{1-az^{-1}} + \frac{1}{1-bz^{-1}}$，$|a| < |z| < |b| \quad (|a| < |b|)$ 🧚‍♀️仔细看 就是把上面的两个公式相加了
				2. 环形带状区域
	4. **稳定性判据 [考点]**：系统稳定的充要条件是 **ROC 包含单位圆** ($|z|=1$)
	5. 双边 $z$ 变换的性质 $X(z) = \mathscr{Z}\{x[k]\}, \quad R_x = \{z; R_{x-} < |z| < R_{x+}\}$，$Y(z) = \mathscr{Z}\{y[k]\}, \quad R_y = \{z; R_{y-} < |z| < R_{y+}\}$ ![[Pasted image 20251125135608.png]]
		1.  PS：$R_{x-}$（内径）：由**右边部分**（$k \ge 0$）决定。它等于右边部分中模最大的极点；$R_{x+}$（外径）：由**左边部分**（$k < 0$）决定。它等于左边部分中模最小的极点。
		2. **位移特性**（与单边 $z$ 变换不一样）：$x[k-n] \overset{\mathscr{Z}}{\leftrightarrow} z^{-n}X(z)$
		3. **时间翻转**：若 $X(z) = \mathscr{Z}\{x[k]\}, \quad R_x = \{z; R_{x-} < |z| < R_{x+}\}$ 则 $x[-k] \overset{\mathscr{Z}}{\longleftrightarrow} X(z^{-1}) \quad \text{ROC: } \frac{1}{R_{x+}} < |z| < \frac{1}{R_{x-}}$ ⚠️翻转收敛域会发生改变‼️
		4. 指数加权：$a^k x[k] \leftrightarrow X(z/a)$ (极点缩放)
		5. 线性加权 (微分)：$k x[k] \leftrightarrow -z \frac{dX(z)}{dz}$
		6. 初值定理：$x[0] = \lim_{z \to \infty} X(z)$ (仅适用于因果序列)。
	6. 双边 **Z 反变换**（求 $x[k]$)）
		1. **方法**:
			1. 部分分式展开 (PFE)：一般只适用于有理分式
			2. 留数法
		2. **核心思想 (必考)**: $x[k]$ 由 $X(z)$ 和 ROC 共同唯一确定。
			1. 【背】基本对 (1) 右边序列的标准公式： $a^k u[k] \iff \frac{1}{1-az^{-1}} = \frac{z}{z-a}$，ROC: **$|z| > |a|$**。
			2. 【背】基本对 (2) 左边序列的标准公式: $-a^k u[-k-1] \iff \frac{1}{1-az^{-1}} = \frac{z}{z-a}$，ROC: **$|z| < |a|$**。
				- 以上左右边序列 z 变换得到的频域公式相同，只有收敛域不一样==>做 $Z$ 反变换时，**必须**先看收敛域 ROC，才能决定是用右边序列的公式，还是用左边序列的公式。
				- $x[k]$ 与 $X(z)$ 是 $z$ 变换的关系，他们之间不构成一一对应的关系，必须加上收敛域 $x[k] \iff X(z) + \text{ROC}$
			3. **考题形式**: 给你 $X(z)$ 和 $3$ 种不同的 ROC (如 $|z|>3$, $|z|<2$, $2<|z|<3$)，求 $3$ 个不同的 $x[k]$。
			4. **[⚠️]** 注意假分式（分子阶数 $\ge$ 分母阶数）需先进行多项式除法化为真分式
			5. 极点：$z$ 平面收敛域不能包含极点，极点是是让 $X(z)$ 的数值“爆炸”（变成无穷大）的那些 $z$ 值
	7. 常见信号的 z 变换对
		1. **单位脉冲**：$\delta[k] \xleftrightarrow{\mathcal{Z}} 1$ \quad $(\text{ROC}: \text{整个 } z \text{ 平面})$
		2. **移位脉冲**：$\delta[k-m] \xleftrightarrow{\mathcal{Z}} z^{-m}$  $(\text{ROC}: z \neq 0 \text{ 若 } m>0)$
		3. **直流常数**：$1 (\text{双边}) \xleftrightarrow{\mathcal{Z}} \text{不存在}$  $(\text{因不满足绝对可和，无收敛域})$
		4. **单位阶跃**：$u[k] \xleftrightarrow{\mathcal{Z}} \frac{1}{1-z^{-1}} = \frac{z}{z-1}$ $(\text{ROC}: |z| > 1)$
		5. **【背】因果指数**：$a^k u[k] \xleftrightarrow{\mathcal{Z}} \frac{1}{1 - a z^{-1}} = \frac{z}{z-a}$  $(\text{ROC}: |z| > |a|)$
		6. **反因果指数**：$-a^k u[-k-1] \xleftrightarrow{\mathcal{Z}} \frac{1}{1 - a z^{-1}} = \frac{z}{z-a}$ $(\text{ROC}: |z| < |a|)$
		7. **双边指数**：$a^{|k|} \xleftrightarrow{\mathcal{Z}} \frac{1-a^2}{1 - a(z + z^{-1}) + a^2}$  $(\text{ROC}: |a| < |z| < \frac{1}{|a|}, |a|<1)$
		8. **对称矩形窗**：$R_{2M+1}[k] \xleftrightarrow{\mathcal{Z}} \frac{z^M - z^{-(M+1)}}{1 - z^{-1}}$ $(\text{ROC}: z \neq 0)$
		9. **通用矩形窗**：$R_N[k] \xleftrightarrow{\mathcal{Z}} \frac{1 - z^{-N}}{1 - z^{-1}}$  $(\text{ROC}: z \neq 0)$
		10. **复指数**：$e^{j\Omega_0 k} u[k] \xleftrightarrow{\mathcal{Z}} \frac{1}{1 - e^{j\Omega_0} z^{-1}} = \frac{z}{z - e^{j\Omega_0}}$  $(\text{ROC}: |z| > 1)$
		11. **【背】余弦**：$\cos(\Omega_0 k) u[k] \xleftrightarrow{\mathcal{Z}} \frac{1 - z^{-1}\cos(\Omega_0)}{1 - 2z^{-1}\cos(\Omega_0) + z^{-2}} = \frac{z(z - \cos\Omega_0)}{z^2 - 2z\cos\Omega_0 + 1}$  $(\text{ROC}: |z| > 1)$
		12. **正弦**：$\sin(\Omega_0 k) u[k] \xleftrightarrow{\mathcal{Z}} \frac{z^{-1}\sin(\Omega_0)}{1 - 2z^{-1}\cos(\Omega_0) + z^{-2}} = \frac{z\sin\Omega_0}{z^2 - 2z\cos\Omega_0 + 1}$  $(\text{ROC}: |z| > 1)$
		13. **斜坡加权**：$k a^k u[k] \xleftrightarrow{\mathcal{Z}} \frac{a z^{-1}}{(1 - a z^{-1})^2} = \frac{az}{(z-a)^2}$  $(\text{ROC}: |z| > |a|)$
20. 离散 LTI系统的复频域分析：
	1. 系统函数 $H(z)$ (Z 域的系统描述)
		1. 语言描述：一个离散非周期信号表达为 $z^{k}$ （读作： $z$ 的 $k$ 次幂）的形式（复指数信号形式），这个基本信号通过系统的响应，利用线性时不变系统是和 $h[k]$ 卷积的关系==>输入是 $z^{k}$，输出还是 $z^{k}$，再乘上一个系统函数 
			1. $H(z)$$x[k] = z^k \longrightarrow \boxed{h[k]} \longrightarrow y[k]$
			2. $y[k] = z^k * h[k] = \sum_{n} h[n] \cdot z^{k-n}= z^k \left( \sum_{n} h[n] \cdot z^{-n} \right)$
			3. $H(z) = \sum_{n} h[n] \cdot z^{-n}$ $H(z)$ 是双边 z 变换
		2. **定义**: $H(z) = \mathcal{Z}\{h[k]\}$。$H(z) = \frac{Y(z)}{X(z)}$，由零极点图可以分析系统特性。
			1. $H(z)$ 本质上是输出信号的 $z$ 变换 $Y(z)$ 与输入信号的 $z$ 变换 $X(z)$ 的比值。$H(z)$ 是一个有理分式，也就是两个多项式的商 $H(z) = \frac{Y(z)}{X(z)} = \frac{N(z)}{D(z)}$
				1. $N(z)$ 是**分子多项式**（Numerator）
				2. $D(z)$ 是**分母多项式**（Denominator）
		3. **核心关系**: $Y(z) = H(z) \cdot X(z)$ (零状态响应)。
			1. 证明：
				1. 反 $z$ 变换  $x[k] = \frac{1}{2\pi \mathrm{j}} \oint_{C} \underbrace{X(z)}_{\text{这里放Z域函数}} z^{k-1} \mathrm{d}z$
				2. $y_{\text{zs}}[k] = \frac{1}{2\pi \mathrm{j}} \oint_{C} \underbrace{X(z)H(z)}_{\text{这里原本是放 Y(z) 的位置}} z^{k-1} \mathrm{d}z$
			2. 目的：引入系统函数和 $z$ 变换不是用来求响应的，更多是引入系统函数用来表述系统
		4. 描述离散 LTI 系统的差分方程与系统函数之间有什么关系
			1. 对差分方程两边进行双边 z 变换：$\sum_{n=0}^{N} a_n y[k-n] = \sum_{n=0}^{M} b_n x[k-n]$ ==> $\left( \sum_{n=0}^{N} a_n z^{-n} \right) Y(z) = \left( \sum_{n=0}^{M} b_n z^{-n} \right) X(z)$, 
			2. 系统描述离散 LTI 系统的差分方程和 $H (z)$ 之间的关系：$H(z) = \frac{Y(z)}{X(z)} = \frac{\sum_{n=0}^{M} b_n z^{-n}}{\sum_{n=0}^{N} a_n z^{-n}}$
			PS：尽管 $H (z)$ 可以由输入和输出信号的 $z$ 变换之比求解，但是系统函数 $H (z)$ 是描述系统的物理量，和系统的输入输出没有关系，是由系统的结构和内部的参数来决定的
		5. 系统函数多种表现形式：不同表现形式在分析、实现、设计系统的时候有相应的作用
			1. $z^{-1}$ 有理多项式表示： $H(z) = \frac{b_0 + b_1 z^{-1} + \dots + b_M z^{-M}}{a_0 + a_1 z^{-1} + \dots + a_N z^{-N}}$
			2. $z$ 有理多项式表示： $H(z) = z^{(N-M)} \frac{b_0 z^M + b_1 z^{M-1} + \dots + b_M}{a_0 z^N + a_1 z^{N-1} + \dots + a_N}$
			3. 零点、极点和增益常数表示：$H(z) = K z^{(N-M)} \frac{(z-z_1)(z-z_2)\dots(z-z_M)}{(z-p_1)(z-p_2)\dots(z-p_N)}$
			4. 二阶因子表示：$H(z) = \prod_{i=1}^{L} \frac{b_{0i} + b_{1i} z^{-1} + b_{2i} z^{-2}}{a_{0i} + a_{1i} z^{-1} + a_{2i} z^{-2}}$
		6. 一个离散 LTI 系统可以通过 $h[k]$ 和 $H(z)$ 来描述，他们之间的关系：$H(z)= \frac{Y(z)}{X(z)} = \frac{\mathcal{Z}\{h[k]\}}{\mathcal{Z}\{\delta[k]\}}=\mathcal{Z}\{h[k]\}$
			1. $\delta[k] \overset{\mathscr{Z}}{\longleftrightarrow} 1$ 收敛域 (ROC)：整个 $z$ 平面
			2. $h[k] \overset{\mathscr{Z}}{\longleftrightarrow} H(z)$
				1. **$h[k]$** 是系统的**单位脉冲响应**（即输入是 $\delta[k]$ 时的系统输出），是离散 LTI 系统的时域描述
				2. **$H(z)$** 是**系统函数**，是离散 LTI 系统的 $z$ 域描述
	2. **FIR vs IIR 系统**:
		1. **FIR**: Finite Impulse Response 有限脉冲响应
			1. 公式：$H(z) = \frac{b_0 + b_1 z^{-1} + \dots + b_{M-1}z^{-(M-1)} + b_M z^{-M}}{a_0 + a_1 z^{-1} + \dots + a_{N-1}z^{-(N-1)} + a_N z^{-N}} = \mathscr{Z}\{h[k]\}$ 当 $N=0, a_0 \neq 0$ (设 $a_0=1$) 时 $H(z) = b_0 + b_1 z^{-1} + \dots + b_{M-1}z^{-(M-1)} + b_M z^{-M} = \sum_{n=0}^{M} b_n z^{-n}$ ==> $h[k] = b_0 \delta[k] + b_1 \delta[k-1] + \dots + b_M \delta[k-M]$ 长度为 $M+1$
			2. 脉冲响应 $h[k]$ 有限长。$H(z)$ 是 $z^{-1}$ 的多项式 (极点都在 $z=0$)。
			3. **FIR系统总是稳定的**。
				1. 时域证明：$\sum_{k=-\infty}^{\infty} |h[k]| < \infty$：FIR 系统的 $h[k]$ 只有有限个非零值（假设有 $N$ 个）。
				2. 频域/极点证明：$H(z) = \frac{b_0 z^M + b_1 z^{M-1} + \dots + b_M}{z^M}$
					1. **极点在哪里？** 令分母为 0，即 $z^M = 0$。
					2. 这意味着该系统有 $M$ 个极点，全部重合在 **$z=0$ (原点)** 处。
					3. **结论：** 原点 $0$ 显然在单位圆（半径为 1）的**内部**。
		2. **IIR**: Infinite Impulse Response
			1. 公式：$H(z) = \frac{b_0 + b_1 z^{-1} + \dots + b_{M-1}z^{-(M-1)} + b_M z^{-M}}{a_0 + a_1 z^{-1} + \dots + a_{N-1}z^{-(N-1)} + a_N z^{-N}} = \mathscr{Z}\{h[k]\}$ 表达式分母有反馈项（有 $z$ 的幂次存在）即 $N>0, \{a_0 \neq 0, \text{且 } a_k; k=1,2,\dots,N\}$ 中至少有一项非零时， $H(z) = K \frac{(z-z_1)(z-z_2)\cdots(z-z_M)}{(z-p_1)(z-p_2)\cdots(z-p_N)} = \sum_{i=1}^{n} \frac{A_i z}{z-p_i}$ ==> $h[k] = \mathscr{Z}^{-1}\{H(z)\} = \sum_{i=1}^{N} A_i (p_i)^k u[k]$
				1. PS：公式里的累加项数确实是有限的（只有 $N$ 项），但每一项在时间轴 $k$ 上持续的时间是无限的。即无限在于 $u[k]$是无限的
			2. $h[k]$ 无限长。$H(z)$ 有非零极点 (有分母)
		3. “分母中有 $z$ 的幂次存在” $\iff$ “时域中有反馈” $\iff$ “IIR (无限长)” ![[截屏2025-11-25 17.32.52.png]]
		4. 因为分母中有 $z$ 的项，导致差分方程里出现了 $y[k-n]$（以前的输出），就是反馈 (Feedback) —— 昨天的输出，变成了今天的输入。
	3. ~={red}**系统的特性 (Z 域判据) (必考中的必考)**:=~
		1. **因果性**: $\iff$ $H(z)$ 的 ROC 是**某个圆的外部** (包含 $z=\infty$)。
			- 分析：因果 LTI 系统 $h[k]$ 是因果序列，而因果序列是右边序列所以收敛域一定是大于某一个值-->大于 $H(z)$ 所有极点的最大值
			- **因果性判据**：ROC 为 $|z| > R$ (圆外)。
		2. **稳定性**: $\iff$ $H(z)$ 的 ROC **包含单位圆 $|z|=1$**。
			-  时域：LTI 系统稳定充要条件==> $\sum_{k=-\infty}^{\infty} |h[k]| < \infty$
		3. **稳定且因果的 LTI 系统**（**因果稳定系统**）: $\iff$ $H(z)$ 的 **所有极点**    都在 **单位圆内部**。
		【总结】稳定性判别
			1. 对于因果系统既可以通过收敛域判断也可以通过极点位置判断稳定性
			2. 只知道是 LTI 系统，则只能通过收敛域判断
	4. **系统函数 $H(z)$ 与系统频率响应 $H(e^{j\Omega})$ 的关系**:
		1. 公式：$H(e^{j\Omega}) = H(z)|_{z=e^{j\Omega}}$。
		2. 前提: 此关系仅在系统**稳定** (ROC 包含单位圆) 时成立，因为稳定的时候 ROC 才包含单位圆==>有了系统函数之后可以更加全面的描述这个系统，在设计系统的时候最后给出的结果也是以 $H (z)$ 的形式给出的，实现系统的时候是针对 $H (z)$ 进行实现的
		3. 应用 (几何法): $z$ 平面上的零极点决定了频响曲线。
			1. 零点在单位圆上 $\implies$ 对应频率 $\Omega$ 处 $H(e^{j\Omega})=0$。
			2. 极点靠近单位圆 $\implies$ 对应频率 $\Omega$ 处 $H(e^{j\Omega})$ 产生峰值。
21. 全通系统与最小相位系统 
	1. **全通系统 $A_m(z)$**:
		1. 定义：表示 m 阶**实系数**全通滤波器的系数函数。全通滤波器分子分母的阶数是一样高的$A_m(z) = \frac{d_m + d_{m-1}z^{-1} + \cdots + d_1 z^{-(m-1)} + z^{-m}}{1 + d_1 z^{-1} + \cdots + d_{m-1} z^{-(m-1)} + d_m z^{-m}}$
		2. 最重要的‼️特点: 幅频响应恒为常数 1 (公式： $|A_m(e^{j\Omega})|=1$)，只改变相位。
		3. 从时域、频域、复频域分析一阶复系数全通数字滤波器：
			0. 实际应用当中用的是实系数的数字滤波器，为什么讨论复系数的呢？
				A：$m$ 阶**实系数**全通滤波器可以分解为 $m$ 个一阶的复系数全通数字滤波器（也不是必须要分解成都是一阶复系数全通数字滤波器，可以混着来，有实有复$A_m(z) = \prod_{k=1}^{m} \frac{z^{-1} - p_k^*}{1 - p_k z^{-1}}$）
			1. 公式：$A_1(z) = \frac{z^{-1} - d^*}{1 - d z^{-1}} \qquad |d| < 1 \qquad d = r e^{j\theta}$（复系数）
			2. 幅度响应：$A_1(e^{j\Omega}) = \frac{e^{-j\Omega} - d^*}{1 - d e^{-j\Omega}} = e^{-j\Omega} \frac{1 - d^* e^{j\Omega}}{1 - d e^{-j\Omega}} = e^{-j\Omega} \frac{1 - r e^{-j\theta} e^{j\Omega}}{1 - r e^{j\theta} e^{-j\Omega}}$
				1. 【背✅】幅度响应是常数 $\left| A_1(e^{j\Omega}) \right| = 1$
			3. **~={red}【背】级联时幅度相乘，相位相加   m个级联起来幅度响应还是1=~**
				1. 证明：$\begin{aligned} H(e^{j\omega}) &= \left( |H_1| e^{j\varphi_1} \right) \cdot \left( |H_2| e^{j\varphi_2} \right) \\ &= \underbrace{ \left( |H_1| \cdot |H_2| \right) }_{\text{幅度相乘}} \cdot \underbrace{ e^{j(\varphi_1 + \varphi_2)} }_{\text{相位叠加}} \end{aligned}$
				2. 总幅度$|H(e^{j\omega})| = |H_1(e^{j\omega})| \cdot |H_2(e^{j\omega})|$ （幅度相乘）
				3. 总相位$\varphi(\omega) = \varphi_1(\omega) + \varphi_2(\omega)$ （相位相加）
			4. 相位响应：$A_1(e^{j\Omega}) = e^{-j\Omega} \frac{1 - r e^{-j\theta} e^{j\Omega}}{1 - r e^{j\theta} e^{-j\Omega}} = e^{-j\Omega} \frac{1 - r \cos(\Omega - \theta) - \mathrm{j} \cdot r \sin(\Omega - \theta)}{1 - r \cos(\Omega - \theta) + \mathrm{j} \cdot r \sin(\Omega - \theta)}$ ==> $\varphi(\Omega) = -\Omega - 2 \arctan \frac{r \sin(\Omega - \theta)}{1 - r \cos(\Omega - \theta)}$ 
				1. 🧚‍♀️相位:arctan(分子虚部比实部)-arctan(分母虚部比实部)   这里合并成2倍了（2 倍是因为分子和分母之间存在一种特殊的“共轭关系”）  ，加上最前面 $e^{-j\Omega}$ 带来的线性相位 $-\Omega$
				2. $\frac{\mathrm{d}\varphi(\Omega)}{\mathrm{d}\Omega} = - \frac{1 - r^2}{(1 - r \cos(\Omega - \theta))^2 + r^2 \sin^2(\Omega - \theta)} < 0$ ==>【背✅】一阶全通数字滤波器的相位响应为单调递减
				3. 全通数字滤波器 - 相位响应图 ![[截屏2025-11-25 22.16.45.png|300]]
			5. 一阶全通滤波器的极点和零点:
				1. **极点为:** $p_1 = d = r e^{j\theta}$
					1. 极点求法：让分母=0，求根
				2. **零点为:** $z_1 = 1/d^* = (1/r) e^{j\theta}$
					1. 零点：系统函数 $H(z)$ 的数值变成**零**的那些 $z$ 的取值
					2. 绝对拦截（当零点在单位圆上时）
						1. 系统的频率响应 $H(e^{j\omega})$ 是系统函数 $H(z)$ 在 $z$ 平面单位圆（$|z|=1$）上的取值
						2. 系统有一个零点 $z_0$ 刚好位于单位圆上，即 $z_0 = e^{j\omega_0}$
						3. 输入一个频率正好为 $\omega_0$ 的复正弦信号 $x[n] = e^{j\omega_0 n}$ 时，系统的响应（增益）就是 $H(e^{j\omega_0})$$z_0$ 是零点，所以：$H(e^{j\omega_0}) = 0$
						4. $|y[n]| = |H(e^{j\omega_0})| \cdot |x[n]| = 0 \cdot 1 = 0$
					3. 频率响应的“波谷”或“深坑”（当零点不在单位圆上时）
						1. 零点 $z_k$ 在单位圆内部或外部，虽然不能让频率响应完全为0，会起到“拉低”的作用。
						2. 系统幅频响应的大小与“单位圆上的频点到零点的距离”成正比：$|H(e^{j\omega})| \propto \prod_{k} |e^{j\omega} - z_k|$
							1. $e^{j\omega}$ 是单位圆上代表当前频率的点
							2. $z_k$ 是零点的位置
							3. $|e^{j\omega} - z_k|$ 是它们之间的距离（向量长度）
					4. 零点通过“拉低”幅频响应，抑制或消除特定频率的信号。零点越靠近单位圆，系统对该频率附近的信号衰减就越厉害，该频率的“波谷”就越深。
				3. 结构：【背】零极点一定是成对出现的，且是**共轭倒数**分布 ($p_k = (1/z_k)^*$)
					1. 理解：即“零极点关于单位圆对称”，即极点和零点像是一对“双胞胎”，它们站在同一个角度的方向上，一个在圆内，一个在圆外，且它们到圆心的距离互为倒数。
				4. 🧚‍♀️极点使系统函数为无穷的点，也就是分母为0的点，零点就是分子为零，可以用极点来界定收敛域
		4. $m$ 阶实系数全通滤波器
			1. 公式： $A_m(z) = \frac{d_m + d_{m-1}z^{-1} + \dots + d_1 z^{-(m-1)} + z^{-m}}{1 + d_1 z^{-1} + \dots + d_{m-1} z^{-(m-1)} + d_m z^{-m}} = \frac{z^{-m} D_m(z^{-1})}{D_m(z)}$ PS：$D_m(z)$ 就是全通滤波器系统函数的“分母多项式”
			2. 幅度响应：$A_m(z)A_m(z^{-1}) = \frac{z^{-m} D_m(z^{-1})}{D_m(z)} \frac{z^{m} D_m(z)}{D_m(z^{-1})} = 1$，利用实序列 DTFT 的对称性$\left| A_m(e^{j\Omega}) \right|^2 = A_m(z)A_m(z^{-1}) \Big|_{z=e^{j\Omega}} = 1$
			3. 零点和极点的关系：$m$ 阶实系数全通滤波器的零极点是成对出现的，且每一个零极点都是呈现共轭倒数关系
				1. 公式：$A_m(z) = K \frac{(z-z_1)(z-z_2)\cdots(z-z_m)}{(z-p_1)(z-p_2)\cdots(z-p_m)}= K \prod_{i=1}^{m} \underbrace{\frac{(z-z_i)}{(z-p_i)}}_{\text{1st Order All-Pass一阶全通}}$
				2. 倒数关系：$p_i = \left( \frac{1}{z_i} \right)^*$
				3. 总结：$m$ 阶实系数全通系统可以分解为 $m$ 个一阶全通系统的级联(串联) $$A_m(z) = K \prod_{i=1}^{m} A_{1,i}(z)$$$A_{1,i}(z)$ 是由系统的极点 $p_i$ 和对应的零点 $1/p_i$ 构成的一阶全通子系统。（==>可以通过一阶全通滤波器的特点分析 $m$ 阶实系数全通系统的相位响应）
			4. 相位响应：
				1. 公式： $\varphi(\Omega) = \varphi_1(\Omega) + \varphi_2(\Omega) + \dots + \varphi_m(\Omega)$
				2. 特殊点：$A_m(e^{j0}) = 1$ 则$\varphi_1(\Omega) < 0, \quad \text{当 } \Omega \in (0, \pi)$（“1”这个数在复平面上不仅代表大小（模），还隐含了方向（相位）。）因为在直流处，输入和输出没有相位差。
				3. $\varphi_1(\Omega) < 0, \varphi_2(\Omega) < 0, \dots, \varphi_m(\Omega) < 0$. 相位函数 $\varphi(\Omega)$ 在 $0$ 到 $\pi$ 这个范围始终小于 0==>全通滤波器一个非常重要的物理特性：**它是一个相位滞后系统**。**相位 < 0**：在信号处理中，负相位通常意味着**时间延迟**（滞后）。【背】**因果稳定的全通系统是一个相位滞后系统**。
				4. 二阶实系数全通滤波器相位响应示意图：单调减、零点相位等于 0 ![[截屏2025-11-25 23.04.48.png|300]] 
		5. **应用:** **~={pink}相位均衡器=~**
			1. 问题： 一个信号通过一个非线性相位系统（非线性相位系统完全可以是 LTI 系统，且绝大多数的 LTI 系统（特别是 IIR 滤波器）都是非线性相位的），也就是说这个系统对信号会产生相位失真；如图在通带范围内（输入信号的频率范围内）：例如频率分量100 Hz 和 300 Hz 两个频率信号叠加起来，在此范围内幅度是常数，在通带里相位响应不是线性的，是弧度的 ![[Pasted image 20251126153452.png]]
				1. 0dB 是 1 倍增益。即信号经过这个系统，其幅度既没有被放大，也没有被衰减
				2. **通带（Passband）**：指的是滤波器允许信号通过的那个频率范围。
				3. 相位失真：即便一个系统的幅度响应是完美的（0dB，不放大也不衰减），信号出来后依然可能变得“面目全非”。  由**相位响应的非线性**导致的信号波形畸变，就叫作相位失真
			2. **相位均衡 (校正非线性相位)**。极点使系统函数为无穷的点，也就是分母为0的点，零点就是分子为零，可以用极点来界定收敛域 ![[截屏2025-11-26 15.58.50.png|300]]
				1. 全通系统与原系统级联，整个系统可以保留原幅度响应（全通系统的幅度响应等于 1，级联之后不会改变原系统的幅频响应），但是全通系统的相位响应是存在的，与原系统相位响应是相加的关系
				2. 两个系统的相位响应相加之后使得在需要的频率范围内相位近似为线性的即可，将完成这样功能的全通滤波器称之为相位均衡器==>调节相位，调节和前面相级联的系统的相位的，不改变原系统的幅度响应 (只会延时，基本不会失真) ![[截屏2025-11-26 16.02.21.png]] ![[截屏2025-11-26 16.02.58.png]]
				【总结】幅值不变相位相加，接近直线
		6. **全通滤波器 (All-pass Filter)**
			1. 定义：幅度响应恒为常数 $|H(e^{j\Omega})| = 1$ 的系统
			2. 零极点特征 [考点]：零点和极点关于单位圆镜像对称。若极点为 $p_k$，则必有零点 $z_k = 1/p_k^*$
	2. **最小相位系统 $H_{min}(z)$**:
		1. 定义: 
			1. **因果稳定**，且 **所有零点** 也都在 **单位圆内部**。
			2. 系统函数**零点和极点都在 $z$ 平面单位圆内**的**因果系统** 称为**最小相位系统**。记为 $H_{\min}(z)$。
			3. **最小相位系统**（Minimum-Phase System）是所有具有相同幅度特性的因果稳定系统中，**相位延迟最小**（能量延迟最小）的那个“完美模型”。
		2. 特性: 
			1. 在所有具有相同幅频响应的系统中，它的**群延迟最小**。
			2. 在具有相同幅频特性的同阶系统中，最小相位系统具有最大的相位，最小的延时。
				【理解】 因果系统中，相位通常是负值，表示滞后，所谓“最大”，是因为它减得最少，所以负数反而排在最前面（最大）。
			3. **[考点] 逆系统稳定性**：只有最小相位系统的逆系统（$1/H(z)$）才是因果稳定的（因为原系统的零点变为逆系统的极点，仍在圆内）
		3. 符号：
			1. $H_1(z)$：通用的“一阶系统函数” 
				 - **$H$ (Transfer Function):** 是**最通用**的符号，代表任意线性时不变系统的系统函数。
				1. 可以是低通、高通、带通，或者全通。
				2. 例如（低通）：$H_1(z) = \frac{1}{1 - 0.5z^{-1}}$
				3. 例如（高通）：$H_1(z) = \frac{1 - z^{-1}}{1 + 0.5z^{-1}}$
			2. $A_1(z)$：专用的“一阶全通滤波器” All-pass
				1. 它的幅度响应必须恒为 1 ($|H(e^{j\Omega})|=1$)。
				2. 分子和分母必须满足“共轭倒数”对称关系$A_1(z) = \frac{z^{-1} - d^*}{1 - d z^{-1}}$
		4. 计算：利用全通系统与最小相位系统的关系 (将非最小相位系统转化为最小相位系统)
			1. $A_{ap}(z) = \frac{a + z^{-1}}{1 + a z^{-1}}$ 全通滤波器的核心特性是幅度响应在所有频率上恒等于 1 ($\frac{z^{-1} - d^*}{1 - d z^{-1}} \quad \xrightarrow{d = -a, \, a \in \mathbb{R}} \quad \frac{a + z^{-1}}{1 + a z^{-1}}$)![[截屏2025-11-26 13.54.00.png|300]]
			2. **系统分解 (考点)**: 任何因果稳定系统 $H(z)$ 都可以分解为最小相位系统和全通系统的级联。‼️实系数因果系统可以表示为一个最小相位系统和全通系统的级联：如果最小相位系统和全通系统的特性，包括幅频响应和相位响应都清楚==>分析任意因果系统的频率特性都简单；如果一个系统 $H(z)$ 不是最小相位的，说明它的某些零点跑到了单位圆外，导致了额外的相位滞后。可以把这个“额外的相位滞后”提取出来，封装成一个全通系统。分解公式是：$\underbrace{H(z)}_{\text{原系统}} = \underbrace{H_{min}(z)}_{\text{幅度相同，相位最小}} \cdot \underbrace{H_{ap}(z)}_{\text{幅度为1，提供额外相位}}$
				![[截屏2025-11-26 14.00.53.png|300]]
				1. 公式：$H(z) = H_{min}(z) \cdot A_m(z)$
					当原来的系统和最小相位系统具有相同幅度响应的相位和延时之间的关系：$H(z) = H_{\min}(z) A(z) \implies |H(e^{j\Omega})| = |H_{\min}(e^{j\Omega})| |A(e^{j\Omega})|$
					1. 相位：$\varphi(e^{j\Omega}) = \varphi_{\min}(e^{j\Omega}) + \varphi_A(e^{j\Omega}) \implies \varphi_{\min}(e^{j\Omega}) = \varphi(e^{j\Omega}) - \varphi_A(e^{j\Omega})$
					2. 延迟：$\tau = -\frac{d\varphi(e^{j\Omega})}{d\Omega} \quad \implies \quad \tau = \tau_{\min} + \tau_A \quad \implies \quad \tau_{\min} = \tau - \tau_A$ 又因为 $\tau_A>0$, 负导数 PS：这里延迟前面的符号是因为延迟已经表示了-符号所以说延迟多少不会说延迟负多少
					【结论】“最小相位系统”拥有“最大”的相位，最小的延时；最小代表最小的相位滞后，是绝对值最小（其实最好叫最小延迟系统，因为相位反而是最大的，当初就是这么说规定下来了）
				2. 证明：$H(z) = H_1(z) (z - a)$（$z=a$ 是 $H (z)$ 跑到单位圆外面的零点）==> $H(z) = H_1(z) (z - a) \cdot \frac{z - 1/a^*}{z - 1/a^*}$ ==> $H(z) = \underbrace{\left[ H_1(z) (z - 1/a^*) \right]}_{H_{\min}(z)} \cdot \underbrace{\left[ \frac{z - a}{z - 1/a^*} \right]}_{A_1(z)}$
				3. $A_{1}(z)$ 吸收了原系统中“不听话”的零点带来的**多余相位延迟**
				4. 方法: 将 $H(z)$ 在单位圆外的零点 $z_k$ (非最小相位零点) "翻转"到圆内的 $1/z_k^*$ 处构造 $H_{min}(z)$，并乘以一个全通 $A_m(z)$ 来保持幅度不变。🧚‍♀️就看哪个零点或者极点是在单位圆外的，就用这个点的共轭来配；全通系统的零点和极点是倒数，所以配在大于1的数的倒数就好啦
		5. 应用: ~={pink}**幅度均衡器**=~
			1. 最小相位系统可以暴力构建逆系统 $H(z) = \frac{b(z)}{a(z)} \quad \implies \quad H^{-1}(z) = \frac{1}{H(z)} = \frac{a(z)}{b(z)}$（实际应用当中的系统要求是稳定的系统，由于最小相位系统零点也全部在单位圆内，所以其系统零点变成极点，极点变成零点之后系统依然是稳定的，因为零点和极点还是全部在单位圆内）
				1. 公式原理： $H(z)H^{-1}(z) = 1$
					-  均衡器： $H^{-1}(z)$ 可以抵消  $H(z)$ 的影响，称为其均衡器
				2. 应用：回声消除系统->若一个系统 $H(z)$ 造成了信号失真（比如产生了回声），想消除这个影响==>两个系统相级联之后总的系统函数等于 1，意味着信号原封不动地出来了，回声被完美消除。
					1. 回声系统公式：是一个典型的线性时不变（LTI）系统，且因为它只依赖于当前的和过去的**输入**，所以是一个 **FIR（有限冲激响应）** 系统。
						1. 差分方程：$y[n] = x[n] + \alpha x[n-D]$，$|\alpha| < 1$
						2. 两边同时进行 $z$ 变换，利用 $z$ 变换的**线性性质**和**时移性质**（$x[n-D] \leftrightarrow z^{-D}X(z)$）：$Y(z) = X(z) + \alpha z^{-D} X(z)$ 则 $H(z) = 1 + \alpha z^{-D}$
						3. 看清该系统对不同频率信号的影响 $H(e^{j\Omega}) = H(z) \big|_{z=e^{j\Omega}} = 1 + \alpha e^{-j\Omega D}$
							1. $\begin{aligned} |H(e^{j\Omega})| &= |1 + \alpha (\cos(\Omega D) - j\sin(\Omega D))| \\ &= \sqrt{(1 + \alpha \cos(\Omega D))^2 + (-\alpha \sin(\Omega D))^2} \\ &= \sqrt{1 + 2\alpha \cos(\Omega D) + \alpha^2 \cos^2(\Omega D) + \alpha^2 \sin^2(\Omega D)} \\ &= \sqrt{1 + \alpha^2 + 2\alpha \cos(\Omega D)} \end{aligned}$
							2. **波峰**（最大值）：$1+\alpha$
							3. **波谷**（最小值）：$1-\alpha$
					2. 回声产生系统 $|\alpha| < 1$ 故零点全在单位圆之内，因为回声的能量通常比原声小；**自然回声（$|\alpha| < 1$）是最小相位系统**
					3. 相乘等于 1，即整个系统函数等于 1，即输出=输入 $y[k] = x[k]$
			2. $H(z)$ 是**非最小相位系统**，有零点在单位圆外，无法直接构成逆系统，又因为任意一个因果稳定系统总可以表示为一个最小相位系统和全通系统的级联 $$H(z) \xrightarrow{\text{求出与 } H(z) \text{ 具有相同幅度响应的 } H_{\min}(z)} H_{\min}(z) \xrightarrow{\text{获得稳定的逆系统}} H^{-1}_{\min}(z)$$$H_{equal}(z) =\underbrace{H^{-1}_{\min}(z)}_{\text{均衡器}} \cdot \underbrace{H(z)}_{\text{原系统}}= H^{-1}_{\min}(z) \cdot [H_{\min}(z) \cdot A_{ap}(z)] = \underbrace{[H^{-1}_{\min}(z) \cdot H_{\min}(z)]}_{=1} \cdot A_{ap}(z) = A_{ap}(z)$ 最后等于一个全通系统，幅度响应为 1 ，对幅度进行了均衡==>  $H^{-1}_{\min}(z)$ 可以称为均衡器

22. 信号的抽样 (ADC 过程)
	1.  信号的时域抽样 
		1. 等间隔抽样公式：$x[k] = x_c(t)|_{t=kT}$。 $T$ 是抽样周期，$f_{sam} = 1/T$ 是抽样频率==>对时域信号进行的离散化处理
		2. 生活例子：
			1. 数字化音乐
			2. 数码相机拍到的数字图像是一个个空间离散的样点值叠加起来的。相机多少万像素即一个画面中过能采集到多少个空间连续图像的样本点
		3. 戴奎斯特：对于一个带宽有限的连续时间信号，如果按照一定的时间抽样间隔进行抽样，离散样点值能代表连续信号的全部信息，样点值能恢复原来的连续时间信号
		4. 利用数字技术处理模拟信号流程图：$$\text{输入 } x(t) \longrightarrow \boxed{\text{抽样}} \xrightarrow{x[k]} \boxed{\text{离散系统}} \xrightarrow{y[k]} \boxed{\text{重建}} \longrightarrow \text{输出 } y(t)$$
		5. 离散信号与系统的主要优点
			1. 信号稳定性好： 数据用二进制表示，受外界影响小。
			2. 信号可靠性高： 存储无损耗，传输抗干扰。
			3. 信号处理简便：信号压缩，信号编码，信号加密等。
			4. 系统的精度高： 可通过增加字长提高系统的精度。模拟系统若要提高精度一般要通过电感电容提高，成本高难度大
			5. 系统灵活性强：改变系统的系数 ($H(z)$ 的系数)使系统完成不同功能。
			6. **易于存储**：数字存储容量大、稳定性好（晶体管基础和集成电路技术的发展）
				1. 模拟：录像带、磁带、相机胶卷
				2. 数字存储：DVD、SD 卡、硬盘
			7. **易于传输**：数字传输 0、1 编码，收到干扰恢复更容易
			8. **易于处理**：指纹识别和人脸识别数字化之后才能进行大量的计算。大量人脸扫描进系统进行特征提取之后从中将需要的人找出来，指纹对比和基因对比，数字化之后用数字系统处理
		6. 周期冲击串的频谱密度函数由**一系列等间隔的狄拉克 $\delta$ 函数（冲击）** 组成，间隔为基频 $\omega_0$
		7. $\delta$ 函数卷积特性：卷积的单位元，任何函数和 $\delta$ 函数卷一下它自己；如果 $\delta$ 函数移位了，原来的函数也跟着移位；任何函数与冲激函数卷积，就是把该函数“搬移”到冲激函数所在的位置；冲激函数具有**筛选性质**，$\delta(t-kT)$ 筛选出 $t=kT$
	2. 时域抽样理论分析：
		1. 抽样频域关系：
			1. 时域出发：$x_{\text{sam}}(t) = x(t)\delta_T(t) = \sum_{k=-\infty}^{+\infty} x(kT)\delta(t-kT)$
				1. 连续信号 $x(t)$ 乘以周期冲激串 $\delta_T(t)$，$\delta_T(t)$ 的频谱是间隔为 $\omega_{\text{sam}} = \frac{2\pi}{T}$ 的冲激串。
				2. 利用冲激函数的性质，将其写成一系列加权冲激函数的和，权重为采样时刻的值 $x(kT)$ 
			2. 频域卷积法：时域相乘对应频域卷积 $X_{\text{sam}}(j\omega) = \frac{1}{2\pi} X(j\omega) * \left( \omega_{\text{sam}} \sum_{n=-\infty}^{+\infty} \delta(\omega - n\omega_{\text{sam}}) \right) = \frac{1}{T} \sum_{n=-\infty}^{+\infty} X\left[j(\omega - n\omega_{\text{sam}})\right]$ 频谱的周期化：抽样后的频谱是原模拟信号频谱 $X(j\omega)$ 的周期延拓，幅度缩放了 $1/T$。
			3. 时域表达式傅里叶变换积分：$\int_{-\infty}^{+\infty} \left[ \sum_{k=-\infty}^{+\infty} x(kT)\delta(t-kT) \right] e^{-j\omega t} dt$
				1. 利用 $\delta(t-kT)$ 筛选出 $t=kT$，**~={red}积分变为求和=~**：$= \sum_{k=-\infty}^{+\infty} x(kT) e^{-j k \omega T}$
				2. 引入离散序列 $x[k] = x(kT)$ 和数字频率 $\Omega = \omega T$，$= \sum_{k=-\infty}^{+\infty} x[k] e^{-j k \Omega} = X(e^{j\Omega})$
				3. **离散时间傅里叶变换（DTFT）** 的定义式。表明抽样信号的频谱在数学形式上就是序列 $x[k]$ 的 DTFT
				4. IDTFT: $x[k] = \frac{1}{2\pi} \int_{<2\pi>} X(e^{j\Omega}) e^{j\Omega k} d\Omega$； DTFT: $X(e^{j\Omega}) = \sum_{k=-\infty}^{\infty} x[k] e^{-j\Omega k}= \sum_{k=-\infty}^{\infty} x[k] \cdot \left(e^{j\Omega k}\right)^*$ 
				5. 1 和 2 相等：~={red} $X(e^{j\Omega}) \bigg|_{\Omega=\omega T} = \frac{1}{T} \sum_{n=-\infty}^{+\infty} X\left( j(\omega - n\omega_{\text{sam}}) \right)$ =~ $\Omega = \omega T$
					1. $T$ 为抽样间隔
					2. $\omega_{\text{sam}} = \frac{2\pi}{T}$ 为抽样角频率
					3. $\Omega = \omega T$ 根据不同需要可以根据模拟角频率或者数字角频率来表示横坐标
					4. 【背】离散序列的频谱 $X(e^{j\Omega})$，本质上就是原模拟信号频谱 $X(j\omega)$ 经过**周期延拓**（周期为采样频率 $\omega_{\text{sam}}$）并**幅度缩放**（乘以 $1/T$）后的结果；频域 $x[k]$ 频谱等于原来 $x(t)$ 的频谱的周期化，周期为抽样角频率
			【总结】时域的离散化对应频域的周期化
		2. **时域抽样定理 (Nyquist)**:
			1. 一个最高频率为 $f_m$ (带限) 的连续信号 $x_c(t)$，
			2. 若 $f_{sam} \ge 2f_m$ (Nyquist 速率)，则 $x_c(t)$ 可以被 $x[k]$ 完全重建，不失真。
			3. **混叠 (Aliasing)**:
			      - 若 $f_{sam} < 2f_m$，频域上周期延展的 $X_c(j\omega)$ 发生重叠，高频分量混入低频，导致失真。
			      - **抗混叠滤波器 (考点)**: 为防止混叠，在抽样前 (ADC前)，必须先让 $x_c(t)$ 通过一个**模拟低通滤波器**，滤除 $f_{sam}/2$ 以上的频率。
		3. 带限信号抽样
			1. 带限信号特点：信号 $x(t)$ 的频谱不是无限宽的，它有一个**最高角频率** $\omega_m$（或者对应的最高频率 $f_m$，其中 $\omega_m = 2\pi f_m$）。
			2. 保证抽样后的离散序列能够**唯一表示**（即无失真恢复）原始连续信号，抽样参数需满足不等式：【必要条件】🧚‍♀️只要经抽样后的频谱不发生混叠且能通过滤波器滤过就可以复原
				1. 抽样间隔 $T$ 的要求（越小越好）：$T \le \frac{\pi}{\omega_m} = \frac{1}{2f_m}$
				2. 抽样频率 $f_{\text{sam}}$ 的要求（越大越好）：$f_{\text{sam}} \ge 2f_m \quad (\text{或 } \omega_{\text{sam}} \ge 2\omega_m)$
		4. 窄带高频信号抽样🧚‍♀️调制一下就行了；带通采样或先下变频再采样
			1. 应用：通信中常用，调制信号
			2. 完全重叠：周期化过程中发生混叠但不一定会失真，只要完全重叠就不会失真，完全重叠不会破坏原来信号频谱的结构

23. 信号的重建 (DAC 过程)：[信号的重建]https://gemini.google.com/gem/212b424da3cc/90dc638ffff2e4c2?hl=zh
	1. 定义：抽样的目的是将连续信号变成离散信号之后通过离散的信号进行处理，离散系统处理之后的信号依然是离散的信号，处理之后的离散信号在很多情况下需要将其变成连续时间信号。信号时域抽样的逆过程
	2. 工程实现：D/A 芯片，将数字转换成模拟（与 A/D 转换芯片是相反功能）；实际中用 **零阶保持器 (ZOH)** (DAC芯片) + **模拟低通滤波** (平滑) 来近似。
		PS: 零阶保持器（Zero-Order Hold, ZOH）的单位脉冲响应（Impulse Response）在时域上就是一个门函数（Gate Function），或者更准确地说是矩形脉冲（Rectangular Pulse）。
	3. 理论实现：$x[k] \longrightarrow \boxed{\begin{matrix} \text{序列转换} \\ \text{为冲激串} \end{matrix}} \xrightarrow{\quad x_{\text{rec}}(t) \quad} \boxed{\begin{matrix} \text{平滑滤波} \\ H(j\omega) \end{matrix}} \longrightarrow x_{\text{r}}(t)$
		1. 理论过程公式：
			1. **序列转冲激串**: $x_{rec}(t) = \sum_{k=-\infty}^{\infty} x[k]\delta(t-kT)$
			2. **理想低通滤波**: $x_r(t) = x_{rec}(t) * h(t)$
			3. **重建滤波器 $h(t)$**:
				1. **理想低通滤波器**: 增益为 $T$，截止频率为 $\omega_c = \omega_{sam}/2 = \pi/T$
				2. 时域: $h(t) = Sa(\frac{\pi}{T}t)$ (Sinc 函数)
				3. 重建公式 (Sinc 插值): $x_r(t) = \sum_{k=-\infty}^{\infty} x[k] Sa(\frac{\pi}{T}(t-kT))$
		2. 序列转换为冲激串：频谱一样，只有横坐标有变换==>数字角频率-->模拟角频率 (离散信号频谱 -->连续信号频谱) 中间信号 $x_{\text{rec}}(t) = \sum_{k=-\infty}^{\infty} x[k]\delta(t-kT)$
		3. 平滑滤波：是去周期化的。低通滤波器频域乘积
		4. 时域分析：
			1. 理想低通滤波器冲激响应
				1. 公式：$h(t) = \frac{\sin(\omega_c t)}{\pi t} = \frac{\omega_c}{\pi} \text{sinc}\left(\frac{\omega_c t}{\pi}\right)=\frac{\omega_c}{\pi} \text{Sa}\left(\frac{\omega_c t}{\pi}\right)$ 
					1. 截止频率：$\omega_c$
					2. 幅度：$1$
				2. 内插函数:
					1. 公式：$h(t) = \frac{\omega_{\text{sam}}T}{2\pi}\text{Sa}\left(\frac{\omega_{\text{sam}}}{2}t\right) = \text{Sa}\left(\frac{\pi}{T}t\right)$
					2. 截止频率：采样角频率-->每秒钟转了多少角度（弧度）$\frac{\omega_{\text{sam}}}{2}$
					3. 幅度：$T$
			2. 抽样间隔和重建间隔的关系：
				1. 没有进行处理的理论情况下：抽样间隔=重建间隔。否则不可以恢复原来的连续信号
				2. 重建=2 抽样，得到的信号是原信号的展宽
				3. 重建=抽样/2，得到的信号是原信号的压缩
				4. 实际应用：同时处理多路抽样信号的时候，抽样频率往往不一样，如果要使用同样的一个重建系统重建，需要将所有多路信号的抽样频率都要转换到和重建间隔相一致的水平上（多速率信号处理：调整抽样频率到重建系统输入端信号的频率等于重建间隔的倒数，否则会失真）
24. 连续信号的离散化处理 **(经典考题框架)**：
	1. Why：实际应用中都是连续时间信号，利用传感器将非电的信号转换成电的信号时，传感器输出信号都是连续信号，通过抽样变成离散信号，再通过离散系统处理分析，又离散系统的输出依然是离散信号，需要将离散信号通过重建再变成连续信号，用于观察或者驱动
	2. **系统框图**: $x_c(t) \xrightarrow{\text{抽样 } T} x[k] \xrightarrow{H(z)} y[k] \xrightarrow{\text{重建 } T} y_c(t)$
		1. **输入 $x_c(t)$**: 频谱为 $X_c(j\omega)$, 横坐标是 $\omega$，有信号最高角频率 $\omega_m$
		2. **抽样后 $x[k]$**: $x[k]$ 是 $x_c (t)$ 的等间隔抽样，抽样间隔（周期）为 $T$，其频谱是原来 $x_c (t)$ 频谱的周期化，幅度变成原来的 $\frac{1}{T}$，周期是 $\omega_{sam}$ 。**离散时间傅里叶变换 (DTFT)** 与 **连续时间傅里叶变换 (CTFT)** 之间的关系
			1. $X (e^{j\Omega}) = \frac{1}{T} \sum_{n=-\infty}^{+\infty} X\left[j (\omega - n\omega_{\text{sam}})\right] \quad (\Omega = \omega T)$ 即 $x[k]$ 频谱的横坐标数字角频率是模拟角频率的抽样间隔 $T$ 倍
			2.  $X(e^{j\Omega}) = \frac{1}{T}\sum X_c(j(\frac{\Omega}{T} - n\frac{2\pi}{T}))$ (注意 $\Omega=\omega T$, 观察是否混叠)。
				【关系】 $X(e^{j\Omega}) = \frac{1}{T} \sum_{n=-\infty}^{+\infty} X\left[j\left( \underbrace{\frac{\Omega}{T}}_{\omega} - n \underbrace{\frac{2\pi}{T}}_{\omega_{\text{sam}}} \right)\right]$
		3. **数字滤波 $y[k]$**: 频谱 $Y(e^{j\Omega}) = X(e^{j\Omega}) \cdot H(e^{j\Omega})$
		4. **重建前 $y_{rec}(t)$**: $Y_{rec}(j\omega) = Y(e^{j\omega T})$。 (将 $Y(e^{j\Omega})$ 的 $\Omega$ 轴替换为 $\omega T$)
		5. **重建后 $y_c(t)$**: $Y_c(j\omega) = Y_{rec}(j\omega) \cdot H_{rec}(j\omega)$。 ( $H_{rec}$ 是截止频率为 $\pi/T$、增益为 $T$ 的理想低通)
			PS: 1. 重建前后频谱一样，只是横坐标从数字角频率变成了模拟角频率
				2. 重建有两个步骤：1️⃣将离散序列变成冲激串信号再 2️⃣ 通过低通滤波器重建
	3. **等效模拟滤波器 (考点)**: 整个 "ADC-DF-DAC" 系统的等效模拟频率响应 $H_{eff}(j\omega)$，就是 $Y_c(j\omega) / X_c(j\omega)$。
	     - 在不混叠的前提下 ($|\omega| < \pi/T$)， $H_{eff}(j\omega) = H(e^{j\omega T})$。
	     - **考点**: 数字滤波器的数字截频 $\Omega_c$ 对应模拟截频 $\omega_c = \Omega_c / T$  
	     - 输入输出都是连续的，但核心内容是通过离散系统实现的 ![[截屏2025-11-27 10.54.05.png|300]]
		1. DF 数字滤波器 Digital Filter：已知 AF 截频 $\omega_c$，DF 截频是$\Omega_c= \omega_c / T$ 
		2. AF 模拟滤波器 Analog Filter：已知 DF截频 $\Omega_c$ 对应等效 AF模拟截频 $\omega_c = \Omega_c / T$  
【总结】抽样：连续-->离散；重建：离散-->连续
	 理论模型：![[截屏 2025-11-27 11.29.46.jpeg]]
	 语言描述：
		 抽样： $x (t)$ 通过周期冲激串乘积之后变成信号 $x_{sam}(t)$, 是原来信号频谱 $x (t)$ 的周期化，他们均为连续信号，还不构成抽样，必须将连续时间信号通过冲激串转化为序列变成离散信号，才是 $x (t)$ 到 $x[k]$ 的过程
		 重建：$x[k]$ 通过将序列转化为冲激串得到信号 $x_{rec}(t)$，再将重建信号通过低通滤波得到原信号 $x_{r}(t)$，输入是离散，输出是连续

### 第五部分：习题课中的重点变换 (必考)

- **$H_1(z) = H(-z)$**
    
    - $z \to -z$
        
    - $z = e^{j\Omega} \to -e^{j\Omega} = e^{j\pi}e^{j\Omega} = e^{j(\Omega-\pi)}$
        
    - **效果**: 频谱**搬移 $\pi$**。理想低通 ($0 \sim \pi/3$) $\implies$ 理想高通 ($\pi - \pi/3 \sim \pi$，即 $2\pi/3 \sim \pi$)。
        
- **$H_2(z) = H(z^2)$**
    
    - $z \to z^2$
        
    - $z = e^{j\Omega} \to e^{j2\Omega}$
        
    - **效果**: 频谱在 $0 \sim 2\pi$ 内**压缩一倍，出现 2 次**。
        
    - 理想低通 ($0 \sim \pi/3$) $\implies$ 理想带阻 (阻带在 $0 \sim \pi/6$ 和 $\pi - \pi/6 \sim \pi$)。 (因为原 $0 \sim \pi/3$ 映射到 $0 \sim \pi/6$；原 $\pi/3 \sim \pi$ 映射到 $\pi/6 \sim \pi$。)
