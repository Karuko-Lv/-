####  概率论常见分布及其数字特征速查表
##### 1.离散型随机变量 (Discrete)

| 分布名称      | 记号           | 分布律 $P(X=k)$                        | 期望 $E(X)$      | 方差 $D(X)$         | 备注                                |
| --------- | ------------ | ----------------------------------- | -------------- | ----------------- | --------------------------------- |
| **0-1分布** | $B(1, p)$    | $p^k(1-p)^{1-k}, k=0,1$             | $p$            | $p(1-p)$          | 基础伯努利试验 $E(X^k)=p$                |
| **二项分布**  | $B(n, p)$    | $C_n^k p^k (1-p)^{n-k}$             | $np$           | $np(1-p)$         | $n$ 次独立重复试验-->可拆分为 $n$ 个独立0-1分布之和 |
| **泊松分布**  | $P(\lambda)$ | $\frac{\lambda^k e^{-\lambda}}{k!}$ | $\lambda$      | $\lambda$         | 稀有事件计数；$E(X^2)=\lambda^2+\lambda$ |
| **几何分布**  | $G(p)$       | $(1-p)^{k-1}p$                      | $\frac{1}{p}$  | $\frac{1-p}{p^2}$ | 首次成功, 无记忆性;推导需用到级数求导技巧            |
| **超几何分布** | $H(N,M,n)$   | $\frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$ | $n\frac{M}{N}$ | $\approx np(1-p)$ | 不放回抽样                             |
##### 2. 连续型随机变量 (Continuous)
| 分布名称     | 记号                 | 概率密度 $f(x)$                                                               | 分布函数 $F(x)$                             | 期望 $E(X)$           | 方差 $D(X)$             | 备注                          |
| -------- | ------------------ | ------------------------------------------------------------------------- | --------------------------------------- | ------------------- | --------------------- | --------------------------- |
| **均匀分布** | $U(a, b)$          | ~={red}$\frac{1}{b-a}, a<x<b=~$（偶函数）                                      | $\frac{x-a}{b-a}, a \le x < b$          | $\frac{a+b}{2}$     | $\frac{(b-a)^2}{12}$  | 必须熟记公式                      |
| **指数分布** | $E(\lambda)$       | $\lambda e^{-\lambda x}, x>0$                                             | ~={red} $1-e^{-\lambda x}, x \ge 0$ =~  | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ | 积分技巧的核心                     |
| **正态分布** | $N(\mu, \sigma^2)$ | ~={red} $\frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ =~ | ~={red} $\Phi(\frac{x-\mu}{\sigma})$ =~ | $\mu$               | $\sigma^2$            | $E(X^2) = \sigma^2 + \mu^2$ |
| **标准正态** | $N(0, 1)$          | $\frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$（偶函数）                           | $\Phi(x)$                               | $0$                 | $1$                   | $E(X^2) = 1$                |
##### 3. 核心关系速记 ‼️重要极限 $\lim_{n \to \infty} (1 + \frac{x}{n})^n = e^x$
1. **0-1 与 二项**：$n$ 个相互独立的 $0-1$ 分布变量之和 $\to B(n, p)$。
2. **二项与泊松**：当 $n \to \infty, p \to 0, np = \lambda$ 时，$B(n, p) \to P(\lambda)$。离散$\to$离散
3. **二项与正态**：当 $n$ 很大时（棣莫弗-拉普拉斯定理），二项分布近似于正态分布。离散 $\to$ 连续
4. **泊松与指数**：如果事件发生的次数服从泊松分布 $P(\lambda)$，那么相邻两次事件发生的时间间隔服从指数分布 $E(\lambda)$
5. **几何与指数**：几何分布的极限就是指数分布
#####  4. 高级积分技巧：凑分布法
在计算形如 $\int_0^{+\infty} x^k e^{-\lambda x} dx$ 或 $\int_{-\infty}^{+\infty} x^k e^{-(x-\mu)^2} dx$ 的积分时，**不要硬算**。
1. **指数型**：凑成指数分布密度的积分（值为 1）或期望/二阶矩公式。书 P77
    - 例：$\int_0^{+\infty} x^2 e^{-\lambda x} dx = \frac{1}{\lambda} \int_0^{+\infty} x^2 (\lambda e^{-\lambda x}) dx = \frac{1}{\lambda} E(X^2) = \frac{2}{\lambda^3}$。
2. **正态型**：凑成正态分布密度的积分。遇到类似 $e$ 的负平方次幂对整个实数域进行积分，可以凑成正态密度函数的形式，原理是 $\int_{-\infty}^{+\infty} \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}dx = 1$
    - 例：$\int_{-\infty}^{+\infty} e^{-(x-\mu)^2} dx = \sqrt{\pi}$ (利用标准正态化)。
$\left\{\begin{array}{l}\int_{-\infty}^{+\infty}x\cdot\lambda e^{-\lambda x}dx=\frac{1}{\lambda}\\\int_{-\infty}^{+\infty}xe^{\lambda x}dx=\frac{1}{\lambda^2}\\\int_{-\infty}^{+\infty}x^2 e^{\lambda x}dx=\frac{1}{\lambda}\int_{0}^{+\infty}x^2\lambda e^{-\lambda x}dx=\frac{2}{\lambda^3}\\\int_{-\infty}^{+\infty}xe^{-(x-\mu)^2}dx=\sqrt{\pi}\mu\\\int_{-\infty}^{+\infty}x^2 e^{-(x-\mu)^2}dx=\left (\frac{1}{2}+\mu^2\right)\sqrt{\pi}\\\int_{-\infty}^{+\infty} e^{-(x-\mu)^2} dx = \sqrt{\pi} \end{array}\right.$

	2. **二项分布如何“变成”正态分布？** (这对应《复习全书》中的 **棣莫弗-拉普拉斯定理** )
2. 随机变量分布函数的概念与性质
	1. **分布函数定义 (考点)** 
		- 全称：累积分布函数
	    - **公式**： $F(x) = P\{X \le x\}, \quad -\infty < x < +\infty$ 【背✅】对于任意一个实数 $x$，分布函数 $F(x)$ 是随机变量 $X$ 的取值小于或等于 $x$ 的概率。
	2. **分布函数的性质 (考点)**
	    1. **单调性**：$F(x)$ 是**单调非减**函数。【背✅】分布函数在可导点的导数大于等于 0
	    2. **有界性**：$0 \le F(x) \le 1$。【背✅】分布函数的值是概率值，在 0 到 1 之间
	    3. **极限**：$F(-\infty) = \lim_{x \to -\infty} F(x) = 0$，$F(+\infty) = \lim_{x \to +\infty} F(x) = 1$。【背✅】分布函数正无穷的值为 1，负无穷的值为 0
	    4. **右连续性**：$F(x) = F(x^+)$，即 $\lim_{\epsilon \to 0^+} F(x+\epsilon) = F(x)$ 即 ~={red} $\lim_{x\to x_0^+} F(x) = F(x_0)$ =~
		    - 【背✅】 $\lim_{x\to x_0^+} F(x) = F(x_0)$ 当自变量 $x$ 从**右侧**（即数值大于 $x_0$ 的方向）无限趋近于某一点 $x_0$ 时，分布函数 $F(x)$ 的极限值等于该点处的实际函数值 $F(x_0)$
		    - 【记忆】 $F(x_0^+)$ 等价于 $\lim_{x\to x_0^+} F(x)$, $F (x^+)$ 等价于 $\lim_{\epsilon \to 0^+} F(x+\epsilon)$ 。$F(x_0^-)$ 等价于 $\lim_{x\to x_0^-} F(x)$
	    - **[考点]** 满足这四条性质的函数才能是分布函数 (例题 1, 2)
	    - PS：当随机变量是连续型的时候，随机变量 $X$ 对应的分布函数一定是连续的，但是不一定是处处可导的。连续性随机变量在任意一点取值的概率为 0，其分布函数正哥定义域是左连续的，而🆚**离散型随机变量**：在可能取值点 $x_k$ 处，$P(X=x_k) > 0$，此时 $F(x)$ 发生跳跃，**不满足**左连续
	3. **用分布函数计算概率 (考点)**【背✅】
	    - $P\{X \le a\} = F(a)$。
	    - ~={red} $P\{X < a\} = F(a^-) = \lim_{x \to a^-} F(x)$ (左极限)。=~
	    - ~={red} $P\{X = a\} = F(a) - F(a^-)$ =~(在 $a$ 处的跳跃高度)。
	    - $P\{x_1 < X \le x_2\} = F(x_2) - F(x_1)$。
		    - ⚠️左开右闭：【考点】由分布函数求概率标准写法区间左开右闭
		    - ‼️**分布函数 $F(x)$ 的分段表达式**时（特别是针对**离散型**随机变量），区间的写法通常确实是 **“左闭右开”**，即 $[a, b)$ 或 $a \le x < b$。
	    - ~={red} $P\{x_1 \le X \le x_2\} = F(x_2) - F(x_1^-)$。=~【重点背】
	    - ~={red}  $P\{x_1 < X < x_2\} = F(x_2^-) - F(x_1)$。=~【重点背】
	    - ~={red} $P\{x_1 \le X < x_2\} = F(x_2^-) - F(x_1^-)$。=~
	    - $P\{X > a\} = 1 - P\{X \le a\} = 1 - F(a)$。
	    - ~={red} $P\{X \ge a\} = 1 - P\{X < a\} = 1 - F(a^-)$。=~
	4. **[考点]** 连续型随机变量的分布函数 $F(x)$ 是连续函数，此时 $P\{X = a\} = 0$。【背✅】**连续函数一点处的概率为 0**
3. 随机变量的数学期望 (Expectation)
	1. 定义与本质
		- **离散型**：$E(X) = \sum x_i p_i$（要求级数绝对收敛）。
		- **连续型**：$E(X) = \int_{-\infty}^{+\infty} x f(x) dx$（要求积分绝对收敛）。
		- **解读**：期望的本质是**加权平均值**。存在的前提是级数或积分绝对收敛，不是所有随机变量都有期望。
	2. 随机变量函数的期望 (无意识的统计学家公式)
		- **一维情形**：
		    - 离散：$E[g(X)] = \sum g(x_i) p_i$
		    - 连续：$E[g(X)] = \int_{-\infty}^{+\infty} g(x) f(x) dx$ **【考点：直接代入密度公式计算，不要先求$g(X)$的分布】**
		- **二维情形**
		    - $Z = g(X,Y) \Rightarrow E(Z) = \iint g(x,y) f(x,y) dx dy$
	3. 期望的性质
		1. $E(C) = C$
		2. $E(CX) = C E(X)$
		3. $E(X \pm Y) = E(X) \pm E(Y)$ **【高频考点：永远成立，无论是否独立】**❓
		4. 若 $X, Y$ **相互独立**，则 $E(XY) = E(X)E(Y)$。（注意：这是独立的必要非充分条件，反之不成立）
	4. 经典解题技巧：“调整法”（拆分法） **【高频考点/解题神器】**
		- **核心思想**：将复杂变量拆解为若干个简单的随机变量（通常是0-1变量）之和。
		- **经典例题（号码布问题/匹配问题）**：
		    - $n$ 个人随机拿 $n$ 个号码布，求拿到自己号码布的人数 $X$ 的期望。
		    - **解法**：令 $X_i=1$ (第 $i$ 人拿到自己号码)，$X_i=0$ (否则)。则 $E(X_i) = 1/n$。
		    - $E(X) = E(\sum X_i) = \sum E(X_i) = n \times \frac{1}{n} = 1$。
4. 随机变量的方差 (Variance)
	1. 定义与计算公式
		- **定义**：$D(X) = E\{[X - E(X)]^2\}$
		- **计算公式**：$D(X) = E(X^2) - [E(X)]^2$ **【考点：做题最常用公式】**
		    - **解读**：求 $E(X^2)$ 是历年考试高频点，常通过 $E(X^2) = D(X) + [E(X)]^2$ 反求。
	2. 方差的性质
		1. $D(C) = 0$
		2. $D(CX) = C^2 D(X)$ （注意平方）
		3. $D(X+C) = D(X)$ **【考点：方差具有平移不变性】**
		4. $D(X \pm Y) = D(X) + D(Y) \pm 2Cov(X,Y)$
		    - **特例**：若 $X, Y$ **相互独立**，则 $D(X \pm Y) = D(X) + D(Y)$。（推广：有限个独立变量和的方差等于方差的和）
5. 随机变量的矩(Moments)
	1. $k$ **阶原点矩**：$E(X^k)$。期望是1阶原点矩。
	2. $k$ **阶中心矩**：$E\{[X - E(X)]^k\}$。方差是2阶中心矩。
	3. $k+l$ **阶混合中心矩**：$E\{[X - E(X)]^k [Y - E(Y)]^l\}$。协方差是1+1=2阶混合中心矩。
6. 二维随机变量协方差与相关系数
	1. 协方差 (Covariance)
		- **定义**：$Cov(X,Y) = E\{[X-E(X)][Y-E(Y)]\}$
		- **计算公式**：$Cov(X,Y) = E(XY) - E(X)E(Y)$ **【考点】**
		- **性质**：
		    1. $Cov(X,Y) = Cov(Y,X)$
		    2. $Cov(X,X) = D(X)$
		    3. $Cov(aX, bY) = ab Cov(X,Y)$
		    4. **分配律**：$Cov(X_1 \pm X_2, Y) = Cov(X_1, Y) \pm Cov(X_2, Y)$ **【解题神器】**
		    5. $D(X \pm Y) = D(X) + D(Y) \pm 2Cov(X,Y)$
	2. 相关系数 (Correlation Coefficient)
		- **定义**：$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$
		- **性质**：
		    1. $|\rho_{XY}| \le 1$。
		    2. $|\rho_{XY}| = 1 \iff Y = aX + b$ (线性关系，几乎处处成立)。
		        - $a > 0 \Rightarrow \rho = 1$
		        - $a < 0 \Rightarrow \rho = -1$
		- **【考点】**：$X,Y$ 服从二维正态分布时，$\rho=0 \iff X,Y$ 独立。
	3. 不相关性与独立性辨析 **【易错点/难点】**
		1. 关系图谱
			- **独立** $\Rightarrow$ **不相关** ($\rho = 0$)。
			- **不相关** $\nRightarrow$ **独立** (一般情况)。
			- **不相关** $\iff$ **独立** (仅在 **二维正态分布** 下成立)。
		2. 经典反例模型 **【小众点/反例素材】**
			1. $Y=X^2$ **模型**：若 $X \sim N(0,1)$，令 $Y=X^2$。
			    - 显然 $X, Y$ 不独立（有函数关系）。
			    - 但 $Cov(X,Y) = E(X^3) - E(X)E(X^2) = 0 - 0 = 0$，即 **不相关**。
			2. **圆域均匀分布模型**：
			    - $(X,Y)$ 在圆 $x^2+y^2 \le r^2$ 上均匀分布。
			    - 计算可得 $\rho_{XY} = 0$（不相关），但边缘密度 $f_X(x)$ 与 $f_Y(y)$ 乘积不等于 $f(x,y)$，故 **不独立**
			3. 事件独立性与指示变量相关性 **【小众点】**
				- 对于事件 $A, B$，定义指示变量 $X, Y$（发生取1，不发生取0）。
				- **结论**：事件 $A, B$ 独立 $\iff$ 随机变量 $X, Y$ 的相关系数 $\rho = 0$。
			4. 正态分布下的特殊考法 **【考点】**
				- 若 $(X,Y)$ 服从二维正态分布，则 $U = aX + bY$ 与 $V = cX + dY$ 独立的充要条件是 $Cov(U,V) = 0$。
				- **特例**：若 $X, Y$ i.i.d. $\sim N(\mu, \sigma^2)$，则 $X+Y$ 与 $X-Y$ **独立**。
				    - 证明：$Cov(X+Y, X-Y) = D(X) - D(Y) = \sigma^2 - \sigma^2 = 0$。
				    - **考法**：若题目问 $X+Y$ 与 $X-Y$ 何时独立？答：需 $D(X)=D(Y)$。
			5. 易错考点‼️：
				1. **级数求导技巧**：几何分布 $E(X)$ 推导中，利用 $(\sum x^k)' = \sum k x^{k-1}$。
					1. 高等数学里最熟悉的**无穷等比级数公式**：$\sum_{k=0}^{\infty} x^k = 1 + x + x^2 + x^3 + \cdots = \frac{1}{1-x}$ --> $\sum_{k=1}^{\infty} k x^{k-1} = \frac{1}{(1-x)^2}$
					2. $E(X) = p \cdot \sum_{k=1}^{\infty} k q^{k-1}$ ($q = 1-p$ 即失败概率)，$\sum_{k=1}^{\infty} k q^{k-1} = \frac{1}{p^2}$ ==> $E(X) = p \cdot \frac{1}{p^2} = \frac{1}{p}$
					3. 🌟该技巧核心：当看到求和式子里有 $k$、$k(k-1)$ 这种系数时，立刻想到它们是 $x^k$ 求一阶导、二阶导产生的系数。
				2. **区域复杂时的协方差**：如三角形区域、圆域，优先用 $E(XY)$ 的二重积分计算，注意积分限的设定。
				3. $D(2Y+1)$ **类计算**：先求 $Y$ 的分布，算出 $D(Y)$，再用 $4D(Y)$，千万别忘了系数平方。

10. 一维离散型随机变量及其分布
	1. **分布律 (概率质量函数) (考点)**
	    - 定义：$P\{X = x_i\} = p_i, \quad i = 1, 2, \dots$
	    - 性质：$p_i \ge 0$ 且 $\sum_{i=1}^\infty p_i = 1$ 【背✅】。
	    - 分布函数 $F(x) = \sum_{x_i \le x} p_i$，是一个阶梯函数【背✅】离散型随机变量的分布函数具有阶梯结构（每到一个离散点分布函数会多一次累计）
	2. 🌟**常见离散分布**
		1. **0-1 分布 (伯努利分布)** $X \sim B(1, p)$：$P\{X=1\} = p, P\{X=0\} = 1-p$ 
		2. **二项分布** $X \sim B(n, p)$ **(考点)** Binomial Distribution
			1. 功能：描述一个只有两种结果情况的事情的，例如成功和失败，若成功概率是 $p$ 则失败是 $1-p$，做这件事情 $n$ 次成功了 $k$ 次的概率是 $C_n^k p^k (1-p)^{n-k}$
	        - 分布律（概率分布表达式）：$P\{X = k\} = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, \dots, n$。
	        - 应用场景：
		        1. 试验总共进行的次数预先确定，一般是 $n$ 次
		        2. 每次试验只有两种可能结果
		        3. 每次试验“成功”概率一样
		        4. 试验之间相互独立
				【注】n 次独立重复伯努利试验，“成功”的次数服从二项分布。
	    3. ~={red}**泊松分布** =~$X \sim P(\lambda)$ **(考点)** Poisson Distribution
		    1. 功能：描述比较长的时间段里面的小概率事件发生的次数的概率分布。例如商场每天的客流量是 1 万人的概率是多少、一段马路每天的车流量是 1 万辆的概率是多少
	        - 分布律：~={red} $P\{X = k\} = \frac{\lambda^k e^{-\lambda}}{k!}=\frac{\lambda^k }{e^{\lambda}k!}, \quad k = 0, 1, 2, \dots$ =~
	        - 数学原理：难度高，要善于利用泰勒展开求和
		        - $\sum_{k=0}^{\infty} P(X=k) = \sum_{k=0}^{\infty} \frac{\lambda^k e^{-\lambda}}{k!} = e^{-\lambda} \underbrace{\sum_{k=0}^{\infty} \frac{\lambda^k}{k!}}_{这正是 e^\lambda} = e^{-\lambda} \cdot e^\lambda = 1$ ==>原理：$e^x = \sum_{k=0}^{\infty} \frac{x^k}{k!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \dots$
		        - $\sum_{k=0}^{\infty} \frac{x^{2k}}{(2k)!}=\frac{\sum_{k=0}^{\infty} \frac{x^k}{k!}+\frac{(-x)^k}{k!}}{2}=\frac{e^x+e^{-x}}{2}$ ，$ k$ 为偶时合并，$k$ 为奇时抵消。随机变量取偶数之和。
		        - 泊松分布推导（泊松过程）其实是用微分方程（泊松过程的核心动力学方程）解出来的（$P'_k(t) = -\lambda P_k(t) + \lambda P_{k-1}(t)$）。它描述了“随着时间流逝，事件发生次数变为 $k$ 次的概率是如何变化的”**。
		    - **[考点] 泊松定理 (泊松近似)** 
		        - 当 $n$ 很大，$p$ 很小，且 $\lambda = np$ 适中时，二项分布 $B(n, p)$ 可用泊松分布 $P(\lambda)$ 近似。
		        - 原理：$X \sim B(n, p)$，在 $n \to \infty, p \to 0$ 且 $\lambda = np$ 保持为常数的情况下，有以下极限公式：$\lim_{n \to \infty} C_n^k p^k (1-p)^{n-k} = \frac{\lambda^k e^{-\lambda}}{k!}$
	    4. **几何分布** $X \sim G(p)$ Geometric Distribution
	        - 分布律：$P\{X = k\} = (1-p)^{k-1} p, \quad k = 1, 2, \dots$ (首次成功在第k次试验)。
	        - **[考点] 无记忆性**：【背✅】
		        - $P\{X > m+n | X > m\} = P\{X > n | X > 0\}=P\{X > n\}$ 。
		        - $P\{X < m+n | X > m\} = P\{X < n | X > 0\}=P\{X < n\}$
			【注】在 $n$ 次伯努利试验中，定义的随机变量 $Y$ 是“第几次才出现第1次成功”，那这就变成了**几何分布**（或者截断的几何分布）
		5. **截断的几何分布**：在一个有最大次数限制（$n$）的试验中，寻找第一次成功发生在第几次。
			- **标准几何分布**（理想情况）：取值范围是 $1, 2, 3, \dots, \infty$（理论上可能永远等不到，虽然概率很小）。    
			- **截断的几何分布**（现实情况）：取值范围是 $1, 2, 3, \dots, n$（等第 1 次成功，**最多只试 $n$ 次**。如果 $n$ 次都没成功，就放弃（或者被强制停止）了。）
11. 一维连续型随机变量及其分布
	1. **概率密度函数  (考点)**
	    - 定义：$f(x)$对于随机变量 $X$ 的分布函数 $F(x)$，存在一个非负函数 $f(x)$，使得对于任意实数 $x$，都有：$F(x) = \int_{-\infty}^{x} f(t) \, dt$，称 $X$ 为**连续型随机变量**，而函数 $f(x)$ 就称为 $X$ 的**概率密度函数**。
	    - 性质：$f(x) \ge 0$ 且 $\int_{-\infty}^{+\infty} f(x) dx = 1$ (规范性)。PS: 当概率密度函数是 $g(x^2)$, 则得满足 $\int_{-\infty}^{+\infty} g(x^2) dx = 1$ 而不是 $\int_{-\infty}^{+\infty} g(x^2) d(x^2) = 1$ 后者是关于随机变量平方的累积，不是 $x$ 本身的分布。
		    - 【背✅】分布函数密度函数必须保证非负性。PS：所以密度函数只可能是偶函数不可能是极函数，分布函数是单调不减函数所以顶多是偶函数
	    - $F(x) = \int_{-\infty}^x f(t) dt$，且 $F'(x) = f(x)$ (在 $f(x)$ 连续点)。
	    - 概率的计算： $P\{x_1 < X \le x_2\} = \int_{x_1}^{x_2} f(x) dx = F(x_2) - F(x_1)$，即对应区间上 $f(x)$ 与 $x$ 轴形成的面积。
	    - 对于连续型变量，$P\{X=a\}=0$，因此 $P\{a < X \le b\} = P\{a \le X \le b\} = P\{a < X < b\} = P\{a \le X < b\}$。
	2. 🌟**常见连续分布** 心得：可以多结合期望和方差做
	    1. ~={red}**均匀分布** $X \sim U(a, b)$ =~ Uniform Distribution
	        - 概率密度函数：$f(x) = \begin{cases} \frac{1}{b-a}, & a < x < b \\ 0, & \text{其他} \end{cases}$
	    2. ~={red}**指数分布** $X \sim E(\lambda)$ **(考点)**=~ Exponential Distribution
	        - 概率密度函数：$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \le 0 \end{cases}$
	        - 分布函数：$F(x) = \begin{cases} 1 - e^{-\lambda x}, & x \ge 0 \\ 0, & x < 0 \end{cases}$
	        - **[考点] 无记忆性**：$P\{X > a+b | X > b\} = P\{X > a\}$ (例题 22)。
	    3. ~={red}**正态分布** $X \sim N(\mu, \sigma^2)$ **(考点)**=~ Normal Distribution 也称高斯分布 Gaussian Distribution。因为这种分布在自然界中非常普遍（Norm）。
	        - 概率密度函数：$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。
	        - $f(x)$ 关于 $x=\mu$ 对称 (例题 25)。
	        - **[考点] ~={red}标准正态分布=~** $N(0, 1)$
	            - 概率密度函数：$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$ (偶函数)。
	            - 分布函数：$\Phi(x) = \int_{-\infty}^x \varphi(t) dt$。
	            - 性质：【背✅】~={red} $\Phi(-a) = 1 - \Phi(a)$ =~，$\Phi(0) = 0.5$ (例题 19, 20)。
				PS：概率密度函数是偶函数，对应的分布函数都满足 $F (-a)=1-F (a)= \frac{1}{2}-\int_{0}^a f(x) dx$
	        - **[考点] 标准化**：若 $X \sim N(\mu, \sigma^2)$，则 $Z = \frac{X-\mu}{\sigma} \sim N(0, 1)$ ，$F(z)=\Phi (\frac{x-\mu}{\sigma})$
	        - **[考点] 线性组合**：若 $X \sim N(\mu, \sigma^2)$，则 $Y = aX+b \sim N(a\mu+b, a^2\sigma^2)$ 
	        - **[考点] 对称性**：若 $X \sim N(\mu, \sigma^2)$，对称关系是关于均值 $\mu$ 的：$F(\mu - x) = 1 - F(\mu + x)$；特别的，标准正态分布的分布函数 $\Phi(x)$：$\Phi(-x) + \Phi(x) = 1$
	3. 连续性随机变量要充分运用几何特性，要将几何特性发挥出来
12. **一维随机变量函数的分布**
	1. **分布函数法 (定义法) (考点)**
	    - 最通用、最基本的方法。适用于任何函数 $Y = g(X)$。
	    - 步骤：$F_Y(y) = P\{Y \le y\} = P\{g(X) \le y\}$。
	    - 然后，解不等式 $g(X) \le y$，将其转化为 $X$ 的范围 (可能是多个区间)。
	    - 最后，在 $X$ 的这些范围上对 $f_X(x)$ 积分，即 $F_Y(y) = \int_{g(x) \le y} f_X(x) dx$。
	    - **[考点]** 对于分段函数或非单调函数 $g(x)$，必须用此方法 (例题 29, 30)。
	2. **公式法 (单调函数法) (考点)**
	    - 前提：$y = g(x)$ 处处可导且**严格单调**。
	    - 设 $x = h(y)$ 是 $y=g(x)$ 的反函数。
	    - 公式：$f_Y(y) = f_X[h(y)] \cdot |h'(y)|$，$y$ 的范围是 $g(x)$ 的值域。
	    - (例题 31)
	3. **[小众点/考点]** 设 $Y$ 是连续型随机变量，其分布函数为 $F_Y(y)$。令 $U = F_Y(Y)$，则 $U$ 服从 $[0, 1]$ 上的均匀分布 $U(0, 1)$。(例题 32 笔记②)

14. 二维随机变量的分布函数
	1. **联合分布函数 (考点)**
	    - 定义：$F(x, y) = P\{X \le x, Y \le y\}$。
	    - 性质：
	        - $0 \le F(x, y) \le 1$。
	        - $F(x, y)$ 关于 $x$ 和 $y$ 均单调不减。
	        - $F(-\infty, y) = F(x, -\infty) = F(-\infty, -\infty) = 0$。
	        - $F(+\infty, +\infty) = 1$。
	        - 关于 $x$ 和 $y$ 均右连续。
	    - **[考点]** 矩形区域概率：$P\{x_1 < X \le x_2, y_1 < Y \le y_2\} = F(x_2, y_2) - F(x_2, y_1) - F(x_1, y_2) + F(x_1, y_1)$ (例题 1)。
	2. **边缘分布函数 (考点)**
	    - $F_X(x) = P\{X \le x\} = P\{X \le x, Y \le +\infty\} = F(x, +\infty)$。
	    - $F_Y(y) = P\{Y \le y\} = P\{X \le +\infty, Y \le y\} = F(+\infty, y)$。
	    - (例题 3)
	3. **独立性 (考点)**
	    - $X$ 和 $Y$ 相互独立的**充要条件**是：$F(x, y) = F_X(x) F_Y(y)$ 对**任意** $x, y$ 成立 (例题 3, 5, 6)。
15. 二维离散型随机变量
	1. **联合分布律 (考点)**
	    - $p_{ij} = P\{X = x_i, Y = y_j\}$。
	    - 性质：$p_{ij} \ge 0$ 且 $\sum_i \sum_j p_{ij} = 1$。
	    - 核心是建立**联合分布表** (例题 7, 8, 9)。
	2. **边缘分布律 (考点)**
	    - $P\{X = x_i\} = p_{i \cdot} = \sum_j p_{ij}$ (第 $i$ 行的和)。
	    - $P\{Y = y_j\} = p_{\cdot j} = \sum_i p_{ij}$ (第 $j$ 列的和)。
	3. **独立性 (考点)**
	    - $X$ 和 $Y$ 相互独立的**充要条件**是：$p_{ij} = p_{i \cdot} \cdot p_{\cdot j}$ 对**所有** $i, j$ 成立 (例题 10)。
	4. **条件分布 (考点)**
	    - $P\{X = x_i | Y = y_j\} = \frac{P\{X = x_i, Y = y_j\}}{P\{Y = y_j\}} = \frac{p_{ij}}{p_{\cdot j}}$ (例题 8)。
16. 二维连续型随机变量
	1. **联合概率密度 (PDF) (考点)**
	    - $f(x, y)$。
	    - 性质：$f(x, y) \ge 0$ 且 $\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) dx dy = 1$ (规范性)。
	    - $P\{(X, Y) \in D\} = \iint_D f(x, y) dx dy$。	        
	    - $F(x, y) = \int_{-\infty}^x \int_{-\infty}^y f(u, v) dv du$，且 $\frac{\partial^2 F}{\partial x \partial y} = f(x, y)$。
	2. **边缘概率密度 (考点)**
	    - $f_X(x) = \int_{-\infty}^{+\infty} f(x, y) dy$ (对 $y$ 积分，消去 $y$)。
	    - $f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) dx$ (对 $x$ 积分，消去 $x$)。
	    - (例题 11, 12)
	3. **独立性 (考点)**
	    - $X$ 和 $Y$ 相互独立的**充要条件**是：$f(x, y) = f_X(x) f_Y(y)$ (几乎处处成立)。
	    - **[考点] 快速判断**：
	        1. $f(x, y) \ne 0$ 的区域必须是一个**矩形** (可以无界)。
	        2. $f(x, y)$ 必须可以**分离变量**，即 $f(x, y) = g(x) h(y)$。
	        - (例题 13, 14, 15)
	4. **条件概率密度 (考点)**
	    - $f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$ (要求 $f_Y(y) > 0$)。
	    - $f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$ (要求 $f_X(x) > 0$)。
	    - 注意：条件概率密度也是一个一维概率密度，关于 $x$ (或 $y$) 的积分为 1 (例题 11, 12)。
17. 二维均匀分布与二维正态分布
	1. **二维均匀分布 (考点)**
	    - 若 $(X, Y)$ 在区域 $G$ (面积为 $A$) 上服从均匀分布。
	    - PDF：$f(x, y) = \begin{cases} 1/A, & (x, y) \in G \\ 0, & \text{其他} \end{cases}$
	    - **[考点]** 概率与面积成正比：$P\{(X, Y) \in G_1\} = \frac{\text{Area}(G_1)}{\text{Area}(G)}$ (其中 $G_1 \subset G$) (例题 17)。
	    - **[考点]** 若 $G$ 是矩形区域 $[a, b] \times [c, d]$，则 $(X, Y)$ 在 $G$ 上均匀 $\iff$ $X \sim U(a, b), Y \sim U(c, d)$ 且 $X, Y$ 相互独立 (例题 16)。
	    - **[小众点]** 若 $G$ 不是矩形 (如圆形、三角形)，则 $X$ 和 $Y$ **不独立**，且其边缘分布**不是**均匀分布 (例题 17)。
	2. **二维正态分布** $(X, Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; \rho)$ **(考点)**
	    - **[考点] 边缘分布**：边缘分布仍是正态分布。$X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$。
	    - **[考点] 独立性**：$X$ 和 $Y$ 相互独立的**充要条件**是 $\rho = 0$ (例题 23)。
	    - **[考点] 线性组合**：$X$ 和 $Y$ 的任意线性组合仍服从正态分布。
	        - $Z = aX + bY \sim N(a\mu_1 + b\mu_2, a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2)$ (例题 20, 22)。
	    - **[考点]** 只要 $(X, Y)$ 服从二维正态分布，则 $X+Y$ (或 $aX+bY$) **必定**服从一维正态分布 (例题 21)。
	    - **[小众点]** $X, Y$ 分别服从一维正态分布，**不能**推出 $(X, Y)$ 服从二维正态分布	        
	    - **[小众点]** $X, Y$ 分别服从一维正态分布 (不独立)，$X+Y$ **不一定**服从正态分布 (例题 21)。
	    - **[小众点]** $X, Y$ 分别服从一维正态分布且**独立**，则 $(X, Y)$ 服从二维正态分布 (此时 $\rho=0$)，因此 $X+Y$ 服从正态分布 (例题 21)。
	    - **[小众点]** 条件分布也是正态分布：$Y|X=x \sim N(\dots, \dots)$ (例题 18, 19)。
18. 二维离散型随机变量函数的分布
	1. **求** $Z = g(X, Y)$ **的分布**
	    - 方法：列举 $Z$ 所有可能的取值 $z_k$。
	    - 计算 $P\{Z = z_k\} = \sum_{g(x_i, y_j) = z_k} P\{X = x_i, Y = y_j\}$。
	    - (例题 24, 25)
	2. **求** $(U, V)$ **的联合分布 (其中** $U=g_1(X, Y), V=g_2(X, Y)$**)**
	    - 方法：列出 $(X, Y)$ 的所有取值组合 $(x_i, y_j)$。
	    - 计算每一组合对应的 $(u, v)$ 值。
	    - 将相同 $(u, v)$ 值的概率 $p_{ij}$ 相加，得到 $P\{U=u, V=v\}$ (例题 24)。
19. 二维连续型随机变量函数的分布
	1. **求** $Z = g(X, Y)$ **的分布 - 定义法 (考点)**
	    - 步骤 1：$F_Z(z) = P\{Z \le z\} = P\{g(X, Y) \le z\}$。
	    - 步骤 2：计算二重积分 $F_Z(z) = \iint_{g(x, y) \le z} f(x, y) dx dy$。
	    - 步骤 3：求导 $f_Z(z) = F_Z'(z)$。
	    - **[考点]** 此方法通用，特别适用于 $Z = \sqrt{X^2+Y^2}$ (例题 30) 或 $Z=\max/\min$ (见题型八)。
	2. **求** $Z = g(X, Y)$ **的分布 - 卷积公式法 (考点)**
	    - **[考点]** $Z = X+Y$：$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z-x) dx$。
	    - **[小众点]** $Z = X/Y$：$f_Z(z) = \int_{-\infty}^{+\infty} f(yz, y) |y| dy$。
	    - **[小众点]** $Z = XY$：$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z/x) |\frac{1}{x}| dx$。
	    - **[考点]** 通用公式 (以 $Y=h(X, Z)$ 为例)：$f_Z(z) = \int_{-\infty}^{+\infty} f(x, h(x, z)) \left| \frac{\partial h}{\partial z} \right| dx$。
	    - **注意**：此方法**不需要** $X, Y$ 独立。最关键的步骤是根据 $f(x, y)$ 的非零区域和变量代换，确定积分的**上下限** (例题 26, 27, 28)。
	    - **[小众点]** 对于 $Z = |X-Y|$，可以看作 $Z = X-Y$ 和 $Z = Y-X$ 两个互斥事件的和，使用“双卷积” (例题 29)。
20. 二维混合型随机变量
	1. **混合型 (一个离散** $X$**，一个连续** $Y$**) (小众点/考点)**
	    - **[考点]** 核心方法：**全概率公式**，以离散变量 $X$ 为条件。
	    - **求** $Y$ **的分布**：$F_Y(y) = P\{Y \le y\} = \sum_i P\{X=x_i\} P\{Y \le y | X=x_i\}$ (例题 33)。
	    - **求** $Z = g(X, Y)$ **的分布**：
	        - $F_Z(z) = P\{Z \le z\} = P\{g(X, Y) \le z\}$
	        - $F_Z(z) = \sum_i P\{X=x_i, g(x_i, Y) \le z\}$
	        - $F_Z(z) = \sum_i P\{X=x_i\} P\{g(x_i, Y) \le z | X=x_i\}$
	        - (例题 31, 32, 33)
21. Max 和 min 型随机变量的分布
	1. $U = \max\{X, Y\}$**,** $V = \min\{X, Y\}$ **(考点)**
	    - **[考点] 定义法 (通用)**
	        - **Max**: $F_U(z) = P\{U \le z\} = P\{\max\{X, Y\} \le z\} = P\{X \le z, Y \le z\} = F(z, z)$。
	        - **Min**: $F_V(z) = P\{V \le z\} = 1 - P\{V > z\} = 1 - P\{\min\{X, Y\} > z\} = 1 - P\{X > z, Y > z\}$
	    - **[考点]** $X, Y$ **相互独立 (常用)**
	        - **Max**: $F_U(z) = P\{X \le z\} P\{Y \le z\} = F_X(z) F_Y(z)$。
	        - **Min**: $F_V(z) = 1 - P\{X > z\} P\{Y > z\} = 1 - [1 - F_X(z)][1 - F_Y(z)]$。
	        - (例题 35, 38)
	    - **[考点]** 若 $X_1, \dots, X_n$ 独立同分布于 $E(\lambda)$，则 $\min\{X_1, \dots, X_n\} \sim E(n\lambda)$ (例题 37)。
	    - **[小众点]** $X, Y$ 不独立时，只能用 $F(z, z)$ 或 $P\{X > z, Y > z\}$ 计算 (例题 36)。


一维随机变量函数 $Y=g(X)$ 结合常见分布（正态、均匀、指数）是概率论考试中**计算题的重灾区**

这类题目通常不会考偏难怪的函数，而是考查你对**“分布函数法（定义法）”**和**“公式法”**的掌握程度，以及对常见分布性质的熟悉程度。

以下是这类考题最常出现的**4种经典套路**：

### 1. 正态分布 + 线性函数（送分题/基础题）

这是最简单但最常考的类型，主要考察正态分布的线性性质。

- **考点原理**：若 $X \sim N(\mu, \sigma^2)$，且 $Y = aX + b$，则 $Y \sim N(a\mu+b, a^2\sigma^2)$。
    
- **典型题目**：
    
    > 已知 $X \sim N(1, 4)$，求 $Y = 2X + 3$ 的概率密度函数。
    
- **解题策略**：
    
    - **秒杀法**：直接算 $E(Y) = 2E(X)+3 = 5$， $D(Y) = 2^2 D(X) = 4 \times 4 = 16$。直接写出 $Y \sim N(5, 16)$ 的密度函数公式。
        
    - **稳健法**：用公式法推导一遍。
        

### 2. 均匀分布/正态分布 + 平方函数 $Y=X^2$（经典陷阱题）

这是**最容易出错**的题型，因为 $g(x) = x^2$ 不是单调函数，**不能直接套公式**，必须用**分布函数法（定义法）**，且往往涉及区间的分段。

- **典型题目 A（均匀分布）**：
    
    > 设 $X \sim U(-1, 2)$，求 $Y = X^2$ 的概率密度函数。
    
    - **难点**：$X$ 的取值范围跨越了 0（从负到正），导致 $Y=X^2$ 的反函数在不同区间有不同解个数。需要分 $0 < y < 1$（有两个解）和 $1 < y < 4$（只有一个解）讨论。
        
- **典型题目 B（标准正态分布）**：
    
    > 设 $X \sim N(0, 1)$，求 $Y = X^2$ 的概率密度。
    
    - **重要结论**：这是 $\chi^2$ 分布（卡方分布）的来源。推导过程考察了对偶函数积分的处理能力。
        

### 3. 指数分布 + 线性或指数函数（常规计算题）

这类题目通常用来考察**公式法**的使用条件（单调可导）。

- **典型题目**：
    
    > 设 $X \sim E(\lambda)$（指数分布），求 $Y = e^X$ 或 $Y = 1 - e^{-X}$ 的概率密度。
    
- **解题策略**：
    
    - 这些函数通常是严格单调的，直接套用公式 $f_Y(y) = f_X[h(y)] \cdot |h'(y)|$ 即可。
        
    - **注意**：一定要先定好 $Y$ 的取值范围（例如 $Y=e^X$ 且 $x>0$，则 $y>1$），这是最容易丢分的地方。
        

### 4. 均匀分布 + 三角函数（高阶计算题）

这类题目考察较复杂的积分区间分析。

- **典型题目**：
    
    > 设 $X \sim U(0, \pi)$，求 $Y = \sin X$ 的概率密度。
    
- **难点**：$\sin X$ 在 $(0, \pi)$ 上先增后减，不是单调的。
    
    - 解 $Y \le y$ 时，对应 $X$ 的区间是两头（比如 $[0, \arcsin y] \cup [\pi - \arcsin y, \pi]$），需要利用对称性求解。
        

### 总结：解这类题的“万能钥匙”

不管题目怎么出，心中要有两套方案：

1. **首选方案（若 $g(x)$ 单调）**：**公式法**。
    
    - $f_Y(y) = f_X[h(y)] \cdot |h'(y)|$。
        
    - **核心**：别忘了标明 $Y$ 的取值范围。
        
2. **保底方案（若 $g(x)$ 不单调，如平方、绝对值、正弦）**：**分布函数法（定义法）**。
    
    - 第一步：写出 $F_Y(y) = P(Y \le y) = P(g(X) \le y)$。
        
    - 第二步：画图，把 $g(X) \le y$ 转化成 $X$ 的区间。
        
    - 第三步：在 $X$ 的密度函数上积分。
        
    - 第四步：求导得 $f_Y(y)$。
