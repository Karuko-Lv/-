$ay'' + by' + cy = 0$可以用降阶来求解吗

**是的，可以，而且“降阶法” (Reduction of Order) 正是用来严格证明“重根”情况解的来源！**

不过，这里有一个“先有鸡还是先有蛋”的问题：

**使用“降阶法”的前提，是您必须已经知道 $ay'' + by' + cy = 0$ 的一个解。**

对于常系数方程，我们是如何知道第一个解的呢？—— 答案是**通过特征方程 $ar^2 + br + c = 0$**。

---

### 降阶法的逻辑

降阶法的完整逻辑是：

1. 我们通过求解特征方程 $ar^2 + br + c = 0$，得到两个根 $r_1$ 和 $r_2$。
    
2. 我们**至少**能得到**一个**解，即 $y_1(x) = e^{r_1 x}$。
    
3. 我们利用降阶法，假设通解为 $y(x) = v(x) y_1(x) = v(x) e^{r_1 x}$，来**寻找第二个**线性无关的解。
    

### 演示：降阶法如何推导出所有三种情况

我们来看降阶法是如何完美地推导出我们熟知的三种解的。

我们已经有了一个解 $y_1 = e^{r_1 x}$。我们设 $y = v(x) e^{r_1 x}$。

- $y' = v'e^{r_1 x} + r_1 v e^{r_1 x}$
    
- $y'' = v''e^{r_1 x} + 2r_1 v'e^{r_1 x} + r_1^2 v e^{r_1 x}$
    

把它们代入 $ay'' + by' + cy = 0$：

$$a(v''e^{r_1 x} + 2r_1 v'e^{r_1 x} + r_1^2 v e^{r_1 x}) + b(v'e^{r_1 x} + r_1 v e^{r_1 x}) + c(v e^{r_1 x}) = 0$$

把 $e^{r_1 x}$（它不为0）除掉，并按 $v$ 的导数整理：

$$a v'' + (2ar_1 + b) v' + (ar_1^2 + br_1 + c) v = 0$$

关键一步： 因为 $r_1$ 是特征方程的根，所以 $ar_1^2 + br_1 + c = 0$。

上面方程的最后一项 $(..._...)v$ 直接消失了！

我们得到了一个**降阶**后的方程：

$$a v'' + (2ar_1 + b) v' = 0$$

令 $w = v'$，这就是一个**一阶**方程：

$$a w' + (2ar_1 + b) w = 0$$

这是一个可分离变量的方程，解为 $w = C_1 e^{-\frac{2ar_1+b}{a} x}$。

---

#### 核心分析：对指数 $-\frac{2ar_1+b}{a}$ 的分析

我们知道，一元二次方程的根 $r_1 = \frac{-b + \sqrt{\Delta}}{2a}$ 和 $r_2 = \frac{-b - \sqrt{\Delta}}{2a}$。

它们满足 $r_1 + r_2 = -b/a$。

所以 $b = -a(r_1+r_2)$。代入指数：

$$-\frac{2ar_1+b}{a} = -\frac{2ar_1 - a(r_1+r_2)}{a} = -(2r_1 - (r_1+r_2)) = -(r_1 - r_2) = r_2 - r_1$$

所以我们得到 $w = C_1 e^{(r_2-r_1)x}$。

因为 $v' = w$，所以 $v = \int w \, dx = \int C_1 e^{(r_2-r_1)x} dx$。

**现在，我们分情况讨论：**

#### 情况一：$r_1 \neq r_2$ (两个不相等的根)

- $v(x) = \int C_1 e^{(r_2-r_1)x} dx = \frac{C_1}{r_2-r_1} e^{(r_2-r_1)x} + C_2$
    
- (令 $K_1 = C_1 / (r_2-r_1)$)
    
- $y(x) = v(x) e^{r_1 x} = (K_1 e^{(r_2-r_1)x} + C_2) e^{r_1 x}$
    
- $y(x) = K_1 e^{r_2 x} + C_2 e^{r_1 x}$
    
- **结论：** 这就是我们背过的“两个不等实根”或“一对共轭复根”的通解形式。
    

#### 情况二：$r_1 = r_2 = r$ (两个相等的重根)

- **这就是降阶法最有价值的地方！**
    
- 此时 $r_2 - r_1 = 0$。
    
- $w(x) = C_1 e^{0 \cdot x} = C_1$
    
- $v(x) = \int C_1 dx = C_1 x + C_2$
    
- **见证奇迹：**
    
    - $y(x) = v(x) e^{r_1 x} = (C_1 x + C_2) e^{rx}$
        
    - $y(x) = C_1 x e^{rx} + C_2 e^{rx}$
        
- **结论：** 降阶法**证明**了为什么在重根情况下，第二个解**必须是 $x e^{rx}$**。
    

---

### 总结

- **可以吗？** **可以。**
    
- **怎么做？** 先用特征方程 $ar^2+br+c=0$ 求出**一个**根 $r_1$，得到**一个**解 $y_1=e^{r_1 x}$。然后设 $y = v(x) y_1(x)$，代回原方程，即可“降阶”求解。
    
- **有必要吗？** 在实际做题时**没必要**。我们之所以用特征方程法，就是因为它已经帮我们“预先”做完了降阶法的所有工作。但在理论上，降阶法是特征方程法（特别是重根情况）的坚实基础。
