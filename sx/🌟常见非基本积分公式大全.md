
## 一、对数函数积分

1. 🌟 $\int \ln x  dx = x \ln x - x + C$
2. 🌟$\int x \ln x  dx = \frac{1}{2}x^2 \ln x - \frac{1}{4}x^2 + C$
3. 🌟$\int x^n \ln x  dx = \frac{x^{n+1}}{n+1} \ln x - \frac{x^{n+1}}{(n+1)^2} + C \quad (n \neq -1)$
4. 🌟 $\int (\ln x)^2  dx = x (\ln x)^2 - 2x \ln x + 2x + C$
5. 🌟 $\int \frac{\ln x}{x}  dx = \frac{1}{2} (\ln x)^2 + C$
6. $\int \frac{dx}{x \ln x} = \ln |\ln x| + C$

## 二、指数函数积分

7. $\int x e^{ax}  dx = \frac{e^{ax}}{a^2} (ax - 1) + C$ ==> 🌟 $\int x e^{x}  dx = {e^{x}} (x - 1) + C$
8. $\int x^2 e^{ax}  dx = e^{ax} \left( \frac{x^2}{a} - \frac{2x}{a^2} + \frac{2}{a^3} \right) + C$ ==> 🌟 $\int x^2 e^{x}  dx = e^{x} \left( {x^2} - {2x} + {2} \right) + C$
9. 🌟$\int e^{ax} \sin bx  dx = \frac{e^{ax}}{a^2 + b^2} (a \sin bx - b \cos bx) + C$
10. 🌟 $\int e^{ax} \cos bx  dx = \frac{e^{ax}}{a^2 + b^2} (a \cos bx + b \sin bx) + C$
11. 🌟 $\int a^x  dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \neq 1)$

## 三、反三角函数积分

12. $\int \arcsin x  dx = x \arcsin x + \sqrt{1-x^2} + C$
13. $\int x \arcsin x  dx = \frac{1}{4} \left( (2x^2-1) \arcsin x + x \sqrt{1-x^2} \right) + C$
14. $\int \arctan x  dx = x \arctan x - \frac{1}{2} \ln (1+x^2) + C$
15. $\int x \arctan x  dx = \frac{1}{2} \left( (x^2+1) \arctan x - x \right) + C$
16. $\int \operatorname{arccot} x  dx = x \operatorname{arccot} x + \frac{1}{2} \ln (1+x^2) + C$

## 四、分式与根式积分

17. 🌟$\int \frac{dx}{x^2 + a^2} = \frac{1}{a} \arctan \frac{x}{a} + C$
18. 🌟 $\int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{|a|} + C \quad (a \neq 0)$
19. 🌟 $\int \sqrt{a^2 - x^2}  dx = \frac{x}{2} \sqrt{a^2 - x^2} + \frac{a^2}{2} \arcsin \frac{x}{a} + C$
20. 🌟 $\int \sqrt{x^2 \pm a^2}  dx = \frac{x}{2} \sqrt{x^2 \pm a^2} \pm \frac{a^2}{2} \ln |x + \sqrt{x^2 \pm a^2}| + C$
21. $\int \frac{dx}{x\sqrt{x^2 - a^2}} = \frac{1}{|a|} \operatorname{arcsec} \frac{|x|}{a} + C$

## 五、双曲函数积分

22. $\int \sinh x  dx = \cosh x + C$
23. $\int \cosh x  dx = \sinh x + C$
24. $\int \tanh x  dx = \ln |\cosh x| + C$
25. $\int \sinh^2 x  dx = \frac{\sinh 2x}{4} - \frac{x}{2} + C$
26. $\int \cosh^2 x  dx = \frac{\sinh 2x}{4} + \frac{x}{2} + C$

## 六、特殊组合积分

28. **$\int \frac{\ln (x + \sqrt{x^2 + a^2})}{\sqrt{x^2 + a^2}}  dx = \frac{1}{2} [\ln (x + \sqrt{x^2 + a^2})]^2 + C$**
29. **$\int \frac{x}{\sqrt{x^2 + a^2}} \ln (x + \sqrt{x^2 + a^2})  dx = \sqrt{x^2 + a^2} \ln (x + \sqrt{x^2 + a^2}) - x + C$**
30. **$\int e^{ax} \ln x  dx = \frac{e^{ax}}{a} \ln x - \frac{1}{a} \int \frac{e^{ax}}{x} dx$ (需用指数积分 Ei)**
31. **$\int \sin (\ln x)  dx = \frac{x}{2} [\sin (\ln x) - \cos (\ln x)] + C$**
32. **$\int \cos (\ln x)  dx = \frac{x}{2} [\sin (\ln x) + \cos (\ln x)] + C$**

## 七、实用技巧

### 递推公式
- $\int x^n e^{ax}  dx = \frac{x^n e^{ax}}{a} - \frac{n}{a} \int x^{n-1} e^{ax}  dx$
- $\int (\ln x)^n  dx = x (\ln x)^n - n \int (\ln x)^{n-1}  dx$

### 特殊函数表示
- $\int \frac{e^{ax}}{x}  dx = \operatorname{Ei}(ax) + C$ (指数积分)
- $\int \frac{\sin x}{x}  dx = \operatorname{Si}(x) + C$ (正弦积分)
- $\int \frac{\cos x}{x}  dx = \operatorname{Ci}(x) + C$ (余弦积分)

> 注：以上公式均假设积分在定义域内有效，实际应用时需注意定义域和积分常数 $C$。
