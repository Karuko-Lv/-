你问的这个问题非常好。零向量和零矩阵是线性代数中的“0”，但它们的性质和我们熟悉的数字 0 既有相似之处，又有**极大的不同**。

你提到线代结论多、记不住，根本原因之一就是把矩阵和向量的运算与数字的运算混淆了。搞清楚 0 向量和 0 矩阵的特殊性，是学好线代的第一步。

根据我的《复习全书》，我帮你总结一下它们最重要的性质：

---

## 1. 0 向量 (零向量)

零向量 $\mathbf{0}$ (或 $O$) 是指所有分量都为 0 的向量。

- **线性相关性 (最重要的性质)：**
    
    - **一个 0 向量**组成的向量组 $\{\mathbf{0}\}$ 是**线性相关**的 1。
        
    - **任何包含 0 向量**的向量组（例如 $\{\alpha_1, \alpha_2, \mathbf{0}\}$）**必定是线性相关**的 2。
        
    - **原因：** 我们可以找到一组不全为 0 的系数（比如 $c_1=0, c_2=0, c_3=1$），使得 $0\alpha_1 + 0\alpha_2 + 1 \cdot \mathbf{0} = \mathbf{0}$ 成立，这完全符合线性相关的定义 3。
        
- **线性表示 (你刚问过)：**
    
    - 0 向量**可以被任意一个向量组** $\alpha_1, \dots, \alpha_s$ 线性表示（只要取系数全为 0 即可） 4。
        
- **线性方程组 (零解)：**
    
    - 齐次线性方程组 $AX = \mathbf{0}$ **一定有解**，就是**零解**（$X=\mathbf{0}$） 555555。
        
    - 如果 $AX = \mathbf{0}$ 只有零解，这恰好说明 $A$ 的列向量组是线性无关的 6666666666。
        
- **特征向量 (一定不是)：**
    
    - 根据定义，特征向量**必须是非零向量** 777777777。
        
    - 因此，0 向量**永远不能**作为特征向量 888888888。
        
- **内积与长度：**
    
    - 0 向量与**任何向量** $\alpha$ 的内积都为 0，即 $(\alpha, \mathbf{0}) = 0$ 99。
        
    - 0 向量的长度（范数）为 0，即 $\|\mathbf{0}\| [cite_start]= 0$ 1010。它是**唯一**长度为 0 的向量 11111111。
        

## 2. 0 矩阵 (零矩阵 $O$)

零矩阵 $O$ 是指所有元素都为 0 的矩阵。

- **矩阵乘法的“消去律” (最易错)：**
    
    - $AB = O$ **不能** 推出 $A=O$ 或 $B=O$ 12。
        
    - $AC = AD$ 且 $A \neq O$ **不能** 推出 $C=D$ 13。
        
    - 这是矩阵运算和数字运算的**根本区别**，你一定要牢牢记住。
        
- **矩阵的秩 (Rank)：**
    
    - 矩阵 $A$ 的秩 $r(A) = 0$ 的**充要条件**是 $A = O$（零矩阵）14。
        
    - 这是秩的性质中最基本的一条。
        
- **矩阵运算：**
    
    - $A + O = A$（加法单位元）
        
    - $AO = O$ (右乘)
        
    - $OA = O$ (左乘)
        
- **特征值：**
    
    - $n$ 阶零矩阵 $O$ 的特征值是 $n$ 个 0。
        
    - 它的特征多项式是 $|\lambda E - O| = |\lambda E| = \lambda^n$。

### 总结

1. **0 向量**：自身是**线性相关**的 15。任何包含 0 向量的组都线性相关 16。
    
2. **0 向量**：永远**不能**是特征向量 171717171717171717。
    
3. **0 向量**：是齐次方程组 $AX = \mathbf{0}$ 的**平凡解**（零解）18。
    
4. **0 矩阵**：是**唯一**秩为 0 的矩阵 19。
    
5. **0 矩阵**：$AB = O$ **不能**推出 $A=O$ 或 $B=O$ 20。这是矩阵乘法**没有消去律**的体现。
