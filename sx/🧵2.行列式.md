1. 主对角线为0的行列式：3 阶且主对角线为 0 的行列式，它的值就是“两条非零对角线”上元素的乘积之和。
2. 数字行列式（一阶矩阵行列式）的值就是数字本身
3. 矩阵为 0 可以推行列式为 0，但行列式为 0 推不出对应的矩阵为 0，不过如果是一阶矩阵行列式为 0 可以推出对应的矩阵为 0
4. 行列式的乘法定理： $|AB| = |A| |B|$ ，$A$ 和 $B$ 必须是 _同阶_ 的方阵
5. 带参数的行列式划 0 降阶会面临参数是否为 0 的讨论（行和相等可能还是会有点麻烦），所以如果是三阶的直接用对角线法则（三阶行列式有两个 0 就可以考虑用对角线法则）或者直接展开
6. 非 0 方阵行列式可能为 0
