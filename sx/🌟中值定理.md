

## 1. 罗尔定理 (Rolle's Theorem)

如果函数 $f(x)$ 满足以下条件：
1.  在闭区间 $[a, b]$ 上连续；
2.  在开区间 $(a, b)$ 内可导；
3.  在区间端点处的函数值相等，即 $f(a) = f(b)$。

那么，在 $(a, b)$ 内至少存在一点 $\xi$（$a < \xi < b$），使得该点的导数为零：
$$f'(\xi) = 0$$

（几何意义：曲线上至少有一点的切线平行于 x 轴。）

## 2. 拉格朗日中值定理 (Lagrange's Mean Value Theorem)

如果函数 $f(x)$ 满足以下条件：
1.  在闭区间 $[a, b]$ 上连续；
2.  在开区间 $(a, b)$ 内可导。

那么，在 $(a, b)$ 内至少存在一点 $\xi$（$a < \xi < b$），使得：
$$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$

或者写作：
$$f(b) - f(a) = f'(\xi)(b - a)$$

（几何意义：曲线上至少有一点的切线平行于连接端点 $(a, f(a))$ 和 $(b, f(b))$ 的割线。）

## 3. 柯西中值定理 (Cauchy's Mean Value Theorem)

如果函数 $f(x)$ 和 $g(x)$ 满足以下条件：
1.  在闭区间 $[a, b]$ 上连续；
2.  在开区间 $(a, b)$ 内可导；
3.  对任意 $x \in (a, b)$，$g'(x) \neq 0$。

那么，在 $(a, b)$ 内至少存在一点 $\xi$（$a < \xi < b$），使得：
$$\frac{f'(\xi)}{g'(\xi)} = \frac{f(b) - f(a)}{g(b) - g(a)}$$

（注：拉格朗日中值定理是柯西中值定理在 $g(x) = x$ 时的特殊情况。）

## 4. 积分中值定理 (Mean Value Theorem for Integrals)

如果函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，
那么，在 $[a, b]$ 上至少存在一点 $\xi$（$a \leq \xi \leq b$），使得：
$$\int_{a}^{b} f(x) \,dx = f(\xi)(b - a)$$

（几何意义：以 $f(\xi)$ 为高的矩形面积，等于曲线 $f(x)$ 在 $[a, b]$ 上与 x 轴围成的曲边梯形的面积。）
