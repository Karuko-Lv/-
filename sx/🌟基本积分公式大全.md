**所有不定积分的结果都需要加上一个任意常数 $C$。**

---
#### 1. 常数与幂函数 

- 零:$\int 0 \,dx = C$
- 常数 k: $\int k \,dx = kx + C$
- 1️⃣幂函数:$\int x^n \,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$
    $$e.g: \begin{cases}\int\frac{1}{x^{2}}dx=-\frac{1}{x}+C\\\int\frac{1}{\sqrt{x}}dx=2\sqrt{x}+C&\end{cases}$$
- 2️⃣特殊幂函数 $(n = -1)$:$\int x^{-1} \,dx = \int \frac{1}{x} \,dx = \ln|x| + C$
#### 2. 3️⃣指数函数
- 自然指数函数:$\int e^x \,dx = e^x + C$
- 一般指数函数:$\int a^x \,dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \neq 1)$
#### 3. 7️⃣对数函数（通常通过分部积分得到，但 $\int \ln x$ 很常见）
- 自然对数函数:$\int \ln x \,dx = x \ln x - x + C$
- 一般对数函数:$\int \log_a x \,dx = \frac{1}{\ln a} (x \ln x - x) + C = x \log_a x - \frac{x}{\ln a} + C$
#### 4. 4️⃣三角函数
- 基本正余弦: $\int \sin x \,dx = -\cos x + C$，$\int \cos x \,dx = \sin x + C$
- 正割平方 / 余割平方: $\int \sec^2 x \,dx = \tan x + C$，$\int \csc^2 x \,dx = -\cot x + C$
- 正割正切 / 余割余切: $\int \sec x \tan x \,dx = \sec x + C$, $\int \csc x \cot x \,dx = -\csc x + C$
- 正切 / 余切: $\int \tan x \,dx = -\ln|\cos x| + C = \ln|\sec x| + C$, $\int \cot x \,dx = \ln|\sin x| + C = -\ln|\csc x| + C$
- 正割 / 余割 (复杂): $\int \sec x \,dx = \ln|\sec x + \tan x| + C$,$\int \csc x \,dx = \ln|\csc x - \cot x| + C = -\ln|\csc x + \cot x| + C$
#### 5. 反三角函数（形式通常出现在被积函数中
这些公式是**由反三角函数的导数**反推得到的，非常重要：
- 6️⃣反正弦相关:
$$\int \frac{1}{\sqrt{1-x^2}} \,dx = \arcsin x + C \quad (= -\arccos x + C')$$
$$\int \frac{1}{\sqrt{a^2-x^2}} \,dx = \arcsin\left(\frac{x}{a}\right) + C \quad (a > 0)$$
- 5️⃣反正切相关:
$$\int \frac{1}{1+x^2} \,dx = \arctan x + C \quad (= -\text{arccot } x + C')$$
$$\int \frac{1}{a^2+x^2} \,dx = \frac{1}{a}\arctan\left(\frac{x}{a}\right) + C \quad (a > 0)$$
- 9️⃣反正割相关: $\int \frac{1}{x\sqrt{x^2-1}} \,dx = \text{arcsec } |x| + C \quad (= -\text{arccsc } |x| + C')$，$\int \frac{1}{x\sqrt{x^2-a^2}} \,dx = \frac{1}{a}\text{arcsec}\left|\frac{x}{a}\right| + C \quad (a > 0)$
#### 6. 8️⃣双曲函数 (Hyperbolic Functions)

- 基本双曲正余弦: $\sinh x = \frac{e^x - e^{-x}}{2}$，$\cosh x = \frac{e^x + e^{-x}}{2}$
$\int \sinh x \,dx = \cosh x + C$ ，$\int \cosh x \,dx = \sinh x + C$
PS：为什么叫“双曲”函数？==>与三角函数（也叫“圆函数”）相对应。
	- **圆函数：** $x = \cos t, y = \sin t$ 满足 $x^2 + y^2 = 1$（单位圆方程）。
	- **双曲函数：** $x = \cosh t, y = \sinh t$ 满足 $x^2 - y^2 = 1$（单位双曲线方程）

    - **分部积分法 (Integration by Parts)**
    - **三角代换 (Trigonometric Substitution)**
    - **部分分式分解 (Partial Fraction Decomposition)**
