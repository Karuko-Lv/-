好的，我们来举几个 $M(x, y)$ 和 $N(x, y)$ 是**同次数的齐次函数**的例子。

回顾定义：

一个函数 $f(x, y)$ 被称为 k 次齐次函数，如果对于任意 $t > 0$，都满足：

$$f(tx, ty) = t^k f(x, y)$$

其中 $k$ 是该齐次函数的次数 (degree)。

“$M(x, y)$ 和 $N(x, y)$ 是**同次数**的齐次函数”意味着它们都是齐次函数，并且它们的次数 $k$ 是**相同**的。这通常出现在形如 $M(x, y)dx + N(x, y)dy = 0$ 的一阶齐次微分方程中。

---

### 例子

#### 例 1：次数 k = 1

- $M(x, y) = x + y$
    
    - 检验: $M(tx, ty) = (tx) + (ty) = t(x+y) = t^1 M(x, y)$
        
    - 次数为 1。
        
- $N(x, y) = x - y$
    
    - 检验: $N(tx, ty) = (tx) - (ty) = t(x-y) = t^1 N(x, y)$
        
    - 次数为 1。
        
- **结论**: $M$ 和 $N$ 都是 1 次齐次函数，次数相同。
    
- **对应微分方程**: $(x+y)dx + (x-y)dy = 0$
    

#### 例 2：次数 k = 2

- $M(x, y) = x^2 + y^2$
    
    - 检验: $M(tx, ty) = (tx)^2 + (ty)^2 = t^2x^2 + t^2y^2 = t^2(x^2+y^2) = t^2 M(x, y)$
        
    - 次数为 2。
        
- $N(x, y) = xy$
    
    - 检验: $N(tx, ty) = (tx)(ty) = t^2(xy) = t^2 N(x, y)$
        
    - 次数为 2。
        
- **结论**: $M$ 和 $N$ 都是 2 次齐次函数，次数相同。
    
- **对应微分方程**: $(x^2+y^2)dx + (xy)dy = 0$
    

#### 例 3：次数 k = 0

- $M(x, y) = \frac{y}{x} + 3$
    
    - 检验: $M(tx, ty) = \frac{(ty)}{(tx)} + 3 = \frac{y}{x} + 3 = t^0 M(x, y)$
        
    - 次数为 0。
        
- $N(x, y) = \cos\left(\frac{y}{x}\right)$
    
    - 检验: $N(tx, ty) = \cos\left(\frac{ty}{tx}\right) = \cos\left(\frac{y}{x}\right) = t^0 N(x, y)$
        
    - 次数为 0。
        
- **结论**: $M$ 和 $N$ 都是 0 次齐次函数，次数相同。
    
- **对应微分方程**: $(\frac{y}{x} + 3)dx + \cos(\frac{y}{x})dy = 0$
    
    - (这种形式通常更容易看出 $\frac{dy}{dx} = F(\frac{y}{x})$ 的结构)
        

#### 例 4：次数 k = 3

- $M(x, y) = x^3 - 3xy^2$
    
    - 检验: $M(tx, ty) = (tx)^3 - 3(tx)(ty)^2 = t^3x^3 - 3t^3xy^2 = t^3(x^3 - 3xy^2) = t^3 M(x, y)$
        
    - 次数为 3。
        
- $N(x, y) = y^3 - 3x^2y$
    
    - 检验: $N(tx, ty) = (ty)^3 - 3(tx)^2(ty) = t^3y^3 - 3t^3x^2y = t^3(y^3 - 3x^2y) = t^3 N(x, y)$
        
    - 次数为 3。
        
- **结论**: $M$ 和 $N$ 都是 3 次齐次函数，次数相同。
    
- **对应微分方程**: $(x^3 - 3xy^2)dx + (y^3 - 3x^2y)dy = 0$
    

关键点：

判断一个多项式函数是否为 k 次齐次函数，只需要看它的每一项 $x^a y^b$ 的总次数 $a+b$ 是否都等于 k 即可。

这个“同次数齐次”的性质，正是允许我们使用 $y=ux$ 代换将微分方程 $Mdx+Ndy=0$ 转化为可分离变量方程的基础。
