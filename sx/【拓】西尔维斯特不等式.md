你问的“西尔维斯特不等式”（Sylvester's Inequality）是线性代数中关于**矩阵乘积的秩**的一个基本不等式。

它说明了矩阵 $\mathbf{A}$ 乘以矩阵 $\mathbf{B}$ 后，乘积 $\mathbf{A} \mathbf{B}$ 的秩**损失**了多少。

### 西尔维斯特不等式 (Sylvester's Inequality)

设 $\mathbf{A}$ 是 $m \times n$ 矩阵，$\mathbf{B}$ 是 $n \times p$ 矩阵。

**定理形式：**

$$r(\mathbf{A}) + r(\mathbf{B}) - n \le r(\mathbf{A} \mathbf{B}) \le \min \{r(\mathbf{A}), r(\mathbf{B})\}$$

- $r(\mathbf{A})$ 和 $r(\mathbf{B})$ 是矩阵 $\mathbf{A}$ 和 $\mathbf{B}$ 的秩。
    
- $r(\mathbf{A} \mathbf{B})$ 是矩阵乘积的秩。
    
- $n$ 是中间维数（即 $\mathbf{A}$ 的列数， $\mathbf{B}$ 的行数）。
    

---

### 1. 几何理解 (为什么秩会被“限制”)

这个不等式可以分为两个部分来理解：

#### A. 上界理解 ( $\mathbf{A} \mathbf{B}$ 的秩不超过 $\mathbf{A}$ 和 $\mathbf{B}$ 中较小的那个)

$$r(\mathbf{A} \mathbf{B}) \le \min \{r(\mathbf{A}), r(\mathbf{B})\}$$

这个上界非常好理解，因为它体现了“信息逐级过滤”的原则：

1. **$r(\mathbf{A} \mathbf{B}) \le r(\mathbf{A})$：** 矩阵 $\mathbf{A} \mathbf{B}$ 的列空间 $C(\mathbf{A} \mathbf{B})$ 总是 $\mathbf{A}$ 的列空间 $C(\mathbf{A})$ 的**子空间**。子空间的维度（秩）不可能大于父空间的维度。
    
2. **$r(\mathbf{A} \mathbf{B}) \le r(\mathbf{B})$：** 矩阵 $\mathbf{A} \mathbf{B}$ 的行空间 $R(\mathbf{A} \mathbf{B})$ 总是 $\mathbf{B}$ 的行空间 $R(\mathbf{B})$ 的**子空间**。
    

#### B. 下界理解 (西尔维斯特的核心)

$$r(\mathbf{A}) + r(\mathbf{B}) - n \le r(\mathbf{A} \mathbf{B})$$

这个下界说明了：**除非有必要，否则秩的损失不会太大。**

- **损失项：** $r(\mathbf{A})$ 的秩损失量 $\Delta r = r(\mathbf{A}) - r(\mathbf{A} \mathbf{B})$。
    
- 零空间的关系： 秩损失 $\Delta r$ 正好等于 $\mathbf{B}$ 的列空间 $C(\mathbf{B})$ 和 $\mathbf{A}$ 的零空间 $N(\mathbf{A})$ 的交集的维数。
    
    $$r(\mathbf{A}) - r(\mathbf{A} \mathbf{B}) = \dim(C(\mathbf{B}) \cap N(\mathbf{A}))$$
    

这个下界定理可以由上述关系推导出来，它保证了矩阵乘积的秩**至少**能保持多少。

### 2. 西尔维斯特不等式的一个重要推论

你之前问的那个定理 $r(\mathbf{A}) + r(\mathbf{B}) \le n$ (当 $\mathbf{A} \mathbf{B} = \mathbf{0}$ 时) 正是西尔维斯特不等式的一个**特殊情况**。

- **如果 $\mathbf{A} \mathbf{B} = \mathbf{0}$，** 那么 $r(\mathbf{A} \mathbf{B}) = 0$。
    
- **代入下界：** $r(\mathbf{A}) + r(\mathbf{B}) - n \le r(\mathbf{A} \mathbf{B})$
    
- **得到：** $r(\mathbf{A}) + r(\mathbf{B}) - n \le 0$
    
- **即：** $r(\mathbf{A}) + r(\mathbf{B}) \le n$
