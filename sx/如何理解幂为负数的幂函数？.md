
理解幂为负数的幂函数（例如 $y = x^{-2}$）的关键在于掌握一个核心法则，并由此推导出它在定义域、图像和性质上与正幂次函数的根本区别。

---

#### 一、核心法则：负指数等于正指数的倒数

这是理解一切的出发点。对于任何非零的底数 $x$ 和任何正数 $n$，下面的恒等式永远成立：

$$\Large x^{-n} = \frac{1}{x^n}$$

**简单来说，一个负次幂就是把它对应的正次幂“翻转”到分母的位置。**

**具体示例**:

* **$y = x^{-1}$** 就是  $y = \frac{1}{x^1} = \frac{1}{x}$  (这是我们熟悉的反比例函数)
* **$y = x^{-2}$** 就是  $y = \frac{1}{x^2}$
* **$y = x^{-3}$** 就是  $y = \frac{1}{x^3}$
* 对于负分数幂也一样：**$y = x^{-1/2}$** 就是  $y = \frac{1}{x^{1/2}} = \frac{1}{\sqrt{x}}$

---

#### 二、图像与性质的深刻变化

一旦我们理解了 $x^{-n} = \frac{1}{x^n}$，就能明白为什么负幂函数的图像和性质与正幂函数如此不同。因为 $x^n$ 出现在了**分母**上。

##### 1. 定义域的变化：$x$ 不能等于 0

* 对于正幂函数如 $y=x^2, y=x^3$，自变量 $x$ 可以取 0。
* 但对于负幂函数，由于分母不能为零（$x^n \neq 0$），所以 **$x$ 永远不能等于 0**。这是最根本的区别。
    * **定义域**: $\{x \in \mathbb{R} \mid x \neq 0\}$

##### 2. 渐近线的出现 (Asymptotes)

由于 $x$ 不能等于 0，我们就要关心在 $x$ 靠近 0 和远离 0 时，函数会发生什么。

* **垂直渐近线 (y-轴)**: 当 $x$ 无限趋近于 0 时，$x^n$ 也趋近于 0，因此 $\frac{1}{x^n}$ 的绝对值会趋向于**无穷大**。这就形成了以 **y-轴 ($x=0$)** 为垂直渐近线的图像特征。
* **水平渐近线 (x-轴)**: 当 $x$ 的绝对值无限增大时（$x \to \infty$ 或 $x \to -\infty$），$x^n$ 的绝对值也无限增大，因此 $\frac{1}{x^n}$ 会无限趋向于 **0**。这就形成了以 **x-轴 ($y=0$)** 为水平渐近线的图像特征。

##### 3. 奇偶性与单调性的新特点

* **奇偶性**: 和正幂函数一样，取决于幂指数的奇偶。
    * $y = x^{-1}, x^{-3}$ (奇次幂) 是奇函数，图像关于原点对称。
    * $y = x^{-2}, x^{-4}$ (偶次幂) 是偶函数，图像关于 y-轴对称。
* **单调性**: 由于在 $x=0$ 处断开，单调性需要分段讨论。
    * 例如 $y=x^{-1}=\frac{1}{x}$，它在 $(-\infty, 0)$ 上单调递减，在 $(0, +\infty)$ 上也单调递减。
    * 例如 $y=x^{-2}=\frac{1}{x^2}$，它在 $(-\infty, 0)$ 上单调递增，在 $(0, +\infty)$ 上单调递减。

#### 三、总结与对比

| 特性 (Property) | 正幂函数 ($y = x^n, n > 0$) | 负幂函数 ($y = x^{-n}, n > 0$) |
| :--- | :--- | :--- |
| **核心形式** | $x^n$ | $\frac{1}{x^n}$ |
| **定义域** | 通常是 $\mathbb{R}$ 或 $[0, +\infty)$，**包含 0** | **永远不包含 0**，即 $(-\infty, 0) \cup (0, +\infty)$ |
| **过定点** | 总是过点 $(1, 1)$，通常也过 $(0, 0)$ | 总是过点 $(1, 1)$，**永远不过** $(0, 0)$ |
| **渐近线** | 通常没有渐近线 | **总是有**：y-轴为垂直渐近线，x-轴为水平渐近线 |
| **当 $x \to 0$ 行为** | $y \to 0$ | $y \to \pm\infty$ |
| **当 $x \to +\infty$ 行为** | $y \to +\infty$ | $y \to 0$ |

**一句话总结**：将幂函数的指数变为负数，本质上就是对原有的正幂函数取了一次**倒数**，这导致函数图像不再经过原点，反而以 x 轴和 y 轴为渐近线，其性质也因此发生了根本性的改变。
