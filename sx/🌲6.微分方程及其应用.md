	1. **一阶线性**常微分方程 
		* <span style="color: gray;"> 一阶线性常微分方程（First-Order Linear ODE）严格来说只有一种标准形式。根据等号右边的项是否为 0 将其分为两种情况</span>
		* 先判断是线性的还是非线性的。看 $y$ 和 $y^{\prime}$ 是不是都是一次的，并且它们之间没有乘积项，如果是则是线性，其中线性如果右边为 0 可以变成可分离式的，也可以直接记住答案（更快）
		1. 一阶（齐次）线性常微分方程 **变成可分离形式**
			* 标准形式： $\frac{dy}{dx} + P(x)y = 0$ ==> $y=Ce^{-\int p(x)dx}$ / $y=Ce^{-\int_{x_{0}}^{x}p(t)dt}$ $C$ 为任意常数
		2. 3️⃣一阶（非齐次）线性常微分方程
			* 标准形式：$\frac{dy}{dx} + P(x)y = Q(x) \quad (\text{其中 } Q(x) \neq 0)$ 
				* ⚠️ $P(x)$ 和 $Q(x)$ 为已知的连续函数，确保 $\int p(x)dx$、$\int Q(x)e^{\int p(x)dx}dx$ 必有原函数
			    ==> $\begin{aligned}&\frac{dy}{dx}e^{\int p(x)dx}+P(x)ye^{\int p(x)dx}=Q(x)e^{\int p(x)dx}\\&(y\cdot e^{\int p(x)dx})^{\prime}=Q(x)e^{\int p(x)dx}\\&y\cdot e^{\int p(x)dx}=\int Q(x)e^{\int p(x)dx}dx+C\end{aligned}$ 
				==>原理是将混在一起的 $\frac{dy}{dx}$ 和 $y$ 通过乘以 $e^{\int p(x)dx}$ ,将方程左边转换成函数 $y\cdot e^{\int p(x)dx}$ 的导数形式
			* ⚠️解中 $\int p(x)dx$ 和 $\int Q(x)e^{\int p(x)dx}dx$ 均应理解成某一个**不含任意常数的原函数**，所以可以理解成**变限积分**【变限积分 $\int_{a}^{x} f(t) dt$ 本质上代表了一个特定的原函数。】，🌟所以他们也可以理解成 $\int p(x)dx=\int_{x_0}^xp(t)dt$ 和 $\int Q(x)e^{\int p(x)dx}dx=\int_{x_{0}}^{x}e^{\int_{x_{0}}^{t}p(s)ds}q(t)dt$（这里的第一个积分号的积分下限不一定要为 $x_0$，只是为了简化计算）；$\int_{x_0}^x P(t)dt$ 是 $P(x)$ 在 $x=x_0$ 处值为 0 的特定原函数。
			* 解法思想：找到一个函数 $\mu$，它的导数 $\mu'$ 等于它自己乘以 $P(x)$。 $e^{\int p(x)dx}$ 作为积分因子，是“构造辅助函数”思想的一个完美典范。他的作用是把方程的左边变成一个**可以被“合并”的完美形式**。
	2. **一阶**常微分方程
		1. 可分离变量微分方程 (Separable Equations) 
			* 既可以是线性的，也可以是非线性的。非线性是更常见的情况
			* 标准形式：
				1. $y^{\prime}=\frac{dy}{dx} = f (x) g (y)$ 
				* ==> $\begin{cases}\int\frac{dy}{g(y)}=\int f(x)dx+C\\g(y)\neq0&\end{cases}$ or $\begin{cases}y^{\prime}=\frac{dy}{dx} =0\\g(y)\equiv0&\end{cases}$ 
					* PS：由于考研大纲只要求求通解，不要求求出所有解，所以 $g(y)\neq0$ 一般可以不去考虑。（非线性方程全部解=通解+奇解；线性方程全部解=通解）
					* 如果给出初始条件无法满足第一种情况就得考虑第二种情况了，第二种情况只在初始条件给了 $y^{\prime}=0$ 的时候需要考虑到
				2. $M(x)dx + N(y)dy = 0$ 这个不用移项很容易
				3. 🌟 【换元记得回代】 $y^{\prime}=\frac{dy}{dx}=f(ax+by+c)$ ==>  $\begin{aligned}&u=ax+by+c\\&\frac{du}{dx}=a+\frac{dy}{dx}b\\&b\frac{dy}{dx}=bf(u)=\frac{du}{dx}-a\end{aligned}$ ==> $dx=\frac{du}{a+bf(u)}$ ( $dx$ 的积分函数是 1)
					1. 微分方程中出现以下表达式考虑换元
						1. $f(x\pm y)\Rightarrow t=x\pm y$
						2. $f(xy)\Rightarrow t=xy$
						3. $f(\frac{y}{x})\Rightarrow t=\frac{y}{x}$
						4. $f(x^{2}+y^{2})\Rightarrow t=x^{2}\pm y^{2}$
					2. 求导公式逆用来换元[[求导公式逆用来换元]]
						1. $f^{\prime}[g(x)]g^{\prime}(x)\Rightarrow[f(g(x)]]^{\prime}$
						2. $f^{\prime}(x)g(x)+f(x)g^{\prime}(x)=[f(x)g(x)]^{\prime}$
						* 这两个都只有全部为 $x$ 的式子可以考虑这个
					3. 用自变量、因变量或 $x$, $y$ 地位互换来换元
		2. 齐次型微分方程 (Homogeneous Equations) 
			* 既可以是线性的，也可以是非线性的。是可以通过 $u=y/x$ 代换求解的方程。
			* 标准形式：1️⃣【换元记得回代】 $y^{\prime}=\frac{dy}{dx} = F\left(\frac{y}{x}\right)$ ==> $\begin{aligned}u&=\frac{y}{x}=u(x)\\\frac{dy}{dx}&=u+\frac{du}{dx}x\\u&+\frac{du}{dx}x=F(u)\end{aligned}$ ==> $\begin{cases}\frac{du}{F(u)-u}=\frac{dx}{x}\\F(u)\neq u&\end{cases}$
						2️⃣ $M(x, y)dx + N(x, y)dy = 0$ （其中 $M(x, y)$ 和 $N(x, y)$ 是同次数的齐次函数 [[10.28 同次数的齐次函数]]）
						3️⃣【换元记得回代】 $\frac{2}{y^{\prime}}=\frac{dx}{dy} = F\left(\frac{x}{y}\right)$ ==> $\begin{aligned}u&=\frac{x}{y}=u(y)\\\frac{dx}{dy}&=u+\frac{du}{dy}y\\u&+\frac{du}{dy}y=F(u)\end{aligned}$ ==> $\begin{cases}\frac{du}{F(u)-u}=\frac{dy}{y}\\F(u)\neq u&\end{cases}$
			* **“齐次”的含义**：这里的“齐次”是指方程的结构只依赖于 $y$ 和 $x$ 的**比值**，或者说 $M$和 $N$ 在变量 $(x, y)$ 按比例 $t$ 缩放时，它们的函数值会按相同的 $t^k$ 比例缩放。
			* 和换元后可分离微分方程
				* 区别：齐次型微分方程的 $y^{\prime}$ 等于的函数不是关于 $x$ 和 $y$ 的函数，而是关于 $\frac{y}{x}$ 的函数 ; 最后结果 $dx$ 的积分函数是 $\frac{1}{x}$
				* 相同：令 $u$ 等于一个自变量 $x$ 与 $y$ 的函数，通过解出 $u$ 的函数再反解 $y$ 的函数，即都是使用代换 （这里 $u = \frac{y}{x}$ ,可分离换圆 $u=ax+by+c$），最后都转化为一个**可分离变量方程**。
		3. 伯努利方程 (Bernoulli Equations) **最接近线性方程的一种非线性方程**
			* 看到 $y^n$ 立刻想到是伯努利方程，用 $u=y^{1-n}$ 换元
			* 标准形式：【换元记得回代】 $\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (\text{其中 } n \neq 0 \text{ 且 } n \neq 1)$ ==> $\begin{aligned}z=y^{1-n}\therefore\frac{dz}{dx}\frac{1}{1-n}+p(x)z=q(x)\\\frac{dz}{dx}+(1-n)p(x)z=(1-n)q(x)\\令u=e^{\int(1-n)p(x)dx},(e^{\int(1-n)p(x)dx}\cdot z)^{\prime}=e^{\int(1-n)p(x)dx}q(x)\\\therefore e^{\int(1-n)p(x)dx}\cdot z=\int e^{\int(1-n)p(x)dx}q(x)dx+c\end{aligned}$ 
			* **一定非线性**：因为 $y^n$ 这一项的存在。几乎是一个一阶线性方程，只是在右边多乘了一个 $y$ 的幂。这个方程特殊在他**可以消除非线性项的影响**。（ $u = y^{1-n}$ 转化为一个一阶线性方程，然后再求解）,变量代换以降次/线性化
		4. 全微分方程（有实际几何意义）
			* 可以是线性的，也可以是非线性的。
			* 标准形式：$M(x, y)dx + N(x, y)dy = 0$（$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$）==> $\begin{aligned} F(x, y) = \int M(x, y) dx + g(y)\\\frac{\partial F}{\partial y} = \frac{\partial}{\partial y} \left( \int M(x, y) dx \right) + g'(y)= N(x, y)\\g'(y) = N(x, y) - \frac{\partial}{\partial y} \left( \int M(x, y) dx \right)\\g(y) = \int g'(y) dy\\F(x, y) = C\end{aligned}$ 
				PS：$g(y)$ 是积分常数。但因为它是在对 $x$ 积分，所以这个“常数”实际上可以是一个**只与 $y$ 有关的未知函数**。
			* 全微分定义： 让自变量 $x$ 发生一个微小的变化 $dx$，让发生一个微小的变化 $dy$，函数值 $F$ 会相应地发生一个总的微小变化，这个总变化近似地等于：$dF = \underbrace{\frac{\partial F}{\partial x}}_{\text{x方向斜率}} dx + \underbrace{\frac{\partial F}{\partial y}}_{\text{y方向斜率}} dy$，$dF$ 就叫做函数 $F(x, y)$ 的**全微分**。
			* 全微分方程定义：若函数 $M(x, y)$，$N(x, y)$ 在单连通区域 D 上具有一阶连续偏导数，且在 D 内满足 $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$，则 $Mdx + Ndy$ 是某二元函数 $F(x, y)$ 的全微分，则 $M(x, y) = \frac{\partial F}{\partial x}, N(x, y) = \frac{\partial F}{\partial y}$ 且 $dF = 0$。PS: $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$ 二阶混合偏导数的对称性（假设 $M, N$ 及其偏导数连续）。若 $M = \frac{\partial F}{\partial x}$ 且 $N = \frac{\partial F}{\partial y}$，则 $\frac{\partial M}{\partial y} = \frac{\partial}{\partial y} \left( \frac{\partial F}{\partial x} \right) = \frac{\partial^2 F}{\partial y \partial x}$，$\frac{\partial N}{\partial x} = \frac{\partial}{\partial x} \left( \frac{\partial F}{\partial y} \right) = \frac{\partial^2 F}{\partial x \partial y}$，根据混合偏导数相等的定理，$\frac{\partial^2 F}{\partial y \partial x} = \frac{\partial^2 F}{\partial x \partial y}$，必须有 $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$。
			* 几何意义：$dF = 0$ 意味着什么？==> 意味着当 $x$ 和沿着解曲线微小变化 $dx, dy$ 时，函数 $F(x, y)$ 的值**没有发生变化** ($dF=0$)。这说明什么？说明微分方程的**解曲线** $y(x)$ 正好就是**势函数 $F(x, y)$ 的等值线（或水平曲线 / Contour Lines）**,全微分方程 $Mdx + Ndy = 0$ 本质上是某个势函数 $F(x, y)$ 的全微分 $dF=0$
	4. **二阶非线性**常微分方程（还有很多复杂的，这里列出的 3个本质上也是一阶常微分方程）
		1. 二阶**可降阶**微分方程 (最高阶前面系数是常数，不同时含有 $x、y、y^{\prime}、y^{\prime\prime}$)
			1. 不显含未知函数 $y$ 的方程：
				* 标准形式：【换元记得回代】 $y^{\prime\prime}=f(x,y^{\prime})$ 型==> $\begin{aligned}&令p=y^{\prime}\therefore p^{\prime}=y^{\prime\prime}\\&\therefore p^{\prime}=f(x,p)\Rightarrow p=\varphi(x,C_{1})\\&\Rightarrow y=\int(\varphi(x,C_{1}))dx+C\end{aligned}$
				* 思想：换元从 $x、y^{\prime}、y^{\prime\prime}$ 的方程变成 $x、p、p^{\prime}$ 的一阶微分方程（$p=p(x)$）（具体是什么类型得具体题目才可判断）
			2. 不显含自变量 $x$ 的方程：
				* 标准形式：【换元记得回代】 $y^{\prime\prime}=f(y,y^{\prime})$ 型==> $\begin{aligned}&令p=y^{\prime}\therefore p^{\prime}=p^{\prime}(x)=y^{\prime\prime}\\&y^{\prime\prime}=\frac{dp}{dx}=\frac{dp}{dy}·\frac{dy}{dx}=p\frac{dp}{dy}\therefore p\frac{dp}{dy}=f(y,p)\Rightarrow p=\varphi(y,C_{1})\\&\Rightarrow \int\frac{dy}{\varphi(y,C_{1})}=x+C\end{aligned}$
				* 如果 $y、y^{\prime}、y^{\prime\prime}$ 是线性的，优先考虑用二阶齐次常系数线性微分方程
				* 原则：（把 $p$ 看成 $p (x)$，明确 $p^{\prime}=p^{\prime}(x)$）解这类题关键是**多一步写出 $y^{\prime\prime}$ 与 $\frac{dp}{dy}$ 的关系**
				* 错误解决方案：沿用对不显含未知函数的方程的处理，若令 $y^{\prime\prime}=\frac{dp}{dx}$ ，原方程变成 $\frac{dp}{dx}=f(y,p)$ 会变成关于 $x,y,p$ 的方程，增强方程复杂性
				* 思想：换元从 $y、y^{\prime}、y^{\prime\prime}$ 的方程变成 $y、p、p^{\prime}$ 的一阶微分方程（$p=p(y)$）（具体是什么类型得具体题目才可判断）
			3. 既不显含 y 也不显含 x，当成 1 处理
				 * 标准形式：【换元记得回代】 $y^{\prime\prime}=f(y^{\prime})$ 型==> $\begin{aligned}令y^{\prime}&=p\therefore y^{\prime\prime}=p^{\prime}\\p^{\prime}&=f(p)\Rightarrow p=\varphi(x,C_1)\\\therefore y&=\int\varphi(x,C_1)dx+C\end{aligned}$
			4. $f(x、y、y^{\prime}、y^{\prime\prime})=0$
				* 有时如果含有二阶导数且微分方程不是线性的，可以考虑利用构造成全微分形式==>利用全微分法进行高阶方程降阶是常用的方法之一。 $e.g$ 🐙强化 15 讲的例 15.7
	5. **二阶线性**常微分方程（线性：所以与现代的解的结构一样）
		* $a_2(x)y'' + a_1(x)y' + a_0(x)y = f(x)$ ，可以将其标准化为 $y'' + P(x)y' + Q(x)y = R(x)$
		* "线性"意味着 $y, y', y''$ 都是一次的，并且它们之间没有乘积项，$\therefore$ 线性组合
		* $y, y', y''$ 的系数 ($P(x), Q(x)$) 只依赖于自变量 $x$。主要的分类是根据**系数是否为常数**以及**等号右边的是否为零**。
		* 微分方程解的存在唯一性定理：对于一个 $n$ 阶常微分方程（在“良好”条件下），你需要**恰好 $n$ 个初始条件**（通常是 $y(x_0), y'(x_0), \dots, y^{(n-1)}(x_0)$）才能**唯一地**确定一个满足这些条件的解。
			* 二阶线性方程：方程是**线性**的，并且是**二阶**的，微分方程理论指出它的**所有解**构成一个**向量空间**（函数空间）。这个向量空间的**维度 (Dimension)** 恰好**等于方程的阶数**。$\therefore$ 二阶线性齐次方程，解空间是一个**二维**向量空间。
			* **线性代数**的基本原理：一个 $n$ 维的向量空间，需要**恰好 $n$ 个线性无关**的向量才能构成它的一组**基 (Basis)**。
		1. **二阶常系数线性**常微分方程（必考🌟）
			* 标准形式： $ay'' + by' + cy = f(x)$ ，其中 $a, b, c$ 是常数，且 $a \neq 0$。即 $P(x)$ 和 $Q(x)$ 都是常数
			* 求解思想：找到一种函数 $y(x)$，它的导数 $y'$ 和二阶导数 $y''$ 与 $y(x)$ 本身**形式非常相似**，最好只是相差一个**常数因子**。~={red}为什么？=~因为这样代入方程 $ay'' + by' + cy = 0$ 后，才有可能通过系数 $a, b, c$ 的组合**恰好抵消为零**。
			1. 二阶常系数**齐次**线性常微分方程 
				1. 标准形式：$ay'' + by' + cy = 0$ ($f(x) = 0$) ==> $\begin{aligned}&ar^{2}+br+c=0\\&\Delta=b^{2}-4ac\end{aligned}$ ==> $\begin{cases}\Delta>0、r_{1}\neq r_{2}、y(x)=C_{1}e^{r_{1}x}+C_{2}e^{r_{2}x}\\\Delta=0、r_{1}=r_{2}=r、y(x)=(C_{1}+C_{2}x)e^{rx}\\\Delta<0、r=a\pm i\beta、 y(x) = e^{\alpha x}(C_1\cos(\beta x) + C_2\sin(\beta x))&\end{cases}$
				2. **“齐次”的含义**：这里的“齐次”**仅仅**意味着等号**右边等于零**，即等号右边自由项为 0==>保证**叠加原理**成立，作为解非齐次的基础
				3. 可以把二阶线性齐次微分方程的**所有解**构成的集合想象成一个**二维向量空间 (Vector Space)**。
					- **解 = 向量**：每一个特定的解（比如 $y=e^{2x}$ 或 $y=e^{3x}$）就像是这个空间中的一个向量。
					- **线性组合 = 向量加法和数乘**：$C_1y_1 + C_2y_2$ 就是向量的线性组合。
					- **线性无关的解 = 基向量 (Basis Vectors)**：$y_1$ 和 $y_2$ 如果线性无关，它们就扮演了**基向量**的角色（就像二维平面中的 $\hat{i}=(1,0)$ 和 $\hat{j}=(0,1)$）。
					- **通解 = 所有向量的表示**：$y(x) = C_1y_1(x) + C_2y_2(x)$ 这个公式，就像是用基向量表示二维空间中**任意**一个向量 $\vec{v} = C_1\hat{i} + C_2\hat{j}$。

			2. 二阶常系数**非齐次**线性常微分方程 
				1. 标准形式：$ay'' + by' + cy = f(x) \quad (f(x) \neq 0)$ 自由项 $f (x)$ 不为 0，为已知的连续函数
					* 通解 $y(x) = y_h(x) + y_p(x)$

### 题目一 (对应策略 1)

> 设曲线 $y = f(x)$ ($x > 0$) 经过点 $P(1, 2)$。已知该曲线上任意一点 $(x, y)$ 处的切线，其**在 $y$ 轴上的截距** 恰好等于该点**横坐标的 3 倍**。求函数 $f(x)$ 的表达式。

#### 为什么对应策略 1？

此题完美符合策略 (1)：“信息点A、B为...导数...表达式且为等量关系时，令A=B，建立方程。”

- **信息点 A (导数表达式)：** 曲线上任意点 $(x, y)$ 处的切线在 $y$ 轴上的截距。
    
- **信息点 B (普通表达式)：** 该点横坐标的 3 倍，即 $3x$。
    
- **等量关系：** $A = B$。
    
- **建立方程：** 我们需要用 $y$ 和 $y'$ 来表示“信息点A”，然后令它等于 $B$，从而得到一个关于 $y$ 和 $y'$ 的微分方程。
    

#### 解题思路

1. **求信息点 A：**
    
    - 曲线上任意点 $(x, y)$ 的切线斜率为 $k = f'(x) = y'$。
        
    - 切线方程为：$Y - y = y'(X - x)$。
        
    - 令 $X=0$，求 $y$ 轴截距 $Y_0$：$Y_0 - y = y'(0 - x) \implies Y_0 = y - xy'$。
        
    - 所以，信息点 A 的表达式为 $y - xy'$。
        
2. **令 $A = B$，建立方程：**
    
    - 根据题意 $A = B$，即：
        
        $$ y - xy' = 3x$$
        
    - 整理得：$xy' - y = -3x$。
        
    - $y' - \frac{1}{x} y = -3$。
        
3. **解微分方程：**
    
    - 这是一个一阶线性微分方程。使用积分因子法，积分因子 $\mu(x) = e^{\int -1/x dx} = e^{-\ln x} = \frac{1}{x}$。
        
    - 方程两边同乘 $\frac{1}{x}$：
        
        $$ \frac{1}{x} y' - \frac{1}{x^2} y = -3 \cdot \frac{1}{x}$$
        
    - 左边正好是 $(\frac{1}{x} y)'$：
        
        $$ \left( \frac{1}{x} y \right)' = -\frac{3}{x}$$
        
    - 两边积分：$\frac{1}{x} y = \int -\frac{3}{x} dx = -3 \ln x + C$。
        
    - 通解为 $y = -3x \ln x + Cx$。
        
4. **代入初始条件：**
    
    - 曲线过点 $(1, 2)$，代入 $x=1, y=2$：
        
    - $2 = -3(1) \ln(1) + C(1) \implies 2 = 0 + C \implies C = 2$。
        
5. **最终解：**
    
    $$f(x) = -3x \ln x + 2x$$

### 题目二 (对应策略 2)

> 设函数 $f(x), g(x)$ 在 $x \in \mathbb{R}$ 上可导，且满足如下关系组：
> 
> $$\begin{cases} f'(x) - g(x) = e^x \\ g'(x) - f(x) = -e^x \end{cases}$$
> 
> 若 $f(0) = 2, g(0) = 1$，求 $f(x)$ 和 $g(x)$ 的表达式。

#### 为什么对应策略 2？

此题完美符合策略 (2)：“信息点为 $f(x), g(x)$ 及 $f'(x), g'(x)$ 的等量关系组，用求导、消元建方程。”

- **等量关系组：** 题目给定了两个方程，混合了 $f, g, f', g'$。
    
- **消元建方程：** 我们的目标是消去一个函数（例如 $g(x)$），来得到一个只关于 $f(x)$ 的高阶微分方程。
    

#### 解题思路

1. 观察方程组：
    
    (1) $f'(x) - g(x) = e^x$
    
    (2) $g'(x) - f(x) = -e^x$
    
2. **求导、消元：**
    
    - 我们对 **方程 (1) 两边同时求导**：
        
        $$ (f'(x) - g(x))' = (e^x)' \implies f''(x) - g'(x) = e^x$$
        
    - 我们得到了一个新的方程 (3)：$f''(x) - g'(x) = e^x$。
        
3. **建立 $f(x)$ 的方程：**
    
    - 我们现在的目标是消去 $g(x)$ 和 $g'(x)$。
        
    - 从 (1) 中，我们得到：$g(x) = f'(x) - e^x$。
        
    - 从 (2) 中，我们得到：$g'(x) = f(x) - e^x$。
        
    - 将 $g'(x) = f(x) - e^x$ **代入** 新方程 (3)：
        
        $$ f''(x) - (f(x) - e^x) = e^x$$
        
    - $f''(x) - f(x) + e^x = e^x$
        
    - $f''(x) - f(x) = 0$
        
4. **解 $f(x)$ 的微分方程：**
    
    - 这是一个二阶常系数齐次线性微分方程。
        
    - 特征方程：$r^2 - 1 = 0 \implies r_1 = 1, r_2 = -1$。
        
    - $f(x)$ 的通解为：$f(x) = C_1 e^x + C_2 e^{-x}$。
        
5. **求 $g(x)$ 的表达式：**
    
    - 利用 $g(x) = f'(x) - e^x$。
        
    - $f'(x) = C_1 e^x - C_2 e^{-x}$
        
    - $g(x) = (C_1 e^x - C_2 e^{-x}) - e^x = (C_1 - 1)e^x - C_2 e^{-x}$。
        
6. **代入初始条件：**
    
    - 已知 $f(0) = 2, g(0) = 1$。
        
    - $f(0) = C_1 e^0 + C_2 e^0 = C_1 + C_2 = 2$
        
    - $g(0) = (C_1 - 1)e^0 - C_2 e^0 = C_1 - 1 - C_2 = 1 \implies C_1 - C_2 = 2$
        
    - 解方程组：
        
        $$ \begin{cases} C_1 + C_2 = 2 \\ C_1 - C_2 = 2 \end{cases}$$
        
    - 两式相加：$2C_1 = 4 \implies C_1 = 2$。
        
    - 代入 $C_1$：$2 + C_2 = 2 \implies C_2 = 0$。
        
7. **最终解：**
    
    - $C_1 = 2, C_2 = 0$
        
    - $f(x) = 2e^x + 0e^{-x} = 2e^x$
        
    - $g(x) = (2-1)e^x - 0e^{-x} = e^x$
