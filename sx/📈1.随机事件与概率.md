1. **运算定律 (考点)：**
    - 交换律：$A \cup B = B \cup A$，$A \cap B = B \cap A$。
    - 结合律：$A \cup (B \cup C) = (A \cup B) \cup C$，$A \cap (B \cap C) = (A \cap B) \cap C$。
    - 分配律：$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$，$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$。
    - 对偶律 (德·摩根定律) **(考点)**：
	    - $\overline{A \cup B} = \overline{A} \cap \overline{B}$ (和的非 = 非的积) 
	    - $\overline{A \cap B} = \overline{A} \cup \overline{B}$ (积的非 = 非的和)
	⚠️同时有 $∪$ 和 $∩$ 是没有结合律，结合律是在运算都是 $∪$ 或者都是 $∩$ 才可以用结合律，同时有 $∪$ 和 $∩$ 则只可以用分配律
2. **特殊关系推导**之纯集合 (小众点)：
	- 如果两个本该互不相容（没有任何交集）的集合相等了，那么它们只能同时为空集（$\emptyset$）。
	1. 由 $A \cap B = \overline{A} \cap \overline{B}$ 可推导出 $A \cap B = \overline{A \cup B}$，则 $A \cup B = \Omega$ ，$A \cap B = \emptyset$ ，则 $A$ 和 $B$ 把样本空间 $\Omega$ 瓜分了（互斥且完备）。==>利用集合的互斥性（Disjoint）来判断事件的特殊状态。
	2. “交集 = 差集”型 (导出空集)：由 $A \cap B = A - B$ (即 $A \cap B = A \cap \overline{B}$)，一个元素不可能既“属于B”又“不属于B”，这两个集合天然互斥，则可推导出 $A \cap B = \emptyset$ 且 $A \cap \overline{B} = \emptyset$；$A = (A \cap B) \cup (A \cap \overline{B})$，两部分都是空，所以 **$A = \emptyset$**（A是不可能事件）。
	3. “并集 = 交集”型 (导出相等)：由 $A \cup B = A \cap B$，一般情况下，交集是并集的子集 ($A \cap B \subseteq A \cup B$)，如果并集没有比交集多出任何东西（即 $A$ 独有的部分 $A-B$ 和 $B$ 独有的部分 $B-A$ 都是空的），则 $A = B$。
	4. “对称差 = 空集”型 (导出相等)：由 $(A - B) \cup (B - A) = \emptyset$，$A - B$ 是“A有B没有”，$B - A$ 是“B有A没有”，两者的并集为空，说明“A有B没有”的情况不存在，且“B有A没有”的情况也不存在，则 $A = B$
	5. “吞噬”关系型 (导出包含) ：
		1. 并集等于大：$A \cup B = A$，则$B \subseteq A$ （B被A吞掉了，说明B在A里面）。
		2. 交集等于小：$A \cap B = A$，则 $A \subseteq B$（A即是公共部分，说明A全在B里面）。
	6. 全空全全：
		1. 若 $A \cup B = \emptyset$，则 $A = \emptyset$ 且 $B = \emptyset$（并集只有全是空才能是空）。
		2. 若 $A \cap B = \Omega$，则 $A = \Omega$ 且 $B = \Omega$（交集只有全是全集才能是全集）。

3. **任意**两个事件 $A$ 和 $B$ （不管它们是什么关系），**永远成立**的概率等式：
	1. 补事件公式： $P(\overline{A}) = 1 - P(A)$
	2. 加法公式：$P(A \cup B) = P(A) + P(B) - P(AB)$
	3. **加法公式 (3个事件)：** $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$。
	4. 减法公式（差事件公式）：$P(A-B) = P(A \overline{B}) = P(A) - P(AB)$。
	5. 全概率公式： $P(B) = P(AB) + P(\overline{A}B)$
	6. 对偶律（德·摩根定律）（事件关系 $\overline{A \cup B} = \overline{A}\overline{B}$ 和 $\overline{AB} = \overline{A} \cup \overline{B}$  在概率上的体现）：$P(\overline{A \cup B}) = P(\overline{A} \overline{B})$ , $P(\overline{AB}) = P(\overline{A} \cup \overline{B})$
4. **概率等式的重要推论 (考点)：**
    - 若 $B \subset A$，则 $P(B) \le P(A)$，且 $P(A-B) = P(A) - P(B)$。
    - "A或B发生，C必不发生" 意味着 $(A \cup B) \subset \overline{C}$，因此 $P(A \cup B) \le P(\overline{C})$，即 $P(A) + P(B) - P(AB) \le 1 - P(C)$。(见例题4)
    - 若 $A, B$ 互不相容 ($P(AB)=0$) 且 $P(A)=1$，则 $P(B)=0$。(见例题6)
5. **特殊关系推导**之概率数值与事件关系的推导
	1. $P(A \cup B) = P(A) + P(B)$
		**推导结论：** $P(AB) = 0$。注意，这**不代表** $A$ 与 $B$ 互不相容（互斥），只能说 $AB$ 是零概率事件（在连续型随机变量中常见，比如点落在某条线上）。但在离散或基础题目中，常近似理解为互斥。
	2. $P(A|B) = 1$ （且 $P(B)>0$）
		**推导结论：** $P(AB) = P(B)$，即 $B$ 发生的概率全部贡献给了交集。这通常意味着在概率测度上 $B \subseteq A$（即 $B-A$ 是零概率事件）。
6. 互斥 vs 对立 vs 独立：
	1. 事件 $A, B$：
		1. 互斥： $P(A \cup B) = P(A) + P(B)$
		2. 对立：$P(A \cup B ) = P(A) + P(B) = P(\Omega) = 1$
		3. 独立：$P(AB) = P(A)P(B)$
	2. 事件 $A, B, C$ 
		1. 互斥： $P(A \cup B \cup C) = P(A) + P(B) + P(C)$
		2. 对立：$P(A \cup B \cup C) = P(A) + P(B) + P(C) = P(\Omega) = 1$
		3. 独立：
			1. 两两独立 $P(AB) = P(A)P(B)、P(AC) = P(A)P(C)、P(BC) = P(B)P(C)$ 
			2. 相互独立再加一个 $P(ABC) = P(A)P(B)P(C)$
	3. 概率为 1 的随机事件与任何时间均独立【背✅】
7. 事件的~={orange}**包含=~关系** $A \subset B$ （或 $A \subseteq B$）即事件 $A$ 是事件 $B$ 的一个子集（如果事件 A 发生，那么事件 B _必然_ 发生）的概率关系：
	1. $P(A) \le P(B)$：$A$ 发生的概率不会超过 $B$ 发生的概率。
	2. $P(B - A) = P(B) - P(A)$：$B = A \cup (B - A)$。又因为 $A$ 和 $(B-A)$ 是互不相容的，所以 $P(B) = P(A) + P(B - A)$。
8. **概率不等式 (小众点)：**$P(A \cup B)P(AB) \le P(A)P(B)$。
9. **概率基本性质：**
    - 非负性：$P(A) \ge 0$。
    - 规范性：$P(\Omega) = 1$。
    - 可加性：若 $A, B$ 互不相容 ($AB = \emptyset$)，则 $P(A \cup B) = P(A) + P(B)$。
10. **条件概率定义 (考点)：** 在 $A$ 发生的条件下 $B$ 发生的概率：$P(B|A) = \frac{P(AB)}{P(A)}$，其中 $P(A) > 0$
11. **条件概率的性质 (考点)：**
    - 条件概率是概率，满足概率的所有性质。例如：
        
    - $P(\overline{B}|A) = 1 - P(B|A)$。
        
    - $P[(B \cup C)|A] = P(B|A) + P(C|A) - P(BC|A)$。
        
    - $P[(A-B)|C] = P(A|C) - P(AB|C)$。
        
12. **重要推论 (考点)：**
    
    - 若 $A \subset B$，则 $P(B|A) = \frac{P(AB)}{P(A)} = \frac{P(A)}{P(A)} = 1$。
        
    - 若 $A \subset B$，则 $P(A|B) = \frac{P(AB)}{P(B)} = \frac{P(A)}{P(B)}$。
        
13. **条件概率的特殊情况 (小众点)：**
    
    - 若 $P(B|A)=1$，则 $P(AB)=P(A)$。这推出 $P(A\overline{B})=0$，即 $P(\overline{B}|A)=0$。**注意：这并不意味着** $A \subset B$。(见例题10)
        
    - $P[(A \cup B)|C] = P(A|C) + P(B|C)$ 成立的充要条件是 $P(AB|C)=0$，即 $P(ABC)=0$。(见例题12)
        
    - $P(AB) = P(ABC)$ 意味着 $P(AB\overline{C})=0$，这推出 $P(AB|\overline{C})=0$。 (见例题13)
        
14. **乘法公式 (考点)：**
    
    - $P(AB) = P(A)P(B|A) = P(B)P(A|B)$。
        
    - 推广：$P(ABC) = P(A)P(B|A)P(C|AB)$。
        
15. **全概率公式 (考点)：**
    
    - 若 $B_1, B_2, ..., B_n$ 是一个完备事件组（互不相容且并集为 $\Omega$），则 $P(A) = \sum_{i=1}^{n} P(B_i)P(A|B_i)$。
        
    - (讲义例题21和22隐式地使用了这个公式)
        
16. **贝叶斯公式 (逆概公式) (考点)：**
    
    - $P(B_i|A) = \frac{P(B_i A)}{P(A)} = \frac{P(B_i)P(A|B_i)}{\sum_{j=1}^{n} P(B_j)P(A|B_j)}$。
        
17. **条件概率与独立性的关系 (小众点)：**
    
    - $P(A|B) > P(A|\overline{B})$ 等价于 $P(AB) > P(A)P(B)$。(见例题18)
        

**题型四：三大概型**

18. **古典概型 (考点)：**
    
    - 特点：(1) 样本空间有限（有限性）；(2) 每个基本事件等可能（等可能性）。
        
    - 公式：$P(A) = \frac{A \text{ 包含的基本事件数}}{ \text{样本空间基本事件总数}}$。
        
    - **跨学科考点：** 结合线性代数（如例题19，判别 $Ax=b$ 有解）或排列组合（如例题20）。
        
19. **几何概型 (考点)：**
    
    - 特点：(1) 样本空间无限（无限性）；(2) 样本点落在某区域的概率与该区域的“测度”（长度、面积、体积）成正比（等可能性）。
        
    - **考点：** 常常需要计算二维面积，有时涉及积分（如例题23中计算 $XY < \frac{3}{64}$ 的面积）。
        
20. **伯努利概型 (n重独立重复试验) (考点)：**
    
    - 特点：(1) 试验独立重复 $n$ 次；(2) 每次试验只有两个结果 $A$ 和 $\overline{A}$；(3) 每次试验 $P(A)=p$ 保持不变。
        
    - 公式：$n$ 次试验中 $A$ 发生 $k$ 次的概率为 $P_n(k) = C_n^k p^k (1-p)^{n-k}$。
        
    - **考点：** 求最值问题（如例题25），将"甲乙命中次数相等"的概率表示为 $p$ 的二次函数，求其最大值。
        
21. **抽样问题 (考点)：**
    
    - **放回抽样 (伯努利)：** $P(\text{乙取到}) = P(\text{甲取到})$。(见例题22)
        
    - **不放回抽样：** $P(\text{乙取到}) = P(\text{甲取到})$。(见例题22) 这是一个重要结论：在不放回抽样中，只考虑第 $k$ 次抽样，任何一个球被抽到的概率是相同的。
        
22. **“A在B前发生”问题 (小众点)：**
    
    - 在 $A, B, C$ 三种结果的独立重复试验中，A在B前发生的概率，可以忽略C。$P(D) = P(A_1 \cup C_1 A_2 \cup C_1 C_2 A_3 ...) = p + rp + r^2p + ... = \frac{p}{1-r} = \frac{p}{p+q}$。(见例题24)
        

**题型五：随机事件的独立性**

23. **独立性定义 (考点)：**
    
    - **两个事件：** $A, B$ 相互独立 $\iff P(AB) = P(A)P(B)$。
        
    - **三个事件 (两两独立)：** $P(AB) = P(A)P(B)$，$P(AC) = P(A)P(C)$，$P(BC) = P(B)P(C)$。
        
    - **三个事件 (相互独立)：** 在两两独立的基础上，还需满足 $P(ABC) = P(A)P(B)P(C)$。
        
    - **重要辨析 (考点)：** 相互独立 $\implies$ 两两独立；两两独立 $\not\implies$ 相互独立。(见例题33，抛硬币 $A_1, A_2, A_3$ 的经典例子)
        
24. **独立性的等价条件 (考点)：**
    
    - 若 $P(B) > 0$，$A, B$ 独立 $\iff P(A|B) = P(A)$。
        
    - $A, B$ 独立 $\iff P(A|B) = P(A|\overline{B})$。 (见例题26)
        
    - **(小众点)** $P(A|B) + P(\overline{A}|\overline{B}) = 1 \implies A, B$ 独立。(见例题27)
        
25. **独立性的传递性 (考点)：**
    
    - 若 $A, B$ 独立，则 $\overline{A}$ 与 $B$，$A$ 与 $\overline{B}$，$\overline{A}$ 与 $\overline{B}$ 也都相互独立。
        
26. **独立性的极端情况 (考点)：**
    
    - 若 $P(A)=0$ 或 $P(A)=1$，则 $A$ 与任意事件 $B$ 都相互独立。
        
    - **(小众点)** 若 $A \subset B$，$A, B$ 独立 $\iff P(A)=0$ 或 $P(B)=1$。
        
    - **(考点)** 若 $A, B$ 互不相容 ($P(AB)=0$) 且 $P(A)>0, P(B)>0$，则 $A, B$ 必不独立（相依）。
        
27. **复合事件的独立性 (考点)：**
    
    - 若 $A, C$ 独立，$B, C$ 独立，则 $(A \cup B)$ 与 $C$ 独立的充要条件是 $AB$ 与 $C$ 独立。(见例题32)
