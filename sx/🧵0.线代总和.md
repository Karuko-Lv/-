4. “线性方程组”等价于“向量方程”与“矩阵方程”<==>“向量的线性组合”等价于“线性方程组”
5. 矩阵🆚向量
	1. 向量是“对象”，而矩阵是作用在对象上的“动作”
	2. 计算机编程（如 NumPy 库）中，向量很多时候直接用一维数组表示，而矩阵用二维数组
	3. 矩阵看作是由一堆向量“拼”起来的。比如一个 $3 \times 3$ 的矩阵，你可以把它看成是由 3 个列向量并排站在一起，或者 3 个行向量上下叠在一起组成的
	4. 向量代表“空间中的一个点”或“一个带方向的箭头”；矩阵代表一种“空间的运动或变换法则”
	5. 矩阵和向量相乘 $Ax = y$：
		1. **$A$（矩阵）** 是一台加工机器或变换指令（比如：“逆时针旋转 90 度，并长度翻倍”）
		2. **$x$（向量）** 是被操作的原材料对象，**$y$（新向量）** 是变换后的最终产品
