1. **方程组的三种形式**：
	1. 矩阵形式：$AX = \beta$（非齐次）或 $AX = 0$（齐次）。
		矩阵形式的解可直接写成列向量形式： $n$ 维向量 $\xi$ 是 $AX = \beta$ 解 $\Leftrightarrow$ $A\xi = \beta$ ; $n$ 维向量 $\eta$ 是 $AX = 0$ 的解 $\Leftrightarrow$ $A\eta = 0$.例如 $\eta = (1, -2, 1, 0)^T$
	2. 向量形式：（$\alpha_1$ 到 $\alpha_n$ 是系数矩阵 A 的列向量组，$\beta$ 也是列向量，他们的纬度是一样的  ）
		1. 非齐次： $x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = \beta$ ==>表示 $\beta$ 能否由向量组 $\alpha_1, \ldots, \alpha_n$ 线性表示。
		2. 齐次：$x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n =0$ 
2. 两种方程组解的情况判别 【考点】
	1. 非齐次方程组 $AX = \beta$（$n$ 为未知数个数）其中 $\mathbf{A}_{m \times n}$ 首先明确 $r(A) \leq r(A|\beta)$
		- **无解** $\Leftrightarrow r(A) < r(A|\beta)$。
		- **有唯一解** $\Leftrightarrow r(A) = r(A|\beta) = n$。（当 $A$ 是方阵时就是克莱姆法则==>主要用于理论证明或求解 $n=2, 3$ 的小规模问题）
		- **有无穷多解** $\Leftrightarrow r(A) = r(A|\beta) < n$。
		- ‼️方程个数即系数矩阵行数~={red} **$m$ 是 $r(A)$ 和 $r(A|\beta)$ 的上界=~**
			- 如果 $A$ 是 $m \times n$ 矩阵，当 ~={red} $r(A) = m$ =~（行满秩）时，~={red} $AX = \beta$ 一定有解=~
			- 如果~={red}  $m < n$ =~（方程个数小于未知数个数），$AX = \beta$ ~={red}**一定不是唯一解**=~（要么无解，要么无穷多解）。
	2. **齐次方程组** $AX = 0$（$n$ 为未知数个数）：首先明确**齐次方程组一定有解**
	    - **只有零解** $\Leftrightarrow r(A) = n$。（**考点**：$A$ 列满秩，齐次方程组只有零解）。
	    - **有非零解** $\Leftrightarrow r(A) < n$。
	    - 如果 $A$ 列满秩（$r(A) = n$），则 $A$ 在矩阵乘法中有**左消去律**：$AB = 0 \Rightarrow B = 0$ ==> $AX = 0 \Rightarrow X=0$，$AB = AC \Rightarrow B = C$。
         - 如果 $A$ 列满秩，则 $r(AB) = r(B)$。应用【可出题】：当 $\beta_1, \ldots, \beta_s$ 可用**线性无关**向量组 $\alpha_1, \ldots, \alpha_r$ 线性表示时，则 $r(\beta_1, \ldots, \beta_s)$ 等于**表示矩阵 $C$ 的秩**。（$C$ 是表示矩阵，也即系数矩阵 $\beta_j = c_{1j}\alpha_1 + c_{2j}\alpha_2 + \cdots + c_{rj}\alpha_r$，$C = \begin{pmatrix} c_{11} & \cdots & c_{1s} \\ \vdots & \ddots & \vdots \\ c_{r1} & \cdots & c_{rs} \end{pmatrix}_{r \times s}$）
	3. 【背✅】
3. 齐次方程组 $AX=0$ 的解（解空间）
	1. 解的性质：如果 $\eta_1, \eta_2+ \cdots +  \eta_s$ 都是 $AX=0$ 的解，则它们的任意线性组合 $c_1\eta_1 + c_2\eta_2+\cdots +  c_s\eta_s$ 也是 $AX=0$ 的解。
	2. **基础解系(考点)**：齐次方程组 $AX=0$ 的解集 $J$ （解空间）的最大无关组。
		1. 解集的秩：解空间的秩 $r(J) = n - r(A)$  即基础解系中包含的线性无关解的个数 $s = n - r(A)$，A 秩是多少就会将 n 维空间压缩到 r（A）维空间
		2. **基础解系的判断条件 (考点)**：一组向量 $\eta_1, \ldots, \eta_s$ 是 $AX=0$ 的基础解系，必须同时满足：
			- (1) 它们都是的解（$A\eta_i = 0$）。  
		    - (2) 基础解系必须是**线性无关**的
		    - (3)~={red} 基础解系包含的向量个数 $s = n - r(A)$。=~
			**【总结】** 基础解系是原方程组任意一个方程的解，基础解系线性无关，个数为 $n$ 减去系数矩阵的秩，并且基础解系不唯一!！【书】基础解系不唯一，但个数 $s=n-r(A)$ 必须正确，且必须线性无关，且必须是方程组的解。
	3. 推论：~={red}  $\mathbf{A}\mathbf{B} = \mathbf{0}$ =~ (其中 $\mathbf{A}_{m \times n}, \mathbf{B}_{n \times p}$)，则
		- 角度一: 秩 
			 ~={red}1)  $r(\mathbf{A}) + r(\mathbf{B}) \le n$ ( $\mathbf{A}$ 的列数 $\mathbf{B}$ 的行数)=~
		- 角度二: 线性方程组
		    1) $\mathbf{B}$ 的每一列 $\vec{\beta}_j$ 都是线性方程组 $\mathbf{A}\mathbf{x} = \mathbf{0}$ 的解。 (由 $\mathbf{A}(\vec{\beta}_1, \vec{\beta}_2, \dots) = (\mathbf{0}, \mathbf{0}, \dots)$) 右边的每一列都是左边方程组的解
		    2) $\iff \mathbf{B}$ 的列向量与 $\mathbf{A}$ 的行向量两两正交。
		- 注: 一个重要常识 $\mathbf{x} = \begin{bmatrix} x_1 \\ \vdots \\ x_n \end{bmatrix} \text{ 是齐次线性方程组 } \mathbf{A}\mathbf{x} = \mathbf{0} \text{ 的解 } \iff \mathbf{x} \text{ 与 } \mathbf{A} \text{ 的任一行向量正交。}$
		  ==>是齐次方程组的解即找一个与系数矩阵所有行向量两两正交的
	4. 齐次方程组的通解： $\eta_1, \ldots, \eta_s$ 是 $AX=0$ 的基础解系，则通解为 $X = c_1\eta_1 + c_2\eta_2 + \cdots + c_s\eta_s$，其中 $c_i$ 为任意常数。
	5. 对于齐次方程组 $AX=0$ 来说，齐次方程组的“解空间”就是矩阵 $A$ 的“零空间”。
4. 非齐次方程组 $AX=\beta$ 的解
	1. **解的性质 (考点)**：
	    - 设 $\xi_1, \xi_2$ 都是 $AX=\beta$ 的解，则 $\xi_1 - \xi_2$ 是对应的齐次方程组 $AX=0$ 的解。
	    - 设 $\xi$ 是 $AX=\beta$ 的解，$\eta$ 是 $AX=0$ 的解，则 $\xi + \eta$ 仍然是 $AX=\beta$ 的解。
	    - 【总结】设 $\xi_1,\xi_2, \ldots, \xi_s$ 是 $AX=\beta$ 的解：
	        - $c_1\xi_1 + \cdots + c_s\xi_s$ 仍是 $AX=\beta$ 的解 $\Leftrightarrow c_1 + \cdots + c_s = 1$。
	        - $c_1\xi_1 + \cdots + c_s\xi_s$ 是 $AX=0$ 的解 $\Leftrightarrow c_1 + \cdots + c_s = 0$。
	    - 应用：通解中的“特解”部分必须是 $AX=\beta$ 的解（如 $(\xi_1+\xi_2)/2$），“基础解系”部分必须是 $AX=0$ 的解（如 $\xi_1-\xi_2$），且线性无关。
	2. **非齐次方程组的通解 (考点)**：
	    - **结构**：$X = \xi_0 + c_1\eta_1 + \cdots + c_s\eta_s$。
	    - **组成**：
	        - $\xi_0$：非齐次方程组 $AX=\beta$ 的一个**特解**。
	        - $c_1\eta_1 + \cdots + c_s\eta_s$：对应的齐次方程组（导出组）$AX=0$ 的**通解**。
	3. **非齐次解集的秩 (小众点)**：
	 - 当 $AX=\beta$ 有解时，其解集（一个仿射空间）的秩为 $n - r(A) + 1$。
	 - 证明：要证明解集的秩是 $s+1$，只需要证明两件事 ：
		1. 我们**能**从 $AX=\beta$ 的解集中找到 $s+1$ 个线性无关的解。
		2. $AX=\beta$ 的解集中**任意** $s+2$ 个解都是线性相关的。[[🌟非齐次方程组的解集的秩]]
	 - **应用**：若 $AX=\beta$ 有 3 个线性无关的解，且 $n=4$，则 $n-r(A)+1 \ge 3 \Rightarrow 4-r(A)+1 \ge 3 \Rightarrow r(A) \le 2$。（例 4.17）
5. 求通解的具体步骤 (核心计算)
	1. **求解** $AX=0$：
	    - (1) 将系数矩阵 $A$ 作初等**行**变换，化为**行最简形矩阵**（或阶梯形）。
	    - (2) 确定 $r(A)$，计算基础解系个数 $s = n - r(A)$。
	    - (3) 选定自由未知量（通常是~={red}**阶梯形中非主元对应的变量**=~），用自由未知量表示主元（台角）未知量。
	    - (4) 对自由未知量轮流赋值（如 $(1, 0, \ldots), (0, 1, \ldots), \ldots$），求出 $s$ 个线性无关的解 $\eta_1, \ldots, \eta_s$
	    - (5) 写出通解 $X = c_1\eta_1 + \cdots + c_s\eta_s$。
	2. **求解** $AX=\beta$：
	    - (1) 对**增广矩阵** $(A|\beta)$ 作初等**行**变换，化为阶梯形。
	    - (2) 判断 $r(A)$ 和 $r(A|\beta)$，确定是否有解。
	    - (3) **求特解** $\xi_0$：在化简后的方程组中，令所有自由未知量为 $0$，解出主元未知量的值，得到 $\xi_0$。
	    - (4) **求** $AX=0$ **的通解**：根据 (A) 化简后的阶梯形，按步骤 13 求出 $AX=0$ 的基础解系 $\eta_1, \ldots, \eta_s$。
	    - (5) 写出通解 $X = \xi_0 + c_1\eta_1 + \cdots + c_s\eta_s$。
6. 整个方程组研究的都是正交向量组 $e.g Ax=0$ 即 $x$ 和 $A$ 的行是正交向量组（整个线代都是玩正交的）
7. 方程组和解是一些可以互相转化的向量组
8. 解题思维：方程组方面的选择题首先考虑是否可以用特例进行排除
9. $AX=0$ 的解都是 $BX=0$ 的解（解集 $J_A \subseteq J_B$）$\Rightarrow n-r(A) \le n-r(B) \Rightarrow r(A) \ge r(B)$。
10. **含参方程组讨论**：
	1. 对增广矩阵作初等行变换，化为阶梯形。
	2. 根据阶梯形中含参的行，讨论和 $r(A|\beta)$ 的关系。
11. **利用已有解构造通解**：不直接解方程，而是利用解的性质。
	1. 已知 $AX=\beta$ 的两个不同解 $\xi_1, \xi_2$，则 $\eta = \xi_1 - \xi_2$ 是 $AX=0$ 的一个解。
	2. 已知 $r(A)$，从而知道 $s = n-r(A)$。如果 $s=1$，那么 $\xi_1 - \xi_2$（或其非零倍数）就是 $AX=0$ 的基础解系。
	3. **例4.21 (技巧)**：$A = (\alpha_1, \alpha_2, \alpha_3, \alpha_4)$，$\alpha_1 = 2\alpha_2 - \alpha_3 \Rightarrow \alpha_1 - 2\alpha_2 + \alpha_3 + 0\alpha_4 = 0$，这说明 $\eta = (1, -2, 1, 0)^T$ 是 $AX=0$ 的一个解。$\beta = \alpha_1 + \alpha_2 + \alpha_3 + \alpha_4 \Rightarrow X = (1, 1, 1, 1)^T$ 是 $AX=\beta$ 的一个特解。

    - $AX-XA=B$（例 4.37）：将 $X$ 设为含 $n^2$ 个未知数（如 $x_1, x_2, x_3, x_4$）的矩阵，将矩阵方程展开为 $n^2$ 个线性方程构成的方程组求解。
    - $AX=B$（例 4.38）：$B$ 有 $p$ 列，这等价于 $p$ 个线性方程组 $AX = \beta_i$（$\beta_i$ 是 $B$ 的第 $i$ 列）共用同一个系数矩阵 $A$。
13. **公共解问题 (考点/难点)**：
    - **(I)** $AX=0$ **和 (II)** $BX=0$ **的公共解**：
        - 方法 1：求 (I) 的通解 $X = c_1\eta_1 + \cdots$。
        - 方法 2：将通解代入 (II) 中（$B(c_1\eta_1 + \cdots) = 0$），解出 $c_i$ 之间的关系。
        - 方法 3：联立两个方程组 $\begin{pmatrix} A \\ B \end{pmatrix} X = 0$，求解。
    - **(I) 和 (II) 有公共非零解** $\Leftrightarrow$ 联立方程组有非零解 $\Leftrightarrow r \begin{pmatrix} A \\ B \end{pmatrix} < n$。
    - **技巧 (例 4.26)**：若 (I) 的基础解系为 $\gamma_1, \gamma_2$，(II) 的基础解系为 $\eta_1, \eta_2$。它们有公共非零解 $\Leftrightarrow$ 向量组 $\gamma_1, \gamma_2, \eta_1, \eta_2$ 线性相关。
14. **同解问题 (考点/难点)**：
    - (I) $AX=0$ 和 (II) $BX=0$ 同解 $\Leftrightarrow r(A) = r(B) = r \begin{pmatrix} A \\ B \end{pmatrix}$
    - (I) $AX=\beta$ 和 (II) $BX=\gamma$ 同解 $\Leftrightarrow r(A|\beta) = r(B|\gamma) = r \begin{pmatrix} A & \beta \\ B & \gamma \end{pmatrix}$。
    - **技巧 (例4.28)**：求 (II) 的一个解（特解），代入 (I) 中，反解出参数 $m, n, t$。
15. **反求方程组**（例4.30, 4.31）：
    - 已知基础解系 $\eta_1, \ldots, \eta_s$（行向量），构造矩阵 $B = \begin{pmatrix} \eta_1^T \\ \vdots \\ \eta_s^T \end{pmatrix}$。
    - 求解齐次方程组 $BX=0$。
    - $BX=0$ 的系数矩阵（或其基础解系的行向量构成的矩阵）$A$ 即为所求。
    - **原理 (例4.30)**：$A$ 的行向量是 $AX=0$ 解空间的“法向量”。$B$ 的行向量（即 $\eta_i^T$）是解空间的“基向量”。$A$ 的行向量必须与所有 $\eta_i$ 正交，即 $A \eta_i = 0$，转置得 $\eta_i^T A^T = 0$，即 $B A^T = 0$
16. **与特征向量的联系 (重要考点)**：
    - 求矩阵 $A$ 属于特征值 $\lambda$ 的特征向量，就是求解齐次线性方程组 $(A - \lambda E)X = 0$ 的非零解。
    - 属于 $\lambda$ 的线性无关的特征向量的个数，就是该方程组的基础解系的个数 $s = n - r(A - \lambda E)$
17. **行列满秩 (重要考点)：“**行满秩”和“列满秩”都是指矩阵的秩达到了**它可能的最大值**，这个“最大值”是受限于它的行数 $m$ 和列数 $n$ 的。
	1. 假设我们有一个 $m \times n$ 矩阵 $A$（$m$ 行，$n$ 列），它的秩为 $r(A)$。
	2. 行满秩：矩阵的秩等于它的**行数 $m$**。
		1. 数学表示：$r(A) = m$。
		2. 前提：只可能在 $m \le n$ 时发生（行数小于等于列数）。
		3. **向量含义**：矩阵的 $m$ 个**行向量**是线性无关的。
		4. **在方程组 $AX = \beta$ 中的意义**（这是最重要的）：当 $r(A) = m$ 时，$r(A|\beta)$ 的秩最大也只能是 $m$。所以，$r(A) = r(A|\beta) = m$ 恒成立。
		    - **结论：$AX = \beta$ 对于任何 $\beta$ 都一定有解**。（解可能是唯一的，也可能是无穷多的，取决于 $n$ 是否等于 $m$）。
	3. 列满秩：矩阵的秩 $r(A)$ 等于它的**列数 $n$**。
		1. **数学表示**：$r(A) = n$。
		2. **前提**：只可能在 $n \le m$ 时发生（列数小于等于行数）。
		3. **向量含义**：矩阵的 $n$ 个**列向量**（即 $\alpha_1, \ldots, \alpha_n$）是线性无关的。
		4. **在方程组中的意义**（这是最重要的）：
		    1. **对于 $AX = 0$**：$r(A) = n$，所以基础解系的个数 $s = n - r(A) = n - n = 0$。      
		        - **结论：齐次方程组 $AX = 0$ 只有唯一的零解**。
	    5. **对于 $AX = \beta$**：如果这个方程组有解（即 $r(A) = r(A|\beta)$），那么这个解一定是**唯一解**，因为 $r(A) = n$。
	    6. 总结对比表

		| **特征**               | **行满秩 (Row Rank)**    | **列满秩 (Column Rank)** |
		| -------------------- | --------------------- | --------------------- |
		| **定义**               | 秩 = **行数** ($m$)      | 秩 = **列数** ($n$)      |
		| **数学表示**             | $r(A) = m$            | $r(A) = n$            |
		| **向量相关性**            | **行向量**线性无关           | **列向量**线性无关           |
		| **对 $AX=0$ 的意义**     | 如果 $n > m$，必有非零解      | **必只有零解**             |
		| **对 $AX=\beta$ 的意义** | **对任意 $\beta$ 都一定有解** | **解如果存在，必是唯一解**       |
