### 为什么 $AX=\beta$ 的解集的秩是 $s+1$？

我们设 $s = n - r(A)$。要证明解集的秩是 $s+1$，我们只需要证明两件事 5：

1. 我们**能**从 $AX=\beta$ 的解集中找到 $s+1$ 个线性无关的解。
    
2. $AX=\beta$ 的解集中**任意** $s+2$ 个解都是线性相关的。
    

#### 1. 📖证明：可以找到 $s+1$ 个线性无关的解

- 根据定理4.6，非齐次方程组 $AX=\beta$ 的通解 = 1个特解 + 导出组 $AX=0$ 的通解 666。
    
- 我们设 $\xi_0$ 是 $AX=\beta$ 的一个特解（$A\xi_0 = \beta$）。
    
- 我们设 $\eta_1, \eta_2, \dots, \eta_s$ 是 $AX=0$ 的一个基础解系（它们是 $s = n - r(A)$ 个线性无关的解向量）77。
    
- 现在，我们构造 $s+1$ 个向量：
    
    $$\xi_0, \quad \xi_1 = \xi_0 + \eta_1, \quad \xi_2 = \xi_0 + \eta_2, \quad \dots, \quad \xi_s = \xi_0 + \eta_s$$
    
- 根据解的性质（定理4.6 ②），这 $s+1$ 个向量**全都是 $AX=\beta$ 的解** 。
    
- 接下来我们证明这 $s+1$ 个解是**线性无关**的。
    
    - 我们来考察它们所对应的另一个向量组 $\{\xi_0, \eta_1, \eta_2, \dots, \eta_s\}$。
        
    - 向量组 $\{\xi_0, \xi_1, \dots, \xi_s\}$ 与向量组 $\{\xi_0, \eta_1, \dots, \eta_s\}$ 可以相互线性表示（例如 $\eta_i = \xi_i - \xi_0$），所以它们是**等价向量组**。
        
    - 等价的向量组有相同的秩，所以 $r(\xi_0, \xi_1, \dots, \xi_s) = r(\xi_0, \eta_1, \dots, \eta_s)$ 99。
        
    - 因为 $\eta_1, \dots, \eta_s$ 是 $AX=0$ 的解（$A\eta_i = 0$），而 $\xi_0$ 是 $AX=\beta$ 的解（$A\xi_0 = \beta$）。只要 $\beta \ne 0$（即方程组真正是“非齐次”的），$\xi_0$ 就不可能由 $\eta_1, \dots, \eta_s$ 线性表示（否则 $A\xi_0$ 会等于0，而不是 $\beta$）。
        
    - 根据《线代》P372 的定理3.2 ，既然 $\eta_1, \dots, \eta_s$ 线性无关，且 $\xi_0$ 不能被它们表示，那么 $\{\xi_0, \eta_1, \dots, \eta_s\}$ 这 $s+1$ 个向量就是线性无关的。
        
    - 因此， $r(\xi_0, \xi_1, \dots, \xi_s) = s+1$ 10。我们成功找到了 $s+1$ 个线性无关的解。
        

#### 2. 📖证明：任意 $s+2$ 个解都线性相关

- 根据通解结构， $AX=\beta$ 的任意一个解 $\xi$ 都可以表示为：
    
    $$\xi = \xi_0 + c_1\eta_1 + c_2\eta_2 + \dots + c_s\eta_s$$
    
    其中 $c_i$ 是常数。
    
- 这说明，$AX=\beta$ 的**所有解**构成的集合，都可以由 $\{\xi_0, \eta_1, \dots, \eta_s\}$ 这 $s+1$ 个向量线性表示。
    
- 根据《线代》P373 定理3.8的推论 ：如果一个向量组（这里是任意 $s+2$ 个解）可以被一个包含 $s+1$ 个向量的向量组 $\{\xi_0, \eta_1, \dots, \eta_s\}$ 线性表示，那么这个向量组（这 $s+2$ 个解）一定是线性相关的 。
    
- 所以，任意 $s+2$ 个解都线性相关。
    

**结合(1)和(2)，$AX=\beta$ 的解集中，线性无关向量的个数最多为 $s+1$，所以解集的秩为 $s+1 = (n - r(A)) + 1$。**
