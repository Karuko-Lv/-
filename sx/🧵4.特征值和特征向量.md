1. 特征值和特征向量：
	1. 定义：设 $A$ 是 $n$ 阶矩阵，如果存在一个数 $\lambda$ 和一个 **非零** $n$ 维向量 $\eta$，使得 $A\eta = \lambda\eta$ 成立，则称 $\lambda$ 为 $A$ 的一个**特征值**，$\eta$ 为 $A$ 的对应于特征值 $\lambda$ 的**特征向量**。
		1. 【读】特征向量是在整个 n 维空间中，矩阵改变该向量只能改变其大小而不改变其方向，且是非 0 向量。
		2. 【读】矩阵是线性变换，在改变特征向量过程中展现出的数值化体现就是特征值。所以一般特征向量一般不变，特征值会随着矩阵的改变而改变。特征值和行列式都是矩阵的影子，是矩阵的数值化体现
		3. $n$ 阶矩阵有 $n$ 个特征值，但是特征值不一定是有 $n$ 个不一样的数字。特征值可以是重复的，即**代数重数**大于 1。【易错】若题目告知一个 4 阶矩阵的特征值不能是 0，1，-1 以外的数字，不代表他只有这 3 个特征值
	2. **计算方法**：
		1. 求特征值：求解 $A$ 的**特征方程** $|\lambda E - A| = 0$。这个方程的根就是 $A$ 的全部特征值。$|\lambda E - A|$ 称为 $A$ 的**特征多项式**。
			PS：
			- $n$ 阶矩阵的特征多项式是一个 $n$ 次多项式，一般来说求 $n$ 次多项式的根是困难的
			-  $n$ 次多项式有 $n$ 个根，故 $n$ 阶矩阵 A 的特征值共有 $n$ 个，其中有相同的和虚数。
			- 特征值重数：对应的特征多项式根的重数==>A 全体不同特征值的重数和=$n$
		2. 求特征向量：对于每一个特征值 $\lambda_i$，求解齐次线性方程组 $(A - \lambda_i E)X = 0$。这个方程组的 **全部非零解** 就是对应于 $\lambda_i$ 的全部特征向量。[考点]
		3. ‼️**注意**：属于特征值 $\lambda_i$ 的特征向量，是由其对应的齐次方程组 $(A - \lambda_i E)X = 0$ 的**基础解系**（$\eta_1, \eta_2, ...$）线性表示的，即 $k_1\eta_1 + k_2\eta_2 + ...$ （其中 $k_i$ 不全为零）。
		    - 一个特征值对应的特征向量是一个**向量空间**（$(A - \lambda E)X = 0$ 的解空间）中**除零向量之外**的所有向量。
		    - 一个特征值 $\lambda$ 对应的特征向量**一定不是唯一**的。如果 $\lambda$ 对应的基础解系包含多个线性无关的向量（即 $\eta_1, \eta_2, ...$），那么它们的**任意非零线性组合** $k_1\eta_1 + k_2\eta_2 + ...$ 都是 $\lambda$ 的特征向量。
	3. **特征值的快速计算（~={orange}特殊矩阵=~）**：
	    - ~={red}**三角矩阵/对角矩阵**：特征值就是其主对角线上的元素。=~[考点]
		    - 证明： 对于一个上三角矩阵或下三角矩阵，它的特征多项式也是一个三角矩阵。而三角矩阵的行列式就等于其主对角线上元素的乘积。所以：$|\lambda E - A| = (\lambda - a_{11})(\lambda - a_{22})\cdots(\lambda - a_{nn}) = 0$
	    - **秩为 1 的矩阵**：如果 $r(A)=1$（$n>1$），则 $A$ 的特征值为 $0, 0, ..., 0, tr(A)$（$n-1$ 个 0）。[考点]
		    - 元素全部一样的矩阵秩为 1
	    - $A = \alpha\beta^T$ **型矩阵**：其中 $\alpha, \beta$ 均为 $n$ 维列向量。
	        - $A$ 的特征值为 $0, 0, ..., 0, \beta^T\alpha$。
	        - 当 $\beta^T\alpha \neq 0$ 时，$\alpha$ 是 $A$ 对应于特征值 $\beta^T\alpha$ 的一个特征向量。[考点]
			PS：【背】 $\beta^T\alpha=tr(\alpha\beta^T)$
	4. **特征值与特征向量的~={red}性质**=~ [考点]：
	    - 设 $\lambda_1, \lambda_2, ..., \lambda_n$ 是 $A$ 的 $n$ 个特征值。
	        - $\sum \lambda_i = tr(A) = \sum a_{ii}$ （特征值之和等于矩阵的迹）。【背‼️】
	        - $\prod \lambda_i = |A|$ （特征值之积等于矩阵的行列式）。【背‼️】PS：几何理解就是特征向量是
	    - $A$ 可逆 $\Leftrightarrow$ $A$ 的所有特征值均不为 0。（ $A-\lambda E$ 可逆 $\Leftrightarrow$ $A$ 的特征值不为 $\lambda$）
		    - $A$ 不可逆 $\Leftrightarrow$ 0 是特征方程的根 $\Leftrightarrow$ 0 是特征值。
			~={red}【背✅】=~**矩阵可逆的充要条件是 0 不是他的特征值**，矩阵不可逆的充要条件是 0 是他的特征值
	    - 若 $A\eta = \lambda\eta$，则：
	        - $f(A)\eta = f(\lambda)\eta$。即 $f(A)$ 有一个特征值 $f(\lambda)$。
	        - $A^T$ 有与 $A$ 相同的特征值（但特征向量不一定相同）。
	        - 若 $A$ 可逆，则 $A^{-1}\eta = \frac{1}{\lambda}\eta$。
	        - $A^*\eta = \frac{|A|}{\lambda}\eta$。[考点] (伴随矩阵的特征值)
	5. **多项式与特征值**：
		1. 如果 $\boldsymbol{\eta}$ 是 $\boldsymbol{A}$ 的特征向量，特征值为 $\lambda$，则 $\boldsymbol{\eta}$ 也是 $\boldsymbol{A}$ 的任何多项式 $f(\boldsymbol{A})$ 的特征向量，特征值为 $f(\lambda)$；
			1. 如果 $A$ 的全体特征值为 $\lambda_1, \lambda_2, \cdots, \lambda_n$，则 $f(A)$ 的特征值是 $f(\lambda_1), f(\lambda_2), \cdots, f(\lambda_n)$。【背✅】例如：-1 ，1，2 是 $A$ 的特征值，$A^3-5 A^2$ 的三个特征值为 $-1 ,1 ,2$ 带入 $f (x)=x^3-5 x^2$, 为$-6,-4,-12$
				特别的 
				- $A + cE$ 的特征值是 $\lambda_1 + c, \lambda_2 + c, \cdots, \lambda_n + c$ 【背✅】
				- $cA$ 的特征值是 $c\lambda_1, c\lambda_2, \cdots, c\lambda_n$【背✅】
				- $A^n$ 的特征值是 $\lambda_1^n, \lambda_2^n, \cdots, \lambda_n^n$【背✅】
		2.  如果 $\boldsymbol{A}$ 可逆，则 $\lambda \neq 0$，并且 $\boldsymbol{\eta}$ 也是 $\boldsymbol{A}^{-1}$ 和 $\boldsymbol{A}^*$ 的特征向量，特征值分别为 $1/\lambda$ 和 $|\boldsymbol{A}|/\lambda$。
			1. 【背✅】~={red} $\boldsymbol{A}$ 可逆时，$\boldsymbol{A}, \boldsymbol{A}^{-1}$ 和 $\boldsymbol{A}^*$ 的特征向量完全一样。=~因为 $A^{-1}$ 和 $A^*$ 本质上是对矩阵 $A$ 变换的**逆向操作**或**常数缩放**（由 $A^* = |A|A^{-1}$ 可知），这些操作只会改变向量伸缩的**倍数**（特征值），而不会改变向量指向的**方向**（特征向量）。逆矩阵 $A^{-1}$ 的任务是**撤销** $A$ 所做的事情恢复原状，$A$ 的特征向量，$A^{-1}$ 也只是在同一方向上对它进行了缩放（拉回来），而没有改变方向。
			2. 【背✅】 $A + cE$ 的特征向量与 $A$ 的特征向量是一致的，只是**特征值**发生了变化。证明：$\begin{aligned} (A + cE)\boldsymbol{x} &= A\boldsymbol{x} + cE\boldsymbol{x} \\ &= A\boldsymbol{x} + c(E\boldsymbol{x}) \\ &= \lambda \boldsymbol{x} + c\boldsymbol{x} \quad \text{(因为 } E\boldsymbol{x} = \boldsymbol{x} \text{)} \\ &= (\lambda + c)\boldsymbol{x} \end{aligned}$
			3. 如果 $A$ 可逆，则 $A^{-1}$ 的特征值是 $1/\lambda_1, 1/\lambda_2, \cdots, 1/\lambda_n$； $A^*$ 的特征值是 $|A|/\lambda_1, |A|/\lambda_2, \cdots, |A|/\lambda_n$。【背✅】
			4. $A^T$ 的特征值也是 $\lambda_1, \lambda_2, \cdots, \lambda_n$。【背✅】
			5. 相似矩阵的特征值是一样的，特征向量往往是不一样的。若 $B = P^{-1}A P$，$\boldsymbol{x}$ 是 **$A$** 的特征向量，则$P^{-1}\boldsymbol{x}$ 是 **$B$** 的特征向量。
		3. 如果 $f(A)=0$，则 $A$ 的任一特征值 $\lambda$ 必满足 $f(\lambda)=0$（例如 $A^2+2A=0$ $\lambda^2+2\lambda=0$，$\lambda$ 只能是 0 或 -2）。
		4. ⚠️反之不成立。即 $f(\lambda)=0$ 的根 **不一定** 都是的特征值。因为他只是零化多项式不是最小多项式
			-  “特征值”、“零化多项式 ” 和 “最小多项式” 之间的核心关系。
				- **特征多项式** $f(\lambda)$：$|\lambda E - A|$。由 凯莱-哈密顿定理 可知，矩阵 $A$ 满足其自身的特征方程，即 $f(A) = O$。
				- **零化多项式** $g(\lambda)$：任何满足 $g(A) = O$ 的多项式，都称为 $A$ 的零化多项式。特征多项式是零化多项式的一种。
				- **最小多项式** $m(\lambda)$：在 $A$ 的所有零化多项式中，**次数最低** 且 **首项系数为 1** 的多项式。
				- 能把矩阵 $A$ “零化”的多项式 $f(\lambda)$，即 $f(A) = O$ (零矩阵)。称 $f(\lambda)$ 为 $A$ 的一个**零化多项式**
				- 如果 $f(A) = O$，那么 $A$ 的所有特征值 $\lambda_i$ **必定**是多项式方程 $f(\lambda) = 0$ 的根（即 $f(\lambda_i) = 0$）。
				- 多项式方程 $f(\lambda) = 0$ 的根，**不一定**是 $A$ 的特征值（它可能包含“多余的根”）。
			- 总结：特征值一定满足**零化多项式**，满足**零化多项式**的不一定是特征值
	6. **特征向量的线性无关性**：
	    - 属于【背】 **不同** 特征值的特征向量必定线性无关。[考点]
		    - 特征向量的线性无关性，只需要在“同一个特征值内部”检查即可，不同特征值之间是天然无关的。
	    - ‼️属于 **相同** 特征值的特征向量（及零向量）构成一个向量空间，其维度等于 $n - r(A - \lambda E)$，称为特征值 $\lambda$ 的**几何重数**。
		🤨如果一个特征值只有一个线性无关的特征向量，即特征多项式对应的齐次方程的基础解系只有一个，所以 $n - r(A - \lambda E)=1$，构成了一个一维的向量空间
	7. **重数关系** [考点]：（与 7 相关）
	    - **代数重数**：设 $\lambda$ 是特征多项式的 $k$ 重根（$k$ 称为代数重数）
	    - 几何重数： $\lambda$ 对应的线性无关的特征向量的个数 $t = n - r(A - \lambda E)$
	    - **重要性质**：$1 \le t \le k$。即~={red}  **$1 \le n - r(A - \lambda E) \le k$** =~
2. 矩阵的相似关系
	1. **定义**：设 $A, B$ 均为 $n$ 阶矩阵，若存在 **可逆矩阵** $P$，使得 $P^{-1}AP = B$，则称 $A$ 与 $B$ **相似**
	2. 记号： $A \sim B$
	3. **相似矩阵的~={red}性质**=~ [考点]：若 $A \sim B$，则：
	     - $r(A) = r(B)$ 【背✅】
		     - 满秩矩阵不改变矩阵的值，满秩矩阵代表一个**可逆变换**（就好比旋转、拉伸，但没有压扁成更低维）
	     - $|A| = |B|$
	     - $tr(A) = tr(B)$
	     - 有 **完全相同的特征值**。【背✅】 $A$ 和有相同的特征多项式，因为 $\lambda \boldsymbol{E}$ 可以被写成 $\boldsymbol{P}^{-1} (\lambda \boldsymbol{E}) \boldsymbol{P}$ 
	     - $f(A) \sim f(B)$。
	     - $A^{-1} \sim B^{-1}$，$A^* \sim B^*$ (当 $A$ 可逆时)。
	 4. [小众点]：$AB$ 与 $BA$ 不一定相似，但它们有 **相同的特征值**。如果 $A, B$ 中至少有一个可逆，则 $AB \sim BA$。
	 5. [小众点]：若 $A \sim B$ 且 $A\eta = \lambda\eta$，则 $B(P^{-1}\eta) = \lambda(P^{-1}\eta)$。即 $P^{-1}\eta$ 是 $B$ 对应于 $\lambda$ 的特征向量。
3. 相似对角化：
	1. **定义**：若 $n$ 阶矩阵 $A$ 与一个对角矩阵 $\Lambda$ 相似，则称 $A$ **可相似对角化**。
	    - 数学表达：存在可逆矩阵 $P$，使得 $P^{-1}AP = \Lambda = \text{diag}(\lambda_1, \lambda_2, ..., \lambda_n)$。
		    - 此时，$\lambda_1, ..., \lambda_n$ 必是 $A$ 的 $n$ 个特征值。
		    - $P$ 矩阵的 $n$ 个列向量~={red}  $P = (\eta_1, \eta_2, ..., \eta_n)$ =~ 必是 $A$ 的 $n$ 个线性无关的特征向量，且 $\eta_i$ 的顺序与 $\lambda_i$ 在 $\Lambda$ 中的顺序一一对应。   
	2. **判别法则（充要条件）** [考点]：$A$ 可相似对角化的 **充要条件** (二者等价)：
	    - **判据一 (向量)**：$A$ 有 $n$ 个线性无关的特征向量。
	    - **判据二 (重数)**：$A$ 的 **每一个** 特征值 $\lambda_i$，其特征方程根的重数 $k_i$**等于** 它的几何重数 $t_i = n - r(A - \lambda_i E)$。
		    - **注意**：对于代数重数为 1 的特征值，此条件（$1 \le t_i \le k_i=1$）自动满足，因此 **只需检验所有代数重数大于1的特征值**。
	3. **充分条件**：
	    - 若 $A$ 有 $n$ 个 **互不相同** 的特征值，则 $A$ 必可相似对角化。
	    - [小众点]：反之不成立。如单位矩阵 $E$ 特征值全为1（有重根），但 $E$ 显然可对角化（它本身就是对角阵）。    
	4. **特殊矩阵的对角化** [小众点]：
	    - $r(A)=1$ 的矩阵 $A$ 可对角化 $\Leftrightarrow tr(A) \neq 0$。
	    - 若 $(A-aE)(A-bE)=0$ 且 $a \neq b$，则 $A$ 必可对角化。
	    - 推论：若 $A^2=A$（幂等矩阵），则 $A$ 可对角化（特征值只能是0或1）。
	    - 推论：若 $A^2=E$（对合矩阵），则 $A$ 可对角化（特征值只能是1或-1）。
4. **实对称矩阵的相似对角化（重要考点）**
	1. **实对称矩阵**：$A^T = A$，且 $A$ 中元素均为实数。
	2. **重要性质** [考点]：
	    - **性质一**：实对称矩阵 $A$ 的特征值 **必为实数**。
	    - **性质二**：~={red}【 背‼️】实对称矩阵 $A$ **必可相似对角化**。=~（对所有特征值，代数重数=几何重数）
	    - **性质三**：实对称矩阵的对应于 **不同** 特征值 $\lambda_1, \lambda_2$ 的特征向量 $\eta_1, \eta_2$ **必定正交**，即 $\eta_1^T \eta_2 = 0$。（3 阶实对称矩阵知道 2个线性无关的特征向量就等于知道了 3 个）
	3. **正交对角化** [考点]：
	    - 对于任意 $n$ 阶实对称矩阵 $A$，**必定存在** 一个 **正交矩阵** $Q$（$Q^T = Q^{-1}$），使得 $Q^{-1}AQ = Q^T A Q = \Lambda$（对角矩阵）。
	4. **构造正交矩阵** $Q$ **的步骤** `[考点]`：
	    - **i. (求特征值)**：解 $|\lambda E - A| = 0$，求出全部实数特征值。
	    - **ii. (求特征向量)**：对每个特征值 $\lambda_i$，求 $(A - \lambda_i E)X = 0$ 的基础解系。
	    - **iii. (正交化)**：
	        - 如果 $\lambda_i$ 是 **单根**，其特征向量 $\eta_i$ 单独放一边。
	        - 如果 $\lambda_j$ 是 **k重根**（$k>1$），则求出的 $k$ 个线性无关的特征向量 $(\alpha_1, ..., \alpha_k)$ **不一定正交**。必须对这 $k$ 个向量进行 **施密特（Gram-Schmidt）正交化**，得到 $k$ 个相互正交的向量 $(\beta_1, ..., \beta_k)$。
	        - （不同特征值对应的向量组之间已自动正交，无需处理）。
	    - **iv. (单位化)**：将 iii 中得到的 $n$ 个相互正交的特征向量 $\eta_i, \beta_j, ...$ 全部 **单位化**（$\gamma = \frac{1}{||\eta||}\eta$），得到 $n$ 个单位正交向量 $\gamma_1, ..., \gamma_n$。
	    - **v. (构造Q)**：令 $Q = (\gamma_1, \gamma_2, ..., \gamma_n)$，则 $Q$ 即为所求的正交矩阵。
	5. [小众点]：
	    - $A^T A$ 是实对称矩阵，其特征值 **均为非负实数**（$\lambda \ge 0$）。
	    - 实反对称矩阵（$A^T = -A$）的特征值只能是 0 或纯虚数。
