证明不等式是数学中的核心技能之一，其方法灵活多样，常常需要结合代数、微积分和几何等多个领域的知识。下面总结了一些最基本和最强大的证明方法。

---

### 一、经典代数方法

这类方法主要依赖于代数恒等变形和不等式的基本性质。

#### 1. 作差法 (Comparison by Difference)

* **核心思想**: 证明 `A > B` 等价于证明 `A - B > 0`。这是最基本、最直接的方法。
* **步骤**:
    1.  构造差式 `D = A - B`。
    2.  通过通分、配方、因式分解等手段，将 `D` 变形为一个或多个易于判断符号的表达式的积或和。
    3.  根据已知条件判断 `D` 的符号，得出结论。
* **适用场景**: 表达式为多项式、分式等结构，作差后容易化简。
* **示例**: 证明 `a² + b² ≥ 2ab`
    * 作差：`a² + b² - 2ab = (a - b)²`
    * 由于任何实数的平方都非负，所以 `(a - b)² ≥ 0`。
    * 因此，`a² + b² ≥ 2ab` 成立。

#### 2. 作商法 (Comparison by Quotient)

* **核心思想**: 当要比较的两个表达式 `A` 和 `B` 都为**正数**时，证明 `A > B` 等价于证明 `A / B > 1`。
* **适用场景**: 表达式为乘积、幂、指数或阶乘形式，作商后易于约分和放缩。
* **示例**: 证明 `3³⁰⁰ > 4²⁰⁰`
    * 作商：`3³⁰⁰ / 4²⁰⁰ = (3³)¹⁰⁰ / (4²)¹⁰⁰ = 27¹⁰⁰ / 16¹⁰⁰ = (27/16)¹⁰⁰`
    * 由于 `27/16 > 1`，所以 `(27/16)¹⁰⁰ > 1`。
    * 因此，`3³⁰⁰ > 4²⁰⁰` 成立。

#### 3. 分析法与综合法 (Analysis and Synthesis)

* **分析法 (Working Backward)**: 从**待证明的不等式**出发，反向推导，寻找使其成立的**充分条件**，直到推导出一个已知的、显然成立的结论（如公理、定理或 `(x-y)² ≥ 0`）。这是一种寻找思路的方法。
* **综合法 (Working Forward)**: 从**已知条件**出发，利用公理、定理，一步步正向推导，直到得出待证明的不等式。这是书写证明过程的标准格式。
* **关系**: 通常先用分析法找到解题思路，再用综合法写出严谨的证明过程。

#### 4. 配方法 (Completing the Square)

* **核心思想**: 将表达式构造成一个或多个完全平方式的和，利用 `(expression)² ≥ 0` 来证明不等式。
* **适用场景**: 涉及二次或更高次的代数式。

---

### 二、微积分方法 (Calculus-Based Methods)

当不等式涉及超越函数（如 `sin(x)`, `e^x`, `ln(x)`）或变量是连续的时，微积分是极其强大的工具。

#### 5. 构造函数与利用单调性

* **核心思想**: 证明 `f(x) > g(x)` 对于 `x > a` 成立，可以构造辅助函数 `h(x) = f(x) - g(x)`。只需证明 `h(x)` 在 `x > a` 时恒为正。
* **步骤**:
    1.  构造函数 `h(x) = f(x) - g(x)`。
    2.  求导数 `h'(x)`，并判断其在目标区间内的符号，从而确定 `h(x)` 的单调性。
    3.  考察区间端点处的函数值（例如 `h(a)` 或 `lim h(x)` as `x→a⁺`）。
    4.  结合单调性和端点值，得出 `h(x)` 在整个区间内的符号。
* **示例**: 证明当 `x > 0` 时，`e^x > 1 + x`。
    * 构造 `h(x) = e^x - 1 - x`。
    * 求导：`h'(x) = e^x - 1`。当 `x > 0` 时，`e^x > 1`，所以 `h'(x) > 0`。
    * 这说明 `h(x)` 在 `(0, +∞)` 上是严格单调递增的。
    * 考察端点值 `h(0) = e⁰ - 1 - 0 = 0`。
    * 函数从 `h(0)=0` 开始，严格单调递增，因此对于所有 `x > 0`，都有 `h(x) > h(0) = 0`。
    * 即 `e^x - 1 - x > 0`，所以 `e^x > 1 + x`。

#### 6. 利用拉格朗日中值定理

* **核心思想**: `f(b) - f(a) = f'(c)(b-a)`。通过估计导数值 `f'(c)` 的范围，来获得函数增量 `f(b) - f(a)` 的范围。
* **适用场景**: 将不等式变形后，可以构造成符合中值定理的形式，尤其适用于对数和三角函数。

#### 7. 利用泰勒公式

* **核心思想**: 将复杂的函数用一个多项式和一个余项来近似。通过控制余项的符号，可以得到非常精确的不等式。
* **适用场景**: 证明在某一点（通常是 `x=0`）附近成立的不等式，特别是涉及 `e^x`, `sin(x)`, `cos(x)`, `ln(1+x)` 等函数。
* **常见泰勒不等式 (x>0)**:
    * $x - \frac{x^3}{6} < \sin(x) < x$
    * $1 - \frac{x^2}{2} < \cos(x) < 1$
    * $x - \frac{x^2}{2} < \ln(1+x) < x$

---

### 三、应用著名不等式

在熟悉基本方法后，直接应用一些著名不等式可以大大简化证明过程。

#### 8. 均值不等式 (AM-GM Inequality)

* **核心内容**: 对于 $n$ 个非负实数 $a_1, ..., a_n$：
    $$
    \frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}
    $$
    (算术平均值 ≥ 几何平均值)。当且仅当所有数都相等时，等号成立。
* **适用场景**: 变量为正数，表达式中同时含有“和”与“积”的形式，常用于求最值。

#### 9. 柯西-施瓦茨不等式 (Cauchy-Schwarz Inequality)

* **核心内容 (代数形式)**: 对于两组实数 $a_1, ..., a_n$ 和 $b_1, ..., b_n$：
    $$
    (a_1^2 + \dots + a_n^2)(b_1^2 + \dots + b_n^2) \ge (a_1b_1 + \dots + a_nb_n)^2
    $$
* **适用场景**: 表达式中含有平方和以及乘积和的形式。

#### 10. 伯努利不等式 (Bernoulli's Inequality)

* **核心内容**: 当 `x > -1`，`α > 1` 或 `α < 0` 时，`(1+x)ᵅ ≥ 1+αx`。当 `0 < α < 1` 时，不等号反向。
* **适用场景**: 证明与幂函数 `(1+x)ᵅ` 相关的放缩。

---

### 四、其他常用策略

* **数学归纳法 (Mathematical Induction)**: 证明与**自然数 n** 相关的不等式。
* **分类讨论法 (Proof by Cases)**: 当表达式中含有绝对值、或者变量的取值范围需要被划分时，将问题分解成有限个情况分别证明。

### 总结

| 方法名称         | 核心思想              | 适用场景/关键词                 |
| :----------- | :---------------- | :----------------------- |
| **作差/作商法**   | 转化为判断符号或与 1 的大小关系 | 结构简单，易于通分、变形、约分          |
| **构造函数法**    | 将不等式看作函数，利用单调性    | 连续变量，超越函数，证明 `f(x)>g(x)` |
| **均值不等式**    | 和与积的转换            | 变量为正，求最值                 |
| **柯西-施瓦茨**   | 平方和与积和的关系         | 平方和，点积                   |
| **拉格朗日中值定理** | 用导数范围控制函数增量范围     | 证明 `f(b)-f(a)` 的界        |
| **泰勒公式**     | 用多项式逼近函数          | `x→0` 附近的超越函数不等式         |
| **数学归纳法**    | 证明对所有自然数 n 成立     | `n`, 递归，数列               |


# 常见不等式的证明方法示例

这是一系列经典且常见的不等式，按照它们最典型的证明方法进行分类。掌握这些例子有助于深化对各种证明技巧的理解和应用。

---

### 1. 作差法 (Comparison by Difference)

* **基本二次不等式**:
    $$a^2 + b^2 \ge 2ab \quad (a, b \in \mathbb{R})$$
    * **证明**: 作差 $a^2 + b^2 - 2ab = (a-b)^2 \ge 0$。

* **排序不等式 (基本形式)**:
    $$a^2 + b^2 + c^2 \ge ab + bc + ca \quad (a, b, c \in \mathbb{R})$$
    * **证明**: 作差后乘以 2，再配方 $2(a^2+b^2+c^2 - ab-bc-ca) = (a-b)^2 + (b-c)^2 + (c-a)^2 \ge 0$。

---

### 2. 作商法 (Comparison by Quotient)

* **指数与幂的比较**:
    $$e^x > x^e \quad (\text{当 } x \neq e)$$
    * **证明**: 构造函数 $f(x) = \frac{\ln x}{x}$，利用导数可证其在 $x=e$ 处取最大值 $\frac{1}{e}$。因此 $\frac{\ln x}{x} < \frac{\ln e}{e}$，整理即得。

* **斯特林公式相关**:
    $$n! > (\frac{n}{e})^n \quad (n \in \mathbb{Z}^+)$$
    * **证明**: 通常使用数学归纳法，并在归纳步骤中利用 $(1+\frac{1}{n})^n < e$ 进行作商比较。

---

### 3. 分析法与综合法 (Analysis and Synthesis)

* **涉及根式的代数不等式**:
    $$\sqrt{a^2+b^2} + \sqrt{c^2+d^2} \ge \sqrt{(a+c)^2 + (b+d)^2}$$
    * **分析思路 (倒推)**: 两边都是正数，考虑平方。平方后得到 $2\sqrt{(a^2+b^2)(c^2+d^2)} \ge 2(ac+bd)$。再次平方，整理后可得 $(ad-bc)^2 \ge 0$，此式显然成立。书写证明时将此过程反向写出（综合法）。

---

### 4. 构造函数与利用单调性

这个方法非常强大，尤其适用于超越不等式。

* **指数函数基本放缩**:
    $$e^x \ge 1+x \quad (\text{对所有 } x \in \mathbb{R})$$
    * **证明**: 构造 $f(x) = e^x - 1 - x$，求导 $f'(x)=e^x-1$。讨论 $x>0$ 和 $x<0$ 时的单调性，发现 $f(x)$ 在 $x=0$ 处取得最小值 0。

* **对数函数基本放缩**:
    $$\ln(x) \le x-1 \quad (\text{当 } x > 0)$$
    * **证明**: 构造 $f(x) = \ln(x) - (x-1)$，求导 $f'(x)=\frac{1}{x}-1$。讨论单调性可知 $f(x)$ 在 $x=1$ 处取得最大值 0。

* **三角函数基本放缩**:
    $$\sin(x) < x < \tan(x) \quad (\text{当 } 0 < x < \frac{\pi}{2})$$
    * **证明**: $sin(x) < x$ 可构造 $f(x) = x - sin(x)$。$x < tan(x)$可构造$g(x) = tan(x) - x$。分别求导证明其在区间内单调递增即可。

---

### 5. 利用拉格朗日中值定理

* **对数函数的经典界**:
    $$\frac{1}{x+1} < \ln(1+\frac{1}{x}) < \frac{1}{x} \quad (\text{当 } x > 0)$$
    * **证明**: 对函数 $f(t)=\ln(t)$ 在区间 $[x, x+1]$ 上使用拉格朗日中值定理。

* **正弦函数的李普希茨连续性**:
    $$|\sin(b) - \sin(a)| \le |b - a|$$
    * **证明**: 对 $f(x)=\sin(x)$ 使用中值定理，`|f(b)-f(a)| = |f'(c)||b-a| = |cos(c)||b-a|`。因为 `|cos(c)| ≤ 1`，结论得证。

---

### 6. 利用泰勒公式

提供在 $x=0$ 附近非常精确的局部不等式。

* **余弦函数**:
    $$1 - \frac{x^2}{2} \le \cos(x) \le 1 - \frac{x^2}{2} + \frac{x^4}{24} \quad (\text{对所有 } x)$$
* **正弦函数**:
    $$x - \frac{x^3}{6} \le \sin(x) \le x \quad (\text{当 } x \ge 0)$$
* **指数函数**:
    $$1+x+\frac{x^2}{2} \le e^x \quad (\text{当 } x \ge 0)$$
* **对数函数**:
    $$x - \frac{x^2}{2} \le \ln(1+x) \le x \quad (\text{当 } x \ge 0)$$

---

### 7. 均值不等式 (AM-GM)

* **基本应用**:
    $$x + \frac{1}{x} \ge 2 \quad (\text{当 } x > 0)$$
    * **证明**: $\frac{x+1/x}{2} \ge \sqrt{x \cdot \frac{1}{x}} = 1$。

* **多变量应用**:
    $$(a+b)(b+c)(c+a) \ge 8abc \quad (a,b,c \ge 0)$$
    * **证明**: 对 $a+b, b+c, c+a$ 分别应用均值不等式：$a+b \ge 2\sqrt{ab}$, $b+c \ge 2\sqrt{bc}$, $c+a \ge 2\sqrt{ca}$。三式相乘即得。

---

### 8. 柯西-施瓦茨不等式

* **基本形式应用**:
    $$(a+b+c)^2 \le 3(a^2+b^2+c^2)$$
    * **证明**: 应用柯西不等式于向量 $(a,b,c)$ 和 $(1,1,1)$。

* **Titu-Engel 形式**:
    $$\frac{x^2}{a} + \frac{y^2}{b} \ge \frac{(x+y)^2}{a+b} \quad (a,b > 0)$$
    * 这是柯西不等式的一个常用推论。

* **倒数和**:
    $$(x+y+z)(\frac{1}{x}+\frac{1}{y}+\frac{1}{z}) \ge 9 \quad (x,y,z > 0)$$
    * **证明**: 展开后使用均值不等式，或直接使用柯西不等式于向量 $(\sqrt{x}, \sqrt{y}, \sqrt{z})$ 和 $(\frac{1}{\sqrt{x}}, \frac{1}{\sqrt{y}}, \frac{1}{\sqrt{z}})$。

---

### 9. 伯努利不等式

* **证明 $e$ 的定义序列单调性**:
    $$\left(1+\frac{1}{n}\right)^n < \left(1+\frac{1}{n+1}\right)^{n+1}$$
    * **证明**: 对 $\frac{(1+\frac{1}{n+1})^{n+1}}{(1+\frac{1}{n})^n}$ 作变形，关键一步会用到 $(1-\frac{1}{(n+1)^2})^{n+1} > 1-\frac{1}{n+1}$，这里就需要伯努利不等式。

---

### 10. 数学归纳法

* **幂与多项式比较**:
    $$2^n > n^2 \quad (\text{对于整数 } n \ge 5)$$
    * **证明**: 先验证 $n=5$ 的基础情况，然后假设 $2^k > k^2$ 成立，推导 $2^{k+1} > (k+1)^2$ 的过程。

---

### 11. 几何法

* **三角形不等式**:
    $$|\vec{a}| + |\vec{b}| \ge |\vec{a}+\vec{b}|$$
    * **证明**: 几何上，三角形两边之和大于第三边。

* **托勒密不等式 (Ptolemy's Inequality)**:
    对于任意四边形 $ABCD$，有 $AB \cdot CD + BC \cdot DA \ge AC \cdot BD$。当且仅当四边形为圆内接四边形时取等号。
