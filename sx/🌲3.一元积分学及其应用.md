1. 变限积分
	1. 莱布尼茨积分法则 (在积分号下求导)：用于计算一个积分对某个参数的导数。
		- **用途：** 当积分的**上下限**和**被积函数**都依赖于某个变量（如 $x$）时，求该积分对 $x$ 的导数。
		- **公式：**$\frac{d}{dx} \left( \int_{a(x)}^{b(x)} f(x, t) dt \right) = f(x, b(x)) \cdot b'(x) - f(x, a(x)) \cdot a'(x) + \int_{a(x)}^{b(x)} \frac{\partial f}{\partial x} dt$
			- **第一项：** 被积函数在**上界**的值，乘以**上界**的导数。
		    - **第二项：** 被积函数在**下界**的值，乘以**下界**的导数。
		    - **第三项：** 被积函数对 $x$ 的**偏导数**，在原区间上的积分。
2. 表格法：适用于 $\int u(x)v(x)dx$。其中要求：
	- $u(x)$ 是一个**求导几次后会变成 0** 的函数（最典型的就是**多项式**，比如 $x^2, x^3$）。
	- $v(x)$ 是一个**容易被连续积分**的函数（最典型的就是 $e^{ax}, \sin(ax), \cos(ax)$）。
	==> 分布积分次数比较多的，遇到幂函数和指数函数相乘或者幂函数和三角函数相乘或者指数函数和三角函数相乘，这种情况都可以用**表格法**找到他的原函数：上面求导下面积分，上面求到 0 为止。例如：$x^2 e^{-x}=$
	
	| **符号 (Sign)** | **求导 (D)** | **积分 (I)** |
	| ------------- | ---------- | ---------- |
	| +             | $x^2$      | $e^{-x}$           |
	| -             | $2x$       | $-e^{-x}$       |
	| +             | $2$        | $e^{-x}$           |
	| -             | $0$        |  $-e^{-x}$       |
