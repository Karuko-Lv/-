#### 6️⃣Vue 的 **SFC (单文件组件)** 模式
本质上就是为了把一个组件的“三要素”打包在一起：

- **`<template>` (HTML 增强版)**：定义页面的**结构**。它比普通 HTML 多了 Vue 指令（如 `v-if`, `v-for`, `{{ }}`），让页面能动起来。
- **`<script>` (JS/TS 逻辑层)**：定义页面的**行为**。用了 `lang="ts"`，说明它是 TypeScript（JS 的增强版，多了类型检查）。
	- ‼️ts 和 js 的区别：ts 多了接口，写法更加规范，类型检查提高代码正确性
- **`<style>` (CSS 样式层)**：定义页面的**外观**。代码中用了 `lang="scss"` 和 `scoped`，意味着支持更高级的 CSS 语法，且样式只对当前组件生效。
