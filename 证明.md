这是一个非常巧妙的问题，它将 **矩阵的迹 (Trace)**、**伴随矩阵 (Adjugate)**、**特征值 (Eigenvalue)** 和 **余子式 (Minor)** 这四个核心概念完美地联系在了一起。

首先，题目中 $A$ 是一个 $3 \times 3$ 矩阵（因为它有3个特征值）。

我们要计算的 $M_{11} + M_{22} + M_{33}$ 是矩阵 $A$ 的主对角线上所有元素的余子式之和。

**答案是： $\lambda_1 + \lambda_2 + \lambda_3$**

---

### 证法一：利用“迹”与“伴随矩阵”的定义（最直接）

这是最快、最直接的证法。

1. **目标：** 计算 $M_{11} + M_{22} + M_{33}$。
    
2. 关键关系 (1)：余子式与代数余子式
    
    根据代数余子式 $C_{ij}$ 和余子式 $M_{ij}$ 的定义：
    
    $C_{ij} = (-1)^{i+j} M_{ij}$
    
    当 $i=j$ 时 (主对角线)，$i+j = 2i$ (偶数)，所以 $(-1)^{i+j} = 1$。
    
    因此：
    
    - $C_{11} = (-1)^{1+1} M_{11} = M_{11}$
        
    - $C_{22} = (-1)^{2+2} M_{22} = M_{22}$
        
    - $C_{33} = (-1)^{3+3} M_{33} = M_{33}$
        
        所以，我们的目标 $\mathbf{M_{11} + M_{22} + M_{33}} = \mathbf{C_{11} + C_{22} + C_{33}}$。
        
3. 关键关系 (2)：伴随矩阵与代数余子式
    
    根据伴随矩阵 $A^*$ 的定义， $A^*$ 是 $A$ 的代数余子式矩阵 $C$ 的转置 ($A^* = C^T$)。
    
    $$A^* = \begin{pmatrix} C_{11} & C_{21} & C_{31} \\ C_{12} & C_{22} & C_{32} \\ C_{13} & C_{23} & C_{33} \end{pmatrix}$$
    
    一个矩阵的迹 (Trace)，是其主对角线元素之和。
    
    因此，$\text{tr}(A^*) = C_{11} + C_{22} + C_{33}$。
    
4. 关键关系 (3)：迹与特征值
    
    任何一个方阵的迹，都等于它所有特征值的总和。
    
    题目已知 $A^*$ 的特征值为 $\lambda_1, \lambda_2, \lambda_3$。
    
    因此，$\text{tr}(A^*) = \lambda_1 + \lambda_2 + \lambda_3$。
    
5. 结论：
$$M_{11} + M_{22} + M_{33} = C_{11} + C_{22} + C_{33} = \text{tr}(A^*) = \lambda_1 + \lambda_2 + \lambda_3$$

### 证法二：利用“特征多项式”与“韦达定理”（最根本）

此方法从 $A$ 本身的特征值出发，揭示了更深层的关系。

1. 设 $A$ 的特征值为 $\mu_1, \mu_2, \mu_3$。
    
    $A$ 的特征多项式为：
    
    $$p(\mu) = \det(A - \mu I) = \mu^3 - (\text{tr}(A))\mu^2 + (M_{11} + M_{22} + M_{33})\mu - \det(A)$$
    
    (注：$\mu$ 的一次项系数，即为 $A$ 的所有 2 阶主子式（即 $M_{ii}$）之和)
    
2. 根据韦达定理 (Vieta's formulas)：
    
    多项式的根（$A$ 的特征值）与系数之间存在关系。
    
    $\mu_1 \mu_2 + \mu_1 \mu_3 + \mu_2 \mu_3 = M_{11} + M_{22} + M_{33}$
    
3. $A$ 与 $A^*$ 的特征值关系：
    
    我们有公式 $A A^* = |A| I$。假设 $A$ 可逆 ($|A| \neq 0$)。
    
    $A^* = |A| A^{-1}$
    
    - $A$ 的特征值为 $\mu_1, \mu_2, \mu_3$。
        
    - $A^{-1}$ 的特征值为 $1/\mu_1, 1/\mu_2, 1/\mu_3$。
        
    - $A^*$ 的特征值（即 $\lambda_1, \lambda_2, \lambda_3$）为：
        
        $\lambda_i = |A| \cdot (1/\mu_i)$
        
    - 又因为 $|A| = \mu_1 \mu_2 \mu_3$，所以：
        
        $\lambda_1 = (\mu_1 \mu_2 \mu_3) / \mu_1 = \mu_2 \mu_3$
        
        $\lambda_2 = (\mu_1 \mu_2 \mu_3) / \mu_2 = \mu_1 \mu_3$
        
        $\lambda_3 = (\mu_1 \mu_2 \mu_3) / \mu_3 = \mu_1 \mu_2$
        
4. **结论：**
    
    - $M_{11} + M_{22} + M_{33} = \mu_1 \mu_2 + \mu_1 \mu_3 + \mu_2 \mu_3$ (来自步骤 2)
        
    - $\lambda_1 + \lambda_2 + \lambda_3 = \mu_2 \mu_3 + \mu_1 \mu_3 + \mu_1 \mu_2$ (来自步骤 3)
        
    - 两者完全相等。
        
    
    _(注：当 $A$ 不可逆时，此关系通过连续性或单独讨论秩为2和秩小于2的情况，同样成立。)_

### 证法三：利用“对角矩阵”特例与“相似不不性”

此方法通过一个简单特例来验证，并推广到一般情况。

1. 假设 $A$ 是一个对角矩阵：
    
    $A = \begin{pmatrix} a & 0 & 0 \\ 0 & b & 0 \\ 0 & 0 & c \end{pmatrix}$
    
    （$A$ 的特征值 $\mu_1=a, \mu_2=b, \mu_3=c$）
    
2. **计算 $M_{11} + M_{22} + M_{33}$：**
    
    - $M_{11} = \begin{vmatrix} b & 0 \\ 0 & c \end{vmatrix} = bc$
        
    - $M_{22} = \begin{vmatrix} a & 0 \\ 0 & c \end{vmatrix} = ac$
        
    - $M_{33} = \begin{vmatrix} a & 0 \\ 0 & b \end{vmatrix} = ab$
        
    - $\text{Sum} = bc + ac + ab$
        
3. 计算 $A^*$ 及其特征值 $\lambda_i$：
    
    $A$ 的代数余子式矩阵 $C$ 为：
    
    $C = \begin{pmatrix} C_{11} & C_{12} & C_{13} \\ C_{21} & C_{22} & C_{23} \\ C_{31} & C_{32} & C_{33} \end{pmatrix} = \begin{pmatrix} bc & 0 & 0 \\ 0 & ac & 0 \\ 0 & 0 & ab \end{pmatrix}$
    
    $A^* = C^T = C$ (因为 $C$ 是对角阵)
    
    $A^*$ 是一个对角矩阵，它的特征值就是它的对角线元素。
    
    - $\lambda_1 = bc$
        
    - $\lambda_2 = ac$
        
    - $\lambda_3 = ab$
        
4. **比较结果：**
    
    - $M_{11} + M_{22} + M_{33} = bc + ac + ab$
        
    - $\lambda_1 + \lambda_2 + \lambda_3 = bc + ac + ab$
        
    - 两者相等，结论在对角矩阵上成立。
        
5. 推广：
    
    我们要证明的等式 $M_{11} + M_{22} + M_{33} = \text{tr}(A^*)$ 是一个恒等式。
    
    $\text{tr}(A^*)$ 是一个相似不变量（因为 $A^*$ 的特征值和不变）。
    
    $M_{11} + M_{22} + M_{33}$ 也是一个相似不变量（因为它 $A$ 的特征多项式的系数）。
    
    由于这个等式对所有可对角化的矩阵都成立（它们都相似于对角矩阵），并且可对角化矩阵在所有矩阵中是稠密的，因此该等式对所有 $3 \times 3$ 矩阵均成立。
