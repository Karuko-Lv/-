## 1.1JS 基本数据类型
弱类型动态语言。同一变量保存多个不同类型
一共七种，其中 ES 5 包含了 undefined、null、Boolean、Number 和 String，ES 6 多了 Symbol，ES 10 多了 bigint
### 1.1.1Undefined
定义变量的默认值是 undefined
 1. 定义了未初始化的变量
 2. 调用了对象中没有的属性
 3. 没有返回值的函数的结果
 4. TypeScript 的可选变量（？）

### 1.1.2Null
空对象的指针 `typeof null` 返回的是 `object`

#### Null 🆚 Undefined
一个是声明的变量未赋值，状态是 undefined
一个主动将声明的变量清空，告诉大家这个变量用于装对象

### 1.1.3Boolean
只有两个值，用于条件判断逻辑运算
JS 里面有 6 种假值 false（隐式类型转换）
1. undefined
```javascript
let a = undefined
if(a) //a是false
```
2. null
3. 数字 0（不是字符串 0，字符串 0 是 true）
4. NaN：无法计算结果时出现，表示非数值 Not a number 但是 `typeof NaN === "number"` 返回 1
5. “”或‘’（空字符串，中间即使有空格也是 true）
6. `false`
