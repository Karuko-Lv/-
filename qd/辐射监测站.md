## 📚 分阶段学习路径 (Step-by-Step Learning Path)

建议按照以下顺序深入学习，每个阶段都直接对应面试考点。

### 第一阶段：工程化与构建基础 (Engineering)

**目标**：理解项目是如何启动、构建和配置的。 **对应招聘要求**：熟悉 Webpack/Vite 等打包工具，了解前端工程化。

1. **Vite 配置**:
    
    - **学习重点**: 查看 [vite.config.ts](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/vite.config.ts)。理解 `server.proxy` 如何解决跨域问题（连接后端 API），`alias` 别名配置（`@/` 指向 `src/`），以及插件配置（自动导入 Element Plus）。
        
    - **面试题**: "Vite 为什么比 Webpack 快？", "如何配置开发环境的反向代理？"
        
2. **TypeScript 基础**:
    
    - **学习重点**: 项目全面使用 TS。关注接口定义（`interface`）、类型推断。
        
    - **面试题**: "TypeScript 的 `interface` 和 `type` 有什么区别？", "为什么要使用 TypeScript？"
        

### 第二阶段：Vue 3 核心与组件化 (Core Framework)

**目标**：掌握现代 Vue 开发模式。 **对应招聘要求**：熟练掌握 Vue.js 框架，熟悉组件化开发。

1. **Composition API**:
    
    - **学习重点**: 查看 [App.vue](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/App.vue) 和组件文件。理解 `<script setup lang="ts">` 语法糖，`ref`, `reactive`, `computed`, `watch` 的用法。
        
    - **面试题**: "Vue 3 Composition API 和 Vue 2 Options API 的区别是什么？", "ref 和 reactive 的区别？"
        
2. **生命周期**:
    
    - **学习重点**: 理解 `onMounted`, `onUnmounted` 等钩子函数在组件中的执行时机。
        
    - **面试题**: "Vue 3 的生命周期钩子有哪些？父子组件生命周期执行顺序是怎样的？"
        

### 第三阶段：状态管理与路由 (State & Routing)

**目标**：管理应用数据流和页面跳转。 **对应招聘要求**：掌握 Vue Router、Vuex/Pinia。

1. **Pinia 状态管理**:
    
    - **学习重点**: 查看 [src/stores/device/device.ts](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/stores/device/device.ts)。学习如何定义 Store (`defineStore`)，如何修改状态 (`actions`)，以及如何在组件中使用。
        
    - **面试题**: "Pinia 和 Vuex 有什么区别？（Pinia 去掉了 mutation，支持 TS 更友好，体积更小）"
        
2. **Vue Router**:
    
    - **学习重点**: 查看 [src/router/index.ts](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/router/index.ts)。理解路由配置、路由守卫（`beforeEach` 做权限控制）、动态路由。
        
    - **面试题**: "Vue Router 的 `hash` 模式和 `history` 模式有什么区别？"
        

### 第四阶段：UI 开发与样式 (UI & Styling)

**目标**：构建美观、响应式的界面。 **对应招聘要求**：熟悉 HTML/CSS，熟练使用 Element UI/Plus。

1. **Element Plus**:
    
    - **学习重点**: 项目使用了 `unplugin-auto-import`，组件无需手动导入。学习常用组件（Table, Form, Dialog）的属性和事件。
        
2. **CSS 预处理与原子类**:
    
    - **学习重点**: 查看 [tailwind.css](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/assets/css/tailwind.css) 和 [main.scss](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/assets/css/main.scss)。理解 SCSS 变量与嵌套语法，以及 Tailwind CSS 的原子类写法（如 `flex`, `items-center`）。
        
    - **面试题**: "CSS 预处理器（Sass/Less）解决了什么问题？", "Tailwind CSS 的优缺点？"
        

### 第五阶段：网络请求与后端交互 (Networking)

**目标**：实现前后端数据对接。 **对应招聘要求**：熟练使用 Axios，具备前后端协作经验。

1. **Axios 封装**:
    
    - **学习重点**: 查看 [src/api/](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/api) 目录。理解如何封装 Axios 实例，配置请求拦截器（添加 Token）、响应拦截器（统一处理错误），以及 API 模块化管理。
        
    - **面试题**: "如何封装 Axios？", "如何处理 Token 过期或 401 错误？"
        

### 第六阶段：进阶与优化 (Advanced)

**目标**：提升代码质量与性能。 **对应招聘要求**：关注性能优化、代码规范。

1. **数据可视化**: 学习 [src/plugins/echarts](file:///Users/lvxiaoxu/Documents/工作/hbj/FSJcz/src/plugins/echarts) 如何按需引入 ECharts 减小包体积。
    
2. **性能优化**: 关注组件懒加载（路由中的 `import()`）、静态资源处理。
