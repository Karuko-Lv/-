### 初始化循环单链表
需要将头结点的 next 指针指向头结点
```cpp
typedef struct LNode{
	int data;
	struct LNode *next;
}LNode,*LinkList;
bool InitList(LinkList &L){
	L = new LNode();
	if(L==nullptr) return false;//分配内存失败 内存不足
	L->next = L;//🏆
	return true;
}
```
![[截屏2026-04-10 18.07.28.png]]
#### 判断是否为空
```cpp
bool isEmpty(LinkList L){
	if(L->next==L)
		return true;
	else
		return false;
}
```

#### 判断是否为循环单链表的表尾结点
```cpp
bool isTail(LinkList L,LNode *p){
	if(p->next==L)
		return true;
	else
		return false;
}
```
