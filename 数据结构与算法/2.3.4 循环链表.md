## 循环单链表
![[截屏2026-04-10 17.59.20.png]]
最后一个结点的 next 指针指回头结点
### 初始化循环单链表
需要将头结点的 next 指针指向头结点
```cpp
typedef struct LNode{
	int data;
	struct LNode *next;
}LNode,*LinkList;
bool InitList(LinkList &L){
	L = new LNode();
	if(L==nullptr) return false;//分配内存失败 内存不足
	L->next = L;//🏆
	return true;
}
```
![[截屏2026-04-10 18.07.28.png]]
#### 判断是否为空
```cpp
bool isEmpty(LinkList L){
	if(L->next==L)
		return true;
	else
		return false;
}
```

#### 判断是否为表尾结点
```cpp
bool isTail(LinkList L,LNode *p){
	if(p->next==L)
		return true;
	else
		return false;
}
```

写代码遍历循环单链表的时候判断扫描指针 p 是否到表尾和普通的单链表也不一样

| **特性**    | **指针版 LNode *p (推荐)** | **变量版 LNode p (不推荐)** |
| --------- | --------------------- | --------------------- |
| **访问成员**  | `p->next`             | `p.next`              |
| **内存开销**  | 极小（只传地址）              | 较大（复制整个结点）            |
| **调用便利性** | 直接传指针名                | 需解引用 `*p`             |
| **逻辑语义**  | 操作链表中的“某个位置”          | 操作一个“孤立的数据块”          |

#### `LNode *p` 和 `LNode *&p`  
1️⃣情况 A：使用 `LNode *p` ~={red}**修改指针指向的变量**=~
1. 主函数有个指针 `p` 指向 `0x100`
2. 调用函数时，产生一个临时变量 `p_copy`，也存着 `0x100`
3. 你在函数里写 `p = new LNode;`
4. **结果**：`p_copy` 变成了 `0x200`，但**主函数的 `p` 纹丝不动**，依然是 `0x100`
    
2️⃣使用 `LNode *&p` ~={red}**修改指针变量本身**=~
1. 主函数有个指针 `p` 指向 `0x100`。
2. 函数里的 `p` 就是主函数 `p` 的别名。
3. 你在函数里写 `p = new LNode;`。
4. **结果**：主函数的 `p` **直接变成了 `0x200`**。
### 单链表 vs 循环单链表
单链表给其中的一个结点 p 只能知道 p 后续的结点，除非知道头结点的指针；循环单链表只要给一个结点 p 可以找到整个循环单链表中的任意一个结点
![[截屏2026-04-10 18.10.44.png]]
作用：例如删除某一个结点，需要修改这个结点的前一个结点的指针

#### 单链表操作
很多时候是在链表的头部或者尾部（头插法和尾插法建立链表）
普通单链表最后一个结点指针指向 null
![[截屏2026-04-10 20.11.22.png]]
如果应用场景需要经常对表头表尾进行操作-->使用循环单链表，让单链表的指针 L 指向表尾的元素
PS：⚠️这么做当需要在表尾插入或者删除一个元素的时候**需要修改这个指针 L 的指向**

## 循环双链表
所有的 next 指针和 prior 指针都形成了一个循环（闭环）
![[截屏2026-04-10 20.14.00.png]] 
### 初始化循环双链表
```cpp
typedef struct DNode{
	int data;
	struct DNode *next, *prior;
}DNode,*DLinklist;

bool InitDLinklist(DLinklist &L){
	L = new DNode();
	if(L==nullptr) return false;//初始化失败
	L->next = L;
	L->prior = L;
	return true;
}
void testDLinklist(){
	DLinklist L;
	InitDLinklist(L);
	//后续代码
}
```

bool isEmpty(DLinklist L){
