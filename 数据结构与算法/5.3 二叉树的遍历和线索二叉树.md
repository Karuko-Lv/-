### 手算二叉树三种遍历（分支结点逐层展开法）
重点练习容易错
![[截屏2026-05-05 18.02.16.png]] 🧚‍♀️后序遍历就是把树往右压平；这个讲解顺序就是广度优先遍历；中序遍历就相当于把树压平
如果一个结点是分支而不是叶子就需要嵌套递归的按照特定的遍历序列展开到下一级
####  中序遍历的“垂直投影法”（最快）

在阳光正好的正午，太阳从树的正上方往下照。 **把这棵树所有的结点直直地垂直掉落到地上**，掉落后它们在地面上从左到右排成的序列，就是**中序遍历序列**。

- _应用：_ 如果这是一棵二叉搜索树（BST），投影下来的序列绝对是一个**从小到大的递增序列**！

#### 后序遍历的“剪叶子法”

像秋风扫落叶一样，**从左到右**寻找树上的叶子结点，把它“剪下来”写进序列。剪完之后，原本叶子的父结点就变成了新的叶子，继续从左到右剪，直到最后剪掉根结点。这就是后序遍历序列。

### 代码写三种遍历
#### 先序遍历
递归函数
![[截屏2026-05-05 21.17.10.png]]
```cpp
typedef struct BiTNode{
	ElemType data;
	struct BiTNode *lchild,*rchild;
}BiTNode,*BiTree;
void visit(BiTNode *p)
{
	if(p == nullptr)
		return;
	cout<<p->value<<" ";
}
void PreOrder(BiTree T){
	if(T!=nullptr){
		visit(T);
		PreOrder(T->lchild);
		PreOrder(T->rchild);
	}
}
```
🤨两个递归的 PreOrder 写在 if 条件语句里面是同时确保了如果 T 为 nullptr 就不能访问其左右孩子了
#### 中序遍历
![[截屏2026-05-05 21.19.55.png]]
```cpp
typedef struct BiTNode{
	ElemType data;
	struct BiTNode *lchild,*rchild;
}BiTNode,*BiTree;
void visit(BiTNode *p)
{
	if(p == nullptr)
		return;
	cout<<p->value<<" ";
}
void InOrder(BiTree T){
	if(T!=nullptr){
		InOrder(T->lchild);
		visit(T);
		InOrder(T->rchild);
	}
}
```

### 三种遍历的扩展应用
‼️考察树这一章的算法题大多数是从这三种算法进行一种变换得到的
例如
1. 求一个树的深度
A：由于二叉树有递归特性根结点、左子树和右子树，求一棵树的深度或者高度应该先递归的求出左子树的高度，之后右子树的高度，再取左右子树更高的一边即可
```cpp
int treeDepth(BiTree T){
	//int h = 0;
	if(T!=null){
		//h+=1;
		int l = treeDepth(T->lchild);
		int r = treeDepth(T->rchild);
		return l>r?l+1;r+1;
	}else{
		return 0;
	}
}
```
🧚‍♀️如 C 结点返回 0+1；d 结点返回 0+1；-结点得到两个一的数据, 返回上一层为 1+1=2；这样的话是不是高度是从底向上加的呢？
![[截屏2026-05-05 22.07.58.png]]
`int l = treeDepth(T->lchild);` 是访问左，`int r = treeDepth(T->rchild);` 是访问右；`return l>r?l+1;r+1;` 相当于访问根==>左右根是后序遍历算法的变种
🧚‍♀️else 里面最后一行改成 ++l和 ++r就好理解了。;同理由叶节点 d 能够得到上一个根节点-的 R=1

这一小节三种遍历算法很重要‼️

### 二叉树的层序遍历
#### 逻辑
一层一层遍历二叉树的各个结点
![[截屏2026-05-05 22.19.28.png]]
1️⃣需要设置辅助队列
2️⃣让根结点入队
3️⃣判断队列是否为空 
4️⃣如果队列不是空的则让**队头结点**出队再访问出队的结点，将其左右孩子插入队尾（如果有）
🧚‍♀️广度优先搜索；这就是 bfs 算法；和中序不一样

#### 代码
用代码实现算法思想如下
```cpp
//层序遍历
void LevelOrder(BiTree T){
	//1️⃣需要设置辅助队列
	LinkQueue Q;
	//「PS」使用链队列而不是顺序队列 因为难以估计要访问的这棵树到底多少结点==>方便队列结点的拓展
	InitQueue(Q);
	BiTree p;//记录出来的结点
	EnQueue(Q,T);//2️⃣让根结点入队
	while(!isEmpty(Q)){//3️⃣判断队列是否为空 
		//4️⃣如果队列不是空的则让**队头结点**出队再访问出队的结点
		DeQueue(Q,p)；
		visit(p);
		if(p->lchild!=nullptr)
			EnQueue(Q,p->lchild);//4️⃣左孩子入队
		if(p->rchild!=nullptr)
			EnQueue(Q,p->rchild);//4️⃣右孩子入队	
	}
}
```
