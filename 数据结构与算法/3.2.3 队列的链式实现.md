队列是单链表的阉割版
队列中由于每次插入元素都只能在表尾的位置插入，因此可以用专门的尾指针指向最后一个结点
出队：从头结点找到第一个数据结点，将其删除即可
==>声明队列时即需要保存对首指针也需要保存队尾指针

## 初始化（带头结点）
![[截屏2026-04-14 22.18.55.png]]
```cpp
typedef struct LinkNode{
	int data;
	struct LinkNode *next;
}LinkNode;

typedef struct{
	LinkNode *front,*rear;
}LinkQueue;

void InitQueue(LinkQueue &Q){
	//初始化 front、rear指向头结点
	Q.front = Q.rear = new LinkNode();
	Q.front->next = nullptr;
}

void testLinkQueue(){
	LinkQueue Q;//声明一个队列 
	InitQueue(Q);//初始化队列
}
```
🧚‍♀️尾指针的循环链表比较好
「PS」有自引用所以需要 `typedef struct [结构体真名] { 成员变量... } [类型别名];`；没有自引用可以偷懒，不写“结构体真名”，直接让编译器创建一个匿名的结构体，然后立刻用 `typedef` 给它贴上一个叫 `LinkQueue` 的别名标签

### 判断是否为空
```cpp
bool isEmpty(LinkQueue Q){
	if(Q.front == Q.rear)
		return true;
	else
		return false;
}
```

```cpp
bool isEmpty(LinkQueue Q){
	if(Q.front == nullptr)
		return true;
	else
		return false;
}
```
头尾指针是否指向同一个结点；或者头结点的 `next` 指针是否指向 `null`

## 入队（带头结点）
新的数据元素入队，该数据元素包含在一个结点当中，所以申请一个新的结点
```cpp
void EnQueue(LinkQueue &Q,int x){
	LinkNode *s = new LinkNode();
	s->data = x;
	s->next = nullptr;
	Q.rear->next = s;//新结点插入到rear之后
	Q.rear = s;//修改表尾指针
}
```
新插入的结点一定是队列当中的最后一个结点，所以新结点的 `next` 指针域要设置为 `nullptr`
由于 `rear`  指针指向的是当前的表尾结点而新插入的结点需要连接到当前的表尾结点之后是新的表尾结点

### 入队（不带头结点）
插入第一个数据元素（第一个数据元素入队）的时候需要特殊处理，因为刚开始 rear 和 front 都是指向 `null` 的，插入第一个元素的时候这两个指针都要进行修改, 如果不是第一个只需要修改 rear 就可以了
```cpp
void EnQueue(LinkQueue &Q,int x){
	LinkNode *s = new LinkNode();//将新的结点s插入链表当中
	s->data = x;
	s->next = nullptr;
	if(Q.front == nullptr){//不带头结点判断是否为空
		Q.front = s;
		Q.rear = s;
	}else{
		Q.
	}
}
```
