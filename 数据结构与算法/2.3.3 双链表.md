单链表每一个结点只包含指向后继结点的指针，如果给一个结点 p 想要找到前驱结点很麻烦，双链表是在单链表的基础上再增加一个指针域

 ## 双链表的初始化（带头结点）
![[截屏2026-04-10 15.46.03.png]]
 ```cpp
 typedef struct DNode{
 	int data;
 	struct DNode *priot,*next;
 }DNode,*DLinklist;
 bool InitDLinkList(DLinkList &L){
 	L = new DNode();
 	if(L==nullptr)
 		return false;//申请内存失败
	L->prior = nullptr;
	L->next = nullptr;
	return true;
 }
 void testDLinkList{
 	DLinkList L;
 	InitDLinkList(L);
 	//...
 }
 ```
 对于带头结点的双链表判断是否为否
 ```cpp
 bool isEmpty(LinkList L){
 	if(L->next == nullptr)//注意不是判断L->data
 		return true;
 }
 ```
