 ## 双链表的初始化（带头结点）
![[截屏2026-04-10 15.46.03.png]]
 ```cpp
 typedef struct DNode{
 	int data;
 	struct DNode *priot,*next;
 }DNode,*DLinklist;
 bool InitDLinkList(DLinkList &L){
 	L = new DNode();
 	if(L==nullptr)
 		return false;//申请内存失败
	L->prior = nullptr;
	L->next = nullptr;
	return true;
 }
 void testDLinkList{
 	DLinkList L;
 	InitDLinkList(L);
 	//...
 }
 ```
 ### 双链表判空
 对于带头结点的双链表判断是否为否
 ```cpp
 bool isEmpty(LinkList L){
 	if(L->next == nullptr)//注意不是判断L->data，头结点不存储“有效数据”，也不用置0之类的
 		return true;
	else
		return false;
 }
 ```

## 双链表插入
1. 在 p 结点后插入 s 结点
```cpp
bool InsertNextDNode(DNode *p,DNode *s){
	DNode t = new DNode;
	t=p->next;
	p->next = s;
	s->next = t;
	return true;
}
```
