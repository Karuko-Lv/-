## 双链表插入
1. 在 p 结点后插入 s 结点
```cpp
bool InsertNextDNode(DNode *p,DNode *s){
	DNode t = new DNode;
	t=p->next;
	p->next = s;
	s->next = t;
	return true;
}
```
❌完全忘了这是双链表了还有一个前驱结点需要做
```cpp
bool InsertNextDNode(DNode *p,DNode *s){
	p->next->prior = s;
	s->next = p->next;
	p->next = s;
	s->prior = p;
	return true;
}
```
❌错误 1️⃣代码逻辑错误如果先让 s 的前驱指向 p 此时的 p 的后继结点还不是 s  最后会导致错误
错误 2️⃣代码健壮性有待提高
Gemini：双链表中插入，口诀是：**“先牵后手，再改前手”**。
```cpp
bool InsertNextDNode(DNode *p, DNode *s) {
    // 0. 安全检查：如果 p 或 s 是空的，直接返回失败
    if (p == nullptr || s == nullptr) return false;

    // 1. 先让 s 认领后继：把 s 后面接上 p 原本的后面
    s->next = p->next; 

    // 2. 处理 p 之后的那个结点（如果有的话）
    if (p->next != nullptr) {
        p->next->prior = s; // 让原后继的前驱指向新来的 s
    }

    // 3. 让 s 认领前驱：把 s 的前面接上 p
    s->prior = p;

    // 4. 最后修改 p 的后继：让 p 指向新来的 s
    p->next = s;

    return true;
}
```
