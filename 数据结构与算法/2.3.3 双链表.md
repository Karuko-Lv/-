## 双链表插入
1. 在 p 结点后插入 s 结点
```cpp
bool InsertNextDNode(DNode *p,DNode *s){
	DNode t = new DNode;
	t=p->next;
	p->next = s;
	s->next = t;
	return true;
}
```
❌完全忘了这是双链表了还有一个前驱结点需要做
```cpp
bool InsertNextDNode(DNode *p,DNode *s){
	p->next->prior = s;
	s->next = p->next;
	p->next = s;
	s->prior = p;
	return true;
}
```
❌错误 ：代码健壮性有待提高
如果后驱结点不存在就不用修改后驱结点的前驱结点了（`nullptr` 也不存在前驱结点，空指针解引用 ）
Gemini：双链表中插入，口诀是：**“先牵后手，再改前手”**。
```cpp
bool InsertNextDNode(DNode *p, DNode *s) {
    // 0. 安全检查：如果 p 或 s 是空的，直接返回失败
    if (p == nullptr || s == nullptr) return false;

    // 1. 先让 s 认领后继：把 s 后面接上 p 原本的后面
    s->next = p->next; 

    // 2. 处理 p 之后的那个结点（如果有的话）‼️
    if (p->next != nullptr) {
        p->next->prior = s; // 让原后继的前驱指向新来的 s
    }

    // 3. 让 s 认领前驱：把 s 的前面接上 p
    s->prior = p;

    // 4. 最后修改 p 的后继：让 p 指向新来的 s
    p->next = s;

    return true;
}
```
3 和 4 可以交换顺序 1 和 2 也可以，但是写代码通常是先存后放所以一般是上面的顺序
🧚‍♀️第二句无所谓四在二前的话可以用 s->next->prior=s 这个是可以的

### 按位序插入
只需要从头结点开始找到一个位序的前驱结点，对这个结点进行后插操作即可

### 前插操作
找到结点的前驱结点，对这个结点进行后插操作即可

==>这两个方便实现主要是因为双链表很容易找到前驱结点

## 双链表的删除
删除指定结点 p 的后继结点 q
删除操作一定不要忘了释放内存; ⚠️注意判断是否会出现空指针的错误  
![[截屏2026-04-10 16.41.33.png]]
```cpp
bool DeleteNextDNode(LNode *p,LNode *q){
	if(p==nullptr) return false;
	p->next = q->next;
	if(q->next != nullptr){
		q->next->prior = p;
	}
	return true;
}
```
❌错误 1️⃣删除结点忘了释放内存
错误 2️⃣不需要传入两个结点的
Re
```cpp
bool DeleteNextDNode(LNode *p){
	if(p==nullptr) return false;
	DNode *q = p->next;
	if(q == nullptr) return false;//p没有后继结点无需删除
	p->next = q->next;
	if(q->next != nullptr){
		q->next->prior = p;//如果q没有下一个结点 说明p变成了尾结点 
	}
	delete q;
	return true;
}
```
