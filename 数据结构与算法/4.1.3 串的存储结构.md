### 静态数组/定长顺序存储
用静态数组存储字符串，length 表示当前字符串的长度
```cpp
#define MAXLEN 255
typedef struct{
	char ch[MAXLEN];
	int length;
}SString;
```
采用这种方式声明静态字符串系统会在内存开辟一片连续的空间以存放各个字符和 length 变量
每个字符占一个字节
定长的顺序存储和静态数组是一样的
缺点：长度不可变，255 长度存不下扩展是不可能的

### 动态数组（可变长）/堆分配存储
在结构体中定义一个基地址指针，用 malloc 申请一整片的连续存储空间，让基地址指针指向这一片空间的起始地址
```cpp
typedef struct{
	char *ch;
	int length;
}HString;

HString S;
S.ch = new char[];
```
`malloc` 函数申请的这一片内存空间是属于内存里面的堆区，这种实现方式也可以称之为“堆分配存储”
