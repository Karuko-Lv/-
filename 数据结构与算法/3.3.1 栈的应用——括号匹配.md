## 算法实现（代码）
```cpp
#define MaxSize 10//定义栈中元素的最大个数
typedef struct{
	char data[MaxSize];
	int top;
}SqStack;
//初始化栈
void InitStack(SqStack &S){
	S.top = -1;//top指针指向栈顶元素
}
//判断栈是否为空
bool StackEmpty(SqStack &S){
	if(S.top==-1)
		return true;
	else
		return false;
}
//新元素入栈
bool Push(SqStack &S,char x){
	//判断栈是否满了
	if(S.top==MaxSize)
		return false;
	S.data[++S.top]=x;
	return true;
}
//栈顶元素出栈
bool Pop(SqStack &S,char &x){
	if(S.top==-1)
		return false;
	S.data[S.top--]=x;
	return true;
}
bool bracketCheck(char str[],int length){
	SqStack S;//声明一个栈
	InitStack(S);
	for(int i = 0;i<length;i++){//1️⃣扫描括号元素
		if(str[i]=='(' || str[i]=='['||str[i]=="{"){//左括号入栈
			Push(S,str[i]);
		}else{
			if(StackEmpty(S))//右括号单身
				return false;
			else{
				char topElem;
				Pop(S,topElem);
				if(topElem != '(' && str[i] == ')')
					return false;//检查完所有的错误情况就是对的了
				else if(topElem != '[' && str[i] == ']')
					return false;
				else if(topElem != '{' && str[i] == '}')
					return false;
			}
		}
	}
	return StackEmpty(S);//检查完所有括号如果栈是空的就是匹配成功了✅
}
```
❌错误
1️⃣ `top` 永远指向当前栈顶元素的真实下标，`MaxSize` 是 10，数组的合法下标是 `0 ~ 9`
修改为
```cpp
bool Push(SqStack &S, char x){ // 判断栈是否满了 
	if(S.top == MaxSize - 1) // 【修改处】：改为 MaxSize - 1 
		return false; 
	S.data[++S.top] = x; 
	return true; 
}
```
2️⃣ `Pop` 函数中的赋值方向反了
`Pop` 函数的目的是把栈顶的元素**取出来**，并赋值给引用参数 `x`。但在你的代码中：`S.data[S.top--] = x;`，这表示把外面传进来的 `x`（可能是一个未初始化的垃圾值）写到了栈里面，这完全起不到出栈读取数据的作用，并且破坏了栈内数据。
3️⃣`bracketCheck` 函数中的字符语法错误
在判断左括号的 `if` 语句中，你写了 `str[i] == "{"`。在 C/C++ 中，单引号 `'{'` 表示单个字符（`char`），而双引号 `"{ "` 表示字符串（本质是 `const char*` 指针）。`str[i]` 是一个字符，它不能和一个字符串指针去进行 `==` 比较，这里会导致编译报错。
