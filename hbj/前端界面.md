#### **各区域详细设计说明**

##### **左侧：设备状态概览区 (优化调整)**

此区域用于快速巡览所有设备的核心状态，建议按重要性重新排序。

1. **高纯锗γ能谱测量分析系统 (HPGe γ-Spectrometry System) - 移至最顶部** 111
    
    - **理由**：这是辐射监测的核心设备，直接关系到辐射水平的衡量 222。它的状态和关键读数（如剂量率）应放在最显眼的位置。
        
    - **建议显示内容**：
        
        - 设备状态：(运行/待机/故障)
            
        - **当前剂量率** (µSv/h 或 nGy/h)：用最醒目的方式展示 3。
            
        - 采集状态：(采集中/分析中)
            
        - 能谱信息概览：可以放一个微缩的能谱图，点击可跳转到中间区域显示详情。
            
2. **空气动力学采样器组 (Aerosol Samplers Group)**
    
    - **理由**：气溶胶和碘是放射性物质的重要载体，将它们组合在一起，便于综合判断空气放射性水平。
        
    - **1号空气碘采样器** ：
        
        - 设备状态、采样流量、累计采样量、采样倒计时。
            
    - **1号气溶胶采样器 (超大流量)** ：
        
        - 设备状态、当前流量、累计流量、工作时长。
            
3. **自动气象仪 (Automatic Weather Station)** 
    
    - **理由**：气象数据（特别是风向、风速）对于判断放射性物质的扩散至关重要。
        
    - **建议显示内容**：
        
        - 风速与风向（可以用风玫瑰图或箭头直观表示） 7。
            
        - 温度、湿度、气压、雨量 8。
            
4. **环境与站房状态 (Environment & Station Status)**
    
    - **干湿沉降采样器** 999：状态、降雨信息、感雨状态。
        
    - **站房状态** 10：站内温度、湿度、门禁状态（开/关） 11111111。
        
    - **电源设备** 121212：电压、电流、工作状态。
        

##### **中间：核心监控与数据展示区 (重点设计)**

这里是整个系统的视觉中心，应展示最重要、最宏观或需要详细分析的信息。

1. **核心设备详情：高纯锗能谱仪实时数据**
    
    - **理由**：替代您截图中原有的“1号高压电离室”，直接展示核心设备——高纯锗能谱仪的详细信息。
        
    - **建议显示内容**：
        
        - **顶部**：显示设备ID、当前状态和最重要的读数——**实时剂量率 (µSv/h)** 13。
            
        - **中部**：一个**动态的、可交互的γ能谱图** 14。这是能谱分析的关键，用户可以缩放、查看特定能量峰。
            
        - **下部**：显示**自动核素识别结果**列表 15，并列出置信度。同时，也可以展示气压、电压等辅助参数。
            
2. **关键参数综合趋势图**
    
    - **理由**：将不同设备的关键数据放在同一个时间轴上对比，有助于分析事件和溯源。
        
    - **建议显示内容**：
        
        - 提供一个多选菜单，允许用户自由组合显示不同设备的趋势曲线。
            
        - **默认显示**：**剂量率、风速、风向、气溶胶/碘采样器流量**。例如，当剂量率出现异常时，可以立刻关联到当时的风向和空气采样情况。
            
        - 提供时间范围选择器（如：近1小时、近24小时、自定义）。
            

##### **右侧：辅助信息与操作区 (补充设计)**

右侧用于放置全局性、需要随时关注的信息和常用操作，让用户无需离开主界面即可完成高频任务。

1. **设备概览与系统状态 (System & Device Overview)** 
    
    - **理由**：提供一个所有接入设备状态的“仪表盘”，一目了然。
        
    - **建议显示内容**：
        
        - 用小图标和颜色（如绿色代表正常，红色代表故障）平铺展示所有设备 。
            
        - 显示通信状态、数据存储状态（如：磁盘剩余空间/可用天数）。
            
2. **实时报警信息列表 (Live Alarm List)** 
    
    - **理由**：报警是系统的核心功能之一，应有专门的区域实时滚动显示 。
        
    - **建议显示内容**：
        
        - 按时间倒序显示最新的报警信息 21。
            
        - 每条信息包含：报警时间、设备名称、报警内容（如：剂量率超限、设备故障、站房门打开 ）。
            
        - 提供快速跳转按钮，点击可查看报警详情或跳转到相关设备控制页面 。
            
3. **设备远程控制面板 (Quick Control Panel)**
    
    - **理由**：将最常用或应急情况下需要使用的设备控制功能提取出来，方便快速响应。
        
    - **建议显示内容**：
        
        - 提供一个下拉菜单选择设备。
            
        - 展示常用控制指令按钮，如对采样器进行“启动采样”、“停止采样”等。
            
        - 下方显示操作指令的反馈信息。
            

通过以上布局调整，您的监测系统界面将变得更加专业和高效，能够让操作人员在第一时间掌握核心信息，并快速做出响应。


### **全新界面布局设计：环境辐射监测仪表盘🌟**

核心思路是提升气溶胶、空气碘以及气象监测数据的权重，将它们作为评估环境辐射水平的核心指标，进行集中展示。

#### **左侧面板：综合空气与气象监测区**

此区域将大气放射性采样数据与关键气象数据相结合，因为这两者在追踪和判断潜在的空气污染扩散时关联性极强。

- **自动气象仪 (置于顶部)**
    
    - **设计理由：** 气象数据（尤其是风向和风速）是判断放射性物质扩散路径和范围的首要因素 。这些信息对于正确解读其他采样器的数据至关重要。
        
    - **建议显示内容：**
        
        - **实时风玫瑰图：** 以图形化方式直观展示风向和风速的分布。
            
        - **核心气象参数：** 突出显示当前的风速(m/s)、风向(°)以及雨量(mm)。
            
        - **其他参数：** 实时显示温度(°C)、气压(hPa)和湿度(%RH) 。
            
- **空气采样设备组**
    
    - **设计理由：** 在无法获取能谱数据的情况下，这些设备采集的样品是衡量空气中放射性物质浓度的主要依据。
        
    - **建议显示内容：**
        
        - **超大流量气溶胶采样器🌟:**
            
            - 设备状态：(例如：“运行正常”、“采样中”、“待机”) 。
                
            - 实时采样流量：(M³/Hour) 。
                
            - 一个显示最近一小时流量变化的动态趋势图。
                
        - **空气碘采样器🌟:**
            
            - 设备状态：(例如：“运行正常”、“采样中”) 。
                
            - 实时采样流量：(L/min) 。
                
            - 当前采样模式与定时设定信息 。

#### **中间面板：站房运维与数据中心**

替代原有的能谱细节，这个核心视觉区域现在将专注于展示整个监测站的运行健康度和宏观数据趋势，成为运维的指挥中心。

- **站房状态总览 (顶部区域)**
    
    - **设计理由：** 提供对监测站实体环境、安防和供配电系统健康状况的“一键式”快速检查。
        
    - **建议显示内容：** 采用大尺寸、色彩鲜明的状态指示器。
        
        - **站房环境：** 显示站内温湿度传感器采集的实时温度和湿度 。
            
        - **供配电状态：** 显示主电源的电压、电流和工作状态 (例如：“市电正常”、“UPS后备”) 。
            
        - **门禁状态：** 一个清晰的图标，用颜色区分站房门的状态（例如：绿色表示“关闭”，红色表示“开启”） 。
            
        - **数据通讯：** 一个图标，用于指示与上级数据中心的连接和数据上传状态 。
            
- **历史数据查询与展示 (底部区域)**
    
    - **设计理由：** 强调软件的数据存储、查询和导出核心功能，使其成为数据分析和报告的主要工具 。
        
    - **建议显示内容：**
        
        - **设备筛选器：** 下拉菜单，允许用户在不同设备（如气象仪、气溶胶采样器等）之间切换选择 。
            
        - **时间范围选择器：** 提供日历控件，方便用户选择查询的起始和结束日期/时间 。
            
        - **数据结果表格：** 清晰地罗列出包含时间戳和数值的查询结果。
            
        - **动态趋势图：** 将查询到的数据动态绘制成折线图，便于进行可视化分析 。
            
        - **数据导出按钮：** 提供一键将查询结果导出为指定格式文件的功能 。

#### **右侧面板：报警、控制与系统日志区**

此区域专注于需要立即响应和处理的事项，包括实时报警、设备遥控和系统消息，是保障系统稳定运行的关键。

- **实时报警与通知列表**
    
    - **设计理由：** 确保操作人员能第一时间注意到任何设备故障或安防事件，这是系统的核心功能之一 。
        
    - **建议显示内容：**
        
        - 一个按时间倒序滚动的最新事件列表 。
            
        - 根据事件严重性进行颜色区分：红色用于紧急报警（如“剂量率超限”、“站房门被打开”），黄色或蓝色用于普通通知 。
            
        - 每条信息都应包含报警时间、设备名称和简要的内容描述 。
            
        - 提供快速跳转按钮，点击可查看报警详情或跳转到历史报警查询页面 。
            
- **设备远程快捷控制**
    
    - **设计理由：** 将最常用或应急时需使用的控制功能提取出来，便于快速响应，无需进入复杂的设置菜单 。
        
    - **建议显示内容：**
        
        - 通过下拉菜单选择需要控制的设备。
            
        - 根据所选设备，动态显示可用的控制按钮。例如：
            
            - **采样器：** “启动采样”、“停止采样”、“参数设置”。
                
            - **站房空调：** “启动制冷”、“调节温度”。
                
        - 在按钮下方设置一个信息反馈区，用于实时显示指令执行后的结果 。
            
- **舱外滚动屏内容控制**
    
    - **设计理由：** 用于管理和控制向外界展示的信息内容 。
        
    - **建议显示内容：**
        
        - 一个文本输入框，用于自定义欢迎信息。
            
        - 提供复选框，让用户可以选择哪些关键监测数据（如实时温度、风速等）在外部显示屏上滚动播放 。
