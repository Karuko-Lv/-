#### 按位序插入
##### 带头结点
好处：可以把头结点当成第 0 个结点，第 1个位置插入元素的代码逻辑和第 2、3 个位置的代码逻辑是一样的
![[截屏2026-03-20 20.21.40.png]]
🧚‍♀️ `s-next=p->next p->next=s`
```cpp
typedef struct LNode{
	int data;
	struct LNode *next;
}LNode,*LinkList;
bool InitList(LinkList &L){
	L = new LNode;
	if(L==NULL)
		return false;
	L->next=NULL;
	return true;
}
void ListInsert(LinkList &L,int i,int e){//第i个位置插入e这个元素
	//1️⃣找到第i-1个元素
	LNode *p = L->next;//第1个元素
	for(int j=1;j<i-1;j++){
		p=p->next;
	}
	LNode *s = new LNode;
	s->next = p->next;
	p->next = s;//顺序不可以换 
	//✅给要插入的元素先赋指针域 再给前面的结点修改指针域
	//❌如果修改前面的结点的指针域就会失去指向后面结点的指针了
}
int main(){
	LinkList L;
	InitList(L);
	ListInsert(L,1,3);
	return 0;
}
```
「PS」上面的代码有很多错误❌
错误 1️⃣：新插入的 s 结点没有赋 data 域
错误 2️⃣：没有判断 i 的合法性
错误 3️⃣：如果是空链表 L->next 可能是 NULL，NULL 不能->next，会发生空指针解引用的报错（`*p` 即解引用）
```cpp
p->next = NULL;          // 方式 A：使用箭头（最常用）
(*p).next = NULL;        // 方式 B：手动解引用 + 点操作符
```
如果是向一个空链表的第 1 个位置插入整数 1，p->next 是空指针解引用

❤️为了规避空指针解引用，每次->next 之前要先问问这个指针是不是 `NULL`

using namespace std;
typedef struct LNode{
	int data;
	struct LNode *next;

		return false;
	L->next=nullptr;
	return true;
}

		return false;

	if(L==nullptr)
		return false;
	LNode *p = L;

	//找到i-1位置的结点
	//看了🌰代码
	while(p!=nullptr && j<i-1){
		p=p->next;
		j++;
	}
	//少写了一段代码 因为下面用了p->next所以要判断一下p是否为空
	if(p==nullptr)
		return false;
	//少了的代码

	s->next = p->next;
	s->data = e;
	p->next = s;
	return true;

	//❌如果修改前面的结点的指针域就会失去指向后面结点的指针了
}
void PrintList(LinkList L){
