```cpp
//初始化线性表中的顺序表
struct SqList{
	int* data;
	int length;//需要一个记录现有长度的变量
	int capacity;
};
void initList(SqList &L,int c){
	L.capacity = c;
	L.data = new int[L.capacity];
	L.length = 0;//刚刚建好的表长度是0
}
int main(){
	SqList L;
	int c = 10;
	initList(L,10);
	return 0;
}
```
🧚‍♀️&这个运算符是 C++里面的引用类型，就是在函数中对引用对象的操作，也会使得被引用的对象属性发生改变； 

![[截屏2026-03-14 22.16.30.png]]
这里是伪代码, 函数接口具有抽象性 ![[截屏2026-03-14 22.16.57.png|100]]
```cpp
#include<stdio.h>
void test(int x){
	x = 1024;
	printf("test函数内部 x=%d\n",x);
}
int main(){
	int x = 1;
	printf("调用test前 x=%d\n",x);
	test(x);
	printf("调用test后 x=%d\n",x);
	return 0;
}
```
🧚‍♀️传值调用和传址调用；* 为取值，&取地址-->* 解引用，&取地址；局部变量拷贝副本；变量作用域；形参无法改变实参；值传递，想要持续修改用指针传递；没有&就是操作的备份数据，原来的没动，加&就是拿原数据直接操作，原来的也改了；除了“查”操作，其他都要加& ；这里可以视为指针传参，指针传传参是双向的，调用后原来的值改变；c++用gcc 会报错，g++才可以；**引用是深拷贝**![[截屏2026-03-14 22.22.48.png]]
```cpp
#include<stdio.h>
void test(int &x){
	x = 1024;
	printf("test函数内部 x=%d\n",x);
}
int main(){
	int x = 1;
	printf("调用test前 x=%d\n",x);
	test(x);
	printf("调用test后 x=%d\n",x);
	return 0;
}
```
PS：c 语言写法如下（址传递）或者用 return x
```cpp
#include<stdio.h>
void test(int *x){
	*x = 1024;
	printf("test函数内部 x=%d\n",*x);
}
int main(){
	int x = 1;
	printf("调用test前 x=%d\n",x);
	test(&x);
	printf("调用test后 x=%d\n",x);
	return 0;
}
```
